本文，笔者将复现开源证券魏建榕老师，在2020年7月21日发布的研报《A股市场中如何构造动量因子中提到的长端动量因子？》

提到动量因子，各位读者一定不会陌生。如果将所有的因子简单分为两类的话，那么一类叫做动量因子，意味着未来会延续这一趋势；另一类则是反转因子。

最典型的动量因子就是过去12个月到过去1个月的累计收益率，但是这个因子在A股市场中的表现可以说是非常一般。

最典型的反转因子就是过去21个交易日的累计收益率，这个因子在A股的表现还是非常不错的。

本文，笔者复现的因子叫做长端动量因子，该因子的计算一共三步。

第一步，对所有标的，回溯去过去160个交易日的最高价、最低价和收盘价数据。

第二步，计算每日的振幅，即最高价/最低价-1。

第三步，选择振幅较低的70%的交易日，对其涨跌幅进行求和，就得到了长端动量因子。

```python
def __call__(self):
    data = BaseDataLoader.load_data('../../data/stock_bar_1day.parquet', fields=['close', 'high', 'low']
                                         ).to_dataframes()
    amp = data['high'] / data['low'] - 1
    ret = data['close'].pct_change()
    res = []
    for i in tqdm(range(160, len(amp))):
        tmp_amp = amp.iloc[i-160:i]
        day = tmp_amp.iloc[[-1]].index.tolist()[0]
        q70 = tmp_amp.quantile(0.7)
        tmp_amp = np.where(tmp_amp < q70, 1, np.nan)
        tmp_ret = (ret.iloc[i-160:i] * tmp_amp).sum()
        tmp_ret.name = day
        res.append(tmp_ret)
    res = pd.concat(res, axis=1).T
    res.index.name = 'datetime'
    res.reset_index(inplace=True)
    res = pd.melt(res, id_vars='datetime', var_name='code', value_name='long_mon')
    res.to_parquet('./long_mon.parquet')
```

第4行，计算每日振幅。

第5行，计算每日收益率。

第7-14行，计算长端动量因子。其中第8行是回溯过去160个交易日的振幅数据；第10行，获取70%分位数；第11行，将小于70%分位数的置为1，其他为nan；第12行，计算振幅小于70%分位数的交易日的收益率之和，也就是长端动量因子。

总结：在2018年到2024年底的回测区间中，这个因子在大部分时间表现都比较平庸，IC绝对值并不是很高。但是，在2023年其IC绝对值却超过了0.1。同时，这个因子在整个回测区间中，分层回测的年化收益率呈现了非常不错的单调性，虽然在2023年前分层回测的净值曲线有很多相交的情况。
