本文继续高频成交量的峰、岭、谷信息这一专题。

本文要介绍的因子，叫做相对加权价格因子。这一因子的构建是在高频成交量峰、岭、谷信息的基础上，结合了聪明钱因子的思想。

即：计算每日量谷的成交量加权价格，与当日总成交量加权价格做比，并计算 20 日均值作为量谷相对加权价格因子。考虑到量谷时点交易情绪低迷，其价格过度反应概率较低，因此其价格相对水平越高，个股未来表现越好。

下面根据代码来详细说明一下，该因子的计算步骤。

```python
def process_single_day(self, idx):
    if idx < 20:
        return pd.DataFrame({})
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    vol = []
    for i in range(idx-20,idx+1):
        file_name = self.files[i]
        full_path = os.path.join(self.file_pth, file_name)
        data = BaseDataLoader.load_data(full_path, fields=['volume', 'close'])
        vol.append(data.to_dataframe('volume'))
    flag = self.cal_flag(vol)

    vol = data.to_dataframe('volume')
    vol = vol.reindex(flag.columns, axis=1)
    close = data.to_dataframe('close')
    close = close.reindex(flag.columns, axis=1)

    weighted_close = (close * (vol.div(vol.sum(), axis=1))).sum()

    res = []
    for i in range(3):
        tmp_flag = np.where(flag == i, 1, np.nan)
        weight = vol * tmp_flag
        res.append((close * (weight.div(weight.sum(), axis=1))).sum() / weighted_close)
    res = pd.concat(res, axis=1)
    res = res.replace(0.0, np.nan)
    res.columns = ['peak', 'ridge', 'valley']
    # res['diff'] = res['amt_foc'] - res['vol_foc']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

这个因子和之前介绍的两个因子的区别代码从第15行开始，第15-18行，获取vol和close字段，然后将其列与flag的列对齐。

第20行，计算成交量加权的价格。

第22-26行，计算峰、岭和谷时间段成交量加权的价格。需要注意的是，这里的价格计算都是基于收盘价的，也可以基于（收盘价+最高价+最低价）/3。

后面的代码就是一些计算完成之后拼接之类的处理了。
