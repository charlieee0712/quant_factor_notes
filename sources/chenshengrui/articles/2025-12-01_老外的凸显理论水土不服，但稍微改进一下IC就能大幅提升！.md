**改进的凸显理论**

在中信建投陈升锐老师的研报《流动性高频因子再构建与投资者注意力因子》中，对老外提出来的凸显理论进行了改进。

一共两种改进方式。

第一种是在计算w，也就是计算惊恐度的时候将收益率改成了换手率，其他不变。

第二种则是在第一种方式上的进一步改进，在最后计算相关性的时候，计算的是用换手率计算的w和换手率之间的相关性。

之前，笔者在股票（[怕自己邪修走火入魔，找了老外论文里的因子复现，结果让人大吃一惊！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247491576&idx=1&sn=766f985afd31848889fd6e8450e12a26&scene=21#wechat_redirect)）和可转债（[转债量化（三）：凸显效应，一个在股票上不行的因子用到转债会“大力出奇迹”吗？](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247492358&idx=1&sn=eff6e26d92d4015ce22d9b9a3f3d1529&scene=21#wechat_redirect)）上都复现了凸显理论因子，但是效果都不行。（注意，笔者的文章中在计算因子的时候都没乘复权因子，但是乘了效果也是不行的。）

**计算公式和代码**

计算公式其实就是一个简单的变量替换，将收益率替换为换手率即可。

**代码**

```python
def __call__(self):
    close = BaseDataLoader.load_data('../../data/stock_bar_1day.parquet',
                                         fields=['close', 'factor']).to_dataframes()
    close = close['close'] * close['factor']
    self.tr = BaseDataLoader.load_data('../../data/capital.parquet', fields=['turnover_ratio']
                                  ).to_dataframe('turnover_ratio')
    self.rtn = close.pct_change()
    res = []
    for i in tqdm(range(21, len(self.rtn)+1)):
        res.append(self.cal_factor(i))
    res = pd.concat(res)
    res.to_parquet('./stv.parquet')
def cal_factor(self, idx):
    rtn = self.rtn.iloc[idx-21:idx]
    tr = self.tr.iloc[idx-21:idx]
    day = rtn.iloc[[-1]].index.tolist()[0]
    tr_mkt = tr.mean(axis=1)
    w = abs(tr.sub(tr_mkt, axis=0)) / (tr.abs().add(tr_mkt.abs(), axis=0) + 0.1)
    w = w.rank(axis=1, ascending=False)
    w = w.div(w.mean(), axis=1)
    res = pd.concat([w.corrwith(rtn), w.corrwith(tr)], axis=1)
    res.columns = ['stv_1', 'stv_2']
    res['datetime'] = day
    return res
```

代码的话就贴出来供大家参考，具体的说明应该是不需要了。毕竟，在之前的文章中已经介绍过了。

主要的区别就是，用换手率替换收益率。其中stv\_1只在计算w的时候继续了替换，stv\_2则全部进行了替换。
