需要注意的是，研报中提到了olhc都是对数价格。但是，笔者在计算中间价格的时候不是直接使用对数价格的，而是使用了原始价格计算完中间价格之后再取对数。因为，感觉这样更符合中间价格的逻辑。

还有一点需要注意的是，变量上面画一条线并不是常规的均值的意思，这里没有任何特殊含义。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    cur = pd.to_datetime(date_str) + timedelta(hours=15)
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['high', 'low', 'close', 'open'])
    h = data.to_dataframe('high')
    l = data.to_dataframe('low')
    c = data.to_dataframe('close')
    o = data.to_dataframe('open')
    tao = 1 - (h == l) * (l == c.shift(1))
    pi_o = ((o != h) * np.where(tao, 1, np.nan)).mean() + ((o != l) * np.where(tao, 1, np.nan)).mean()
    pi_c = ((c.shift(1) != h.shift(1)) * np.where(tao, 1, np.nan)).mean() + (
            (c.shift(1) != l.shift(1)) * np.where(tao, 1, np.nan)).mean()
    pi_o = -8 / pi_o
    pi_c = -8 /pi_c
    mid = np.log((h + l) / 2)
    o = np.log(o)
    c = np.log(c)
    ohl = ((mid - o) * (o - mid.shift(1))).mean() * pi_o
    chl = ((mid - c.shift(1)) * (c.shift(1) - mid.shift(1))).mean() * pi_c
    ohlc = ((mid - o) * (o - c.shift(1))).mean() * pi_o
    chlo = ((o - c.shift(1)) * (c.shift(1) - mid.shift(1))).mean() * pi_c
    res = pd.concat([ohl, chl, ohlc, chlo], axis=1)
    res.columns = ['ohl', 'chl', 'ohlc', 'chlo']
    res['datetime'] = cur
    return res
```

第11行，计算公式中的τ。

第12-16行，计算π。

第17-19行，计算中间价、开盘价和收盘价的对数。

第20-23行，按照公式计算价差因子。
