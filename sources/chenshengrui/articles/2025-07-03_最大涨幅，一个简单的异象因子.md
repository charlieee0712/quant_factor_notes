本文，笔者要复现的因子叫做最大涨幅。

这个因子来自中信建投陈升锐老师在2023年发布的研报《高频选股因子分类体系》。

这篇研报中有一大堆的因子，包括量价类的，流动性类的。

但是，笔者能力有限，弄不到这篇研报的原文，所以只能先复现其中的一个因子。如果未来机缘巧合，看到了这篇研报的原文，那么再复现其他的因子吧。

最大涨幅，这个因子的计算可以说是非常简单了，就是（1+分钟级收益率）累乘，只不过这里并不是考虑所有的分钟收益率，而是只考虑分钟收益最大的10%。

该因子背后的逻辑是，MAX为当天前百分之十涨幅推动的股票具体涨幅。短期内，较大涨幅通常由大资金推动，这类“彩票型股票”也更得到投机性投资者的青睐，短期回报较高；长期内，随着投机者离场，股价又会下跌。

```python
def process_single_day(self, idx):

    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'open'])
    rtn = data.to_dataframe('close') / data.to_dataframe('open') - 1
    thd = rtn.quantile(0.9)
    rtn = 1 + (rtn * (rtn >= thd))
    rtn = rtn.fillna(1)
    rtn = rtn.cumprod()
    rtn = rtn.iloc[-1]
    rtn.name = pd.to_datetime(date_str) + timedelta(hours=15)
    return rtn
```

第9行，计算每个标的涨幅最大的10%的门限。

第10-11行，对收益率+1处理，对收益率为nan的直接填充为1，再累乘的时候相当于这些时刻的收益率为0。

值得一提的是，这个因子还有一个日频的版本，如果没记错的话应该是过去9个交易日的收益率绝对值的最大值。这是一个异象因子，在石川老师的《因子投资》这本书中有相关的介绍，感兴趣的大佬们可以去看看。
