本文，笔者要复现的因子是之前复现的时间网络相对中心度因子的姊妹因子，叫做空间网络相对中心度。

这个因子最早出现在学术论文当中，后来在2021年的时候被曹春晓和杨国平两位老师在《股票网络与网络中心度因子研究》这篇研报当中再次提及。可以说是，曹春晓老师早期的因子之一了。

这个因子的计算可以分为三步。

第一步，计算过去20个交易日股票收益率与其他所有股票收益率的皮尔逊相关系数的均值。

第二步，计算平均距离，即2\*（1-第一步得到的均值），然后开方。

第三步，对距离进行平方然后再取其倒数即可。

该因子衡量了股票在网络中与其余股票的相对距离（关联程度），因子值越高，说明股票在网络中与其他股票的距离越近，越处于网络相对中心的位置，对其他股票的相对影响程度也越高。

其计算代码如下：

```python
def __call__(self):
    data = BaseDataLoader.load_data('../../data/stock_bar_1day.parquet', fields=['close']
                                         ).to_dataframe('close')
    ret = data.pct_change()
    res = []
    for i in tqdm(range(21, len(ret))):
        tmp = ret.iloc[i-21:i]
        corr = tmp.corr().values
        np.fill_diagonal(corr, np.nan)
        corr = np.nanmean(corr, axis=0)
        dist = 1 / (2 * (1 - corr))
        res.append(dist)
    res = pd.DataFrame(data=res, columns=ret.columns.tolist(), index=ret.index.tolist()[21:])
    res.index.name = 'datetime'
    res = pd.melt(res.reset_index(), id_vars='datetime', var_name='code', value_name='scc')
    res.to_parquet('./scc.parquet')
```

第8行，计算过去21个交易日，每只股票与其他所有股票的皮尔逊相关系数。

第9行，将对角线的所有元素置为nan。pandas直接使用corr计算的话，还会计算自相关性，也就是矩阵中的对角线元素了。

第10行，求相关性的均值。

第11行，将上面步骤中的第二步和第三步合并了，直接计算得到了最后的因子。

由于这个因子是基于过去21个交易日的日频数据计算得到的，所以它计算出来的时候就可以说是一个月度因子了，不需要再使用过去21个交易日的均值或者标准差来合成了。

总结：这个因子相对来说中规中矩。有六年的IC超过了0.06，但是只有两年超过了0.08，同时没有任何一年的IC超过了0.1。此外，在分层回测的净值曲线中，虽然因子值最低的一组其净值曲线从2019年开始就与其他四条不相交了，但是因子值最大的一组其收益率并不是最好的，反而是因子值第二大的一组收益率最高。
