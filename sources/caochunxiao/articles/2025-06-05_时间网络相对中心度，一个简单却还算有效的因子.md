这个因子在论文和研报中都有提及。

论文是2020年发表在《经济学》上的《股票网络、系统性风险与股票定价》，作者是张自力、闫红雷和张楠。

研报是华西证券2021年发表的《股票网络与网络中心度因子研究》，作者是曹春晓和杨国平。这个时候的曹老师还没去方正证券呀。

该因子的逻辑是，衡量了股票的网络位置随着时间推移的稳定性，因子值越高，说明过去时间段内股票i对网络中心其他股票的收益率偏离程度越小，股票在网络中的位置越稳定。

该因子的计算主要分为三步。

第一步，计算股票在t时刻截面上相对于样本股票池的平均收益的偏离程度。简单来说，就是对股票的收益率在截面上做z-score标准化。

第二步，计算过去20个交易日股票对其他股票收益率偏离程度的均值。

第三步，计算时间维度的网络相对中心度。

```python
def __call__(self):
    data = BaseDataLoader.load_data('../../data/stock_bar_1day.parquet', fields=['close']
                                         ).to_dataframe('close')
    ret = data.pct_change()
    ret_21 = ret.abs().rolling(21).mean()
    ret = ret.sub(ret.mean(axis=1), axis=0).div(ret.std(axis=1), axis=0)
    ret = np.square(ret)
    ret = 1 / ret.rolling(21).mean()
    dfs = {'tcc': ret, 'ret_21': ret_21}
    df = dfs_to_df(dfs)
    df.to_parquet('./tcc.parquet')
```

第2-3行，加载数据。

第4行，计算日收益率。

第5行，计算对比因子，因为从直觉上觉得这因子和收益率绝对值的均值有很高的相关性。

第6行，执行时间网络相对中心度因子计算的第一步。

第7-8行，执行第二步和第三步。需要注意的是，对过去21个交易日求均值是第二步，第三步是取倒数。这里合并了一下。

总结：这个因子的计算确实挺简单的，但是IC还是很不错的，在整个回测区间里有8年的IC绝对值超过了0.08。

虽然，时间网络相对中心度因子和过去21个交易日日收益率绝对值的均值这两个因子有着很高的相关性，但是从因子评价的结果来看，两者还是有很大差异的，特别是在分层回测的表现中。

在IC上的表现，两者可以说是半斤八两，过去21个交易日日收益率绝对值的均值因子IC绝对值超过0.1的年份有两年，最大的一年甚至超过了0.12。但是，它IC绝对值超过0.08的年份却少于时间网络相对中心度因子。

至于分层回测，时间网络相对中心度因子，因子值最高的一组在回测区间里年化收益率最高，因子值最低的一组在回测区间里年化收益率最低。但是过去21个交易日日收益率绝对值的均值就不是这样了，它的因子值最大的一组和因子值最小的一组年化收益率低于其他的三组。

所以，时间网络相对中心度因子这个因子从IC和分层回测表现来看，还是可以的。尽管，它的计算很简单，原理也并不复杂。
