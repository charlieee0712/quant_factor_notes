在8月27日，方正证券的曹春晓老师发布了一篇研报《个股日内成交量分布特征与日内流动性弹性刻画——多因子选股系列研究之二十三》。

从研报的标题就可以看出，这篇研报中一共有两个因子。

一个是个股日内成交量分布特征的因子，研报中将其命名为了“暗流涌动”因子。

另一个是日内流动性弹性刻画的因子，这个因子就叫做“日内流动性弹性”。

在这篇文章中，笔者将首先复现一下“暗流涌动”因子。

**计算步骤**

熟悉曹春晓老师的读者肯定知道，曹老师的因子一般计算步骤较多，这个因子也不例外。在研报中，一共有六步。不过，笔者自作聪明将其浓缩了一下，得到下面这样一个计算步骤。

第一步，计算个股相对成交量，即个股成交量/全市场所有股票的成交量。

第二步，统计每个五分钟区间内的相对成交量占全部时间区间的比值。

第三步，计算成交量分布的香农熵，得到日度成交量分布熵值因子。

第四步，对第三步得到的熵值因子进行均值距离化处理，即减去截面均值然后去绝对值，就得到了日频的因子值。

**代码**

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]    
    # 加载当日分钟数据    
    full_path = os.path.join(self.file_pth, file_name)    
    data = BaseDataLoader.load_data(full_path, fields=['volume'])    
    vol = data.to_dataframe('volume')    
    vol = vol.div(vol.sum(axis=1), axis=0).fillna(0)    
    vol_5min = vol.rolling(5).sum().iloc[list(range(4, 240, 5))]    
    vol_5min = vol_5min.div(vol.sum(axis=0), axis=1)    
    res = -(vol_5min * np.log(vol_5min)).sum(axis=0)    
    res = (res - res.mean()).abs()    
    res.name = pd.to_datetime(date_str) + timedelta(hours=15)    
    return res
```

第8行，对应第一步。

第9-10行，对应第二步。

第11行，对应第三步。

第12行，对应第四步。

众所周知，上面的计算得到的是日频的因子。而曹老师喜欢用过去20个交易日的均值、标准差以及两者之和来合成月度因子，并对月度因子进行评价。

因此，接下来，因子评价的时候，笔者将分别评价均值合成以及标准差合成的因子。需要注意的是，笔者更偏好用过去21个交易日的数据来合成。

**总结**

从上面的因子评价结果来看，用均值合成的因子表现要好于用标准差合成的因子。

从IC的表现来看，均值合成的因子IC绝对值更高。最直观的感受就是IC累加的曲线。同时，从分年度的IC来看，均值合成的因子有三年的IC绝对值超过了0.06，而用标准差合成的因子只有一年的结果满足这一条件。

从分层回测的结果来看，那么均值合成的因子的表现可以说是更好了。

首先，是单调性。均值合成的因子在整个回测区间中的收益率呈现了完美的单调性，虽然在2021年之前的相交还是比较多。反观用标准差合成的因子，group\_3的收益率超过了group\_2。

其次，是多空收益。很显然也是均值合成的因子更好。
