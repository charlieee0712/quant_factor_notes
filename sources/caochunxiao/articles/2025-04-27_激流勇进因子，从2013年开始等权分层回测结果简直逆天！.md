本文介绍的因子叫做激流勇进因子，它来自方正证券研究所于2024年8月29日发布的报告《个股交易放量期间的买入强度刻画与“激流勇进”因子构建——多因子选股系列研究之十九》。

该因子通过成交量和收益率的涨跌将分钟频数据划分为放量上涨、缩量上涨、放量下跌和缩量下跌四个区间。然后计算放量下跌区间的成交额与总成交额的比值，就得到了日频的因子。

需要注意的是，这里用的是滚动5分钟的收益率来判断上涨还是下跌；放量和缩量则是时刻t-4到时刻t这五分钟的成交量之和与t-5到t-1这五个时刻成交量之和作比较来判断的。

具体的实现代码如下所示：

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'volume', 'turnover'])
    dfs = data.to_dataframes()
    ret = dfs['close'].pct_change(5) > 0
    vol = np.where(dfs['volume'].rolling(5).sum().diff() > 0, 1, np.nan)

    condition = ret + vol
    res = []
    for idx in [1, 2]:
        flag = np.where(condition == idx, 1, np.nan)
        res.append((dfs['turnover'] * flag).sum(axis=0) / dfs['turnover'].sum(axis=0))
    res = pd.concat(res, axis=1)
    res.columns = ['down', 'up']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

第2-8行，加载数据。

第9行，判断收益率是涨还是跌。

第10行，判断是放量还是缩量，这里只保留放量的，对于缩量直接置为nan。

第12行，得到每分钟的状态，1代表放量下跌，2代表放量上涨，其他情况全为nan。

后续，笔者计算了两个成交额之比，一个是放量上涨阶段的，一个则是放量下跌阶段的。

无论是研报的分析，还是笔者自己的测试都表明放量上涨并不是一个好的因子。研报当中是这么说的，相较于放量上涨情形，放量下跌期间买入强度较高的股票后续相对于市场有更为明显的超额收益。我们认为其原因在于一方面由于放量下跌过程中容易产生反应过度，后续反转概率较高；另一方面在个股出现放量下跌期间，那些积极逆势买入的投资者对后市表现较为乐观，我们将个股每日放量下跌情形中投资者买入意愿强度定义为“日激流勇进”因子。
