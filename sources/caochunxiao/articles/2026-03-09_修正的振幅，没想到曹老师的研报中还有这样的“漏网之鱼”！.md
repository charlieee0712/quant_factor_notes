**修正的振幅**

笔者最近发现，在曹老师的众多因子中，有一个因子笔者完全没有提过，它就是修正的振幅因子。

笔者并没有复现完曹老师的所有因子，有些是因为数据问题，有些则是效果不是最好的，比如修正的模糊价差，但是这个因子笔者在文章中提到过[模糊系列因子，有一个因子四年IC绝对值超过了0.1，分层回测也很好，真的值得看！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247487930&idx=1&sn=7058accf1e91539734f39899d6946df8&scene=21#wechat_redirect)

至于，修正的振幅这个因子，它来自曹老师2022年发布的研报《个股股价跳跃及其振幅因子的改进》。在此之前，笔者已经复现过这篇研报中的两个因子，它们在[日内跳跃度因子，这个因子来自研报，果然换成分钟级数据之和有惊喜，IC非常稳定](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247486367&idx=1&sn=833a16be81de07e3afad80eab84ce15a&scene=21#wechat_redirect)和[单复利差因子，IC依旧稳定，甚至超越了日内跳跃度因子](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247486367&idx=2&sn=0607addd0f3d9f66570cad8af90ede73&scene=21#wechat_redirect)这两篇文章中有过相关的介绍。

本文，要介绍的修正的振幅因子，和之前介绍的日内跳跃度因子有一定的关系，其背后的逻辑是这样的。

> 修正振幅因子利用股票日跳跃度对传统振幅的变化过程做了区分；那些未发生跳跃或跳跃程度较小的股票，可能并未吸引到博彩偏好型投资者，股价也未过度反应，可以对这些时刻的振幅做反向修正，增强其低波动异象的属性。

**计算步骤和代码**

这个因子的计算需要用到日内跳跃度因子，但是之前的文章介绍过这个因子了，所以这里将不赘述了，大家可以看之前的文章。

**1**

**计算步骤**

第一步，计算日内跳跃度因子。

第二步，计算日内跳跃度因子的横截面均值，将日内跳跃度小于横截面均值的标的的振幅做修正（取相反数），其他标的的振幅不变。

**2**

**代码**

```python
def __call__(self):
    jump = BaseDataLoader.load_data('../曹春晓/日内跳跃度/jump.parquet', fields=['jumps_1min']).to_dataframe('jumps_1min')
    data = BaseDataLoader.load_data('../../data/stock_bar_1day.parquet', fields=['close', 'high', 'low']).to_dataframes()
    amp = (data['high'] - data['low']) / data['close'].shift()
    dfs = {'amp': amp}
    flag = jump > jump.mean(axis=1).values.reshape(-1, 1)
    flag = flag.reindex(amp.index.tolist())
    flag = flag.reindex(columns=amp.columns.tolist())
    modify_amp = amp * np.where(flag, 1.0, -1.0)
    dfs['modify_amp'] = modify_amp
    df = dfs_to_df(dfs)
    df.to_parquet('./modify_amp.parquet')
```

由于，之前已经计算过日内跳跃度因子了，所以这个因子变成了日频因子了，计算代码就简单了。

第2-3行，读取数据。

第4行，计算振幅。

第5行，保存一下振幅因子，后面用来和修正的振幅因子计算相关性。

第6行，将日内跳跃度因子和截面均值比较，得到大于截面均值的信息。

第7-8行，对其flag和振幅因子的行和列，这样就可以直接相乘了。

第9行，计算修正的振幅因子。

**因子评价**

由于修正的存在，这两个因子的相关性应该是不高的。

![](images/2026-03-09_修正的振幅，没想到曹老师的研报中还有这样的“漏网之鱼”！/002.png)

相关性分析证实了这一点，两者的截面斯皮尔曼相关系数为-0.36，而用世坤的方式计算因子收益率，然后计算因子收益率之间的相关性的话，两者的相关性为0.37。

![](images/2026-03-09_修正的振幅，没想到曹老师的研报中还有这样的“漏网之鱼”！/003.png)

在研报中，这个因子是用均值合成的，所以接下来展示的评价结果，也是用过去21个交易日的均值合成的。

但是，如果用标准差合成的话，因子的IC会高很多。
