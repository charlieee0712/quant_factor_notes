这个因子的计算有点小复杂了。

第一步，对每个标的，计算三个符号。

这三个符号是最近1分钟收益率符号，当前1分钟收益率相对前5分钟收益率（不包括当前分钟）的变化符号，当前1分钟成交量相对前5分钟成交量（不包括当前分钟）的变化的符号。

若最近1分钟收益率、当前1分钟收益率相对前5分钟收益率变化、当前1分钟成交量相对前5分钟的成交量变化为零，则它们的符号根据当前分钟的收盘价相对前5分钟（不包括当前分钟）均价的变化的符号为准，若该符号仍为零，则按照当前分钟的收盘价相对前一日收盘价变化的符号为准。

这样，对于一个标的每分钟有三个符号。

将该标的与其他所有标的比较，将同一时间同一种类符号相同的次数累加，得到该标的与其他标的的协同次数。

第二步，取当天与其协同次数最多的30只股票作为其协同效应股票，将该股票当天收益率减去这30只股票的平均收益率，得到相对协同股票的超额收益率。

虽然，看起来只有两步，但是通过代码实现就有那么一点点复杂了。

在实现中，笔者一共定义了三个方法，我们一个一个来看。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    if idx == 0:
        return
    else:
        last_df = self.last_close.iloc[[idx-1]]

    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'volume'])
    last_df = last_df.reindex(columns=data.codes)
    dfs = data.to_dataframes()

    ret = dfs['close'].pct_change(1)
    ret_ma5 = ret - ret.rolling(5).mean().shift(1)
    vol_ma5 = dfs['volume'] - dfs['volume'].rolling(5).mean().shift(1)
    close_diff = dfs['close'] - dfs['close'].rolling(5).mean().shift(1)
    close_diff_last = dfs['close'].sub(last_df.iloc[0], axis=0)

    self.ret_daily = dfs['close'].iloc[-1] / last_df.iloc[-1] - 1
    self.ret_dir = self.cal_flag(ret, close_diff, close_diff_last)
    self.ret_ma5_dir = self.cal_flag(ret_ma5, close_diff, close_diff_last)
    self.vol_ma5_dir = self.cal_flag(vol_ma5, close_diff, close_diff_last)
    res = Parallel(n_jobs=-1, verbose=10)(
        delayed(self.cal_factors)(code)
        for code in data.codes
    )
    res = pd.Series(data=res, index=data.codes.tolist(), name=pd.to_datetime(date_str) + timedelta(hours=15))
    return res
```

这里需要用到前一天的收盘价，所以多了第5-8行的代码。

第14行之前的代码都是用来加载数据的。

第16行，计算收益率。

第17行，计算收益率相对于前5分钟的变化。

第18行，计算成交量相对于前5分钟均值的变化。

第19行，计算收盘价相对于5分钟前的变化。

第20行，计算收盘价相对于昨日收盘价的变化。

第22行，计算标的当日的收益率，在最后计算超额收益率的时候会用到。

```python
@staticmethod
def cal_flag(data, mask_data1, mask_data2):
    data_dir = data > 0
    mask = (data == 0) & (mask_data1 != 0)
    data_dir[mask] = mask_data1 > 0
    mask = (data == 0) & (mask_data1 == 0) & (mask_data2 != 0)
    data_dir[mask] = mask_data2 > 0
    return data_dir
```

cal\_flag方法就是计算数据的符号，拿收益率举例。

第3行，就是对收益率大于0的置为1，其他为0。

第4行，找到mask\_data1不为0的位置。在这里其实就是当前收盘价和5分钟之前收盘价之差。

第5行，将第4行找到的位置中的数据按照，mask\_data1来填充其符号。如果mask\_data1大于0，就为1，否则继续为0。

第6-7行，就是将4-5行的mask\_data1换成了mask\_data2。在这里，是当前收盘价与前一日的收盘价之差。

```python
def cal_factors(self, tar):
    res = 0
    for key in ['ret_dir', 'ret_ma5_dir', 'vol_ma5_dir']:
        tmp_data = self.__getattribute__(key)
        tar_data = tmp_data[tar]
        tmp_data = tmp_data.drop(columns=[tar])
        res += tmp_data.mul(tar_data, axis=0).sum()
        res += (1 - tmp_data).mul(1 - tar_data, axis=0).sum()
    res.sort_values(inplace=True, ascending=False)
    res = res.iloc[:30]
    ex = self.ret_daily.loc[tar] - self.ret_daily.reindex(res.index).mean()
    return ex
```
