**短线交易拥挤度**

本文，笔者将复现中信建投姚紫薇老师研报《市场微观结构系列研究：从分钟频到Level2的探索与发现》中提到的短线交易拥挤度因子。

这个因子关注的是尾盘的成交量，由最后30分钟成交量的重心和最后3分钟成交量占比组合而来。

这两个因子的相关性其实是很高的，但是姚老师认为这两个因子从不同的角度描述了尾盘资金聚集现象，重心是变化趋势的角度，而另一个则是占比角度。因此，两者可以互补。

**计算步骤和代码**

计算步骤，研报里面写了五步，其实用一句话总结就是最后30分钟成交量的重心和最后3分钟成交量占比这两个因子在截面上排序求和。

**1**

**计算步骤**

第一步，将时间戳转换为序号，9:31对应1，15:00对应240。

第二步，计算最后30分钟，也就是从211-240分钟的成交量重心，公式可以参考[时间和，一个分层回测表现有点离谱的多头因子！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247493059&idx=1&sn=8371648040cfc68b3c850c8bad6bc488&scene=21#wechat_redirect)这篇文章，也可以看后面代码。

第三步，计算最后三分钟成交量占比。

第四步，计算第二步和第三步因子值在截面上的排序并求和。

第五步，取过去21个交易日的均值。

**2**

**代码**

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    full_path = os.path.join(self.file_pth, file_name)
    vol = BaseDataLoader.load_data(full_path, fields=['volume']).to_dataframe('volume')
    ratio = vol.iloc[-3:].sum() / vol.sum()
    vol = vol.iloc[-30:]
    t_list = np.arange(211, 241).reshape(-1, 1)
    gu = (vol * t_list).sum() / vol.sum()
    res = ratio.rank(pct=True) + gu.rank(pct=True)
    res = pd.concat([res, gu, ratio], axis=1)
    res.columns = ['stc', 'vol_gravity', 'ratio']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    return res
```

前5行，读取数据。

第6行，计算最后3分钟成交量占比。

第7-9行，计算最后30分钟成交量重心。

第10行，计算短线交易拥挤度因子。
