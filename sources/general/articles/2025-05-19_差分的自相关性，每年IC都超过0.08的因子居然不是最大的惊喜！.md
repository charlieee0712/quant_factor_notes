这一篇文章要介绍的因子，是将长江证券的首席分析师覃川桃老师的研报和兴业证券的首席分析师郑兆磊老师的研报相结合得到的。

覃川桃老师的研报提到了差分标准差因子，郑兆磊老师的研报介绍过一阶自相关性。

那么，用差分的一阶自相关性会擦出什么样的火花呢？

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume', 'turnover'])
    date_str = file_name.split('.')[0]

    vol = data.to_dataframe('volume')
    money = data.to_dataframe('turnover')
    vol = (vol.diff()).div(vol.mean(axis=0), axis=1)
    money = (money.diff()).div(money.mean(axis=0), axis=1)
    money_vol_dis = (np.square(money - vol)).mean()

    res = [vol.shift(1).corrwith(vol), money.shift(1).corrwith(money), vol.corrwith(money), money_vol_dis]
    res = pd.concat(res, axis=1)
    res.columns = ['vol', 'money', 'mcv', 'money_vol_dis']
    res['diff'] = res['money'] - res['vol']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

代码是很简单的，就是计算成交量和成交额的差分，对应代码低8-10行。

然后，计算它们的自相关性，以及成交量差分和成交额差分的相关性，还有一个就是欧式距离。

此外，为了和之前的因子进行比较，还计算了成交量差分的一阶自相关性和成交额差分的一阶自相关性的差。

先来看一看这一系列因子，和普通的一阶自相关性因子的一个相关性，然后咱们再来进一步分析。
