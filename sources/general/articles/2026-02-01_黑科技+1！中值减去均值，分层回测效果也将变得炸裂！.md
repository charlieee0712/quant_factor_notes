**中值-均值**

在去年12月底的时候，笔者复现了方正证券曹春晓老师的滴水石穿因子（[滴水石穿，金工诗人曹春晓老师最新力作，6年IC超0.1，其中3年IC超0.12！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247492920&idx=1&sn=b5cb8732c94a2198a8ab6dfc307b53c7&scene=21#wechat_redirect)），然后根据自己的专业知识，将该因子计算步骤中的傅里叶变换（FFT）换成了IAA算法（[量化+雷达算法！用IAA（迭代自适应）算法优化滴水石穿因子，分层回测单调性提升！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247493445&idx=1&sn=537a9ff60c31f1665d39cac987155ba7&scene=21#wechat_redirect)）。

虽然，用FFT和IAA计算出来的因子相关性为1，但是从原理上来说，IAA相比FFT能够更精确的估计信号和噪声。所以，笔者想到了用两者相减构建一个新的因子，没想到出现了奇迹（[黑科技！两个相关性很高的因子相减，居然会出现“奇迹”！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247493813&idx=1&sn=0013fbc8f05b504083c7033de269e15c&scene=21#wechat_redirect)）。

最近，笔者想到了，在信号处理中，还有中值滤波和均值滤波两种不同的滤波方式，这两种滤波方式得到的结果相关性也是很高的。同时，中值滤波不易受到极端值的影响。

因此，笔者想到用中值-均值会不会也出现奇迹。于是，就有了这篇文章。

**代码**

在验证这个想法的时候，笔者选择了曹春晓老师的孤雁出群因子（[孤雁出群因子，名字很美，计算步骤略多，分层回测单调性不错](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247486558&idx=1&sn=a6229a110edb6dc6ab914cfb1f8e17ee&scene=21#wechat_redirect)）。

这个因子的计算步骤还是挺多的，这里就不赘述了，因为它并不是本文的重点。如果各位大佬对这个因子的实现细节感兴趣的话，可以移步引用的文章去看一看。

**代码**

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['turnover', 'close', 'open', 'volume'])
    rtn = data.to_dataframe('close') / data.to_dataframe('open') - 1
    sigma = rtn.std(axis=1)
    sigma = sigma < sigma.mean()
    turnover = data.to_dataframe('turnover')
    turnover = turnover[sigma].corr().abs()
    mask = np.eye(turnover.shape[0], turnover.shape[1], dtype=bool)
    res = pd.concat([turnover.mask(mask, np.nan).mean(axis=1),
                     turnover.mask(mask, np.nan).median(axis=1)], axis=1)
    res.columns = ['money_mean', 'money_med']
    res['diff'] = res['money_med'] - res['money_mean']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    return res
```

第2-6行，读取数据。

第7行，计算分钟收益率。

第8-9行，计算截面上收益率的波动，当波动小于全天均值的时刻就是不分化时刻。孤雁出群因子就是计算每个标的与其他所有标的不分化时刻的成交额相关性的绝对值的均值。

第10-11行，就是计算相关性的绝对值。

第12行，将对角线，也就是自己和自己的相关性置为nan。

第13-14行，计算相关性绝对值的均值和中值。

相关性计算的结果是符合预期的，均值（money\_mean）和中值（money\_med）的相关性为1，而两者之差（diff）因子和它们的相关性接近于0。

由于本文主要就是来介绍diff因子的，所以接下来的因子评价结果都是以diff因子为主的，至于money\_mean和money\_med则留给各位大佬自行尝试了。不过，笔者可以简单的说一嘴，这两个因子用均值合成的分层回测最终的单调性好一点，但是用标准差合成的区分度更高（可惜的是group\_1的收益率高于group\_0）。

关于diff因子，笔者发现用标准差合成的话，其IC和分层回测的表现都是好于均值的。
