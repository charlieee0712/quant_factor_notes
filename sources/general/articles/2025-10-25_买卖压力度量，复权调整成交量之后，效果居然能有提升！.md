![Image](images/2025-10-25_买卖压力度量，复权调整成交量之后，效果居然能有提升！/002.jpg)本文，笔者将复现东方证券朱剑涛老师2019年10月29日发布的研报《因子选股系列研究之六十：基于量价关系度量股票的买卖压力》。

2019年的时候，因子选股系列已经发布了60篇了，这含金量无需多言。

研报中，认为市场上存在两类投资者，A类着眼于中长期，B类看中短期。所以，B类投资者的交易是噪声交易。

通过对A类投资者的分析，这类投资者买入的时候，标的在价格低位的成交量大于价格高位；卖出的时候，标的在价格高位的成交量大于价格低位。

基于这样的一个思路，就有了这个叫做买卖压力度量的因子。

vwap很好理解，就是成交量加权平均价格。

此外，在研报中还隐藏了一个细节，那就是vwap和volu都是经过复权调整的。

说实话，笔者不是很理解这一句话，主要是成交量是如何经过复权调整的。

实在想不明白，笔者就简单粗暴，直接将成交量乘上了复权因子。

**计算代码**

```python
def __call__(self):
    data = BaseDataLoader.load_data('../../data/stock_bar_1day.parquet',
                                         fields=['vwap', 'volume', 'factor']).to_dataframes()
    self.vwap = data['vwap'] * data['factor']
    self.vol = data['volume'] * data['factor']
    res = []
    for i in tqdm(range(21, len(self.vwap)+1)):
        res.append(self.cal_factor(i))
    res = pd.concat(res, axis=1).T
    res.index.name = 'datetime'
    res.reset_index(inplace=True)
    res = pd.melt(res, id_vars='datetime', var_name='code', value_name='reverse_prob')
    res.to_parquet('./reverse_prob.parquet')
```

第一段代码，主要就是读取数据。主要是第4行和第5行，需要说明一下，这两行的作用就是对vwap和volu进行复权调整。

```python
def cal_factor(self, idx):
    vwap = self.vwap.iloc[idx-21:idx]
    day = vwap.iloc[[-1]].index.tolist()[0]
    vol = self.vol.iloc[idx-21:idx]
    vol = vol.div(vol.mean(), axis=1)
    res = np.log(vwap.mean() / (vwap * vol).sum())
    res.name = day
    return res
```

第二段代码，就是按照公式计算每21个交易日的因子值。

**因子评价**

由于这个因子在计算的时候已经使用了过去21个交易日的量价数据了，所以它本身就可以看作是一个月度因子了，无需在对过去21个交易日的因子值取均值或者标准差了。

值得一提的是，笔者在复现的时候也尝试过，对vwap进行复权，而不对volu进行复权，得到的因子表现不如都复权处理的。所以，下面的因子评价结果展示的是对vwap和volu都复权处理计算的因子。
