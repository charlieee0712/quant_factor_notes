**日内量价背离斜率**

之前，笔者都是让DeepSeek来生成一些因子。在股票上和基金上都有过尝试，股票上的尝试太久远了（记忆中总是觉得差那么一点点），但是最近在基金上的尝试还是不错的。

于是，这一次，笔者准备让豆老师来生成一个选股因子，看看效果如何。

豆老师参考了12篇资料，其中有两篇研报是笔者复现过的，一篇是魏建榕老师的高频成交量的峰、岭、谷信息，这篇研报笔者分了好几篇文章来复现，其中效果比较好的一篇是[峰岭成交比因子，IC和分层回测都表现不错！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247491403&idx=1&sn=5c2277221d5b56a76afe99902eec6149&scene=21#wechat_redirect)；另一篇还是魏老师的，高频振幅因子的内部切割，这篇研报笔者在[高频振幅因子的内部切割，笔者在3月10日提过的因子出现在了大佬的研报中！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247489922&idx=1&sn=675adc95896fa264f82e23ae0ee43b0a&scene=21#wechat_redirect)这篇文章中提到过。

豆老师给的这个因子的名字叫做日内量价背离斜率，其核心逻辑是用分钟线拆解日内量价同步/背离的“加速度”，捕捉主力资金悄悄调仓的痕迹：日内价量相关系数的线性趋势斜率，负值更强时多空分歧，反转信更明确；正值更强时情绪一致、趋势延续更稳。

**计算步骤和代码**

在计算的时候，笔者不仅计算了beta，还计算了alpha，看看这两个到底哪个更好。

**1**

**计算步骤**

第一步，计算分钟收益率（对数价格相减），这里豆老师是用收盘价计算的，每日第一分钟的收益率需要用到昨日收盘价。根据笔者之前研究情况，用开盘价计算其实效果是差不多的，而且不用乘复权因子了，实现起来简单。

第二步，计算每20分钟收益率和成交量的相关性。这里笔者追问了一下豆老师，豆老师表示需要进行截面Z-Score标准化。

第三步，以相关性序列为被解释变量（y），时间序列从1到221为解释变量（x）进行线性回归。

**2**

**代码**

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'volume', 'open'])
    vol = data.to_dataframe('volume')
    vol = (vol - vol.mean(axis=1).values.reshape(-1, 1)) / vol.std(axis=1).values.reshape(-1, 1)
    close = np.log(data.to_dataframe('close')) - np.log(data.to_dataframe('open'))
    corr_list = []
    for i in range(20, len(vol) + 1):
        tmp_corr = close.iloc[i-20:i].corrwith(vol.iloc[i-20:i])
        corr_list.append(tmp_corr)
    corr = pd.concat(corr_list, axis=1).T
    res = self.cal_beta(corr)
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    return res
```

第一段代码的核心部分在第9-13行，计算每20分钟的相关性。

```python
@staticmethod
def cal_beta(data):
    x = np.arange(1, 222, 1)
    y = data.values
    X_full = np.c_[np.ones((221, 1)), x]
    beta_full = np.linalg.pinv(X_full.T @ X_full) @ X_full.T  @ y
    return pd.DataFrame(beta_full, index=['alpha', 'beta'], columns=data.columns.tolist()).T
```

第二段代码的线性回归，这里笔者直接用最小二乘的公式进行计算了。这样处理比较快，但是需要注意的是，数据里面不能有缺失值，如果有缺失值的话那就会beta为nan了。

第3行，生成解释变量x。

第4行，获取被解释变量y。

第5行，给解释变量x前面增加一个截距项。

第6行，通过最小二乘的公式得到alpha和beta，其中beta就是豆老师提供的日内量价背离斜率因子，后续为了表述简单就称其为beta因子。
