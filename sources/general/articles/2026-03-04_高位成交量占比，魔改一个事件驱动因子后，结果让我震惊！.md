**高位成交量占比**

本文介绍的因子，是笔者在国盛证券沈芷琦老师2025年11月26日发布的研报《“量价淘金”选股因子系列研究（十五）：高低位放量事件簇-正负向信号的有机结合》的基础上魔改的。

为什么是魔改的呢？

因为在研报中，沈老师是通过在高位放量、低位放量信号中分别筛选一批有效且低相关的信号，得到“高位放量事件簇”、“低位放量事件簇”，并将信号互相叠加得到综合信号、构建事件驱动策略。

在这里面，沈老师用到了逐笔成交数据来对激进程度进行划分，这个数据笔者是没有的。

简单来说，就是笔者能力不足，不足以完全复现沈老师的因子。所以，就只能魔改了。

值得一提的是，根据沈老师的设计，笔者想到了一个择时因子。在低位放量的时候买入，在高位放量的时候卖出。后续，笔者想办法弄点指数的数据，然后尝试一下。

**计算步骤和代码**

魔改后因子的计算应该是较为简单的。毕竟，只需要找到高位，然后计算其成交量与全天成交量的比值即可。

**1**

**计算步骤**

第一步，寻找高位。在研报中给出了三种方式，收盘价大于当日所有收盘价90%分位数，3sigma准则或者1.5sigma准则。

这里笔者选择的是1.5sigma准则，因为测试效果比3sigma好。至于为什么不选择90%分位数，因为选择分位数的话就意味着这一天一定有高位了，但是它是可以没有高位的。

第二步，计算高位成交量占全天成交量的比值。

**2**

**代码**

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'volume'])
    date_str = file_name.split('.')[0]
    cur = pd.to_datetime(date_str) + timedelta(hours=15)
    close = data.to_dataframe('close')
    vol = data.to_dataframe('volume')
    high_pos = close > close.mean() + 1.5 * close.std()
    tot_vol = vol.sum()
    res = (np.where(high_pos, 1.0, np.nan) * vol).sum()
    res = res / tot_vol
    res.name = cur
    return res
```

代码应该不需要过多的说明了，就是一个1.5sigma准则判断高位（第10行），然后计算高位成交量占比（第11-13行）就结束了。
