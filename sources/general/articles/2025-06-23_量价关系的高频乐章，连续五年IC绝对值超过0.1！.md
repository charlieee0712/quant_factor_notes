本文要复现的因子来自方正证券2020年2月27日发布的研报《市场微观结构剖析系列6——量价关系的高频乐章》，作者是朱定豪和严佳炜两位老师。

在这篇研报中，主要介绍了四个因子。

分别是，收盘价和成交额的相关性；对数收益率的绝对值和成交额的相关性；滞后一期的成交额与对数收益率绝对值的相关性；滞后一期对数收益率绝对值与成交额的相关性。

本文复现的是最后一个，滞后一期对数收益率绝对值与成交额的相关性。

看到这里，各位大佬肯定会有疑惑，不就是计算一个相关性吗？有什么复杂的呢？

因为，这里的成交额不是直接使用的，而是需要进行一次z-score标准化的。同时，这里的z-score标准化不是截面的，也不是同一天时序上的，而是过去20天同一分钟的成交额进行z-score标准化。

对于这个因子，其背后的逻辑是：分钟成交额在日内呈U型或W型分布，与收益率分布的不对称会影响相关性的计算，对成交额做标准化可以捕捉成交额的相对波动；若存在较多收益为0的样本也会使系数偏高，需将其剔除；修正后的因子能捕捉更真实纯粹的量价异动，因子表现大幅提升。

```python
def process_single_day(self, idx):
    if idx < 20:
        return pd.DataFrame({})
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close']).to_dataframes()
    ret = np.abs(np.log(1+data['close'].pct_change().shift(1)))

    amt = []
    for i in range(idx-20,idx+1):
        file_name = self.files[i]
        full_path = os.path.join(self.file_pth, file_name)
        amt.append(BaseDataLoader.load_data(full_path, fields=['turnover']).to_dataframe('turnover'))
    amt = pd.concat(amt)
    amt['minute'] = amt.index.hour * 60 + amt.index.minute
    amt = amt.groupby('minute', as_index=False, group_keys=False).apply(self.z_score)

    res = ret.corrwith(amt)
    res.name = pd.to_datetime(date_str) + timedelta(hours=15)
    return res

def z_score(self, group):
    group = group.sub(group.mean(axis=0), axis=1).div(group.std(axis=0), axis=1)
    group = group.iloc[[-1]]
    return group.drop(columns=['minute'])
```

第8-10行，计算滞后一阶的对数收益率的绝对值。

第12-17行，读取过去21个交易日的成交额数据。

第18行，计算当前交易时间是当天的第几分钟，用于进行同分钟数据的z-score标准化。

第19行，对分钟相同的数据进行聚合，然后标准化。

这里需要注意的是，笔者计算相关性的时候并没有去掉收益率为0的情况，因为在因子评价的时候发现不去掉效果更好。
