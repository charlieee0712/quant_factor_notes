本文，笔者要复现的因子来自《中银证券》2022年9月2日发布的研报《中银量化多因子选股系列（三）：锚定反转因子构建》。

该因子的构建考虑了心理学的因素。

若个股近期上涨， 反转投资者倾向于选择近期涨幅最小的个股：以股票近期低点作为“锚点” 计算股票的相对涨幅作为因子值；若个股近期下跌， 反转投资者倾向于选取近期跌幅最大的个股：以股票近期高点作为“锚点” 计算股票的相对跌幅作为因子值。

```python
def __call__(self):
    data = BaseDataLoader.load_data('../../data/stock_bar_1day.parquet', fields=['close', 'high', 'low']
                                         ).to_dataframes()
    flag = data['close'] < data['close'].shift(21)
    anchor = data['high'].rolling(21).max() * flag + (1 - flag) * data['low'].rolling(21).min()
    res = data['close'] / anchor - 1
    res.index.name = 'datetime'
    res.reset_index(inplace=True)
    res = pd.melt(res, id_vars='datetime', var_name='code', value_name='anchoring_reverse')
    res.to_parquet('./anchoring_reverse.parquet')
```

这个因子的计算非常简单。

第4行，通过当前收盘价与过去第21天收盘价进行比较，如果当前收盘价小于过去第21天的收盘价则认为个股近期下跌，否则认为个股近期上涨。

第5行，计算锚点，对于下跌的情况，锚点是过去21个交易日的最高价；对于上涨的情况，锚点是过去21个交易日的最低价。

第6行，计算锚定反转。

由于这个因子是直接使用了过去21个交易日的数据计算得到的，所以不存在合成的问题。因此，在因子评价的时候，直接用当日计算到的因子即可。
