**跨周期收益迁移因子**

已经连续两天没有好好休息了，一直在研究新的量化选基因子，有很多构想，但是效果够不好。

于是，只能取pua豆老师了，在无数次的pua之后，豆老师终于给了一个勉强合格的因子，它叫做跨周期收益迁移因子。

传统因子只看 “单一周期收益”，而 CTF 聚焦 “收益能力的跨周期迁移性”—— 优质基金的赚钱能力，能从 “震荡市” 迁移到 “牛市”“熊市”，而运气型基金的收益仅集中在某一周期。

**计算步骤和代码**

这个因子豆老师给的过于复杂了（效果也不好），笔者对其进行了简化，所以下面的步骤是笔者简化之后的。

**1**

**计算步骤**

第一步，将日收益率划分为三个时间段，前期（1-6 个月）、中期（7-12 个月）、后期（13-18 个月）。

第二步，计算前期和中期的收益相关性以及中期和后期的收益率相关性。

第三步，对两者取平均。

**2**

**代码**

```python
@staticmethod
def __cal_factors__(group):
    group = group.dropna()
    col = group.columns.tolist()[0]
    days = group.index.tolist()
    idx_list = []
    res = []
    for year in range(2017, 2025):
        cur = pd.to_datetime('{}-12-31'.format(year)) + timedelta(hours=15)
        idx = bisect.bisect_right(days, cur)
        if idx < 378:
            res.append(np.nan)
        else:
            tmp = group[[col]].iloc[idx-378:idx].dropna(axis=1)
            if tmp[col].abs().mean() == 0:
                res.append(np.nan)
            else:
                x1 = tmp.iloc[:126].reset_index(drop=True)
                x2 = tmp.iloc[126:252].reset_index(drop=True)
                x3 = tmp.iloc[252:].reset_index(drop=True)
                corr = (x2[col].corr(x3[col]) + x1[col].corr(x2[col])) / 2
                res.append(corr)
        idx_list.append(cur)
    res = pd.Series(res, index=idx_list, name=col)
    return res
```

核心代码从第8行开始。

第9行，取当年的最后一天。

第10行，获取小于等于最后一天的索引。

第11-16行，如果没有378条数据，一年半，那就将因子值置为nan。

第18-20行，将收益率分为前中后期。

第19行，执行计算步骤中的第二步和第三步。
