**不对称性**

通常来说，常见的衡量不对称性的方法主要是偏度。在大A这个市场中，偏度已经被广泛使用了，收益率的偏度、成交额的偏度，这些都是很常见的因子了。

最近，笔者找到了一篇关于不对称性的研报，里面的内容可以说是给了我们不少新的工具。这篇研报是东方证券朱剑涛老师在2021年12月25日发布的《因子选股系列研究之八十：收益率的非对称分布与尾部蕴含的Alpha》。

这里的输入数据是一个series，具体为一个标的一天239分钟的收益率。

```python
def __cal_factors__(self, data):
    data = data.dropna()
    if data.std() == 0:
        return 0
    if len(data) < 10:
        return np.nan  # 核密度估计至少需要10个样本
    # 2. 核密度估计（拟合 PDF）
    kde = gaussian_kde(data, bw_method='scott')
    # 3. 构建 x 轴网格（覆盖收益率主要分布区间）
    mu, sigma = data.mean(), data.std()
    x_min = mu - 3 * sigma
    x_max = mu + 3 * sigma
    x_grid = np.linspace(x_min, x_max, 1000)
    # 4. 计算网格上的 PDF 与 CDF
    pdf_grid = kde.evaluate(x_grid)  # PDF 值
    # 梯形法累积积分 → CDF
    dx = x_grid[1] - x_grid[0]
    cdf_grid = cumulative_trapezoid(pdf_grid, dx=dx, initial=0)
    cdf_grid = np.clip(cdf_grid, 0.0, 1.0)  # 数值截断
    # 5. 将原始收益率点映射到 CDF 曲线上
    pdf_xi = kde.evaluate(data)  # 每个 xi 处的 PDF 值
    cdf_xi = np.interp(data, x_grid, cdf_grid)  # 每个 xi 处的 CDF 值
    # 6. 计算相关系数 ρ[ PDF(xi), CDF(xi) ]
    r, _ = pearsonr(pdf_xi, cdf_xi)
    # 7. 最终因子 Asym = -ρ
    asym = -r
    return asym
```
