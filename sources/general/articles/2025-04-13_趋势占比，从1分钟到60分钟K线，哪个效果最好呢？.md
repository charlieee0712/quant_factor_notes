该因子来自广发证券2021年的研报，《高频量化数据因子化方法》，作者是文巧均、安宁宁和罗军三位老师。

该因子可以看作是Ribeiro价格位移与路程之比，衡量了日内股价的趋势强度，取值范围位-1~1，通常与未来收益负相关。

这个因子的计算也是比较简单的一个公式。

每日最后一条（K线的收盘价-第一条K线的收盘价）/（每条K线收盘价差分的绝对值之和）。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close'])
    diff = np.diff(data.data[:, 0, :], axis=0)

    res = data.data[-1, 0, :] - data.data[0, 0, :]
    res = res / np.sum(np.abs(diff), axis=0)
    res = pd.Series(data=res, index=data.codes, name=pd.to_datetime(date_str) + timedelta(hours=15))
    return res
```
