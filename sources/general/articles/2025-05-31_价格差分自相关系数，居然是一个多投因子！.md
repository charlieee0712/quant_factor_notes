这个因子来自东吴证券高子剑老师在2021年3月1日发布的研报《“技术分析拥抱选股因子”系列研究（五）-CPV因子错位版，价量自相关性中蕴藏的选股信息》，该因子和笔者之前介绍的，兴业证券郑兆磊老师研报中提到的一阶自相关性基础因子有点类似，但还是有差别。

高老师的一阶自相关性计算用的是价格的差分，而郑老师用了收益率和成交量占比。同时，高老师还在研报中对差分的数据进行了分类，正差分的自相关性和负差分的自相关性。

具体的分法，咱们根据代码来看。

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close'])
    date_str = file_name.split('.')[0]

    price = data.to_dataframe('close').diff()
    price_delay1 = price.shift(1)
    self_corr = price_delay1.corrwith(price)

    flag = (price > 0) * (price_delay1 > 0)
    self_corr_plus = price_delay1[flag].corrwith(price[flag])

    flag = (price < 0) * (price_delay1 < 0)
    self_corr_minus = price_delay1[flag].corrwith(price[flag])

    res = [self_corr, self_corr_plus, self_corr_minus]
    res = pd.concat(res, axis=1)
    res.columns = ['self_corr', 'self_corr_plus', 'self_corr_minus']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

第8-10行，计算的是价格差分的一阶自相关性。

第12-13行，计算的是正的差分自相关性，这里的正不是说差分为正就行了，而是t时刻的差分和t-1时刻的差分同时为正。

第15-16行，计算的是负差分自相关性。

self\_corr

这个因子用均值合成月度因子的IC略高于用标准差合成的情况。用标准差合成的只有2023年一年的IC绝对值超过了0.08，但是用均值合成的有三年的IC绝对值超过了0.08！

在分层回测方面，用均值合成的因子可以看作是一个多头因子，因为其因子值最低的一组收益率遥遥领先其他四组。而一旦用标准差合成，那么分层回测就没有这么高的区分了，多空收益远低于均值合成的。

总结：这个因子复现的时候其实是没有什么期待的，没想到居然是一个看起来还行的多头因子。这系列因子中需要注意的是，正差分相关性和负差分相关性计算的时候可能会因为数据太短导致相关性为nan的情况。笔者简单粗暴，直接没有填充。然后，合成月度因子的时候是对过去21个交易日当中有值的那几天进行合成的。
