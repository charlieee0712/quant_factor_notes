本文要介绍的因子是参考了两篇研报的。

第一篇，是兴业证券郑兆磊老师在2024年6月5日发布的研报《高频研究系列六——时序信息中的Alpha》。在这篇研报中，郑老师提到了成交量占比的一阶自相关性因子。这个因子在IC上的表现是这样的，2018年、2020年、2021年、2022年和2023年这五年的IC绝对值超过了0.1，其中2018年这一年的IC绝对值达到了0.12。

第二篇，是方正证券朱定豪和严佳炜两位老师在2020年2月27日发布的研报《市场微观结构剖析系列6——量价关系的高频乐章》。在这篇研报中，笔者复现了滞后一期对数收益率绝对值与成交额的相关性。这里的成交额，是对过去21个交易日同一分钟的成交额进行z-score标准化后得到的。这个因子在不去掉收益率为0时刻的情况下，其从2018年到2023年这六年的IC绝对值都超过了0.1。

在这两篇研报的基础上，笔者得到了启示，对成交额进行第二篇研报所述的过去21个交易日同一分钟的成交额进行z-score标准化，然后按照第一篇研报的方式计算其自相关性。

没想到就是这样一个简单的组合居然获得了意想不到的效果。新的因子，在回测区间中有六年的IC绝对值超过了0.1，其中3年的IC绝对值超过了0.12，甚至还有1年的IC绝对值接近0.14了！

根据《专利法》的说法，有意想不到的效果就是有创造性的。但是，《专利法》还说，两个文件的结合是可以破坏创造性的。那么问题来了，这个因子到底有没有创造性呢？

回到正题，这个因子的计算是比较简单的，而且它所用到的方法，笔者在前面的复现中已经介绍过了，所以这里就不聊细节了，直接贴出代码。

```python
def process_single_day(self, idx):
    if idx < 20:
        return pd.DataFrame({})
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    amt = []
    vol = []
    for i in range(idx-20,idx+1):
        file_name = self.files[i]
        full_path = os.path.join(self.file_pth, file_name)
        data = BaseDataLoader.load_data(full_path, fields=['turnover', 'volume'])
        amt.append(data.to_dataframe('turnover'))
        vol.append(data.to_dataframe('volume'))
    amt = self.z_score(amt)
    vol = self.z_score(vol)

    res = [amt.shift(1).corrwith(amt), vol.shift(1).corrwith(vol)]
    res = pd.concat(res, axis=1)
    res.columns = ['amt_foc', 'vol_foc']
    res['diff'] = res['amt_foc'] - res['vol_foc']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()

def z_score(self, data):
    data = pd.concat(data)
    data['minute'] = data.index.hour * 60 + data.index.minute
    data = data.groupby('minute', as_index=False, group_keys=False).apply(self._z_score_)
    return data

@staticmethod
def _z_score_(group):
    group = group.sub(group.mean(axis=0), axis=1).div(group.std(axis=0), axis=1)
    group = group.iloc[[-1]]
    return group.drop(columns=['minute'])
```

这里，笔者还是成交量和成交额以及两者之差都进行了计算。

但是，这里两者之差这个因子的表现比较一般，所以因子评价的时候就没它了。

此外，成交额和成交量计算的因子，相关性是非常高的，所以接下来也就只展示一下基于成交额计算的因子的评价结果了。
