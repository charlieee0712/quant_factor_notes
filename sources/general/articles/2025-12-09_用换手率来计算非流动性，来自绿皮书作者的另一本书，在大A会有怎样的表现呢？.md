**非流动性**

![Image](images/2025-12-09_用换手率来计算非流动性，来自绿皮书作者的另一本书，在大A会有怎样的表现呢？/002.jpg)非流动性，这个因子想必大家都耳熟能详了。

但是，在券商研报中，使用的非流动性因子大都使用成交额计算的，即收益率的绝对值除以成交额。

而在量化绿皮书的作者另一本书《Active Equity Management》中，介绍了另外两种计算方式，第一种是收益率的绝对值除以成交额（该比率衡量的是每单位成交量引起的平均价格变动百分比），另一种则是收益率的绝对值除以换手率（这种方式能降低和市值的相关性）。

因子值越大非流动性越大，标的未来收益率会越高。因为，在同样的风险调整后收益下，投资者更愿意投资流动性高的标的。所以，流动性低的标的要有更高流动性溢价。

值得一提的是，在介绍流动性风险这一部分内容中，作者还提到了另外两种不同类型的非流动性因子，一种是用截面均值标准化之后的非流动性因子，还有一种叫做非流动性比率。

**计算公式和代码**

计算公式在上面已经提过了，这里进行一个简单的总结。

**1**

**计算公式**

illiquidity\_vol=abs(收益率)/成交量。

illiquidity\_money=abs(收益率)/成交额

illiquidity\_tr=abs(收益率)/换手率

**2**

**代码**

```python
def __call__(self):
    data = BaseDataLoader.load_data('../../data/stock_bar_1day.parquet', fields=['close', 'volume', 'turnover'])
    cap = BaseDataLoader.load_data('../../data/capital.parquet', fields=['market_cap', 'turnover_ratio'])
    rtn = np.abs(data.to_dataframe('close').pct_change())
    illiquidity_vol = rtn / data.to_dataframe('volume')
    illiquidity_money = rtn / data.to_dataframe('turnover')
    illiquidity_tr = rtn / cap.to_dataframe('turnover_ratio')
    dfs = {'illiquidity_vol': illiquidity_vol, 'illiquidity_tr': illiquidity_tr,
           'illiquidity_money': illiquidity_money, 'market_cap': cap.to_dataframe('market_cap')}
    df = dfs_to_df(dfs)
    df.to_parquet('./illiquidity.parquet')
```

代码的话没什么难度，都是直接用公式计算出来的。

从相关性来看，illiquidity\_tr和市值因子的相关性（绝对值）确实是最小的。同时，它与illiquidity\_vol和illiquidity\_money的相关性也没有超过0.6。

illiquidity\_money这个因子的IC并不是太高，但是分层回测的单调性是最好的。这个因子大家都挺熟悉的，所以就最后展示一下分层回测的结果。

illiquidity\_vol这个因子的话，IC是最低的，也是最不稳定的，2018-2020年IC为正，2021年-2023年IC为负。

所以，接下来展示的是illiquidity\_tr这个因子的因子评价结果。

需要注意的是，这里展示的因子使用过去21个交易日的均值合成的“月度因子”。
