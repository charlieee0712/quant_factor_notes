整体来说，这一系列的因子还是有价值的。

最近，笔者重新阅读了这篇研报，然后有了一个新的思路，用截面回归来计算模糊价差。

**计算步骤和代码**

这个因子的计算是以模糊性为基础的，模糊性其实就是标准差的标准差。基于这个概念，曹老师提出了5个因子，这些因子的具体计算方式就请各位大佬移步之前的文章查看了。

这里，将聚焦新因子的计算。

**1**

**计算步骤**

第一步，计算模糊残差因子，即以模糊金额比为被解释变量（y），模糊数量比为解释变量（x）进行截面回归，并计算残差。

第二步，计算模糊价差因子（模糊金额比-模糊数量比）与残差价差因子的差值，就得到了本文要介绍的最终的因子。

**2**

**代码**

```python
def __call__(self):
    data = BaseDataLoader.load_data('./dazzling.parquet', fields=['fuzz_volume', 'fuzz_money'])
    self.x = data.to_dataframe('fuzz_volume')
    self.y = data.to_dataframe('fuzz_money')
    res = []
    for i in tqdm(range(len(self.x))):
        res.append(self.cal_factors(i))
    res = pd.concat(res, axis=1).T
    dfs = {'fuzz_resi': res, 'fuzz_diff': self.y - self.x}
    df = dfs_to_df(dfs)
    df.to_parquet('./fuzz_diff.parquet')
```

第一段代码，读取数据，然后调用cal\_factors方法计算残差。

```python
def cal_factors(self, idx):
    x = self.x.iloc[idx]
    y = self.y.iloc[idx]
    data = pd.concat([x, y], axis=1).dropna()
    data.columns = ['x', 'y']
    lr = LinearRegression()
    lr.fit(data[['x']], data[['y']])
    resi = data[['y']] - lr.predict(data[['x']])
    resi.columns = [x.name]
    return resi
```

第二段代码，和之前的残差计算有一点点区别。

这里的线性回归模型输入的是dataframe，这样的话计算的残差直接就是带index的了，且其index就是标的代码。

当然，这种方式计算起来比较慢，不如直接使用最小二乘的公式。
