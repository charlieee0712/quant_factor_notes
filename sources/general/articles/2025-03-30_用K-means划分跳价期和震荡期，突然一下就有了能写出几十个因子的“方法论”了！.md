前段时间，笔者复现了兴证金工首席分析师郑兆磊老师的研报。

在这篇研报中，郑老师通过两个高斯混合分布将标的分钟级别的收益率划分成了震荡期和跳价期。

划分的逻辑很简单，大部分时间是震荡期，小部分时间是跳价期。所以权重大的高斯分布对应震荡期，权重小的高斯分布对应跳价期，高斯分布的均值和方差就是对应时期的均值和方差。

在这篇研报的启发下，笔者将高斯混合模型改成了K-means的聚类算法。用该算法将标的分钟级的数据聚为两类。选取的聚类特征为收益率，成交量的变化率和成交额的变化率。注意，这里都用了变化率也就不需要再做归一化了。也可以不用变化率，然后用z-score或者其他方式对每一个特征进行归一化。

划分的逻辑也是参考郑老师的，多的那一类对应震荡期，少的那一类对应跳价期。

```python
def kmeans_fit(data):
    x = data[1:, :] / data[:-1, :] - 1
    flag = np.all(~(np.isnan(x) | np.isinf(x)), axis=1)
    x = x[flag]
    if len(x) <= 60:
        return np.nan
    model = KMeans(n_clusters=2)
    model.fit(x)
    labels = model.labels_
    if np.mean(labels) > 0.5:
        labels = 1 - labels
    jump_labels = np.where(labels, 1, np.nan)
    return np.nanstd(x[0, :] * jump_labels)
```

第2行，计算收盘价、成交量和成交额的变化率。

第3-4行，去掉有值为nan的行。

第5-6行，对于nan过多标的，直接返回nan。

第7-8行，用K-means对数据进行聚类。

第9-11行，将跳价期的标签置为1。

第12行，是跳价期，那么为1，否则为nan。

第13行，计算跳价期的波动。

这里为什么选择跳价期的波动呢？

因为，在之前高斯混合分布的文章中，笔者最终也是展示了跳价期的波动的评价结果。这里选择跳价期的波动纯粹是为了对比。
