**代码**

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'open'])
    close = data.to_dataframe('close')
    rtn = close / data.to_dataframe('open') - 1
    res = Parallel(n_jobs=-1, verbose=10)(
        delayed(self.cal_factors)(close.iloc[:, idx])
        for idx in range(close.shape[1])
    )
    res = pd.DataFrame(res, index=rtn.columns.tolist(), columns=['snr_layer2', 'snr_layer3'])
    vol = rtn.std()
    vol_max = vol.max()
    vol_min = vol.min()
    vol_std = (vol - vol_max) / (vol_max - vol_min)
    weight = 0.5 + (vol_std - 0.5) / 5
    res['snr'] = weight * res['snr_layer2'] + (1 - weight) * res['snr_layer3']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    return res
```

核心代码是第9-12行，调用cal\_factors方法来计算因子。

```python
@staticmethod
def cal_factors(close):
    emd = EMD(MAX_ITERATION=10)
    imfs = emd(close, max_imf=3)
    snr_layer2 = snr_layer3 = None
    if len(imfs) == 4:
        noise_layer3 = np.sum(imfs[:3, :], axis=0)
        snr_layer3 = np.log(np.std(close - noise_layer3) / np.std(noise_layer3))
        noise_layer2 = np.sum(imfs[:2, :], axis=0)
        snr_layer2 = np.log(np.std(close - noise_layer2) / np.std(noise_layer2))
    elif len(imfs) == 3:
        noise_layer2 = np.sum(imfs[:2, :], axis=0)
        snr_layer2 = np.log(np.std(close - noise_layer2) / np.std(noise_layer2))
    return [snr_layer2, snr_layer3]
```

第3行，实例化EMD，并指定最大迭代次数。默认是1000，笔者改成了10次还需要1505分钟才能计算完2300多天的数据。

第4行，对收盘价进行EMD分解，最多分解三层。得到的结果应该是[4, 数据长度]，1-3代表三层的imfs，4代表r。

后面的就是根据公式计算信噪比了，没有什么好特别说明的了。
