本文，笔者要介绍的因子叫作残差收益噪声偏离因子，这个因子的名字来自兴业证券郑兆磊老师的研报，笔者曾在[用“么么哒”算法计算收益率噪音偏离因子，又是一个差点错过的好因子！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247495254&idx=1&sn=66d729d4108267553f2e6d4dbfc465a4&scene=21#wechat_redirect)这篇文章中有过介绍。

虽然，名字来自郑老师，但是笔者感觉这篇文章将要介绍的这个因子的整个计算过程和郑老师的研报没有太大的关系了，所以大言不惭地来上一句，这应该是笔者的独创性因子了。

**计算步骤和代码**

在郑兆磊老师的研报中，笔者认为其核心思想是zscore标准化后的收益率是服从标准正态分布的，然后衡量其和标准正态分布的差异就得到了收益率噪声偏离因子。

这让笔者想到了研究生阶段看到过的一篇论文，大概是用观测值去估计真实值，而估计误差是服从正态分布的。

基于此，笔者想到了，用CAMP模型来计算残差收益率，而这个残差收益率应该是服从均值为0，标准差为σ的正态分布。于是，就有了本文要介绍的因子。

**1**

**计算步骤析**

第一步，利用CAMP模型分布计算过去21个交易日分钟收益率的残差和当前交易日分钟收益率的残差。这里，市场收益率用的是全市场所有标的分钟收益率的均值。

第二步，用残差收益率除以残差收益率的波动将其标准差变成1。

第三步，计算收益率的噪声偏离因子。这一步通过代码来解释吧，笔者实在想不到怎么用语言来描述了。

**2**

**代码**

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    cur_time = pd.to_datetime(date_str) + timedelta(hours=15)
    if idx < 243 or cur_time >= pd.to_datetime('2026-01-01'):
        return pd.DataFrame(columns=['nos_gs'])
    close = []
    for i in range(idx - 20, idx + 1):
        file_name = self.files[i]
        full_path = os.path.join(self.file_pth, file_name)
        data = BaseDataLoader.load_data(full_path, fields=['close'])
        factor = self.factor.iloc[idx - 243].reindex(data.codes)
        close.append(data.to_dataframe('close') * factor.values.reshape(1, -1))
    his = pd.concat(close).pct_change()
    mkt = his.mean(axis=1)
    mmd = []
    for i in range(len(his.columns)):
        mmd.append(self.cal_factors(his.iloc[:, i], mkt))
    res = pd.DataFrame(mmd, index=his.columns, columns=['mmd'])
    res['datetime'] = cur_time
    return res
```

第一段代码，主要是读取数据。然后调用cal\_factors方法为每个标的计算因子。

```python
def cal_factors(self, data, mkt):
    his = pd.concat([data, mkt], axis=1)
    cur = his.iloc[-240:].dropna()
    if len(cur) < 10:
        return np.nan
    his = self.cal_resi(his.dropna())
    cur = self.cal_resi(cur)
    if cur is None:
        return np.nan
    return self.compute_mmd(his.iloc[-len(cur):].values, cur.values)
```

第2行，将标的收益率和市场收益率拼接为一个dataframe。

第3行，获取当前交易日的标的收益率和市场收益率，并去掉nan。

第4-5行，去掉当前交易日有效交易时间小于10分钟的标的，对于它们直接返回nan。

第6-7行，计算标准化后的残差收益率。

第8-9行，特殊处理。有的标的全天的分钟收益率都是0，此时直接返回nan。

第10行，计算收益率噪声偏离因子。his.iloc[-len(cur):]是当前交易日的数据在21天的时间窗口内的标准化的残差收益率，cur是当前交易日的数据在当天时间窗口内标准化的残差收益率。

这么做的目的是，样本越多估计越准，残差越接近正态分布。所以his比cur更接近正态分布。当然，也可以直接计算his和cur的最大平均偏差，但是这样的话计算量太大了。

```python
@staticmethod
def cal_resi(data):
    if data.iloc[:, 0].sum() == 0:
        return None
    lr = LinearRegression()
    lr.fit(data.iloc[:, 1:], data.iloc[:, 0:1])
    resi = data.iloc[:, 0:1] - lr.predict(data.iloc[:, 1:])
    return resi / resi.std()
```

这部分代码主要就是计算残差了。

第3-4行，对收益率全为0的标的进行处理。

第8行，对残差收益率进行标准化，这里笔者偷懒了，直接用了std，理论上应该按照均值为0计算标准差。但是，差距应该不大。

值得一提的是，对于除第一分钟外其他时间收益率不为0的标的，可以批量用最小二乘的公式直接计算，并不需要循环。关于这点，笔者在早期复现barra因子的时候介绍过，这里就简单提上一句。
