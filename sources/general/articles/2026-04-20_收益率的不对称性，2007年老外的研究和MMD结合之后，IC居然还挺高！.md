**收益率的不对称性**

本文，笔者要复现的因子是东方证券朱剑涛老师在2021年12月25日发布研报《因子选股系列研究之八十：收益率的非对称分布与尾部蕴含的Alpha》中提到的。

这个因子的目的是度量股票收益率分布的不对称性。

通常，大家都是用峰度和偏度来衡量不对称性的。

在这篇研报中，朱老师提出了一种新的衡量不对称性的方式，笔者认为挺有价值的，所以决定分享出来。

这个因子是老外在2007年提出的，其构造是基于熵的原理。简单来说，它就是衡量x和-x+2μ这两个分布的差异。

**代码**

这个因子计算的关键就是在于如何衡量x和-x+2μ这两个序列的概率分布的差异。

统计学中有很多种方法，比如KL散度、JS散度。这里，笔者使用的是最大平均偏差算法（MMD），这个在笔者之前的一些因子当中也有用到过，[用“么么哒”算法计算收益率噪音偏离因子，又是一个差点错过的好因子！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247495254&idx=1&sn=66d729d4108267553f2e6d4dbfc465a4&scene=21#wechat_redirect)

第一段代码，主要就是读取数据，然后计算收益率，并调用\_\_cal\_factors\_\_为每一个标的计算因子。

```python
def __cal_factors__(self, data):
    mu = data.mean()
    data = data.dropna()
    if len(data) < 60:
        return np.nan
    new_data = -data + 2 * mu
    asym = self.compute_mmd(data.values, new_data.values)
    return asym
@staticmethod
def compute_mmd(x, y, gamma=1.0):
    x = x.reshape(-1, 1)
    y = y.reshape(-1, 1)
    k_xx = rbf_kernel(x, x, gamma=gamma)
    k_yy = rbf_kernel(y, y, gamma=gamma)
    k_xy = rbf_kernel(x, y, gamma=gamma)
    return np.mean(k_xx) + np.mean(k_yy) - 2 * np.mean(k_xy)
```

这部分代码包含了两个方法。

\_\_cal\_factors\_\_主要是对数据进行处理，对于有效数据小于60分钟的标的，直接返回nan。对于其他情况，构造一个new\_data，然后计算它和data之间的MMD。

compute\_mmd方法就是按照公式计算两个序列的MMD。

**因子评价**

这个因子和研报中介绍的并不完全一致。

除了两个分布差异的度量方式外，还有以下几点。

第一，研报中使用的是日频数据计算的。

第二，研报中还乘以了一个用左右尾概率作差的符号函数，来代表收益率到底是左偏还是右偏。

笔者也尝试过，表现不如这篇文章改进后的因子。所以，就给各位大佬介绍一下这个简单改进之后的因子。同时，其用均值或标准差进行低频化之后的表现是差不多的，所以这里就展示一下用均值进行低频化的结果。
