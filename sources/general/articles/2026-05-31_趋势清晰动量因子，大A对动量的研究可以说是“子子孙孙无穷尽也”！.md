**趋势清晰动量因子**

每当笔者以为动量因子已经被研究透了，没什么可复现的东西的时候，动量之神就会把一篇和动量相关的因子拍在笔者脸上。

本文，笔者将复现西南证券郑琳琳老师在2024年11月11日发布的研报《因子选股系列之五——价格形成路径与趋势清晰度因子》。

其实，在这篇研报之前，笔者还看到了一篇题为《因子选股系列之一——基于分钟线的高频选股因子》，是信达证券的于明明老师在2022年4月28日发表的。这篇研报中提到了两个反转因子，简单来说就是放量时刻的正收益均值和放量时刻的负收益均值。

这个因子的思想其实挺正常的。

笔者还发现了一个思路清奇的因子，工作日动量因子。当然咯，这么清奇的因子是由老外提出的，然后国内的金工大佬在大A上进行了验证。其思想就是计算相同工作日的收益率的均值，得到周一动量到周五动量一共五个因子。

笔者没有复现这个因子并不是觉得它离谱，而是周一动量预测的是未来一个月周一的收益率。这就导致了其在笔者的框架下复现起来很复杂，所以暂时放弃了它。

def cal_tc(self, data):
    res = []
    for i in range(data.shape[1]):
        res.append(self.__cal_tc__(data.iloc[:, i:i+1]))
    return res
@staticmethod
def __cal_tc__(data):
    data['t'] = list(range(1, len(data) + 1, 1))
    data = data.dropna()
    if len(data) < 200:
        return np.nan
    lr = LinearRegression()
    lr.fit(data.iloc[:, 1:], data.iloc[:, 0:1])
    pred = lr.predict(data.iloc[:, 1:])
    r2 = r2_score(data.iloc[:, 0:1], pred)
    return r2

**3**

**因子逻辑**

在下跌的股票中，趋势清晰度与股票下期收益负相关。即在过去下跌的股票中，下跌路径越明确、趋势越清晰的股票，未来更有可能维持下跌；下跌路径不明确、趋势不清晰的股票，未来更有可能反转上涨。

在上涨的股票中，趋势清晰度与股票下期收益正相关。即在过去上涨的股票中，上涨路径越明确、趋势越清晰的股票，未来更有可能维持上涨趋势；上涨路径不明确、趋势不清晰的股票，未来更有可能下跌下跌。

同样的，站在趋势清晰度因子角度来讲：趋势清晰度越高的股票中过去收益率越高越好，趋势清晰度越低的股票中过去收益率越低越好。
