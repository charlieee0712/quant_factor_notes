**计算步骤**

这个因子计算和之前的几篇文章相比可以说是非常简单了。

即，用前两个极值点来计算收益率。

其背后的逻辑是这样的，日内动量反转因子捕捉了日内交易中价格走势的反转效应；当行情上扬时第一次出现的极值点往往是低点，当行情波动下挫时第一个出现的极值点往往是高点。

代码

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close'])
    date_str = file_name.split('.')[0]
    close = data.to_dataframe('close')
    flag = ((close.diff() > 0) & (close.diff(-1) > 0)) | ((close.diff() < 0) & (close.diff(-1) < 0))

    res = []
    for i in range(len(close.columns)):
        res.append(self.cal_factors(close.iloc[:, i], flag.iloc[:, i]))
    res = pd.DataFrame(data=res, index=close.columns.tolist(), columns=['intraday_mon'])
    res['extreme_cnt'] = extreme_cnt
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()

def cal_factors(self, close, flag):
    idx = []
    for i in range(len(flag)):
        if flag.iloc[i]:
            idx.append(i)
        if len(idx) == 2:
            break
    if len(idx) < 2:
        return np.nan
    intraday_mom = close.iloc[idx[-1]] / close.iloc[idx[0]] - 1
    return intraday_mom
```

第8行，获取极值点对应的时间。这里的极值点的计算分成了两个部分，用|（或）分割。前半部分代表极大值点，后半部分代表极小值点。

有了这个极值点的标志位之后，就可以通过cal\_factors来计算日内动量反转因子了。代码的逻辑也不复杂，就是获取前两个flag为True的索引，然后用它们所对应的价格计算收益率。需要注意的是，如果只有一个极值点的话，笔者将因子直接置为nan处理。

**总结**

这个因子从IC上的表现来看，用均值和标准差合成的差异是不大的。

但是，从分层回测上来看，两者还是稍微有一点点差异的。如果只看整个回测区间的年化收益而不看净值曲线的话，标准差合成的因子可以说是表现完美，因为它的年化收益呈现了完美的单调性。

可是，当我们关注净值曲线的时候，就会发现其实也不过如此，相交的情况很多，除了最后时刻，中间和开始的阶段也并不是单调的。
