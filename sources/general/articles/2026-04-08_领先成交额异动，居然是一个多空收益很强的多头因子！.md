**领先成交额异动**

早就听说过招商证券任曈老师的名号了，但是却很少复现任老师的论文，因为任老师的研报不是那么容易获取。

不过，这次笔者侥幸看到了任老师在2024年12月16日发布的研报《“青出于蓝”系列研究之九：基于分钟数据的价变共振因子》。因为，2026年的《因子日历》中引用了其中的一个因子，叫做个股收益最高时的领先成交量异动。

这是因为，在研报中介绍这个因子的章节名字叫做异常成交量，但是计算的时候却使用了成交额，所以笔者自作主张修改了一下。

此外，这篇研报中还有两个因子，分别是是价格趋势上的一致性和分歧度。这两个因子，咱们后面有机会再聊。

关于这个因子的逻辑，研报中是这样描述的。

> 异常成交量放大往往意味着交易过热的信号，因此交易过度活跃的个股往往未来收益率表现偏弱。延伸前述逻辑，我们可以关注个股在分钟级别下剧烈变化时的成交量情况进行指标构建。如果个股脉冲式收益率是伴随着个股区别于市场的成交量异动出现的，那么这样的价格变化可能是由自身成交量的异动造成的，这将不利于个股与市场保持共振。

在《日子日历》中，其说明是这样写的。

> 若个股的上涨没有伴随市场成交量异动，而是由自身成交量异动造成的（因子值越大，占比越大），未在成交量上与市场形成有效共振关系，则容易受到自身交易过热的反噬，未来收益表现较弱。

**计算步骤和代码**

这个因子的计算倒是不复杂。

**1**

**计算步骤**

第一步，计算每个标的每分钟成交额占全天成交额的比例。

第二步，再将日内最高（低）的N个分钟收益率（研报尝试了5、15、30和60）期间个股成交额比值的均值除以当日240个分钟的成交额比值的均值。但是，这么做出来的因子不行（研报中IC绝对值最高0.0304）。

所以，研报对其进行了改进，换成了日内最高（低）的N个分钟收益率前一分钟的成交额比值。这样做出来的因子，IC绝对值最高能有0.0578，对应为最低30个分钟收益率。

**2**

**项目重点**

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    cur_time = pd.to_datetime(date_str) + timedelta(hours=15)
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'turnover']).to_dataframes()
    rtn = data['close'].pct_change()
    money = data['turnover']
    money = money / money.sum().values.reshape(1, -1)
    mkt = rtn.mean(axis=1)
    resi = []
    for i in range(len(rtn.columns)):
        resi.append(self.cal_resi(rtn.iloc[:, i], mkt))
    resi = pd.concat(resi, axis=1)
    resi = resi.reindex(money.index)
    res = []
    for q in [0.1, 0.9]:
        if q == 0.1:
            flag = rtn < rtn.quantile(q)
            resi_flag = resi < resi.quantile(q)
        else:
            flag = rtn > rtn.quantile(q)
            resi_flag = resi > resi.quantile(q)
        res.append((money * np.where(flag.shift(-1), 1.0, np.nan)).mean() / money.mean())
        res.append((money * np.where(resi_flag.shift(-1), 1.0, np.nan)).mean() / money.mean())
    res = pd.concat(res, axis=1)
    res.columns = ['amt_lead_min10', 'amt_lead_min10_resi', 'amt_lead_max90', 'amt_lead_max90_resi']
    res['datetime'] = cur_time
    return res
```

前7行，读取数据。

第8行，计算收益率。

第9-10行，计算成交额占比。

第11-16行，调用cal\_resi方法计算残差收益率。因为，所有用收益率计算的因子都可以换成残差收益率来试一试。

第17-27行，计算因子值。这里，笔者为了方便计算，没有选择最大或最小的30分钟，而是按比例取最大或最小的10%。同时，按照最大或最小的30分钟来取，可能会遇到收益率相同的情况，这时候取成交额更大的还是更小的呢？

```python
@staticmethod
    def cal_resi(y, x):
        data = pd.concat([y, x], axis=1).dropna()
        if len(data) < 60 or data.iloc[:, 0].sum() == 0:
            return pd.Series(index=y.index, name=y.name)
        lr = LinearRegression()
        lr.fit(data.iloc[:, 1:], data.iloc[:, 0:1])
        resi = data.iloc[:, 0:1] - lr.predict(data.iloc[:, 1:])
        return resi
```
