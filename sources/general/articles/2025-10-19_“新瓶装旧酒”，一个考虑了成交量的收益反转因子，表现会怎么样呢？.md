本文笔者复现的因子来自信达证券于明明老师在2022年4月29日发布的研报《因子选股系列之一——基于分钟线的高频选股因子》。

在这篇研报当中一共介绍了三类因子，分别是收益反转因子、收益波动因子和尾盘成交额占比因子。

在这篇文章中，笔者将复现一下收益反转因子。

关于这个因子，笔者想到了一个文邹邹的词汇，叫做“新瓶装旧酒”。因为，这个因子在国内外已经被研究过无数次了。

最著名的就是反转因子（21天收益率）和动量因子（t-12个月到t-1的月的收益率）。这两个因子，认为股票在短期内呈现反转效应，在长期内呈现动量效应。

还有一个因子认为，有些标的呈现反转效应，有些标的则呈现出动量效应，这两种不同的标的一种叫做球队型，一种叫做硬币型。笔者在["球队硬币"因子，这篇研报又给了一个让因子数量翻倍的方法](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247487891&idx=1&sn=077bca91cff5a86fd7a33bfdf07456bc&scene=21#wechat_redirect)这篇文章中对其进行了详细的介绍，感兴趣的读者可以自行查阅。

在《因子选股系列之一——基于分钟线的高频选股因子》中关于收益反转因子有多个。

第一个，用30分钟收益率的均值预测未来30分钟的收益率。这个因子的频率太高了，笔者暂时先不复现了。

第二个，每个交易日前235分钟的收益率的均值。

第三个，改进的正收益率反转因子，即成交量大于均值+标准差的时刻且收益率为正的均值。

第四个，改进的负收益率动量因子，即成交量大于均值+标准差的时刻且收益率为负的均值。

关于后两个因子，作者是这样考虑的。

> 缩量时段，较小的成交量可以造成较大的价格波动，价格变动失真，信噪比较低，只有当前属于放量时段时，伴随这较大的成交量，价格变动的大小才能充分反映多空博弈的剧烈程度，因此可尝试对分钟数据进行基于成交量的筛选。
>
> 同时，由于 A 股难以做空，通常会造成多头和空头力量的不匹配，当某支股票持续上涨时，会吸引较多资金跟进，股价会出现虚高的情况，有较大概率在后续出现回落至合理价位的情况，出现反转效应。相反，当某支股票持续下跌时，因融券做空较难实现，难以吸引空头资金跟进，从这个角度出发，我们可以对分钟收益进行一个划分，将分钟线收益分为大于 0 和小于 0 的情况分别测算。

本文，笔者将聚焦后面三个因子。

这个因子的计算其实挺简单的，所以笔者决定直接通过代码来说明一下。

**代码**

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'open', 'volume']).to_dataframes()
    rtn = (data['close'] / data['open'] - 1).iloc[:235]
    vol = data['volume'].iloc[:235]
    vol_flag = vol > vol.mean() + vol.std()
    flag_pos = rtn > 0
    flag_neg = rtn < 0
    reverse_imp_pos = -1 * (rtn * np.where(vol_flag * flag_pos != 0, 1.0, np.nan)).mean()
    reverse_imp_neg = (rtn * np.where(vol_flag * flag_neg != 0, 1.0, np.nan)).mean()
    res = pd.concat([reverse_imp_pos, reverse_imp_neg, rtn.mean()], axis=1)
    res.columns = ['reverse_imp_pos', 'reverse_imp_neg', 'rtn']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    return res
```

前5行代码，读取计算因子的相关数据。

第6行代码，计算分钟收益率，并保留前235分钟的数据。

第7-8行，获取放量时间段。

第9-10行，获取收益率为正和收益率为负的时间段。

第11行，计算第三个因子。

第12行，计算第四个因子。

**因子评价**

这三个因子，无论是用过去21个交易日的均值合成月度因子，还是用过去21个交易日的标准差合成的月度因子，在IC和分层回测上的表现都差不多。

笔者经过精心的比较，选择了用过去21个交易日的均值合成的第三个因子。它的分层回测表现好于第二个因子，IC绝对值高于第三个因子。

值得一提的是，研报中计算的是日频因子的IC，同时还去掉了当日涨跌停的标的，次新股（上市不足一年），停牌及ST股票。所以，研报中该因子的IC为正，但是笔者复现的该因子为负。
