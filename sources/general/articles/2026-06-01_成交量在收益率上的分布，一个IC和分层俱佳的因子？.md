**成交量在收益率上的分布**

本文，笔者要介绍的因子是成交量在收益率上的分布。这个因子来自东北证券王琦老师在2023年11月30日发布的研报《因子选股系列之六：日内成交量分布因子及Logsig-Alpha因子生成》。

在这篇研报中，有四类因子。

第一类，日内对数成交量分布相关因子。主要就是对数成交量的波动和偏度。

第二类，日内成交量在时间上的分布相关因子。主要就是对早盘成交量占比或者是早午开盘成交量占比。

第三类，就是本文介绍的因子。

第四类，日内成交量在价格上的分布相关因子。这个因子是有点复杂的，不是一两句话能说清楚的，需要另开一篇文章。所以，各位大佬如果感兴趣的话，通过点赞关注推荐和分享的方式让笔者看到你们的热情呗，热情越高上线时间肯定越快。

**计算公式和代码**

这一类主要有两个因子，然后根据低频化方法的不同，就变成了四个。

所以，笔者的代码只计算了其中两个。

**1**

**计算公式**

![](images/2026-06-01_成交量在收益率上的分布，一个IC和分层俱佳的因子？/001.png)

第一个因子，价格显著上行价稳量比因子。这个因子，是用高频数据计算，然后用过去21个交易日的均值进行低频化的。

所以，第二个因子就是用方差低频化的，叫作价格显著上行价稳量比稳定因子。这里的方差是以均值为0计算的。

第三个因子，同时考虑了价格的波动和成交量的波动，所以它叫作价格显著上行价量波动因子。

![](images/2026-06-01_成交量在收益率上的分布，一个IC和分层俱佳的因子？/002.png)

其用方差低频化之后就叫作价格显著上行价量波动稳定性因子。

**代码**

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    cur_time = pd.to_datetime(date_str) + timedelta(hours=15)
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'open', 'volume']).to_dataframes()
    rtn = data['close'] / data['open'] - 1
    flag = (rtn > 0) & (rtn > rtn.mean() + rtn.std())
    sigma = (rtn * np.where(flag, 1.0, np.nan)).std()
    vol_mu = data['volume'].mean()
    sig_up_p_v_ratio = sigma * (data['volume'] * np.where(flag, 1.0, np.nan)).mean() / vol_mu
    sig_up_v_intraday_std = sigma * (data['volume'] * np.where(flag, 1.0, np.nan)).std() / vol_mu
    res = pd.concat([sig_up_p_v_ratio, sig_up_v_intraday_std], axis=1)
    res.columns = ['sig_up_p_v_ratio', 'sig_up_v_intraday_std']
    res['datetime'] = cur_time
    return res
```

这个代码没有什么难点，就是第9行获取了价格显著上升的flag之后，按照公式计算即可。

**因子评价**

价格显著上行价稳量比因子的IC比价格显著上行价稳量比稳定因子高，但是，前者分层回测的单调性比后者差一点点。

价格显著上行价量波动因子其IC比价格显著上行价稳量比因子低因，而价格显著上行价量波动稳定性因子则是四个因子当中最低的。

从分层回测的结果来看，价格显著上行价稳量比稳定因子是四者当中最好的。所以，因子评价的时候将以它为主。
