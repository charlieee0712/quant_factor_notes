本文，笔者将介绍理想振幅因子。

该因子来自开源证券2020年的研报《振幅因子的隐藏结构》。

之前写过一篇类似的文章，但是用的都是日频数据。本文，笔者将使用1分钟、5分钟、10分钟、30分钟和60分钟的数据一起来看一看这个因子的效果。

首先，简单用文字来介绍一下这个因子，它的计算分为三步。

1. 计算振幅，即最高价/最低价-1。
2. 根据收盘价将数据划分为高价区间和低价区间，具体方式为收盘价高于75%分位数的为高价区间，低于25%分位数的为低价区间
3. 用高价区间的平均振幅减去低价区间的平均振幅就得到了理想振幅因子。

然后，一起来看一看代码是怎么实现的。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['high', 'low', 'close'])
    data.data[:, 0, :] = data.data[:, 0, :] / data.data[:, 1, :] - 1
    
    amp_df = data.to_dataframe('high')
    close_df = data.to_dataframe('close')
    thd = close_df.quantile([0.25, 0.75])
    amp_series = amp_df.where(close_df >= thd.iloc[1]).mean(axis=0) - amp_df.where(close_df <= thd.iloc[0]).mean(axis=0)
    amp_series.name = pd.to_datetime(date_str) + timedelta(hours=15)
    return amp_series
```

第8行，计算振幅。

第12行，计算25%分位数和75%分位数对应的值，返回的是一个dataframe，列名为标的代码，行名对应0.25和0.75。

第13行，计算高价区间和低价区间振幅因子的均值，并相减。

其他的处理，和上一篇文章中介绍的是一样的，这里就不再赘述了。

最后，来看一看因子的效果，从IC上来看确实比上一篇介绍的欧氏距离计算的惊恐度强。虽然市值加权的分层回测还是不能看，但是从这个IC的均值来看，如果是等权的话效果应该还凑合，大家可以自行尝试一下。
