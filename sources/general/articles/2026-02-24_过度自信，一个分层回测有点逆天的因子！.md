**计算步骤**

第一步，计算分钟收益率。研报中的1分钟数据有242条，然后用收盘价计算得到240个收益率。由于笔者的分钟数据只有240条，所以使用收盘价/开盘价-1计算得到242个收益率。

第二步，计算分钟收益率的均值和标准差。

第三步，将大于mu+sigma的作为快速上涨区间，小于mu-sigma的作为快速下跌区间。

第四步，分别计算这两个区间的时间中位数，然后用下跌中位数-上涨中位数。

**2**

**代码**

这个因子的计算步骤挺多的，但是代码并不长。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'open'])
    rtn = data.to_dataframe('close') / data.to_dataframe('open') - 1
    mu = rtn.mean()
    sigma = rtn.std()
    up_flag = (rtn > mu + sigma).reset_index(drop=True)
    down_flag = (rtn < mu - sigma).reset_index(drop=True)
    res = []
    for i in range(rtn.shape[1]):
        res.append(self.cal_med(down_flag.iloc[:, i]) - self.cal_med(up_flag.iloc[:, i]))
    res = pd.Series(data=res, index=data.codes, name=pd.to_datetime(date_str) + timedelta(hours=15))
    return res
```

前6行，读取数据。

第7-9行，计算分钟收益率，及其均值和标准差。

第10-11行，计算快速上涨区间和快速下跌区间。需要注意的是，这里进行了reset\_index的操作，将具体时间转化成了1-240分钟。

第12-14行，对每个标的计算过度自信因子。

```python
@staticmethod
def cal_med(data):
    data = data[data]
    return np.median(data.index.tolist())
```
