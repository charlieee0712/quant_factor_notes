本文笔者将复现中信建投2019年发布的研报《基于市场羊群效应的股票alpha研究》。

这篇研报发表在2019年，虽然它是一个基于日频数据计算的因子，但是这个因子的计算可以说是一点也不简单。笔者用i9开了16个进程来计算都花费了70多分钟。

没想到啊，6年前的因子就这么卷了。

那么一个复杂的因子会有更好的效果吗？

**计算步骤**

这个因子的计算步骤有四步。

第一步，计算当前标的和其他所有标的过去120个交易日的涨跌幅相关系数。

第二步，选取相关系数最大的9只股票，加上当前标的一共10只股票构成一个板块。

第三步，以当前标的为中心，计算板块过去120个交易日的横截面绝对偏差CSAD。即abs（板块所有标的收益率-当前标的收益率），然后在截面上取均值。

第四步，分别计算当前标的过去20个交易日的CSAD标准差和收益率标准差的比值，过去120个交易日的CSAD标准差和收益率标准差的比值，用短周期的除以长周期的就得到了最后的因子。

def process_single_day(self, idx):
    ret = self.ret.iloc[idx - 120:idx].dropna(how='all', axis=1)
    ret = ret.dropna(thresh=20, axis=1)
    ret = ret.loc[:, [col for col in ret.columns if (ret[col] != 0).sum() >= 20]]
    corr = ret.corr()

```python
@staticmethod
def __process_single_day__(corr, ret, code):
    corr = corr.dropna().sort_values()
    if len(corr) < 10:
        return [np.nan] * 4

    corr = corr.iloc[-10:]
    sector_ret = ret[corr.index.tolist()]
    csad = (sector_ret - sector_ret[code].values.reshape(-1, 1)).abs().mean(axis=1)
    csad.sort_index(inplace=True)
    sigma_20 = csad.iloc[-20:].std() / sector_ret[code].iloc[-20:].std()
    sigma_120 = csad.std() / sector_ret[code].std()
    return [sigma_20, sigma_120, sigma_20 / sigma_120, csad.iloc[-21:].std()]
```

**因子评价**

在这四个因子当中，只有CSAD的标准差这个因子的表现稍微好一点，其他的三个因子都比较一般。

它们的IC绝对值偏低，最高的一年也就刚刚超过0.06。

它们的分层回测就更不用提了，可以说是从头相交到尾。
