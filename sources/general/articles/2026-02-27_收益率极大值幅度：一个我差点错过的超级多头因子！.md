**收益率极大值幅度**

前段时间，笔者说自己陷入瓶颈了。

结果，意外发现一个还不错的因子之后就像打开了任督二脉一样。

先是邪修了一下，整合了三篇研报的思想弄出来了一个因子，虽然阅读不多，但至少有东西写了。

然后，突然想到，在去年五月份的时候，笔者曾集中复现过兴业证券首席分析师郑兆磊老师的高频研究系列因子，没有复现完。因为，当时在准备一个专利代理师的资格考试，就把一些复现难度大的，不好理解的因子留了下来。

想到这，笔者又打开了移动硬盘，将郑老师的研报又看了一遍。

没想到，这一看居然让我发现了一个超级多头因子。这个因子叫作收益率极大值幅度，来自2022年5月4日的研报《高频研究系列三：收益率分布中的Alpha（2）》。（在此之前，笔者复现过这篇研报中的部分因子[跳价期波动，原理很复杂，但计算并不难，2023年IC绝对值超过0.1，2021年后半段的分层回测也不错](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247486703&idx=1&sn=243d4b885a2259f41204a0089045755f&scene=21#wechat_redirect)）

关于这个因子背后的经济学逻辑，研报当中有非常详细的说明（保守估计一篇高考作文的长度是有的），各位大佬如果想了解的话可以通过正规渠道下载郑老师的研报学习。

但是，因为一个偶然的机会，得到了一个兴业证券高频因子说明的文档，里面刚好记录了这个因子，所以才得以复现。

**计算公式**

关于收益率极值，研报中一共介绍了两个因子。

一个是标题中的，收益率极大值幅度，它是基于VaR计算分钟收益率的极大值均值，具体为5分钟滚动收益率大于VaR的样本均值/VaR。

另一个是，收益率极小值出现概率，它是基于VaR计算极小值出现的频率，具体为5分钟滚动收益率小于VaR的样本个数。

看到这，笔者有个疑问，如果VaR也是基于5分钟滚动收益率计算的话，那么绝大多数标的的第二个因子的因子值应该相等。所以，笔者用的是1分钟的收益率来计算VaR。

关于VaR的计算，可以参考[CVaR因子，IC表现不错，分层回测有待提升！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247489552&idx=1&sn=d44ca3f3be45b9c25d78ee4c022d980a&scene=21#wechat_redirect)这篇文章。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, codes=self.codes, fields=['close']).to_dataframe('close')
    rtn = data.pct_change(5)
    rtn_1 = data.pct_change(1)
    q = rtn_1.quantile([0.05, 0.95])
    ex_rtn_max_val = (np.where(rtn > q.iloc[-1], 1.0, np.nan) * rtn).mean() / q.iloc[-1]
    ex_rtn_min_freq = (rtn < q.iloc[0]).sum()
    res = pd.concat([ex_rtn_max_val, ex_rtn_min_freq], axis=1)
    res.columns = ['ex_rtn_max_val', 'ex_rtn_min_freq']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    return res
```

前6行，读取数据。

第7行，计算5分钟收益率。

第8行，计算1分钟收益率。

第9行，分别计算5%分位数和95%分位数的VaR。

第10行，计算收益率极大值幅度。

第11行，计算收益率极小值出现概率。

这里也不进行相关性分析了。因为，这两个因子一个是极大值一个是极小值，一个是出现的概率，一个是幅度，看起来就不可能相关性高。
