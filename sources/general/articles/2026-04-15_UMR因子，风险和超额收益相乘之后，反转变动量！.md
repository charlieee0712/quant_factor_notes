**UMR因子**

在上一篇文章中，笔者复现了行业联合动量因子，有不少的大佬留了言。

有的说，这个因子确实提取出了动量信息；也有大佬，希望笔者能复现以下UMR因子，因为他的结果和研报有出入。

于是，笔者放弃了邪修，通过AI找到了和这个因子有关的研报（由东方证券的杨怡玲老师在2023年7月13日发布的《因子选股系列之九十四：UMR2.0，风险溢价视角下的动量反转统一框架》），并对其内容进行复现。

关于这个因子的逻辑，研报是这样描述的。

> 高风险日获得的收益往往是承担风险带来的，其更多源于投资者过度自信导致的价格反应过度，因此未来更倾向于反转效应，而低风险日获得的收益并不源于承担高风险，因此未来更偏向于动量效应，我们以时序均值调整后的风险指标来加权个股每日的溢价，并以此构建统一的动量反转（Unified Momentum and Reversal）因子。

**计算步骤**

由于这个因子是在传统动量因子的基础上考虑了风险，因此用不同的指标来度量风险，就会得到不同的因子。

研报中一共给出了8个风险代理指标。

其中，两个用日频数据计算的，分别是换手率和真实波动；三个用日内数据计算的，分别是大单买入均价偏离、小单主动买入占比和平均单笔成交量；三个用5分钟高频数据计算的，分别是早尾盘成交占比、分钟收益波动率和分钟收益偏度。

日内数据的，笔者没办法复现，所以本文将只复现另外五种风险代理指标计算的UMR因子。

```python
def __call__(self):
    data = BaseDataLoader.load_data('../../data/stock/stock_bar_1day.parquet',
                                    fields=['close', 'factor', 'volume', 'high', 'low']
                                    )
    codes = data.codes
    trade_days = data.trade_days
    data = data.to_dataframes()
    rtn = (data['close'] * data['factor']).pct_change()
    rtn = rtn * np.where(data['volume'] != 0, 1.0, np.nan)
    cap_data = BaseDataLoader.load_data('../../data/stock/capital.parquet',
                                        fields=['market_cap','turnover_ratio', 'circulating_market_cap']
                                        ).to_dataframes()
    weights = cap_data['market_cap'] / cap_data['market_cap'].sum(axis=1).values.reshape(-1, 1)
    mkt_rtn = rtn * weights
    excess_rtn = rtn - mkt_rtn.sum(axis=1).values.reshape(-1, 1)
    tr = cap_data['turnover_ratio']
    tr_risk = tr.rolling(21).mean() - tr
    dfs = {'tr_umr': excess_rtn * tr_risk}
    ture_range = np.max([data['high'] - data['low'], (data['high'] - data['close'].shift(1)).abs(),
                 (data['low'] - data['close'].shift(1)).abs()], axis=0) / data['close'].shift(1)
    ture_range_risk = ture_range.rolling(21).mean() - ture_range
    dfs['ture_range_umr'] = excess_rtn * ture_range_risk
    res = self.run()
    res = pd.concat(res)
    res.index.name = 'code'
    res.reset_index(inplace=True)
    res = BaseDataLoader.from_dataframe(res, codes=codes).to_dataframes()
    for key in res:
        min_risk = res[key].reindex(trade_days)
        if key == 'money':
            min_risk /= (cap_data['circulating_market_cap'] * 100000000)
        new_risk = min_risk.rolling(21).mean() - min_risk
        dfs[key + '_umr'] = new_risk * excess_rtn
    df = dfs_to_df(dfs)
    df.to_parquet('./umr.parquet')
```

第一段代码，主要是处理日频数据。

第2-9行，处理的是行情数据，主要是计算日频的收益率。其中，第9行，将停牌的收益率置为nan，而不是0。

第10-12行，读取市值、换手率和流通市值的数据。

第13-15行，用市值加权计算市场收益，然后计算每个标的的超额收益。

第16-18行，以换手率为风险代理指标计算因子。

第19-22行，以真实波动为风险代理指标计算因子。

第23-27行，计算分钟高频的风险代理指标，并转换为3个dataframe。

第28-33行，计算分钟高频风险代理指标对应的因子。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    cur = pd.to_datetime(date_str) + timedelta(hours=15)
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'turnover']).to_dataframes()
    rtn = data['close'].pct_change().iloc[1:]
    money = data['turnover'].iloc[:6].sum() + data['turnover'].iloc[-6:].sum()
    sigma = rtn.std()
    skew = rtn.skew()
    res = pd.concat([money, sigma, skew], axis=1)
    res.columns = ['money', 'sigma', 'skew']
    res['datetime'] = cur
    return res
```
