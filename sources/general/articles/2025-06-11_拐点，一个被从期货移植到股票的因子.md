本文，想复现的因子叫做价格拐点。

这个因子来自国金证券在2024年发布的研报《基于日内高频博弈信息的商品CTA策略》。

这个因子通过计算t日拐点数量与前五日拐点数量的均值之差。

在期货市场中，这个因子的逻辑是这样的。

拐点因子衡量了短期内期货品种相较过去的交易分歧度；当市场分歧度小（因子值小，做多），当日拐点数量可能相对较少；当市场分歧大（因子值大，做空），博弈频繁时，拐点数量就可能相对较多。

笔者进行了一个简单的尝试，想用股票数据来复现一下这个因子，看看这个因子能不能在股票市场上也是有效的。

关于拐点的计算，和高中数学还是有一点不一样的。

在高中数学里面，拐点的判断有四条定理。

这里所说的拐点就是，最新价格变动方向与上一次价格变动方向不一致的时候，算一次拐点。

在实现中，笔者还进行了一点点简单的改动，就是对价格不变动的时候进行了特殊处理。如果t时刻价格不变动，那么我们一直往前找，找到最近一次价格变动的时候，以它的变动方向（正或负）作为t时刻的变动方向。对应下面代码中的10-14行。

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume', 'close'])
    date_str = file_name.split('.')[0]

    vol = data.to_dataframe('volume').diff()
    price = data.to_dataframe('close').diff()
    vol = vol.replace(0.0, np.nan)
    price = price.replace(0.0, np.nan)

    vol = vol.ffill()
    price = price.ffill()

    vol = ((vol * vol.shift(1)) < 0).sum()
    price = ((price * price.shift(1)) < 0).sum()

    res = pd.concat([vol, price], axis=1)
    res.columns = ['vol', 'price']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

同时，在计算的过程中，笔者还计算了成交量的拐点。因为，在之前复现长江证券覃川桃老师的成交量波峰因子的时候，发现这是一个还算不错的因子。因此，这里参考了那篇研报的思想，看看成交量拐点因子是不是也会有不错的表现。

价格拐点

这个因子的效果很一般，但是如果去掉笔者自作主张的那一点点改动的话，会有些微的提升。但是，这回影响到成交量拐点因子的效果，毕竟成交量拐点因子是这期复现中效果最好的因子了。所以，就保留了那个自作主张的改动。

如果不改动的话，那就是去掉10-14行，然后将16和17行换成下面代码。

```python
    vol = (vol != vol.shift(1)).sum()
    price = (price != price.shift(1).sum()
```

成交量拐点

这个因子用均值合成的效果好。如果用标准差的话，2018年因子的IC值为正，其他年份为负。

总结：这个因子如果是用成交量计算的话，还算是一个不错的因子。IC绝对值有四年超过了0.06，而分层回测方面，因子值最高的一组和最低的一组的净值曲线从2021年开始，与其他三组就很少相交了。
