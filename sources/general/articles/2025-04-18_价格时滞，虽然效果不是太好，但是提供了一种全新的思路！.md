本文介绍的因子来自外国人2005年发表的论文，然后东方证券的朱剑涛老师在2015年的研报《投机、交易行为与股票收益（上）》当中提到了这个因子。

这个因子的计算需要用到线性回归。

回归的被解释变量，也就是通常所说的y是标的日频收益率。

回归的解释变量，也就是通常所说的x由市场收益率组成，分别是当期市场收益率、滞后1期市场收益率、滞后2期的市场收益率、……、滞后n期的市场收益率。这里，n可以取3或者5。

然后计算两个R方。第一个R方是上述回归方差的R方，第二个则是只保留档期市场收益率的权重，滞后的市场收益率权重全为0的R方。

然后，用1-（第一个R方/第二个R方）作为因子。

计算的原理不难，但是如果一个一个标的去线性回归的话，那就太慢了。

```python
def process_single_day(self, idx):
    tmp_y = self.data.iloc[idx - 21:idx].dropna(how='all', axis=1)
    tmp_x = self.mkt_ret.iloc[idx - 21:idx]

    Y_full = tmp_y.dropna(axis=1)
    idx_full, Y_full = Y_full.columns, Y_full.values
    X_full = np.c_[np.ones((21, 1)), tmp_x.values]
    beta_full = np.linalg.pinv(X_full.T @ X_full) @ X_full.T @ Y_full
    r2 = compute_r_squared(X_full @ beta_full, Y_full)
    beta_full[2:, :] = 0
    new_r2 = compute_r_squared(X_full @ beta_full, Y_full)
    factor_full = pd.Series(data=1 - new_r2 / r2, index=idx_full, name=tmp_x.index[-1])

    factor_lack = {}
    for c in set(tmp_y.columns) - set(idx_full):
        lack_tmp_y = tmp_y.loc[:, [c]].copy()
        lack_tmp_y = lack_tmp_y.dropna()
        lack_tmp_x = tmp_x.reindex(lack_tmp_y.index)
        if len(lack_tmp_y) < 10:
            factor_lack[c] = np.nan
            continue
        X_lack = np.c_[np.ones(len(lack_tmp_y)), lack_tmp_x.values]
        Y_lack = lack_tmp_y.values
        beta_lack = np.linalg.pinv(X_lack.T @ X_lack) @ X_lack.T @ Y_lack
        r2 = compute_r_squared(X_lack @ beta_lack, Y_lack)[0]
        beta_lack[2:, :] = 0
        new_r2 = compute_r_squared(X_lack @ beta_lack, Y_lack)[0]
        factor_lack[c] = 1 - new_r2 / r2
    factor_lack = pd.Series(factor_lack, name=tmp_x.index[-1])
    return pd.concat([factor_full, factor_lack]).sort_index()
```

整个因子的计算可以分成两部分，第一部分（对应代码的2-12行）对于没有缺失值的，直接一起用最小二乘的公式计算即可。第二部分（对应代码的14-29行），对于有缺失值的，那就得一个一个计算了。

第2-3行，获取过去21个交易日的数据。然后去掉全为nan的列。

第5行，去掉有nan的列，剩下的数据就是没有缺失值的了。

第6行，保存数据和列名。

第7行，添加截距项。

第8行，通过最小二乘公式计算回归系数。

第9行，计算第一个R方。

第10-11行，计算滞后市场收益率回归系数为0时的R方。

第二部分和第一部分类似，就是需要一个标的一个标的的处理。

需要注意的是第18行，通过reindex对齐解释变量和被解释变量的索引。

这里需要说明的是，3阶滞后和5阶滞后看起来没有太大的差别。
