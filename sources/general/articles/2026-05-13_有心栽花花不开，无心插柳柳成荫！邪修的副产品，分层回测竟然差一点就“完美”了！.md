**计算步骤和代码分析**

这次，笔者想用过去21个交易日每分钟的成交量中位数作为解释变量，当天每分钟的成交量作为被解释变量，进行时序回归，然后回归的系数就是成交量放大倍数。

在进行因子计算的时候，笔者顺便算了一下R方。

结果，在因子评价的时候发现用R方选股的效果更好，所以就有了这个叫做成交量的可预测性的因子。

第二步，计算回归的R方。

def process_single_day(self, idx):

    file_name = self.files[idx]

第7-11行，读取t-20到t日一共21个交易日的成交量数据。

第12行，计算成交量的中位数。

第13行，获取当日的成交量。

第14行，将x和y的数据维度对齐。

第15-17行，进行时序回归，并返回斜率和R方。

```python
def cal_med(self, data):
    data = pd.concat(data)
    data['minute'] = data.index.hour * 60 + data.index.minute
    data = data.groupby('minute', as_index=False, group_keys=False).apply(self._cal_med_)
    return data

@staticmethod
def _cal_med_(group):
    group = group.drop(columns=['minute'])
    group = group.median()
    return group.to_frame().T

@staticmethod
def cal_factors(x, y):
    flag = ~np.isnan(x) & ~np.isnan(y)
    x, y = x[flag].reshape(-1, 1), y[flag].reshape(-1, 1)
    lr = LinearRegression()
    lr.fit(x, y)
    pred = lr.predict(x)
    r2 = r2_score(y, pred)
    return [lr.coef_[0][0], r2]
```

这三个方法都挺简单的，前两个用来计算同分钟数据的中位数，第三个则是调用sklearn的包来进行线性回归，并返回斜率和R方。
