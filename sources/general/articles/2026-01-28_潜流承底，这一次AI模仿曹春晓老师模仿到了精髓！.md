**潜流承底**

说实话，曹春晓老师的因子从各方面来说都很出色，因子表现很好，名字也极具诗意。

于是，笔者有了个想法，让AI来模仿曹老师的风格，生成一个因子。这一次，笔者让豆老师模仿曹老师的风格，它选择了激流勇进因子并对其进行改进得到了潜流承底因子。关于激流勇进因子，各位可以阅读[激流勇进因子，从2013年开始等权分层回测结果简直逆天！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247488083&idx=1&sn=abf322ca2064a600663b49d35e60e9c4&scene=21#wechat_redirect)这篇文章。

关于潜流承底因子，豆老师的构建思想是这样的。

聚焦日内缩量下跌阶段的隐性承接力量刻画，核心挖掘市场反应不足的 alpha：放量下跌的逆势买入是显性承接（曹春晓 “激流勇进” 因子逻辑），而缩量下跌中价格未随量能快速下探的隐性承接，代表资金锁仓意愿强、抛压已充分释放，这类个股后续更易出现修复性上涨，属于未被市场充分定价的交易信号。

因子的经济学含义是这样的。

缩量下跌的价格抗跌性与量能萎缩度的匹配度，代表场内隐性承接资金的 “托底强度”，潜流喻指缩量中未被显性放量体现的承接资金，承底喻指下跌过程中的价格支撑能力。

**计算步骤和代码**

从计算步骤上来说，豆老师也模仿到了曹老师的精髓。

在豆老师的原版因子中去掉了开盘前15分钟和收盘前5分钟的数据，为了是规避集合竞价和尾盘砸盘的干扰。

不过，笔者实测下来，去不去掉对最终的因子评价结果影响微乎其微，所以这里选择不去掉这些数据了。

**计算步骤**

第一步，计算领域成交量、领域收益率趋势和领域价格最低点偏离。

邻域成交量：五分钟成交量之和。

邻域收益率趋势：t分钟收盘价/t-4分钟收盘价-1。

![](images/2026-01-28_潜流承底，这一次AI模仿曹春晓老师模仿到了精髓！/002.png)

邻域价格最低点偏离：5分钟最低价均值/t分钟最低价-1。

![](images/2026-01-28_潜流承底，这一次AI模仿曹春晓老师模仿到了精髓！/003.png)

第二步：判断有效缩量状态。即同时满足邻域收益率趋势<0；邻域成交量相对变化<-0.2;1分钟收益率>-0.01这单个条件。

第三步：对每一个有效缩量时刻，计算承接强度值。

![](images/2026-01-28_潜流承底，这一次AI模仿曹春晓老师模仿到了精髓！/004.png)

此公式的逻辑为：价格抗跌性（LowDev 越大，抗跌性越强）为分子，量能萎缩度为分母，比值越高代表 “缩量越多、价格越不跌”，承接力量越强。

最后就是对每日的承接强度取均值，然后再对过去21个交易日的承接强度取均值得到最后的因子。

**2**

**代码**

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'open', 'low', 'volume'])
    vol = data.to_dataframe('volume')
    vol = vol.rolling(5).sum()
    vol_chg = vol.pct_change()
    close = data.to_dataframe('close')
    open_ = data.to_dataframe('open')
    low = data.to_dataframe('low')
    rtn_5 = close / open_.shift(4) - 1
    rtn_1 = close / open_ - 1
    low_dev = 1 - low.rolling(5).mean() / low
    flag = (rtn_5 < 0) & (vol_chg < -0.2) & (rtn_1 > -0.01)
    cs = low_dev / (abs(vol.diff()) / vol.shift(1))
    res = (cs * np.where(flag, 1, np.nan)).mean()
    res.name = pd.to_datetime(date_str) + timedelta(hours=15)
    return res
```

第7-8行，计算邻域成交量。

第9行，计算邻域成交量相对变化。

第10-13行，计算邻域收益率趋势。

第14行，计算1分钟收益率。

第15行，计算邻域价格最低点偏离。

第16行，获取有效缩量分钟的flag。

第17-18行，计算日承接强度因子。
