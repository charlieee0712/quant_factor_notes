**筹码集中度**

在中信建投陈升锐和姚紫薇两位老师在2025年8月25日发布的研报《筹码分布因子系统构建》中，一共有四个主题的因子。

此前，笔者复现了第一个主题，筹码分布。在这个主题中，表现最好的是偏度因子。碰巧的是，笔者遇到了一位同样复现过这篇研报的大佬，在他的复现结果中，也是偏度表现最好。

同时，他还给了笔者一个信息，其他的因子都不太行。

所以，在笔者复现筹码集中度因子的时候，并没有抱任何期待。只是想着代码的改动不是很大，睡觉前改一下代码然后跑一个晚上就好了。

```python
def cal_factors(self, code, close, volume, factor, tr):
    data = pd.concat([close, volume], axis=1)
    data.columns = ['close', 'volume']
    data = data.groupby('close', as_index=False, group_keys=False).sum()
    last_data = self.last_chip.get(code, None)
    if last_data is None:
        data.iloc[:, -1] *= tr
    else:
        last_data['close'] /= factor
        data = data.merge(last_data, how='outer', on=['close'])
        data = data.fillna(0)
        data['volume'] = data.iloc[:, -1] * (1 - tr) + data.iloc[:, -2] * tr
        data = data[['close', 'volume']]
    chip_high = data['close'].max()
    chip_low = data['close'].min()
    diff = chip_high - chip_low
    close = close.iloc[-1]
    weight = data['volume'] / data['volume'].sum()
    mean = np.average(data['close'], weights=weight)
    q = data['close'].quantile([0.05, 0.95, 0.15, 0.85, 0.25, 0.75])
    q_90 = q.iloc[1] - q.iloc[0]
    q_70 = q.iloc[3] - q.iloc[2]
    q_50 = q.iloc[5] - q.iloc[4]
    chip_distri1 = 2 * q_90 / (q.iloc[1] + q.iloc[0])
    chip_distri2 = q_90 / chip_high
    chip_distri3 = 2 * q_70 / (q.iloc[3] + q.iloc[2])
    chip_distri4 = q_70 / chip_high
    chip_distri5 = 2 * q_50 / (q.iloc[5] + q.iloc[4])
    chip_distri6 = q_50 / chip_high
    data.iloc[:, 0] *= factor
    self.last_chip[code] = data.copy()
    return ((close - mean) / diff, chip_high / chip_low - 1, (mean - chip_low) / diff,
            chip_distri1, chip_distri2, chip_distri3, chip_distri4, chip_distri5, chip_distri6)
```

代码其实也不复杂，就是按照公式来计算。

第20-29行，就是公式的代码表示。

**3**

**因子逻辑**

在研报中关于这六个因子的逻辑介绍是这样的。

筹码集中度越高，往往对于出货阶段，市场炒作明显，未来表现可能不佳。

这就意味着，这是一个负向因子。
