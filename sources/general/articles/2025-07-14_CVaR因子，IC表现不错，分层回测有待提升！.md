本文，笔者将介绍CVaR因子。

这个因子来自严佳炜老师2019年12月25日发布的研报《市场微观结构剖析系列之一：分钟线的尾部特征》。

这个因子的核心思想就是运用风险管理中的VaR模型（在险价值）和CVaR模型（条件在险价值）进行尾部效应分析可以提供有价值的预测信息。

它的计算过程分为两步。

第一步：计算VaR。VaR（X）的含义为给定置信度的情况下，X的最大值是多少。用一个简单的例子进行说明，假设X服从标准正态分布（即均值为0标准差为1的正态分布），那么给定置信度为95%的情况下，X=1.65。

第二步，计算CVaR。CVaR（X）描述了在第一步给定的置信度的情况下，X≤VaR（X）的均值。

笔者的文字表达能力不太行，没看明白的各位待会可以结合代码看看。

在本文要复现的因子中，X是收益率。由于，在股票市场中，损失通常是人们更关心的东西。因此，研报中计算的是左侧5%的CVaR值。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'open']).to_dataframes()
    rtn = data['close'] / data['open'] - 1
    flag = (rtn <= rtn.quantile(0.05))
    cvar = (rtn * np.where(flag, 1.0, np.nan)).mean()
    cvar.name = pd.to_datetime(date_str) + timedelta(hours=15)
    return cvar
```

核心的计算代码从第7行开始。

第7行，计算收益率最小的5%对应的分钟数。

第8行，对上述分钟数的收益率求均值就得到了最后的因子了。

接下来，笔者将展示一下这个因子的因子评价结果。从结果来看，这个因子无论是均值合成月度因子，还是用标准差合成月度因子，它在IC上的表现都是不错的，但是在分层回测上的表现就有那么一点点不尽人意的意思了。

总结：从IC上的表现来看，两者都是不错的，都有一年的IC绝对值超过了0.12。不过，用标准差合成的要更好一些，因为它还有两年的IC绝对值超过了0.1。再降低一点标准，比较双方IC绝对值超过0.08的年份，用标准差合成的因子也是多出来一年。同样，用标准差合成的因子每年的IC绝对值都超过了0.06，而用均值合成的因子有一年的IC绝对值是低于0.06的。

从分层回测上来说，两者都不太突出。用均值合成的，因子值最中间的那组年化收益率是最高的；用标准差合成的，因子值最小的一组收益率最高，因子值最大的一组收益率最小，但是中间的三组单调性完全相反。
