**基于高频时间维度的选股因子**

前段时间，笔者集中复现了一下东北证券王琦老师的研报，复现到后面那阅读真的是惨不忍睹。

在这种情况下，笔者只能将王老师系列研报中最后一个能复现的因子往后挪一挪了，先来复现点其他的东西。

本文，笔者要复现的因子来自兴业证券的研报《基于高频时间维度的国债期货择时因子及个股CTA研究》，由兴证金工团队发布，具体发布日期笔者没去查，在他们的公众号上显示是2026年6月11日发布的。

从标题来看，这个因子应该和选个股不搭嘎，但是笔者还是想强行看看用它来选股的效果。

**计算步骤和代码**

在研报中，一共有两个系列的因子，一类是基于价格的，另一类则是基于成交量的。

根据笔者以往的经验，用成交量的效果往往是好于价格的，所以本文笔者将先复现基于成交量计算的因子。

如果成交量的因子表现不错的话，后续有可能考虑复现一下基于价格的因子。各位大佬感兴趣的话，可以点赞推荐关注和分享支持一下笔者。

**计算步骤**

第一步，先计算j时刻成交量持续期，计算公式如下：

这里面的Threshold，研报里面说这是他们的一个创新点，使用了动态的门限，但是笔者没看懂是怎么计算动态门限的，于是就使用了中位数离差法，具体通过代码来说明。

公式中的i是从j-1到0中第一个满足Vj-Vi绝对值大于门限的值。若没有满足条件的，则成交量持续期直接等于j。

第二步，计算因子值，笔者一共计算了六个因子。分别是，均值，标准差，最大值，最大值-最小值（因为最小值都是0，所以这个其实是等于最大值的），极端持续期占比（持续期>均值+2倍标准差出现的概率）和成交量持续期比值（上午的均值/下午的均值）。

这一段代码主要就是读取成交量数据，然后对于每一个标的都调用cal\_factors计算因子。

```python
def cal_factors(self, data):
    absolute_deviation = np.abs(data - data.median())
    mad = np.nanmedian(absolute_deviation)
    thd = 5 * mad
    # thd = 3 * data.std()
    res = [0, 1]
    for i in range(2, len(data)):
        for j in range(i-1, -1, -1):
            if abs(data.iloc[i] - data.iloc[j]) > thd:
                res.append(i - j)
                break
        if len(res) != i + 1:
            res.append(i)
    res = np.array(res)
    mu, sigma, max_val = np.mean(res), np.std(res), np.max(res)
    extreme = np.mean(res > mu + 2 * sigma)
    return np.mean(res), np.std(res), max_val, max_val - np.min(res), extreme, np.mean(res[:120]) / np.mean(res[120:])
```

第2-4行，计算门限。

第5行，是用3倍标准差作为门限。

第6-14行，计算持续期。这里注意，代码里面的i和j和前面公式里面的i和j刚好是反过来的。

第15-17行，计算因子值。

**因子逻辑**

交易持续期越短，市场信息效率越高、价格冲击越强，且知情交易更为集中；反之，交易间隔拉长则意味着信息到达放缓、知情交易减少。
