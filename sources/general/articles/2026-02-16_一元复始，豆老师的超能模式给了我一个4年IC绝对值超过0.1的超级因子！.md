**计算步骤和代码**

这个因子的计算步骤很多，但并不是很复杂。

**1**

**计算步骤**

第一步，计算t到t-9分钟的最低价，记为low\_9，然后用t分钟收盘价/low\_9-1得到t分钟收盘价相对于t到t-9分钟最低价的涨幅，衡量价格从低位的反转幅度。

第二步，计算t分钟成交量相对于t-9 到 t 分钟成交量的均值的放大比例，衡量资金进场的强度。

第三步，将从窗口低位涨幅超过 1%且成交量较 10 分钟均值放大 20%的分钟记为复始信号分钟。

第四步，计算复始信号分钟数占比因子，并用该因子乘上（1+日内收益率）得到一元复始因子。

**2**

**代码**

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume', 'close', 'low', 'open'])
    date_str = file_name.split('.')[0]
    low = data.to_dataframe('low').rolling(10).min()
    rtn_reverse = data.to_dataframe('close') / low - 1
    rtn_daily = data.to_dataframe('close').iloc[-1] / data.to_dataframe('open').iloc[0] - 1
    vol = data.to_dataframe('volume')
    spring = ((vol / vol.rolling(10).mean() > 1.2) & (rtn_reverse > 0.01)).mean()
    res = pd.concat([spring, rtn_daily, spring * (1 + rtn_daily)], axis=1)
    res.columns = ['spring', 'rtn_daily', 'factor']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

核心代码从第7行开始，该行是获取t分钟到t-9分钟的最低价。

第8行，计算的是价格从低位的反转幅度，和第7行一起对应计算步骤中的第一步。

第9行，计算的是日内收益率。

第10-11行，实现了计算步骤中第2步和第3步的计算。

后面就是整合因子，并返回dataframe了。这里，笔者还返回了日内收益率因子，用来和一元复始因子计算相关性。
