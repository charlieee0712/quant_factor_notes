**凸显理论因子**

有几天没复现研报了，最近一直在邪修，因为实在找不到好复现的研报了。

然而，邪修也没有什么成果。于是，只能倒逼自己去找研报看看。

本文复现的凸显理论因子，是笔者在中信建投陈升锐老师在《流动性高频因子再构建与投资者注意力因子》这篇文章中看到的。陈老师也是引用了2012年的一篇老外发表的论文，他在这个因子的基础上进行了改进。

为了对比改进之后的效果，首先来看看原始因子的表现吧。

关于这个因子的逻辑是这样的，投资者注意力更容易被具有“凸显性”的收益所吸引；当股票因子值较高时，其历史最高收益具有较高的凸显性，此时投资者过度关注其上升潜力，严重高估股价；因子值较低时，历史最低收益具有明显凸显性时，此时投资者过度强调其下行风险，严重低估股价。

**1**

**计算步骤**

第一步，就是按照惊恐度因子的计算公式，计算惊恐度。需要注意的是，这里的市场收益用的是股票池的平均收益。笔者很懒，直接用了全市场标的收益的均值。

第二步，计算惊恐度在截面上的排序，惊恐度最大的排序为n，最小的排序为1，就是倒序。

第三步，取过去21个交易日的惊恐度因子，除以其21个交易日的均值。

第四步，计算第三步得到的因子值和收益率的相关系数。

**2**

**代码**

```python
def cal_factor(self, idx):
    rtn = self.rtn.iloc[idx-21:idx]
    day = rtn.iloc[[-1]].index.tolist()[0]
    rtn_mkt = rtn.mean(axis=1)
    w = abs(rtn.sub(rtn_mkt, axis=0)) / (rtn.abs().add(rtn_mkt.abs(), axis=0) + 0.1)
    w = w.rank(axis=1, ascending=False)
    w = w.div(w.mean(), axis=1)
    res = w.corrwith(rtn)
    res.name = day
    return res
```

第2行，取过去21个交易日的收益率。

第4行，计算市场收益率。

第5行，计算惊恐度。

第6行，计算惊恐度在截面上的排序。

第7行，计算惊恐度排序除以排序的均值。

第8行，计算相关性。
