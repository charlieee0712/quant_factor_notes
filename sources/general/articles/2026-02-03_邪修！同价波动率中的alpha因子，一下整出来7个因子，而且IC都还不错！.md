**同价波动率**

同价波动率这个因子是笔者在尝试复现国盛证券《Alpha掘金系列之六：弹性与投资者耐心-基于高频订单簿的斜率凸性因子》时邪修出来的。

众所周知，笔者是没有逐笔数据，也没有订单簿数据的。因此，对于此类因子只能理解其思想，然后用分钟级数据尝试复现一下。

这个因子，笔者想通过同价成交量的斜率来尝试一下，但是效果不是很好。不过，在这次尝试的过程中，笔者有了一个新的思路，同价波动率，然后根据同一价格不同分钟的收益率来计算波动率，并以此来构建因子。

这个新的邪修出来的因子，其主要思想参考的是兴业证券郑兆磊老师的同价成交量分布中的alpha因子，关于这个因子笔者在[同价成交量分布中alpha因子，IC和分层回测都还不错](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247488301&idx=1&sn=e9338199d3b01c6c34cdc5b5b38b87f3&scene=21#wechat_redirect)这篇文章中复现过。

**计算步骤**

这个因子的计算思想并不是很复杂，但是实现起来也并不是那么的简单，所以一起来看一看吧。

**1**

**计算步骤**

第一步，计算分钟收益率，即收盘价/开盘价-1。

第二步，计算同一收盘价所有分钟的收益率的标准差，即同价波动率。

第三步，取波动率最高和最低的两个价格，分别记为high和low。

第四步，计算high/最高收盘价，high/最低收盘价，high/收盘价，low/最高收盘价，low/最低收盘价，low/收盘价和high/low，一共7个因子。

**2**

**代码**

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

                                                                     'low2min', 'low2max', 'low_ratio', 'high2low'])
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)

    return res.reset_index()
```

第一段代码主要就是读取数据。

第8-11行，按照标的循环然后调用cal\_factors计算因子。

```python
@staticmethod
def cal_factors(data):
    close = data.iloc[-1, 0]
    high = np.nanmax(data.iloc[:, 0])
    low = np.nanmin(data.iloc[:, 0])
    data['open'] = data['close'] / data['open'] - 1
    data = data.groupby('close').std()
    data.sort_index(inplace=True)
    data.reset_index(inplace=True)
    max_idx = np.argmax(data['open'])
    max_price = data.iloc[max_idx]['close']
    min_idx = np.argmin(data['open'])
    min_price = data.iloc[min_idx]['close']
    return [max_price / low, max_price / high, max_price / close,
            min_price / low, min_price / high, min_price / close, max_price / min_price]
```

第3-5行，分别获取收盘价、最高收盘价和最低收盘价。

第6行，计算分钟收益率。

第7行，按照收盘价进行聚合，并计算波动率。

第10-11行，获取波动率最大的收盘价。

第12-13行，获取波动率最小的收盘价。

第14-15行，返回因子。
