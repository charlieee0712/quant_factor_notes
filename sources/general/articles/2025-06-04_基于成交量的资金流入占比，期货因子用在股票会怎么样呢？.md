本文，笔者复现的因子叫做基于成交量的资金流入占比，该因子来自海通证券2018年的研报《高频量价因子在股票和期货中的表现》。

从原理上来说，这个因子和长江证券覃川桃老师在《高频因子（七）：分布估计下的主动成交占比》是类似的。这个因子，笔者在很早之前在5分钟数据和日频数据上复现过。

该因子，通过对比t分钟和t-1分钟的收盘价的大小来确定是买入还是卖出。如果t分钟的收盘价更大，那t分钟就是买入，系数为1；如果t分钟收盘价更小，那么t分钟就是卖出，系数为-1；两者相等则认为既不是卖出也不是买入，系数为0。

第二步，计算成交量\*收盘价\*上一步得到的系数，或者是权重，然后求和，得到当日流入的资金量。

第三步，用第二步得到的当日流入的资金量除以当日的总成交额，得到了基于成交量的资金流入占比。

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume', 'turnover', 'close'])
    date_str = file_name.split('.')[0]

    dfs = data.to_dataframes()
    dfs['flag'] = dfs['close'].diff()
    dfs['flag'] = dfs['flag'] / dfs['flag'].abs()
    dfs['flag'] = dfs['flag'].fillna(0)
    res = dfs['volume'] * dfs['close'] * dfs['flag']
    res = res.sum() / dfs['turnover'].sum()
    res.name = pd.to_datetime(date_str) + timedelta(hours=15)
    return res
```

总结：这个因子无论用均值合成还是用标准差合成，其在IC上的表现都不是很好。用标准差合成的因子IC为负，2018年到2020年这三年的IC接近于0。用均值合成的也没好到哪里去，从上面的图中可以看出，2018年和2019年的IC为负，其他年份为正。

但是，让人意外的是，这个因子在分层回测上居然表现出了不错的单调性，甚至这个分层回测的结果比一些在IC上表现好的因子还要出色。从2021年开始，相交的情况就很少了。遗憾的是，group\_2的年化收益率大于group\_3，影响了分层回测的单调性。

值得一提的是，如果用标准差合成月度因子的话，相交的情况同样不多，但是多空收益率要远低于均值合成的因子。同时，单调性会比均值差一点，因子值最大的一组在整个回测区间中也是亏钱的。
