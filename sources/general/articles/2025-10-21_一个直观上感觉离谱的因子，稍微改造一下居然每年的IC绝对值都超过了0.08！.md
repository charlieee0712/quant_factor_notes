本文复现的因子来自招商证券任曈老师在2323年发布的研报《基于股票代码有序性的“注意力溢出”》。

在开始正文之前，笔者想要道个歉。这篇研报笔者在2024年的上半年的时候在《因子日历》上看到了它，但是一直没有复现它。

因为，这个因子用到了股票代码排序。笔者第一眼看到就觉得这个因子很离谱，效果应该不好。

可是，随着时间的推移，笔者发现《因子日历》上的因子大部分都是非常不错的，所以这次决定看看这个因子到底怎么样。

《因子日历》对这个因子给出的说明是这样的：

> 由于投资者注意力有限，以及行情软件通常是根据代码顺序依次展示股票信息，投资者会轻易地关注到与焦点股票相邻的其他股票，出现“注意力溢出”，焦点股票与相邻股票之间会存在价格与交易情绪的转移；“自信溢出”对投资者因近期获利的过度自信向邻近股票外溢的情况，与未来收益正相关。

**计算步骤及代码**

我们先用文字来简单描述一下这个因子的计算步骤，这样你们可能就会理解为啥笔者第一次看到这个因子的时候会觉得它离谱了。

**1**

**计算步骤**

第一步，将股票按代码从小到大进行排序后，对每只股票计算相邻n只股票在过去T日的日均涨跌幅，并等权合成为焦点股票i的NBR\_ret。

第二步，将焦点股票i的NBR\_ret对其自身的日军涨跌幅进行截面回归，回归得到的残差即为“自信溢出”因子RNBR\_ret。

注意：焦点股票i的相邻股票数对因子效果影响不大，建议使用较少数量的股票，比如10。

上面的是《因子日历》中介绍的因子计算步骤，用这个计算出来的因子比较一般，只有3年的IC绝对值超过了0.06。

于是，笔者给它改造了一下，将第一步中的过去T日改为过去1日，然后每天做截面回归，得到残差，然后用过去21个交易日的残差因子合成月度因子。

**2**

**代码**

```python
def __call__(self):
    data = BaseDataLoader.load_data('../../data/stock_bar_1day.parquet', fields=['close', 'factor', 'volume']
                                         ).to_dataframes()
    close = data['close'] * data['factor']
    vol = data['volume'] != 0
    close = close * np.where(vol, 1.0, np.nan)
    self.rtn = close.pct_change()
    res = []
    for i in tqdm(range(1, len(self.rtn))):
        res.append(self.process_single_day(i))
    res = pd.concat(res, axis=1).T
    res.index = self.rtn.index.tolist()[1:]
    res.index.name = 'datetime'
    res.reset_index(inplace=True)
    res = pd.melt(res, id_vars='datetime', var_name='code', value_name='rnbr_ret')
    res.to_parquet('./rnbr_ret.parquet')
```

第一段代码，主要是加载数据并对数据进行预处理。

第4行，用每日收盘价乘上每日的复权因子，得到股权的价格。

第5-6行，通过成交量是否为0来判断是否停牌，并将其置为nan。

第7行，用pct\_change来计算价格的变换，也就是收益率。

第8-10行，调用process\_single\_day来计算每日的因子。

```python
def process_single_day(self, idx):
    rtn = self.rtn.iloc[idx].dropna()
    rtn.sort_index(inplace=True)
    x = [rtn.iloc[:10].mean()] * 5
    for i in range(5, len(rtn) - 5):
        x.append(rtn.iloc[i-4:i+6].mean())
    x += [rtn.iloc[-10:].mean()] * 5
    x = np.array(x).reshape(-1, 1)
    y = rtn.values.reshape(-1, 1)
    lr = LinearRegression()
    lr.fit(x, y)
    resi = y - lr.predict(x)
    return pd.Series(resi.reshape(-1), index=rtn.index.tolist())
```

第2行，去掉收益率为nan的标的。

第3行，对数据按照代码从小到大排序。

第4-7行，计算第一步中的NBR\_ret。这里，前5个标的的NBR\_ret为前10个标的收益率的均值，最后5个标的的NBR\_ret为最后10个标的收益率的均值。

剩下的就是线性回归然后计算残差了。
