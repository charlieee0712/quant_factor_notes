**前景价值**

![](images/2026-02-17_前景价值，老外的因子能克服水土不服吗？/001.jpg)蛇年的最后一天笔者没有闲着，那么马年的第一天我也不想闲着。

本文，笔者将带来一个老外在2016年的学术论文中发表的因子，论文名叫《Prospect theory and stock returns: A empirical test》。

笔者是在2025年的《因子日历》上看到这一因子的，它的名字叫做前景价值TK。

在《因子日历》中，是这样对这个因子进行说明的。

> 前景价值代表了投资者给每支个股的“打分”，基于前景理论，由于价值量TK高的股票更具吸引力，投资者总会去买入哪些价值量TK高的股票，导致价值高的股票被超买，而价值低的股票由于没人关注而被低估，股票未来的预期收益率和当前的前景价值TK呈负相关。

```python
@staticmethod
def v_func(x):
    if x >= 0:
        return x ** 0.88
    return -2.25 * (-x) ** 0.88
@staticmethod
def w_plus(p):
    return p ** 0.61 / ((p ** 0.61 + (1 - p) ** 0.61) ** (1 / 0.61))
@staticmethod
def w_minus(p):
    return p ** 0.69 / ((p ** 0.69 + (1 - p) ** 0.69) ** (1 / 0.69))
```

第二段代码主要是读取数据，然后计算了一下收益率，最后调用cal\_factors方法来计算前景价值TK因子。

```python
def cal_factors(self, data):
    data = data.dropna()
    if len(data) <= 5:
        return np.nan
    data.sort_values(inplace=True)
    tk = 0
    l = len(data)
    m = len(data[data < 0])
    n = l - m
    for i in range(m):
        val = data.iloc[i]
        tk += self.v_func(val) * (self.w_minus((i+1) / l) - self.w_minus(i / l))
    for i in range(n):
        val = data.iloc[m+i]
        tk += self.v_func(val) * (self.w_plus((n - i + 1) / l) - self.w_plus((n - i) / l))
    return tk
```

最后一段代码也是最核心的代码，可以和各位大佬好好唠一唠。

第2-4行，对于过去21个交易日中有效交易日小于5的标的直接返回nan。

第5行，对数据按照收益率排序。

第6行，初始化tk为0。

第7行，计算有效交易日的数量。

第8行，计算收益率小于0的有效交易日的数量。

第9行，计算收益率大于等于0的有效交易日的数量。

第10-12行，对收益率小于0的交易日计算前景价值，并累加到tk上。这里需要注意的是，收益率最小的在计算公式中对应的索引是-m，而代码中对应的是i，所以需要将公式中的i替换为-m+i。

第13-15行，计算收益率大于等于0的交易日的前景价值，并累加到tk上。
