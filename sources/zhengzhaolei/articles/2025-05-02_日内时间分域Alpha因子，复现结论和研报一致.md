本文，要复现的因子来自兴证金工2024年7月31日发布的研报《高频研究系列九——日内分域信息中的Alpha》，作者郑兆磊。

该研报介绍了日内时间分域因子、日内价格分域因子以及日内显著性相关的因子。

本文将先介绍一下日内时间分域因子，这一系列一共有三个因子。

第一个，lh\_rtnDIff，开盘和尾盘半小时涨跌幅的比值。

第二个，lh\_volDiff，开盘和尾盘半小时成交量的比值。

第三个，lh\_stdDiff，开盘和尾盘半小时波动率的比值。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'open', 'volume'])
    dfs = data.to_dataframes()
    ret = dfs['close'] / dfs['open'] - 1
    lh_rtndiff = (data.data[29, 1, :] / data.data[0, 2, :] - 1) / (data.data[-1, 1, :] / data.data[-30, 2, :] - 1)
    lh_rtndiff = pd.Series(data=lh_rtndiff, index=ret.columns.tolist())
    lh_voldiff = dfs['volume'].iloc[:30].sum() / dfs['volume'].iloc[-30:].sum()

    lh_stdiff = ret.iloc[:30].std() / ret.iloc[-30:].std()
    res = pd.concat([lh_rtndiff, lh_voldiff, lh_stdiff], axis=1)
    res.columns = ['lh_rtndiff', 'lh_voldiff', 'lh_stddiff']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

因子简单，代码自然也就简单。核心计算代码就是从第9行开始，到第14行结束。

第9行，计算的是收益率，在第14行计算lh\_stdiff的时候会用到。

第10-11行，计算的是lh\_rtndiff。

第12行，计算的是lh\_voldiff。

这三个因子的实现很简单，研报中也没有对其具体的逻辑进行说明，直接展示了结果。从研报中的结果来看，第三个因子的效果是最好的。那么，我们来验证一下到底是不是这样的。

在实际的对比中，用标准差合成的因子，第一个的IC会出现不稳定的情况，一些年份为正，一些年份为负。但是，后面两个因子用标准差的话IC更高。因此，在展示因子评价结果的时候，第一个使用均值合成月度因子的，后面两个是用标准差合成月度因子的。
