本文，笔者将复现兴业证券2024年6月5日发布的研报《高频研究系列八—关联信息中的 Alpha》中提到的部分因子，这篇研报的作者是他们金工首席分析师郑兆磊老师。

在这篇研报中，核心内容一共四小节，分别是《关联信息基础型因子》、《关联信息中的全局特征：非线性因果关系》、《关联信息中的局部特征：“交易时间”和持续期特征》、《关联信息中的典型特征：再论“闪电崩盘”》。

本文复现的因子来自《关联信息基础型因子》这一小节，一共四个因子，分别是：

rvc\_cor：分钟收益率与成交量占比的相关系数。

rhl\_cor：分钟收益率与分钟最高价最低价变化率的相关系数。

voh\_cor：正分钟收益率样本下分钟收益率与成交量占比的相关系数。

vol\_cor：负分钟收益率样本下分钟收益率与成交量占比的相关系数。

在研报中，有一段说明。由于个股在开盘和收盘阶段的量价信息波动较大，因此我们剔除了开盘和收盘10分钟的数据。

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume', 'close', 'open', 'high', 'low'])
    date_str = file_name.split('.')[0]

    vol = data.to_dataframe('volume').iloc[2:-2]
    vol = vol.div(vol.sum(axis=0), axis=1)    
    ret = (data.to_dataframe('close') / data.to_dataframe('open')).iloc[2:-2] - 1
    amp = (data.to_dataframe('high') / data.to_dataframe('low')).iloc[2:-2] - 1

    rvc_cor = ret.corrwith(vol)
    rhl_cor = ret.corrwith(amp)

    flag = np.where(ret > 0, 1, np.nan)
    voh_cor = (ret * flag).corrwith(vol * flag)

    flag = np.where(flag==1, np.nan, 1)
    vol_cor = (ret * flag).corrwith(vol * flag)

    res = pd.concat([rvc_cor, rhl_cor, voh_cor, vol_cor], axis=1)
    res.columns = ['rvc_cor', 'rhl_cor', 'voh_cor', 'vol_cor']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

这里的代码是基于5分钟K线的，如果要基于1分钟K线计算，那么需要将2:-2换成10:-10。由于，需要去除开盘和收盘10分钟的数据，所以就没有继续尝试15分钟之后的K线了。

第8-9行，计算成交量占比。

第10行，计算分钟收益率，这里用的是收盘价/开盘价-1。其实还可以用t时刻的收盘价/（t-1时刻 的收盘价）-1，但是两者差距不会太大。

第11行，计算最高价最低价变化率。

第13-20行，分别计算上文提到的四个因子，其中flag用来划分正分钟收益率和负分钟收益率，这里笔者将收益率为0的情况算作负分钟收益率。

rvc\_cor

这个因子是基于5分钟K线计算的，5分钟K线计算的结果IC绝对值略高一点点，但是可以说没有实质性差异。同时，两者的分层回测可以说是一样的差。

同时，这个因子是用标准差合成的。如果用均值的话IC绝对值会更低，而且分层回测的表现也会更差。

rhl\_cor

这个因子是基于1分钟K线计算的，同时也是用均值合成的。

1分钟K线和5分钟K线相比，有两大优势。

第一，IC绝对值更高，1分钟K线计算的因子IC绝对值大部分都超过了0.07，5分钟K线计算的因子则有刚刚过0.06的。

第二，分层回测表现也更好，相交的情况更少。

均值和标准差相比，就好更多了。标准差的IC绝对值最高的也就勉强超过了0.07，最低的只有0.03。分层回测的话两者相差并不大。

用标准差合成的话就更差了，2019年的IC和其他年份的IC不是一个方向的（2019年IC为负，其他年份为正）。

vol\_cor

从上面的因子应该可以得出结论，这个因子也是用1分钟K线计算更好。同时，也是均值合成更好。

实际的结果也是这样的，用标准差合成的IC方向与均值相反，而且IC绝对值的范围在0.01-0.05之间，而均值合成的IC绝对值范围在0.03-0.11之间。分层回测就更不用谈了，用标准差合成的五条净值曲线柔和在一起，很难分开。

如果用5分钟K线计算，IC绝对值的下线会提到到0.04，但是上限也会低。分层回测的单调性更好了，但是多空收益会低。这里补两张图。

总结：这四个因子中，第二个因子和第四个因子的表现还是不错的。这一点和研报的结论有一部分相似，研报中IC最高的是第二个因子，但是最低的是最后一个因子。

第二个因子最好，这在笔者的复现中得到了证实，但是第四个因子最差就和复现结果不同了。
