上一篇文章，笔者介绍了交易时间下的基础因子。

本文，将继续复现该主题下的另外两个因子。

cutVol\_timeCor，这个因子就是时间间隔数据的一阶自相关性。

这个因子背后的逻辑是，一阶自相关性越高，那么该标的的交易聚集性也越高。

cutVol\_entropy，分桶熵。分桶熵的计算，首先要对数据进行分桶，然后计算数据落入每一桶的概率，最后计算熵。

在这个因子当中，分桶的时候同时考虑了时间间隔和收益率的正负。

统计收益率方向一致以及时间间隔一致的数据出现的次数，然后除以所有数据出现的次数，就得到了数据落入到这个桶中的概率。

举个例子，假设时间间隔为3，收益率为正的数据出现过2次，然后总数据长度为50，那么落入这个桶的概率为0.04。

这个因子背后的逻辑，研报是这样描述的。本质上是刻画一组数据的分化程度与随机性：如果同号同大小的时间间隔数据大量出现，如大量出现上涨且间隔绝对值均为 3 的数据，说明该股出现了交易聚集现象，整体交易规则呈现出极强的一致性，反之我们可以认为交易的时间间隔相对随机。

接下来，看看代码。

```python
def cal_factors(self, data):
    close, time_list = [], [0]
    thd = np.nansum(data[:, 0]) / 50
    if thd == 0:
        return [np.nan] * 2
    cur_vol, cur_close = 0, 0

    for i in range(len(data)):
        diff = thd - cur_vol
        if data[i, 0] is None:
            continue
        if diff - data[i, 0] <= 0.01:
            cur_close += data[i, 1] * diff / thd
            close.append(cur_close)
            cur_vol = (data[i, 0] - diff)
            time_list.append(i + 1)
            while thd - cur_vol <= 0.01:
                time_list.append(i+1)
                close.append(data[i, 1])
                cur_vol -= thd
            else:
                cur_close = cur_vol * data[i, 1] / thd
        else:
            cur_close += data[i, 0] * data[i, 1] / thd
            cur_vol += data[i, 0]
    time_list = np.diff(time_list)
    ret = np.diff(close) / np.array(close)[:-1] > 0
    ret = np.insert(ret, 0, close[0] > data[0, 1])
    bucket = np.where(ret, 1, -1) * time_list
    pos_prob = self.cal_entropy(bucket[bucket > 0], len(bucket))
    neg_prob = self.cal_entropy(np.abs(bucket[bucket < 0]), len(bucket))
    cutVol_entropy = -1 * (pos_prob + neg_prob)

    time_series = pd.Series(data=time_list)
    cutVol_timeCor = time_series.shift(1).corr(time_series)
    return cutVol_entropy, cutVol_timeCor
```

这段代码中，大部分的内容是和上一篇文章中一样的。

独属于这两个因子的代码是从27行开始的。

第27行，统计收益率大于0的情况。

第28行，对于第一个区间，将其交易时间价格和第一个收盘价比较来确定方向。

第29行，计算分桶，收益率为正在桶中为正值，收益率为负的在桶中为负值。

第30-31行，分正负值计算各自的熵。因为，numpy的bincount方法只能对正数使用。

第32行，计算分桶熵。

第34-35行，计算时间间隔的一阶自相关性。

```python
@staticmethod
def cal_entropy(data, l):
    prob = np.bincount(data[data >= 0]) / l
    prob = prob[prob > 0]
    return np.sum(prob * np.log(prob))
```

这个就是计算分桶熵的核心代码。

cutVol\_timeCor

这个因子用均值合成的效果是最好的，如果用标准差的话IC会反向，同时绝对值也远低于用均值合成的因子。
