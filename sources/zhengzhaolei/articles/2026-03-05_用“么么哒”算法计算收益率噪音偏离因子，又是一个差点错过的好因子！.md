**收益率噪音偏离因子**

本文，笔者要复现的因子还是来自兴业证券的首席分析师郑兆磊老师，他和他的团队在2022年1月23日的时候发布过一篇题为《高频研究系列二：收益率分布因子构建》的研报。

在这篇研报中，郑老师提到了一个收益率噪声偏离因子。

关于这个因子的逻辑，研报中提供了一张图片。

**计算步骤和代码**

在详细介绍这个因子的计算步骤之前，笔者将简答和各位大佬聊一聊自己对这个因子的理解。

以笔者的知识来看，这个因子的核心思想是认为我们看到的收益率是围绕着真实收益率波动的。在信号处理中，都是通过观测值去预测真实值，最典型的就是卡尔曼滤波。

也就是说，观测到的收益率和真实的收益率之间存在一个误差，我们可以假设这个误差是服从均值为0的正态分布，然后用一系列统计学方法来衡量误差和其服从的正态分布之间的差异就得到了这个因子。

**1**

**计算步骤**

第一步，对收益率进行标准化。减去均值，可以认为得到了误差；然后除以误差的标准差，则是将误差的分布转换为标准正态分布。

第二步，衡量误差和标准正态分布之间的差异。这里将是计算这个因子的关键。

首先，如何获得标准正态分布的数据。这里，笔者提供两种方法。

第一种，随机生成一个标准正态分布的序列。

第二种，对过去21的交易日的分钟收益率序列标准化。因为，这个数据的观测越多，其误差也就越接近标准正态分布。

其次，如何衡量两个序列之间的分布差异。这里，将有多种方法。

第一种，研报中采用的，引入核密度函数估计收益率分布的pdf，再计算估计出的pdf与标准正态分布pdf的距离（差值的平方和）。由于研报中没写怎么引入核密度函数估计pdf的，所以笔者无从下手。直接点对点计算距离，显然是不科学的，因为每个点之间是独立同分布的。

第二种，直接计算序列的偏度。

第三种，统计学方法，KL散度之类的，但是这里笔者用的是最大平均偏差（MMD）。因为，根据笔者读研时的经验，这种方法效果较好。

需要注意的是，计算MMD的时候，两个序列长度是可以不一样的。换句话说，我们可以计算当前交易日标准化后的收益序列和过去21个交易日标准化后收益序列的MMD。但是，这样做的计算时间会大幅拉长，拉长110+倍。

之所以叫它“么么哒”算法，是因为“么么哒”的缩写也是MMD。

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    cur_time = pd.to_datetime(date_str) + timedelta(hours=15)
    if idx < 243 or cur_time >= pd.to_datetime('2026-01-01'):
        return pd.Series(name=cur_time)
    close = []
    for i in range(idx - 20, idx + 1):
        file_name = self.files[i]
        full_path = os.path.join(self.file_pth, file_name)
        data = BaseDataLoader.load_data(full_path, fields=['close'])
        factor = self.factor.iloc[idx - 243].reindex(data.codes)
        close.append(data.to_dataframe('close') * factor.values.reshape(1, -1))
    his = pd.concat(close).pct_change()
    cur = his.iloc[-240:]
    his = (his - his.mean().values.reshape(1, -1))
    his = his / his.std().values.reshape(1, -1)
    cur = (cur - cur.mean().values.reshape(1, -1))
    cur = cur / cur.std().values.reshape(1, -1)
    res = cur.skew().to_frame()
    res.columns = ['skew']
    mmd = []
    for i in range(len(cur.columns)):
        tmp_cur = cur.iloc[:, i].dropna()
        if len(tmp_cur) < 120:
            mmd.append(np.nan)
        else:
            tmp_his = his.iloc[-240:, i].dropna()
            mmd.append(self.compute_mmd(tmp_cur.values, tmp_his.values))
    res['mmd'] = mmd
    res['datetime'] = cur_time
    return res
```

第15行，计算收益率。

第16行，获取当前交易日的收益率。

第17-18行，对过去21个交易日的收益率数据进行标准化。

第19-20行，对当前交易日的收益数据进行标准化。

第21行，计算偏度。

第23-31行，计算MMD。

```python
@staticmethod
def compute_mmd(x, y, gamma=1.0):
    x = x.reshape(-1, 1)
    y = y.reshape(-1, 1)
    k_xx = rbf_kernel(x, x, gamma=gamma)
    k_yy = rbf_kernel(y, y, gamma=gamma)
    k_xy = rbf_kernel(x, y, gamma=gamma)
    return np.mean(k_xx) + np.mean(k_yy) - 2 * np.mean(k_xy)
```
