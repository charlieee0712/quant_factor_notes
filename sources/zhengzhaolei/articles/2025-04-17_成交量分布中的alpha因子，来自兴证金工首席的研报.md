本文介绍的一系列因子来自，兴业证券2022年8月29日发布的研报《高频研究系列四——成交量分布中的alpha》。作者是兴证金工的首席郑兆磊老师。

在这篇研报中，一共介绍了9个因子。其中有三类常见的成交量因子，一共7个。

第一类是成交量对数相关的因子，一共有三个，对数成交量的偏度、对数成交量90%以上占比，对数成交量10%以下占比。

第二类是成交量变化率相关的因子，一共两个，分别是偏度和峰度。

第三个是累积成交量相关的因子，也是两个，分别是累积成交量的均值和标准差。

在研报中，作者是在15分钟的数据上对这7个因子进行测试的。

笔者在复现过程中，对1分钟和5分钟的数据也进行了相应的测试，效果确实不如15分钟数据的好。同时，IC的方向也和15分钟的数据相反。

既然是复现别人的研究成果，那就尊重研报，按照研报中的数据来实现，所以后面展现的所有结果都是基于15分钟的数据的。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume']).to_dataframe('volume')

    vol_change = data.pct_change()
    vol_skew = vol_change.skew(axis=0)
    vol_kurt = vol_change.kurt(axis=0)

    cum_vol = data.cumsum(axis=0)
    cum_vol_mean = cum_vol.mean(axis=0)
    cum_vol_std = cum_vol.std(axis=0)

    log_vol = np.log(1 + data)
    log_vol_skew = log_vol.skew(axis=0)
    log_vol_kurt = log_vol.kurt(axis=0)

    log_vol_sum = log_vol.sum(axis=0)
    q = log_vol.quantile([0.1, 0.9])
    log_vol_tail10 = (log_vol * (log_vol < q.iloc[0])).sum(axis=0) / log_vol_sum
    log_vol_tail90 = (log_vol * (log_vol > q.iloc[1])).sum(axis=0) / log_vol_sum

    res = pd.concat([vol_skew, vol_kurt, cum_vol_std, cum_vol_mean, log_vol_kurt,
                     log_vol_skew, log_vol_tail90, log_vol_tail10], axis=1)
    res.columns = ['vol_skew', 'vol_kurt', 'cum_vol_std', 'cum_vol_mean', 'log_vol_kurt',
                   'log_vol_skew', 'log_vol_tail90', 'log_vol_tail10']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

这个因子的计算是比较简单的，也只需要用到成交量一个字段。

第9-10行，计算成交量变化率相关的两个因子。

第13-15行，计算和累积成交量相关的两个因子。

第17-18行，这里笔者自行增加了一个因子，对数成交量的峰度，效果不好。

第21-23行，计算对数成交量90%以上占比和对数成交量10%以下占比这两个因子。
