**成交量分桶熵**

成交量分桶熵这个因子，笔者也是差不多一年前就知道它了。

它来自兴业证券郑兆磊老师在2022年8月29日发布的研报，《高频研究系列四——成交量分布中的Alpha》。

此前，笔者复现过这篇研报中的部分因子。

比如：[成交量极大值分布，读不懂的研报，效果意外的因子！](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247495439&idx=1&sn=68a83727ece78132984b8a6f091ca05b&scene=21#wechat_redirect)

又比如：[同价成交量分布中alpha因子，IC和分层回测都还不错](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247488301&idx=1&sn=e9338199d3b01c6c34cdc5b5b38b87f3&scene=21#wechat_redirect)

再比如：[成交量分布中的alpha因子，来自兴证金工首席的研报](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247487630&idx=1&sn=54fbe0c0bf30f0971e48acca836947cc&scene=21#wechat_redirect)

这就意味着，这篇文章结束之后，郑老师提出的关于成交量分布中的Alpha因子就全部介绍完了。

值得一提的是，这个因子在2024年-2026年这三年的《因子日历》中至少出现了两次。关于其背后的逻辑，《因子日历》是这么描述的。

> 成交量分桶熵（标准差）衡量了成交量不稳定性的分散程度，该因子值越大，说明该股在近一段时间内成交量不稳定的分散程度较大，股价曾经或正在受到知情交易者的带动，此时个股不稳定性极高，整体下行风险较大；日度成交量分桶熵取值越大，日内成交量分布越均匀。

**计算步骤和代码**

之前，笔者没有介绍这个因子，是因为当时没能理解这个因子的精髓。

说人话就是，不知道该怎么分桶。

但是，最近突然就悟了。

**1**

**计算步骤**

第一步，将成交量按照最小值到最大值的范围等距分桶，并计算成交量落在每一个桶的概率。这里研报是分成了10个桶。同时，笔者测试了一下分成5个桶或者20个桶的情况和10个桶是差不多的。

第二步，计算熵。

第三步，计算近20个交易日熵的标准差。

**2**

**代码**

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    vol = BaseDataLoader.load_data(full_path,
                                    fields=['volume']).to_dataframe('volume')
    start = vol.min()
    step = (vol.max() - start) / 10
    res = []
    for i in range(1, 11):
        end = start + step
        if i != 10:
            flag = (vol >= start) & (vol < end)
        else:
            flag = (vol >= start) & (vol <= end)
        res.append(flag.sum())
        start = end
    res = pd.concat(res, axis=1).T
    res = res / vol.count()
    res = res * np.log(res)
    res = -res.sum()
    res.name = pd.to_datetime(file_name.split('.')[0]) + timedelta(hours=15)
    return res
```

这个因子的计算并不是很复杂，所以代码也不长。

前6行，读取数据。

第7行，获取成交量的最小值，将其作为分桶区间的起始值。

第8行，计算分桶的步长。

第9-17行，计算成交量落在每个桶中的次数。

第19行，计算落在每个桶的概率。

第20-21行，计算熵。
