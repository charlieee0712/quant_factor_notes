**成交量极大值分布**

本文，笔者要复现的因子来自兴业证券的首席分析师郑兆磊老师，他在2022年8月29日发布了一篇题为《高频研究系列四：成交量分布中的Alpha》的研报。

在之前的文章中，笔者已经复现了这篇研报中的部分因子，其中比较知名的应该是和同价成交量相关的，[同价成交量分布中alpha因子，IC和分层回测都还不错](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247488301&idx=1&sn=e9338199d3b01c6c34cdc5b5b38b87f3&scene=21#wechat_redirect)。还有就是[成交量分布中的alpha因子，来自兴证金工首席的研报](https://mp.weixin.qq.com/s?__biz=MzkwNjYzMTEyMg==&mid=2247487630&idx=1&sn=54fbe0c0bf30f0971e48acca836947cc&scene=21#wechat_redirect)这篇文章中介绍的一系列因子。

本文，笔者介绍的因子成交量极大值分布因子。（研报中还有一个成交量分桶熵因子笔者还没有介绍）

为什么把这个因子留到了一年之后呢？因为一年前，笔者也不太理解这个因子是怎么计算的。即使是现在，笔者也没有完全理解，只是凭着感觉尝试了一把，看起来效果并不差，所以就写了篇文章。

值得一提的是，同价成交量和成交量分桶熵这两个因子在《因子日历》上是出现过的，如果没记错的话，2024到2026年这三年的《因子日历》上都有这篇研报的因子，但是成交量极大值分布这个因子却一直没有。笔者想不自量力一把，填补这个空缺。

关于这个因子的具体逻辑，研报中用了整整半页来详细说明，这里就不全贴出来了。只引用其中的一句话。

> 因子值越大，说明该股在近一段时间内成交量极大值的分散程度较大，股价曾经或正在出现成交量多次过大，或极值差异明显。此时个股不稳定性极高，整体下行风险较大。

**计算步骤和代码**

说实话，研报里面没有具体的计算步骤，只是提到了Bootstrap重采样的方法。

这个方法，笔者算是比较熟悉了，它就是从一堆数据里面放回采样，重复n次，然后计算采样数据的均值。值得一提的是，当n趋近无穷大的时候，每一个数据也有1/e的概率没有被采样到。

**1**

**总体分析**

第一步，对每个标的的分钟成交量进行Bootstrap重采样得到极大值样本数据。这里，笔者怀疑过按照成交量占比的概率来进行采样，但是那样的话应该就不是Bootstrap重采样了。可是，等权采样怎么得到极大值样本数据呢？笔者不知道，但还是按照等权实现了一把。

第二步，减去均值后，计算均值，得到第一个因子。该值越大，说明该股成交量极值出现的次数相对较多，且骤增的量较大。

第三步，减去均值后，计算标准差，得到第二个因子。该极大值的样本标准差越小，说明该股成交量极大值的量相对均衡，极值部分差异较小。

**2**

**代码**

这个因子的实现关键在于Bootstrap重采样，但是pandas提供了方法进行这一操作，所以代码就不复杂了。

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]
    cur_time = pd.to_datetime(date_str) + timedelta(hours=15)
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume']).to_dataframe('volume')
    resample_data = []
    for i in range(1000):
        new_data = data.iloc[:]
        resample_data.append(new_data.sample(2400, replace=True).mean())
    resample_data = pd.concat(resample_data, axis=1)
    resample_data = resample_data - resample_data.mean(axis=1).values.reshape(-1, 1)
    res = [resample_data.mean(axis=1), resample_data.std(axis=1)]
    res = pd.concat(res, axis=1)
    res.columns = ['resample_mu', 'resample_sigma']
    res['datetime'] = cur_time
    return res
```

前8行，读取数据。

第9-12行，Bootstrap重采样1000次，然后计算每一次重采样的均值。这里，也可以有多种理解，比如将所有重采样的数据一起减去均值再计算均值和标准差。理论上，笔者应该尝试一下这种方法然后再写文章的，但是计算一次需要10小时，太慢了。
