V型特征，一般用来形容标的先跌后涨。

该因子来自《高频研究系列八—关联信息中的 Alpha》中《关联信息中的典型特征》这一小节。

该因子的计算分成了四步。

第一步，计算分钟量价幅度序列。即分钟级对数收益率与成交量占比的乘积。

第二步，计算累计分钟量价幅度序列。即符号相同的量价幅度的累加，这样会得到一个正负交替的数组。

第三步，刻画V型特征，这里的V型特征是指大幅下跌，小幅上涨，且下跌幅度大于上涨幅度的一段走势。最后，计算满足条件的相邻两个累计分钟量价幅度之和的绝对值。

第四步，得到四个因子，分别如下。

negV\_mean：第三步得到的序列的均值。

negV\_max：第三步得到的序列的最大值。

negVwgt\_mean：第三步得到的序列除以各自对应的时间跨度后的均值。

negVwgt\_max：第三步得到的序列除以各自对应的时间跨度后的最大值。

```python
@staticmethod
def cal_factors(data):
    rv_cum = []
    time_cum = []
    tmp_cum = data.iloc[0]
    for i in range(1, len(data)):
        tmp_rv = data.iloc[i]
        if tmp_cum > 0 and tmp_rv > 0:
            tmp_cum += tmp_rv
        elif tmp_cum < 0 and tmp_rv < 0:
            tmp_cum += tmp_rv
        else:
            rv_cum.append(tmp_cum)
            if not time_cum:
                time_cum.append(i)
            else:
                time_cum.append(i - time_cum[-1])
            tmp_cum = tmp_rv

    rv_v = []
    rv_weight_v = []
    for i in range(1, len(rv_cum)):
        condition = rv_cum[i-1] + rv_cum[i]
        if rv_cum[i-1] < 0  and condition < 0:
            rv_v.append(np.abs(condition))
            rv_weight_v.append(rv_v[-1] / (time_cum[i-1] + time_cum[i]))
    if not rv_v:
        return [np.nan] * 4
    return [np.mean(rv_v), np.max(rv_v), np.mean(rv_weight_v), np.max(rv_weight_v)]
```

该方法的输入为单个标的的分钟量价幅度序列组成的Series。

第3-18行，完成第二步累计量价幅度的计算，同时计算一下每一个累计量价幅度持续的时间。

第20-21行，完成第三步和第四步得到最后的因子。这里需要注意的是第24行，研报中有3个条件，被笔者合并称为了两个。因为rv\_cum是一个正负相间的序列，所以rv\_cum[i-1]小于0的时候，rv\_cum[i]肯定是大于0的。

最后，展现一下评价结果。

说实话，这四个因子的效果可谓是半斤八两，笔者标准差、均值以及均值/标准差都试过了，但是没有发现一个特别好的因子。

所以，就简单展示一下IC绝对值最高的一个因子，negV\_max。

这个因子用均值合成IC略高一点，用标准差合成分层回测略好一点，但是没有实质性差异。所以，分层回测就展示均值合成的了。
