在很久很久之前，笔者将这个因子应用到了日频数据上，取得了比非流动性要好的效果。

本文，笔者将在分钟级的数据上再次复现一下这个因子，看看效果会不会更好。

这篇因子来自长江证券首席分析师覃川桃2019年研报，《高频因子（五）：高频因子和交易行为》

关于这个因子的计算，用一个公式就能表达了。

![](images/2025-04-11_轨迹的非流动性，来自长江证券的首席覃川桃！/001.png)

研报中，将收益率绝对值+1并连乘称之为轨迹。所以，就有了标题当中的因子名，轨迹非流动性。

关于这个因子，还有一大段的解释。但是，笔者认为这些并不重要，只要知道如何计算，然后效果还不错，就行了。

```python
def process_single_day(self, idx):
    file_name = self.files[idx]
    date_str = file_name.split('.')[0]

    # 加载当日分钟数据
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['close', 'open', 'turnover'])
    ret = np.log(np.prod(np.abs(data.data[:, 0, :] / data.data[:, 1, :] - 1) + 1, axis=0))
    turnover = data.to_dataframe('turnover').sum(axis=0)
    res = pd.Series(data=ret/turnover.values, index=data.codes, name=pd.to_datetime(date_str) + timedelta(hours=15))
    return res
```

一个简单的公式，实现起来自然也就不难。这里需要注意的是，笔者在计算收益率的时候用的是收盘价/开盘价-1。
