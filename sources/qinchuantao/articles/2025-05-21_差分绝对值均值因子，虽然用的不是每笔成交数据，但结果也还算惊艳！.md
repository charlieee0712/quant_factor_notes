本文，笔者将继续复现覃川桃老师在《基础因子研究（十四）：高频因子（九），高频波动中的时间序列信息》中提到的差分绝对值均值因子。

在之前的文章中，笔者复现了差分标准差因子，然后说了句“没想到桃子姐也有失手”的时候。

在这篇文章开始之前，笔者先道歉。因为，不是桃子姐失手了，而是笔者复现错了。

主要有两个错误，第一个错误，是合成日频因子的时候，笔者用的是均值/标准差 ，而研报中用到的则是均值。

第二个错误，研报中的成交量用的是每笔成交量，而笔者使用的是每分钟成交量，因为笔者没有每笔成交的数据，所以只能用每分钟的数据来复现。

本文，复现的差分绝对值均值因子也是要用到每笔成交量的，但是笔者同样只能用每分钟的数据来复现，没想到的是结果居然还不错。

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume', 'turnover'])
    date_str = file_name.split('.')[0]

    vol = data.to_dataframe('volume')
    money = data.to_dataframe('turnover')
    vol = np.abs((vol.diff()).div(vol.mean(axis=0), axis=1))
    money = np.abs((money.diff()).div(money.mean(axis=0), axis=1))
    diff = money - vol

    res = [vol.mean(), money.mean(), diff.std(), diff.mean()]
    res = pd.concat(res, axis=1)
    res.columns = ['vol', 'money', 'diff_std', 'diff_mean']
    res['diff'] = res['money'] - res['vol']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

因子的计算是比较简单的，就是在差分标准差因子的代码第10行和第11行加上了绝对值。

其他的地方都没有任何改变。

虽然，改变只有一点点，但是从效果上来看却有很大的差别。

从相关性来看，vol和money这两个因子以及diff和diff\_mean这两个因子的秩相关性为1，因此在接下来的因子评价中，只展示vol、diff\_std和diff这三个因子的结果。

vol

这个因子用均值合成的表现可以说是非常不错的了，有三年的IC绝对值超过了0.1。

如果用标准差合成的话，那效果将大打折扣，IC绝对值最多也就勉强超过了0.03，同时除了2018年、2020年和2021年外，其他年份的IC绝对值都是负数。
