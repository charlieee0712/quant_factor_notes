本文，笔者要复现的因子来自长江证券的首席分析师覃川桃，这篇研报题为《基础因子研究（七）：高频因子（二），结构化反转因子》，发布于2019年6月1日。

传统的反转因子都只考虑了中短期的反转效应，但是实际上对于单个标的来说，在短期内反转效应和动量效率应该是都存在的。基于这一点，研报中提出了结构化反转这个因子。

从上面的描述，大家可以看出，这个因子的关键步骤就是如何划分动量效应和反转效应。

在研报的研究中，发现交易层面的反转效应会和成交活跃程度相关，成交量越低反转效应越弱，且到达一定阈值后转变为动量效应。

于是，就有了划分的办法，比如将成交量10%、15%或者是20分位数以下的时间段认为是动量时间，超过这个门限的则是反转时间。

对于动量时间，对所有这一时间的对数收益率按照成交量的倒数加权求和。

对于反转时间，对所有这一时间的对数收益率按照成交量加权求和。

最后，用反转时间的对数加权收益率之和减去动量时间的对数加权收益率之和，就得到了结构化的反转因子。

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume', 'close', 'open'])
    ret = np.log(data.to_dataframe('close') / data.to_dataframe('open'))
    vol = data.to_dataframe('volume')
    date_str = file_name.split('.')[0]

    thd = vol.quantile(0.1)
    flag = vol > thd
    rev_flag = np.where(flag, 1, np.nan)
    mom_flag = np.where(flag, np.nan, 1)

    rev_weights = vol * rev_flag
    rev_weights = rev_weights.div(rev_weights.sum(axis=0), axis=1)

    mom_weights = 1 / (vol * mom_flag)
    mom_weights = mom_weights.replace(np.inf, np.nan).replace(-np.inf, np.nan)
    mom_weights = mom_weights.div(mom_weights.sum(axis=0), axis=1)

    rev = (rev_weights * ret).sum(axis=0) - (mom_weights * ret).sum(axis=0)
    rev.name = pd.to_datetime(date_str) + timedelta(hours=15)
    return rev
```

核心代码从第10行开始。

第10-13行，通过成交量分位数将分钟级别的数据划分为反转时刻和动量时刻。

第15-16行，计算反转时刻的权重。

第18-20行，计算动量时刻的权重。

第22行，计算结构化反转。

这里需要注意的是，因子的合成方式，之前笔者常用的都是过去21日的均值、标准差以及均值除以标准差。但是，在这里，笔者觉得还有一种方式，就是对过去21个交易日的分钟数据按照成交量划分成两个不同的时刻，然后一起计算。

笔者在测试的时候尝试了这种方式，运行结束了，但是结果没保存。因为中间有点事情，所以也没注意报错信息，猜测可能是内存爆炸了。所以，这里就用成交量10%分位数和1分钟的数据来对这个因子的评价结果进行展示。
