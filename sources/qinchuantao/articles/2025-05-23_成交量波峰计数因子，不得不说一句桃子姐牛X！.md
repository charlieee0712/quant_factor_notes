本文复现的因子还是来自长江证券的研报《长江证券-基础因子研究（十四）：高频因子（九），高频波动中的时间序列信息》。

这个因子叫做成交量波峰计数因子，研报中对这个因子做出了这样的解释日内“放量”现象次数更多的股票可能有更多的趋势或知情交易者参与交易，进而在未来一段时间内有更高的收益率。

这个因子的计算步骤分为两步。

第一步，通过成交量的均值和标准差来标记局部峰值出现的时刻，即成交量大于均值+1倍标准差。

第二步，考虑到相邻的两个峰值应该算作同一个峰值，因此只能计数一次。所以，需要对连续大于均值+1倍标准差的进行处理。

最后，对剩余的局部峰值计数就得到了成交量波峰计数因子。

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume', 'turnover'])
    date_str = file_name.split('.')[0]

    vol = data.to_dataframe('volume')
    money = data.to_dataframe('turnover')

    vol_cnt = (vol >= (vol.mean() + vol.std()))
    vol_cnt = (vol_cnt == True) * (vol_cnt.shift(1) == False)
    money_cnt = (money >= (money.mean() + money.std()))
    money_cnt = (money_cnt == True) * (money_cnt.shift(1) == False)

    res = [vol_cnt.sum(), money_cnt.sum()]
    res = pd.concat(res, axis=1)
    res.columns = ['vol_cnt', 'money_cnt']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

这段代码的核心在第12行和第14行，这两行完成的是第二步的操作。

这里，笔者分别计算了成交量波峰计数和成交额波峰计数，然后看了以下结果，这两个基本上都是相等的，只有少数几个标的在某些交易日两者相差1。所以，我们可以认为这两个因子是完全一样的，因此在接下来的因子评价结果当中，也仅仅展示以下成交量波峰计数因子。

总结：这个因子使用均值合成的。如果用标准差的话，IC绝对值最高的一年也才刚刚超过0.03而已。这个因子可以说是在IC和分层回测上的表现都还是不错的，IC绝对值超过0.1的有三年，没超过0.08的只有2024年这一年。分层回测方面，IC值最小的一组收益率远低于其他四组，不过其他四组的净值曲线相交的情况有点多了。
