本文复现的因子来自长江证券覃川桃老师在2020年发布的研报《高频因子（八）：高位成交因子——从量价匹配说起》。

理论上，这个研报中应该是有多个因子的。但是，笔者没有看到研报的原文，只是从其他渠道获得了这篇研报中的部分因子，或者说是一个因子，它叫做成交额占比熵。

这个因子的计算是比较简单的，就是计算每分钟的成交额占比，然后计算熵。

这个因子背后的逻辑是，以成交额占比作为每一个时间段状态的发生概率，以熵的定义刻画成交的混乱程度，得到成交额占比熵因子；当因子值越小时，体系越混乱，成交额占比彼此之间存在较大分歧，成交在价格高位较为密集，易造成股价高估。

```python
def process_single_day(self, idx):
    # 加载当日分钟数据
    file_name = self.files[idx]
    full_path = os.path.join(self.file_pth, file_name)
    data = BaseDataLoader.load_data(full_path, fields=['volume', 'turnover']).to_dataframes()
    date_str = file_name.split('.')[0]

    vol = data['volume']
    vol = vol.div(vol.sum(axis=0), axis=1)
    vol = -1 * vol * np.log(vol)
    vol = vol.sum()

    amt = data['turnover']
    amt = amt.div(amt.sum(axis=0), axis=1)
    amt = -1 * amt * np.log(amt)
    amt = amt.sum()

    res = pd.concat([vol, amt], axis=1)
    res.columns = ['vol', 'amt']
    res['diff'] = res['amt'] - res['vol']
    res['datetime'] = pd.to_datetime(date_str) + timedelta(hours=15)
    res.index.name = 'code'
    return res.reset_index()
```

笔者在复现的时候，除了计算成交额占比熵之外，还计算了成交量占比熵以及两者之差。

上述代码中，第9行计算的是成交量占比，第14行计算的是成交额占比。

第10-11行，计算的是成交量占比熵。

第15-16行，计算的是成交额占比熵。

从相关性来看，成交额占比熵和成交量占比熵这两个因子的相关性是非常高的，这也符合了咱们的认知。同时，两者相减之后得到的差分因子与两者的相关性都不高，这一点也和之前复现的一些和成交量成交额有关的差分因子一致。

所以，在接下来的因子评价中，笔者将只展现成交额占比熵和两者差分的因子的结果。

成交额占比熵

这个因子用标准差合成的话，2018年、2020年和2021年这三年的IC绝对值不超过0.03。

如果用均值合成的话，2019年和2024年这两年的IC为正，其他年份的IC为负，不是一个稳定的因子。

因此，这里笔者展示一下用标准差合成的结果。

差分

这个因子用均值合成和标准差合成都比成交额占比熵这个因子要好。

用均值合成的话，除了2022年（超过0.04）和2024年（超过0.02），其他年份的IC都超过了0.06。同时，这是一个正向因子。

在分层回测方面，因子值最低的一组收益率远低于其他四组。但是，在其他四组当中，因子值最大的一组反而是收益率最低的，虽然低的并不是太多。

用标准差合成的话，2024年的IC绝对值依旧是最低的，但是也超过了0.06。
