# 方法论总纲 —— 把"读因子"变成"造因子·测因子·入库"

> 跨四本书(ML in Asset Pricing / Navigating the Factor Zoo / Market Microstructure / 因子投资) · 生成 2026-06-30

四本书指向同一条铁律：**成因（风险补偿或错误定价）重于样本内统计显著性**。低 p 值只是因子有效的必要非充分条件——样本内显著、样本外塌缩、讲不出经济故事的因子，多半是 p-hacking、多重检验与高维学习共同造出的伪发现。本总纲把"判真假 / 测因子 / 算成本"压成可执行索引，每个决策点深挖见对应专题文件。

## 三个决策点速查

**① scorecard 真假 → `02_Navigating_the_Factor_Zoo/_真假因子判据.md`**
为何多数"显著"因子是假的有两条独立来源：统计层面的 p-hacking / 多重检验 / 发表偏差（100 个独立因子里"最显著那个"t>2 的概率约 99%），与高维学习诱导的样本内可预测性（投资者实时收缩学习把"反应不足"印进收益，前向 OOS 期望为零）。八条判据逐条过、定 A/B/C：t 门槛抬到 3.0（严格档时序≥3.8 / Fama-MacBeth≥3.4）+ 多重检验 FDR/FDP 校正（金融别用 FWER/Bonferroni，太严）；经济逻辑一票否决；前向 + 反向 OOS 双关；对 FF5 与库内因子回归看增量 α（Buffett 六因子分解：控住 BAB/QMJ 等已知因子前 alpha 5.8%、之后只剩 0.3%——没控住已知因子前没有真 alpha）；与库内 |ρ|<0.8；计成本净收益正。**A**=八条全过，**B**=统计达标有短板（降权小仓监控），**C**=触任一红线（t<3.0 / 无逻辑 / 样本外失效 / 无增量 α / 冗余 / 净负）。两条一票否决：无经济逻辑、样本外（含反向 OOS）失效。对另类数据（概念热度/活跃度/人气）尤须警惕——当年投资者拿不到，等价"过度稀疏"，伪样本外为正也不宜直接归因 alpha。

**② 回测避坑 → `04_因子投资_方法与实践/_回测框架与检验方法.md`**
测因子比挖因子更易错——统计方法用错把噪声当显著，数据处理用错把未来漏进回测。统计核心：标准误一律 Newey-West（重叠收益致假显著）；单调性看 Spearman ρ_s 而非只看 High−Low 价差；合成先正交后合成护尾部。数据命门是 **PIT 财务对齐**——取数函数返回"决策日可见的最新记录"，只用首次披露值信息不足、只用最终修正值=用了未来数据回测虚高；配后复权（前复权时变）、单季度·TTM 派生、防前视的滚动 β。停牌按指标分流（动量可向前填充，波动/β/相关类必须置缺失，否则 0 收益假"无波动")。A股调仓剔涨跌停/一字板/停牌（不可成交）、遵守 T+1、壳价值剔最低 30% 市值（剔/不剔都报）、做空难则只做多头。稳健性必做多窗口（前/后十年/整段）+ 参数敏感 + 加权口径 + 双重排序去混杂。

**③ 成本 & L2 → `03_Market_Microstructure_in_Practice/_成本模型与L2理论.md`**
流动性不是固定 bps 滑点，而是随委托量上升的 round-trip 成本曲线 c(Q)。成本拆显性（佣金+印花税+半价差，线性 cᵢ）+ 隐性（市场冲击，平方根律 Δ≈Y·σ·√(Q/V) 的二次凸近似 dᵢ，按 ADV/换手分位分档）+ 时机成本（快冲高冲击 vs 慢拖高漂移的权衡），**进目标函数同步权衡 max f−γ_TC·TC，而非满仓后事后扣**；区分临时冲击（约 1 日内随弹性回补）与永久冲击（留存进持仓成本）——剔 alpha 与订单流自相关后元订单几乎不留永久冲击，长持有期别用单边冲击高估、误把 alpha 记成成本。微观结构噪声两面：做低频波动/相关时是估计偏差（signature plot 随采样发散、Epps 协方差塌缩）须去偏（按拐点设采样频率、异步用 Hayashi-Yoshida、VPIN 用成交量时钟），做 L2 因子时是 alpha 载体（Kyle's λ、PIN/VPIN、有效价差、OFI/盘口失衡、Hawkes 分支比、弹性）——根因是流动性溢价：低流动性资产要更高预期收益、却在压力期因流动性枯竭剧烈回撤。

## 四本书的方法论独特贡献（各一句话）

- **ML in Asset Pricing（Nagel）**：讲透"为何样本内≠真因子"——资产定价是低信噪比高维预测，正则化=贝叶斯先验=注入经济约束的通道；高维学习即便投资者理性也会印出样本内可预测、而前向（及反向）样本外消失。
- **Navigating the Factor Zoo**：给出因子族的科学框架——分族讲经济直觉+构造公式+实证，统一学界（可交易组合 λ=E[f]）与业界（公司特征 IC/ICIR）两套定义于 E[r]=α+βλ，并配去伪纪律辨真假因子。
- **Market Microstructure in Practice**：执行/流动性/订单簿机理——平方根冲击、临时 vs 永久冲击、价差-波动-活跃度近似关系 ψ≈a+bσ/√N、Hawkes 分支比与队列反应模型，是成本建模与 L2 因子的物理根基。
- **因子投资（石川等）**：最严谨的中文检验配方 + A股落地——以 E[Rᵢᵉ]=αᵢ+βᵢ′λ 统一因子/异象/多因子，给可复制实证工具链（排序/Fama-MacBeth/NW/GRS/正交化）与 A股制度铁律（涨跌停/T+1/壳价值/做空难）。

## 三个反复踩的坑·标准答案速查

- **√252 / 重叠收益年化**：重叠窗口机械制造正自相关，简单乘 √T 默许零自相关 → 低估标准误、虚高 t 与夏普 → 假显著。**标准答案 = Newey-West(HAC)** 标准误，s.e.=√(S/T)、滞后阶 J≥重叠期数；动量等长持有期更稳用**错开子策略**（拆成相位错开 1 期的非重叠子组合取均值，从根上回避重叠自相关）。
- **单调性判定**：用 High−Low 价差及其 t 值代表单调性是错的——价差只用两端两组，中间 8 组可能 U 形/驼峰仍有正价差。**标准答案 = 算收益排位与分组排位的 Spearman 秩相关 ρ_s + p 值**，ρ_s→±1 且显著才算单调；价差 t 与 ρ_s 两个都报（A股 BM 价差 t=1.85 弱、但 ρ_s=0.94、p≈5e-5 极强）。
- **合成稀释尾部**：相关<1 的子信号标准化后等权平均使分布向中心收缩、尾部变薄，而因子溢价恰集中在尾部。**标准答案 = 先正交后合成**：Gram-Schmidt 取各因子增量残差、层级合成（维内→维间）、合成后再标准化拉回 σ=1；同一维的子信号当一个因子用、别假装两个。

## 一页纸全流程 checklist：造因子 → 测因子 → 入库

**造因子**
- [ ] 先写经济故事（风险补偿 or 行为偏差），无故事不立项
- [ ] 数据 PIT：后复权 / 单季度·TTM / 缩尾 / 黑名单（防幸存者偏差）/ 剔不可交易股
- [ ] 多信号先正交化再层级合成、合成后标准化护尾部

**测因子**
- [ ] t 过严格档（时序≥3.8 / FM≥3.4），记搜索空间 N 并做 FDR/FDP 校正
- [ ] 标准误一律 Newey-West；单调性报 Spearman ρ_s + 价差 t 两个
- [ ] 对 FF3/FF5 + 库内因子回归看增量 α（GRS/α 检验）；与库内 |ρ|<0.8
- [ ] pre+post 样本外 + 反向 OOS；先验贝叶斯校正；多窗口/参数/placebo 稳健
- [ ] 成本进目标函数（线性 cᵢ + 二次 dᵢ、平方根冲击分档）算净收益；查 alpha 是否靠小盘空头 / A股做空可行性

**入库**
- [ ] 评 A/B/C：A 主力、B 降权小仓监控、C 淘汰或观察
- [ ] 记成因标签、IC/IR/rank IC/半衰期、相关性与拥挤度四代理（估值价差·配对相关·因子波动·因子反转），纳入 factor_lib
- [ ] 强相关因子群（多价值/活跃度/概念热度度量）宜 L2 收缩 / PC 化加权，而非 lasso 择一

## 一句话总评

四本书合起来给的不是因子清单，而是一条流水线——**先用经济逻辑立项、再以严格（含反向）样本外与成本净值检验去伪、最后按 A/B/C 入库**：MLAP 管"为何样本内不算数"、Factor Zoo 管"因子族与判真假"、Microstructure 管"成本与 L2 物理根基"、因子投资管"A股最严谨配方"，正好把"读因子"变成"造因子·测因子·入库"。
