## 本块讲什么

本块是第1章后半，把"因子"从直觉定义拉回到**资产定价的线性因子模型框架**里，核心是讲清三件事的"为什么"与"怎么做"：

1. **因子到底是什么**（§1.5）——同一个词"factor"/"alpha"在学界和业界含义完全不同；用 APT/SDF 把线性因子模型推一遍，说明为什么"可交易组合因子"的风险溢价 λ 恰等于其超额收益期望 E[f]。
2. **因子识别的标准流程**（§1.6）——学界两步法：portfolio sort（分组排序）造因子 + Fama-MacBeth 两段回归检验显著性；业界则用 IC / ICIR 评价。给出"真因子"的六条标准。
3. **共识因子谱系与业界因子分类**（§1.7–1.10）——CAPM→FF3→Carhart(动量)→+流动性→FF5 的演进逻辑（重点是它们作为"基准模型"如何用来判定新因子是否原创），以及业界因子的六大类、因子的生命周期、explainer(解释) vs predictor(预测) 两种世界观。结尾用 Table 1.3 做学界 vs 业界对照。

贯穿全章的主线：**业界 vs 学界对"因子/alpha/目标/检验/共识模型"的系统性差异**——这是全书反复强调的视角。

## 方法论 / 机理要点（重点·展开）

### 1. "因子是什么"——先把线性因子模型框架立起来（§1.5）

书强调：要调和学界与业界对"factor"的不同用法，必须回到资产定价理论。线性因子模型不是凭空造的，它与三件事**等价**：随机折现因子(SDF, stochastic discount factor)表示、均值-方差有效组合、以及在截面(cross-section)上对资产收益定价的线性因子模型。识别并设定因子模型对学界和资管业都至关重要，正因为此。

**数据生成过程（APT，Ross 1976）**：资产超额收益由

> r = E[r] + βν + e　(1.4)

驱动。其中 r 是 N×1 超额收益向量，β 是 N×K 的**因子暴露(factor exposures，又叫 factor loadings)**矩阵，ν 是 K×1 零均值的**因子创新(factor innovations)**，e 是 N×1 零均值特异误差(idiosyncratic errors)，且 E[e|ν]=0。

**截面定价方程**（APT 对期望超额收益的分解）：

> E[r] = α + βλ　(1.5)

λ 是 K×1 的**因子风险溢价(risk premia)**，α 是 N×1 的**定价误差(pricing errors)**。书原话：这条截面方程是"实证资产定价与因子投资的心脏与灵魂(heart and soul)"——它说明一只资产的期望超额收益由它对一组因子的暴露(β)与这些因子的风险溢价(λ)共同决定（加上一个误差项 α），从而解释截面上不同资产为何有不同的期望收益。

### 2. 识别因子的三条路线——学界 vs 业界 vs 统计因子（§1.5 核心区分）

理论框架不告诉你右边该放哪些因子。识别因子有两大类共三条路线：

- **路线A-学界（可交易组合）**：因子被当作**用公司特征(firm characteristics)构造出来的可交易组合**。例如价值因子 HML 用账面市值比(B/M)、规模因子 SMB 用市值(market cap)构造（Fama-French 1993）。此时"因子实现值"就是这些可交易组合的超额收益；"因子暴露"是用资产超额收益**对因子做回归**得到的系数。
- **路线A-业界（Barra 式）**：因子暴露被认为是**可观测**的。业界（如 MSCI Barra 模型）直接把公司特征（经必要标准化后）当作因子暴露 β，再用资产收益**对因子暴露做回归**反解出因子的风险溢价。
- **路线B-统计因子**：因子与暴露都是**潜在(latent)**的，用主成分分析(PCA)等统计技术识别（Chamberlain & Rothschild 1983；Connor & Korajczyk 1986）。

**这就是"factor"一词在两个圈子里的根本分歧**：学界口中 factor = 可交易组合（HML 这种"高 B/M 组合收益 − 低 B/M 组合收益"的多空价差），是真实可买卖的收益序列；业界口中 factor = 公司特征本身（虽未明说，实际充当线性因子模型里的因子暴露）。本书自始至终强调这个区分。

**本书为何选路线A（可交易组合特征因子）——两个理由**：
1. 金融文献已提出大量"用对应公司特征构造的可交易因子"，且各有**基于风险(risk-based)或行为金融(behavioral)**的解释；把它们分门别类、逐类详述，能让读者全面理解过去几十年学界发现的最重要、最主流的因子。
2. 统计因子模型概念上虽诱人（不要求因子可观测），但**很难准确估计**：资产收益数据样本小、信噪比(signal-to-noise ratio)低。更要命的是它**假设所有相关且被定价的风险因子都已反映在资产收益里并能被设定的因子捕捉**——当某些风险因子较弱、或市场整合度较低时该假设不成立，产生**遗漏变量问题(omitted variable problem)**（Giglio & Xiu 2021），从根本上削弱传统统计因子模型刻画真实风险结构的可靠性。

### 3. 机理深挖：为什么可交易因子的 λ = E[f]（SDF 推导）（§1.5 末–书 p.18）

这是本块最"硬"的方法论，回答"为什么这样建模"。当因子是可交易组合时，因子实现值写成总体均值与创新之和 f = E[f] + ν，可证其风险溢价 λ = E[f]。推导链条：

- **无套利 ⟺ SDF 存在**：定价误差 α=0（即(1.5)无定价误差）等价于存在形如 m = a − b′(f − E[f]) 的 SDF，满足基本定价方程 E[mr]=0。不失一般性归一化 E[m]=1（即 a=1），于是 m = 1 − b′(f − E[f])。
- **由 E[mr]=0 推出线性因子模型**：
  > 0 = E[mr] = E[m]E[r] − cov(r,f′)b = E[r] − cov(r,f′)b　(1.6)
  > ⟹ E[r] = cov(r,f′)b　(1.7)
  代入 β′ ≡ cov(r,f′)var(f)⁻¹：
  > E[r] = cov(r,f′)var(f)⁻¹ var(f) b = β′ var(f) b　(1.8)
  此处 **λ ≡ var(f)b 即因子风险溢价**。
- **再证 λ = E[f]**：把 f′ 乘到 m = 1 − b′(f − E[f]) 两边、取期望、转置，得
  > E[mf] = E[f] − var(f)b　(1.9)
  因 f 是可交易组合（代表超额收益），故 E[mf]=0，代入(1.9)得 E[f] = var(f)b ≡ λ。

**结论（机理要点）**：**可交易因子的风险溢价 = 其超额收益的期望值**，λ = E[f]，且 f = λ + ν。这正是"用可交易组合当因子"在数学上干净的根源——你想知道某因子的溢价，直接看这个多空组合的平均超额收益即可。

### 4. 学界 vs 业界：同一个模型，两种估计方式（§1.5 末–书 p.19，全书关键）

把(1.5)代回(1.4)并用 f=λ+ν，数据生成过程改写为 r = α + βf + e　(1.10)，单资产形式：

> r_it = α_i + Σ_k β_ik f_kt + e_it　(1.11)

K（因子数）远小于 N（资产数）。**α_i 是截距 = 因子未能解释/捕捉的那部分超额收益，学界称之为"误定价(mispricing)"**。注意这与业界的 alpha 定义截然不同——业界 alpha 指策略**跑赢市场基准的超额表现(outperformance)**。

- **学界估计法**：因子可交易、f 已知，故对每只资产做**时间序列回归**（用 t=1..T 把超额收益对因子实现值回归），系数即因子暴露，**截距即 α_i**。
- **业界估计法**：公司特征被当作因子暴露，(1.11) 改写为
  > r_it = γ_t + Σ_k β_ik f_kt + e_it　(1.12)
  注意截距**故意改记为 γ_t 而非 α**——强调它**不是定价误差**。(1.12) 是在给定时间 t 上、跨不同资产的**截面回归**：把资产超额收益对因子暴露(=公司特征)回归，估出的系数是**该时点的因子收益(factor returns)**。而**截距 γ_t 因对截面所有资产相同，不可解读为定价误差；真正的定价误差估计是残差 e_it**。

书的总结：尽管学界、业界对"factor"定义不同、估计方法不同，**实质差异没有看上去那么大**——两者都在找"能解释资产期望收益截面差异、并消除定价误差"的公司特征。因子投资的目标就是在核心公式 **E[r] = α + βλ** 下找到有用的因子；本书的任务正是把不同类别的因子呈现给读者。

### 5. 因子识别的标准流程（§1.6）

**先立警示**：线性因子模型太"通用"，似乎可以随心所欲往右边塞任何因子——Fama 戏称因子模型是"捕鱼执照(fishing license)"，讽刺多因子范式下的"因子挖矿"。为防止退化成纯数据挖掘、保证因子背后有实质经济学逻辑，**一个真且有效的因子应满足六条标准（Shi, Liu & Lian 2020，即本书中文版《因子投资：方法与实践》）**：

1. **强直觉性(intuitiveness)**——有来自风险或行为金融的逻辑根基；
2. **持续性(persistence)**——即便被公开披露后仍长期有效；
3. **增量预测力(incremental predictive power)**——提供其他因子未捕捉的独特视角；
4. **稳健性(robustness)**——对构造方法与参数不敏感，能通过各种稳健性检验；
5. **高可投资性(investable)**——能转化为交易成本可控的可执行策略；
6. **普适性(pervasive)**——跨资产类别、或在单一资产类内跨多国保持一致。

本书介绍的所有因子类别都满足以上标准。

**(a) Portfolio sort（分组排序法）——造因子**（以 SMB 为例，Fama-French 1992,1993）：
- 把全市场股票按某特征（此处市值）**升序排序**；
- 分成若干分位组(quantiles)，每组做**等权或市值加权**组合；最低组=最小市值股，最高组=最大市值股；
- **价差 = 小市值组收益 − 大市值组收益**（通常按月度量），即 SMB premium；
- 下一期**重新排序、重新平衡(rebalance)**，再算一次价差……如此滚动，得到 SMB premium 的**时间序列**。
- **现代做法**：通常用 10 个分位组(deciles)而非 3 组，价差 = 第10组收益 − 第1组收益（等权或市值加权），记为 SP_t。把 SP_t 放到(1.11)右边做时间序列回归估暴露。因为"先在截面上按特征排序、再分配到不同组合"，故学界称之为 **portfolio sort 法**。

**(b) Fama-MacBeth（1973）两段回归——检验因子有效性**（实证标准流程）：
- **第一段（时间序列，逐股）**：对每只股票 i，用 t=1..T 把超额收益 r_it 对 SP_t 回归，估 β_i，得到 N 个 β̂_i。
  > r_it = α_i + β_i SP_t + e_it　(1.13)
- **第二段（截面，逐时点）**：对每个时点 t，把全部股票超额收益对 β̂_i 做截面回归（i=1..N），得 λ̂_t，共 T 个。
  > r_it = γ_t + λ_t β̂_i + e_it　(1.14)，γ_t 为截距。
- **第三段（显著性）**：假设 λ̂_t 独立同分布(i.i.d.)，用 T 个观测算 t 统计量：
  > t_λ = λ̄ / (σ̂_λ / √T)　(1.15)
  其中 λ̄ = Σλ̂_t/T 是**因子平均收益**（因子风险溢价的估计），σ̂_λ = √[Σ(λ̂_t−λ̄)²/(T−1)] 是标准误。**大样本下要求 |t_λ| > 2.0** 才算显著。

**为什么第二段不直接用 SP_t 当因子溢价、而要绕一道截面回归？**——为了提取**单位因子溢价(unit factor premium)**，即一个对该因子暴露恰好为 1（beta=1）的组合的期望收益。跑截面回归(1.14)时，λ_t 正对应这样一个组合的因子溢价（见图1.1）。

**Fama-MacBeth race**：若有许多候选因子要挑最重要的，就对每个因子各做一遍排序得价差、各跑一遍 Fama-MacBeth，看谁的 |t_λ| 最大——这个搜索过程被非正式地称为"Fama-MacBeth 赛跑"。

### 6. 业界的因子评价：IC 与 ICIR（§1.6 末）

业界把因子理解成**公司特征**（如市盈率倒数 EP），而非排序构造的可交易组合。评价标准工具是**信息系数(Information Coefficient, IC)**：

> IC = corr(EP_{i,t}, R_{i,t+1})　(1.16)

即"本期特征值"与"下期收益"的横截面 Pearson 相关——衡量某因子(如 EP)是否能驱动未来收益。除累计 IC 曲线外，还算：

> IC_mean = (IC_1 + … + IC_{T−1}) / (T−1)　(1.17)
> IC_std = √[ 1/(T−2) · Σ_{t=1}^{T−1}(IC_t − IC_mean)² ]　(1.18)
> **ICIR = IC_mean / IC_std**　(1.19)

ICIR（IC 的信息比率）高，意味着平均 IC 高 且/或 其波动小——是业界判断因子"又准又稳"的核心单指标。

### 7. 共识因子谱系——重点是"基准模型"的方法论用途（§1.7）

书的笔法不是罗列因子，而是讲**模型演进的逻辑**与**基准模型如何充当"原创性裁判"**：

- **CAPM（最早、最有影响）**：市场期望超额收益是**唯一**解释截面差异的因子，E[r_i] = α_i + β_i E[r_m]　(1.20)。若 CAPM 成立，样本估出的 α_i 应在统计上与零无显著差异，收益完全由 β_i（对市场因子的**系统性暴露**）解释。**核心贡献=系统性风险概念**：并非所有风险都被定价，只有**不可分散的系统性风险**（证券收益与市场的协方差／市场方差）才获补偿——凸显分散化的作用，纠正了"高风险高回报"的笼统说法。市场因子是"全市场(market-wide)"的，对所有资产相同。
  - **BAB(Betting-Against-Beta) 异象**（Frazzini & Pedersen 2014）：低 beta 证券的风险调整后收益反而更好，**与 CAPM 预测矛盾**。策略=做多低 beta、做空高 beta，构造 beta≈0 组合，理论上无市场风险却能赚超过无风险利率的收益。解释有二：(i) **杠杆约束**——部分投资者不能/不愿用杠杆，为追求高收益就超配高 beta 股，推高其价格、压低未来收益；低 beta 股被低估，未来收益更高。(ii) **行为偏差**——投资者愿为"上涨时可能大赚"的高 beta 股多付钱，致高 beta 高估、低 beta 低估。

- **FF3（Fama-French 三因子）**：单因子太僵化。FF(1992) 发现规模与账面市值比能解释市场 beta 解释不了的超额收益；FF(1993) 正式提出 SMB(小减大)与 HML(高减低)。FF3 = CAPM + 规模 + 价值：
  > E[r_i] = α_i + β_i1 E[r_m] + β_i2 SMB + β_i3 HML　(1.21)

- **Carhart(1997) 四因子 = FF3 + MOM**：基于 Jegadeesh & Titman(1993) 的动量。MOM = 过去12个月赢家与输家的收益差，**跳过最近一个月**以避免序列自相关问题(serial correlation)。
  > E[r_i] = α_i + β_i1 E[r_m] + β_i2 SMB + β_i3 HML + β_i4 MOM　(1.22)

- **FF3 + MOM + LIQ**：Pástor & Stambaugh(2003) 的流动性因子，关联股票超额收益对前期订单流(order flow)的**反转(reversal)**。**反转越大→流动性越低→要求越高的风险溢价**。
  > … + β_i5 LIQ　(1.23)

- **FF5（Fama-French 2015）**：在 FF3 上加**盈利因子 RMW** 与**投资因子 CMA**。RMW=robust-minus-weak（高 vs 低经营盈利能力公司的收益差）；CMA=conservative-minus-aggressive（保守 vs 激进投资公司的收益差）。
  > E[r_i] = α_i + β_i1 E[r_m] + β_i2 SMB + β_i3 HML + β_i4 RMW + β_i5 CMA　(1.24)

**这些标准模型的方法论价值（关键）**：它们是判断"新识别的因子是原创、还是已有因子的复制品"的**基准**。流程是：新因子→做 portfolio sort 检验（排序、取顶部 decile 与底部 decile 收益差，逐期得到价差时间序列）→把该价差对某选定标准模型（如 FF3+MOM+LIQ）做**带截距的时间序列回归**→**若截距显著异于零**，说明价差未被现有模型完全解释，新因子具有一定有效性/原创性。

### 8. 业界因子的六大类（§1.8，非正式分类，转述 Table 1.3 类目）

业界因子定义不如学界严格、数量极多（即"因子动物园"factor zoo），既无共识也无标准分类。作者基于经验给出**非穷尽的六类**：

1. **宏观经济敏感度**：对利率、汇率、信用利差、商品价格等的敏感度。宏观变量是市场总体环境的"风险晴雨表"（如黑天鹅临近常伴随美元走强、信用利差走阔），股票对宏观变量的敏感度本身即因子。
2. **公司基本面**：资产负债表的杠杆率、利润表的财务比率等；由单个或多个财务比率构造的**评分**也可作因子，典型如 **Piotroski F-score**（综合多项指标评估财务健康度）。把基本面用于量化模型不同于只用基本面做决策的基本面分析师——由此衍生出"**Quantamentals(量化基本面)**"概念（第2章详述）。
3. **统计特性**：基于投资者风险偏好，资产定价理论关注风险厌恶如何反映在**统计矩(statistical moments)**中；回归与时间序列分析也能从数据提取信息。第3章讲统计矩如何作为因子。
4. **市场微观结构(microstructure)**：成交量价变动、换手率、高开低收价、流动性、订单流等；对**高频数据(tick 级、买卖价差 bid-ask spread)**的深入研究能反映买方 vs 卖方的动机，驱动短期收益。
5. **技术指标**：从简单均线、相对强弱指标(RSI)，到布林带、随机震荡指标等；意在识别趋势，可能含驱动收益的信息。基于形态的指标（通道突破、头肩形 head-and-shoulders）因信奉者一致行动可能**自我强化(self-reinforcing)**。
6. **行为金融**：题目宽泛、只能松散定义，多源于前景理论(prospect theory)、参照点依赖偏好(reference-dependent preference)、有限理性(bounded rationality)。第8章详述。

**业界新因子准入的实务门槛**：评估新因子时常检查它与库内**已有因子的相关性**，理想情况下不应与任何已有因子相关性超过某阈值（如 **80%**）。

### 9. 因子作为预测器：两点考量 + 生命周期（§1.9，explainer vs predictor）

这是本块对"两种世界观"的点题：**学界视因子为"解释器"(explainer)**——解释超额收益的截面差异；**业界视因子为"预测器"(predictor)**——因子是充当暴露的属性，目标是用它预测未来收益。要把因子当预测器用，必须考虑两点：

1. **因子有效期的生命周期(life cycle)**——大致四阶段：**发现期(discovery) → 使用期(usage) → 拥挤期(crowded) → 退出期(phase out)**。新发现时无人交易过，最有效；越多人发现越**拥挤**，收益逐渐消散；最终失效、被淘汰。书坦言：书中展示的是过去有效、现在仍有效的因子，但**当读者都开始用时它们就会变拥挤、效果减弱**；要持续跑赢市场，必须改进、更新这些因子并发现新因子。（呼应 §1.6 六标准里的"持续性"。）
2. **因子不普适(not universal)**——在某些市场有效不代表在别处有效；美国市场过去有效的因子不能简单移植到其他市场。市场效率、市场结构、交易摩擦、政府监管、市场参与者构成等都会影响。（对 A 股研究是直接警示：照搬美股因子需重新验证。）

### 10. 结论与 Table 1.3（§1.10）

因子投资的价值在于理解超额表现(outperformance)从何而来及其分解。全章反复对照两套语义，最后用 **Table 1.3** 收束（见下"图表"节）。再次强调：alpha 在学界=因子模型解释不了的截距（**高度依赖所选因子模型**），在业界=相对特定基准的超额表现；学界追求 alpha 统计上≈0（完美模型），业界追求 alpha 最大（最强超额）。因子动物园正是源于业界对因子的松散定义，导致因子池迅猛膨胀。

**核心公式速查**：
- 数据生成过程 (1.4)：r = E[r] + βν + e

## 与因子研究 & 回测的关联（构造/检验/成本/L2/A股）

- **构造（portfolio sort 是回测的标准管线）**：本块给出可直接落地的因子构造-检验流程——按特征截面排序→分 decile（现代用10组而非3组）→顶/底组多空价差→逐期 rebalance 得价差时间序列。这是因子库里"单因子回测"的标准骨架。等权 vs 市值加权是构造时必须明确的一个开关。
- **检验双轨**：学界用 Fama-MacBeth 两段回归 + |t|>2 判显著（注意第二段截面回归的截距 γ_t 不可当定价误差，残差才是）；业界用 IC/ICIR。**做 A 股因子库时两套都该实现**：Fama-MacBeth 给"是否被定价"的统计结论，IC/ICIR 给"预测又准又稳吗"的工程化单指标。
- **原创性判定（防止重复造轮子）**：新因子价差对 FF3/FF5/+MOM/+LIQ 等基准模型做带截距回归，**截距显著≠0 才算贡献增量**——这与业界"与已有因子相关性<80%"的准入门槛是同一目的的两种实现，回测框架里值得都做。
- **成本与可投资性**：六标准里的"高可投资性"提醒——价差再漂亮，若换手/冲击成本吃掉收益就无意义；动量因子"跳过最近一个月"正是处理序列自相关/微观结构噪声的工程细节。
- **L2/微观结构**：业界第4类因子（微观结构）明确点名 tick 级、买卖价差、订单流、换手——与本仓库的高频/Level2 数据与微观结构因子方向直接对应；订单流反转→流动性溢价(LIQ)的逻辑(Pástor-Stambaugh)可作 L2 流动性因子的理论锚。
- **A股适配（重要警示）**：§1.9 明确"因子不普适"——美股因子不能直接移植，需考虑市场效率、结构、交易摩擦、监管、参与者构成。叠加"生命周期"（公开即拥挤、逐渐失效），A股因子研究应把"在本土重新验证 + 持续迭代更新"当作默认动作，而非一次性照搬学界清单。
- **统计因子(PCA) 的取舍**：书指出统计因子受小样本、低信噪比、遗漏变量困扰——若在因子库里上 PCA/统计因子模块，要意识到其估计不稳，宜与可交易特征因子互为补充而非替代。
