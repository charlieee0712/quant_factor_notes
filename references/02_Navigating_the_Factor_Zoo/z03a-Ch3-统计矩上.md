## 本块讲什么

全章的核心命题：**资产收益可以看成某个分布的样本，而这个分布的"统计矩"（moments）本身就是天然的因子候选**。第一矩是均值，第二（中心）矩是方差，第三是偏度，第四是峰度。作者先用一套"效用函数 Taylor 展开"的推导，从理论上回答"为什么矩能当因子、投资者对各阶矩的偏好方向是什么"；再把四个矩的定义、样本估计量逐一讲清（含 n−1 自由度的道理）；最后把矩与真实股票收益挂钩，重点讲了两条已被大量实证支持的线索——**波动率（方差）的低波动异象**与**偏度（系统性 vs 特质性）的定价**，并引出只看下行/上行的"偏矩"（Partial Moments, LPM/UPM）。

一句话定位因子：作者开篇就说，因子的本质是"用变量去刻画资产收益的某种属性；若该属性能预测未来收益，它就是一个有效因子"。统计矩就是刻画收益分布属性最基础的一组变量。

## 方法论 / 机理要点（重点·展开）

### 1. 为什么统计矩能当因子？——效用函数的 Taylor 展开（全章理论基石）

这是本块最重要的方法论。作者不靠"经验上有效"来论证，而是从投资者效用出发推导出**投资者对每一阶矩的偏好方向**，从而说明各阶矩为何是合理的因子。

设每个投资者对非负收益 x 有效用函数 u(x)，围绕均值 x̄ 做 Taylor 展开到第四阶（式 3.1）：

```
u(x) = u(x̄) + u'(x̄)(x−x̄) + u''(x̄)(x−x̄)²/2! + u'''(x̄)(x−x̄)³/3! + u''''(x̄)(x−x̄)⁴/4! + …
```

两边取期望（式 3.2）。一阶项因 E[x−x̄]=0 而消失，于是：

```
E[u(x)] = u(x̄) + u''(x̄)·E[(x−x̄)²]/2! + u'''(x̄)·E[(x−x̄)³]/3! + u''''(x̄)·E[(x−x̄)⁴]/4! + …
```

逐项的经济含义（**这是"为什么"的关键**）：

- **第一项 u(x̄)**：均值本身带来的效用值——投资者当然偏好更高的期望收益（第一矩为正向偏好）。
- **第二项（方差，系数 u''(x̄)/2!）**：现代组合理论要求"给定期望收益下最小化方差"，因此系数 u''(x̄) 必须为**负**——即效用函数对风险厌恶者是**凹**的（concave）。投资者厌恶方差。
- **第三项（偏度，系数 u'''(x̄)/3!）**：刻画结果的不对称。金融文献发现投资者偏好"彩票型"结果（lottery outcomes）——以极小概率博取极大正收益的赌局，因此 u'''(x̄) 为**正**。投资者偏好正偏度。
- **第四项（峰度/尾部厚度，系数 u''''(x̄)/4!）**：度量分布尾部的"稠密程度"，即黑天鹅（blackswan）事件发生的可能性，也就是"尾部风险"（tail risk）。实证显示投资者**厌恶尾部风险**，故系数 u''''(x̄) 为**负**。（作者注：尾部风险近年越来越受重视，第7章专门讨论。）

**由此得到一条漂亮的普适规律**：投资者**偏好奇数阶矩**（u 的所有奇数阶导数为正），**厌恶偶数阶矩**（u 的所有偶数阶导数为负）。这就从理论上证成了"统计矩是良好的因子候选"。

> 机理直觉：奇数阶矩（均值、偏度…）是"方向性"的好东西——越大越好（更高均值、更右偏）；偶数阶矩（方差、峰度…）是"分散/极端性"的坏东西——越大越糟（更大波动、更厚尾）。

### 2. 历史脉络：从 MPT 到 CAPM，前两阶矩如何被"钉死"

作者用两个人物框（Markowitz、Sharpe）串起统计矩进入金融的里程碑：

- **Markowitz (1952) Portfolio Selection**：提出投资者只关心收益分布的**前两矩**——均值（衡量期望收益）与方差（衡量风险）。这奠定了 MPT 核心的**均值-方差优化**框架：以方差代理风险，问题就化为均值-方差优化。引申出的关键思想：
  - **有效前沿**（efficient frontier）：给定风险下能取得最高期望收益的组合集合；
  - **分散化**（diversification）：组合那些价格不完全同步变动的资产，可降低整体波动、提升潜在收益。
- **CAPM (Sharpe 1964, Lintner 1965, Mossin 1966)**：在 MPT 之上进一步巩固前两矩的地位。资产期望收益由其**系统性风险 beta** 决定，beta 本质上是"资产收益与市场收益的协方差，再用市场方差标准化"。beta 由此成为解释收益的关键因子。
- **Sharpe Ratio**：风险调整收益的标准刻度 =（组合收益 − 无风险利率）/ 组合标准差；比率越高越好。这又是一次"用第二矩（标准差）做分母"的实践。

> 这段的方法论价值：说明"用矩当因子"不是凭空发明，而是 MPT/CAPM 几十年理论与实践的自然延伸——前两阶矩早已是定价核心，三、四阶矩是顺理成章的扩展。

### 3. 四个矩的定义与"样本估计量"（含自由度的坑）

作者从随机变量讲起，逐阶给出**总体定义**与**无偏样本估计量**——做因子时真正要用的是后者。

**随机变量基础**：资产收益视为连续随机变量（取值不可数、无限）；若假设服从正态，则支撑集（support，密度>0 的取值范围）是整条实轴 R（−∞ 到 +∞）。离散用 PMF P(X=xᵢ)，连续用 PDF f(x)（单点概率为 0），统一可用 CDF F(x)=P(X≤x) 描述。

| 矩 | 总体定义 | 样本估计量 | 备注 |
|---|---|---|---|
| 一阶（均值 μ_x） | E[X]=Σ P(X=xᵢ)xᵢ（离散，式3.3）/ ∫ x f(x)dx（连续，式3.4） | x̄ = Σxᵢ / n （式3.5） | 隐含"每个历史样本等概率 1/n" |
| 二阶中心（方差 σ²_x） | E[(X−μ_x)²] | σ̂²_x = Σ(xᵢ−x̄)² / (n−1) （式3.6） | 分母 n−1，见下 |
| 三阶中心 | E[(X−μ_x)³] | n·Σ(xᵢ−x̄)³ / [(n−2)(n−1)] （式3.8） | 标准化后即偏度 |
| 四阶中心 | E[(X−μ_x)⁴] | n(n+1)·Σ(xᵢ−x̄)⁴ / [(n−3)(n−2)(n−1)] （式3.10） | 标准化后即峰度 |

**为什么方差分母是 n−1 而不是 n？**（实操常踩的坑）公式里用的是**样本均值 x̄（估计值）而非真实 μ_x**，因此损失了一个自由度——直觉上"x̄ 由这 n 个样本算出，约束了它们"。在 i.i.d. 假设下，用 n−1 才得到方差的**无偏估计**。

**标准化矩**（除以 σ 的幂，得到无量纲量）：

- **偏度**（skewness，式3.7）：skewness(X) = E[ ((X−μ_x)/σ_x)³ ]
- **峰度**（kurtosis，式3.9）：kurtosis(X) = E[ ((X−μ_x)/σ_x)⁴ ]

**奇/偶次幂的机理差别**（理解偏度 vs 峰度的关键）：

- 三次幂**保号**（sign-preserving）：高于均值的取值仍为正、低于均值仍为负；且高次幂会让**极端值主导**数值。故偏度衡量"哪一侧有极端尾巴"——正偏=右侧有尖尾（极端正收益有一定概率），负偏=左侧有尖尾（极端负收益）。对称分布偏度=0。
- 四次幂**正负同等对待**（都变正）：峰度衡量两侧极端值的稠密/厚度，与方向无关。高峰度=厚尾（fat tails）。以正态为基准：与正态同峰度=**mesokurtic（常峰）**；高于正态=**leptokurtic（尖峰/厚尾）**；低于正态=**platykurtic（低峰/薄尾）**。

> 作者坦言：四阶矩（峰度）的学术证据"既不丰富也不有力"，因此后文把实证重心放在**波动率（方差）**和**偏度**两条线上。

### 4. 方差（波动率）与收益：低波动异象（low-volatility anomaly）

这是把第二矩做成因子的代表性实证，**与"高风险高收益"的直觉相反**：

- **van Vliet & de Koning (2016)**：86 年美股数据（1929–2015）。
  - **因子构造**：用**过去 3 年月度收益的波动率**作为因子值；为剔除极小市值/低流动性干扰，每期**只取市值前 1000 只**，按因子值分 10 档（每档 100 只），**季度再平衡**（rebalanced quarterly）。
  - **结果**：最低波动档年化收益 **10.2%**，最高波动档仅 **6.4%**；考虑复利后这 3.8% 的年化差距，使低波组累计收益是高波组的 **18 倍**。波动率与收益呈**负相关**。
  - **重要 caveat**：跨各分档看，关系**并非单调**（not monotonic）——低波组显著跑赢高波组，但中间档位次序未必严格递减。
- **Blitz & van Vliet (2007)**：全球范围（1986–2006），在大市值股中"**做多低波、做空高波**"的策略年化约 **12%**，**t 统计量 3.0**。

> 方法论提炼：①低波动异象稳健且跨市场存在；②做因子时要先过滤极小市值/低流动性，否则被噪音污染；③评估因子别只看多空两端，要看**全分档单调性**——单调性弱意味着信号在中段不可靠。

### 5. 偏度与收益：系统性 vs 特质性，及"构造稳健性"陷阱

第三矩做因子的实证更微妙，核心是**区分两类偏度**与**构造方法对结论的颠覆性影响**：

**两类偏度（经济含义不同）**：

- **系统性偏度**（systematic skewness）：高系统性偏度的股票在市场下跌时更具韧性、提供下行保护——像"保险"，理应要求**风险溢价**（持有者愿付溢价 → 预期收益低）。
- **特质性偏度**（idiosyncratic skewness）：能独立于大盘交付巨大回报的能力，类似"彩票型暴富"（lottery-like windfall）。

**理论分歧**（取决于对投资者偏好的假设）：

- 一派认为**只有系统性偏度**该带风险溢价：Rubinstein (1973)、Kraus & Litzenberger (1976)、Harvey & Siddique (2000)。
- 另一派认为**总偏度（系统性+特质性）**都参与定价：Brunnermeier-Gollier-Parker (2007)、Barberis & Huang (2008)、Mitton & Vorkink (2007)。

**Bali, Engle & Murray (2016) 的实证（构造稳健性是重点教训）**：

- 行为金融假说：投资者追逐正偏度 → 把高偏度股**高估** → 其预期收益反而**低**。若成立，则"做空高偏度、做多低偏度"可获超额收益。
- 数据美股 1963–2012，对比**不同频率（日频 vs 月频）**、**不同回看窗口**、**等权 vs 市值加权**。
- **关键发现——结论高度依赖构造方式**：
  - 日频 + 1 个月偏度 + 等权分档：偏度与收益显著**负相关**；
  - 但改成**市值加权后，相关性反转** → 说明这种构造**不稳健**，很可能源于偏度的**高估计误差**（high estimation errors）。
  - 进一步考虑偏度的**持续性**、并在 **Fama-MacBeth 回归**中控制其他变量后，作者**推荐用月频收益 + 更长历史窗口**估偏度。此时偏度与预期收益**负相关**：例如用过去一年月度收益估偏度、市值加权、做多最低偏度做空最高偏度，**月均收益 0.34%，t=2.49**，在控制 Fama-French (1993) 三因子与 Carhart (1997) 动量后仍显著。
- **Langlois (2018)**：提出新方法**预测系统性/特质性偏度的横截面排名**（排名比绝对值更易预测）。预测出的**系统性偏度风险因子**带显著溢价 **7%–12%/年**，且对下行 beta、规模、价值、动量、盈利、投资等因子稳健；而**特质性偏度**的定价作用**不够稳健**。

> 方法论提炼（极其重要）：高阶矩的**估计误差大**，因子结论会因"频率/窗口/加权方式"而翻转。靠谱做法是——**月频、长窗口、市值加权、Fama-MacBeth 控制其他因子**，并检验信号**持续性**。日频+短窗+等权的"漂亮结果"很可能是估计噪音。

### 6. 偏矩（Partial Moments）：只看一侧的风险度量

普通方差对上行/下行**一视同仁**，但投资者真正怕的是下行。偏矩只针对相对某**阈值 t**（无风险利率或最低可接受收益）的一侧波动。

**下偏矩（Lower Partial Moments, LPM）**——只看跌破阈值的部分：

- **LPM0（零阶）**：收益跌破阈值的**概率**；
- **LPM1（一阶）**：低于阈值部分的绝对差之和 = **期望短缺**（expected shortfall）；
- **LPM2（二阶）**：低于阈值部分差的**平方和**，强调更大的偏离，对应**下行方差/半方差**（semivariance）。

通式（式 3.11）：

```
LPM_n = ∫_{-∞}^{t} (t − r)^n · f(r) dr
```

t=目标阈值，r=实际收益，f(r)=收益的概率密度，n=阶数。

- **LPM 的定价应用**：Bawa (1975)、Bawa & Lindenberg (1977) 构建**整合 LPM 的 CAPM**，可适配**任意概率分布**，涵盖传统模型且解释力至少相当，并能扩展为资本投资项目的决策准则；Fishburn (1977) 是"均值-风险分析"的奠基；Harlow & Rao (1989) 把均值-方差框架推广到含 LPM 并做实证检验。

**上偏矩（Upper Partial Moments, UPM）**——只看超出基准的正向部分：

- 度量相对基准的**上行/正向偏离**，隔离"好的"波动（机会而非风险）；
- 帮助评估跑赢基准的**频率与幅度**，可在优化中**倾斜配置**到上行潜力大的资产；
- 融入风险调整指标：**Sortino Ratio、Omega Ratio**（分母只放下行风险），更细致地刻画"投资者欢迎的好波动"；
- 适合**资本增值优先于保本**、偏好彩票型收益的非对称偏好投资者；也用于研究投资者乐观情绪、设计结构化产品/期权策略。

**相关文献群**：偏矩在组合选择（Jarrow & Zhao 2006、Brogan & Stidham 2008、Ling et al. 2014）与资产定价（Ang-Chen-Xing 2006、Anthonisz 2012、Kelly & Jiang 2014、Guastaroba et al. 2016）中受到广泛关注。Huang, Li & Yao (2022) 用 LPM/UPM 度量组合对基准的偏离、以一个**平衡参数**连接两者，构建增强指数与指数跟踪模型，回测优于传统模型，跟踪稳定性接近平均绝对偏差模型。

> 方法论提炼：当投资者效用非对称（怕亏不怕赚）时，**用半方差/LPM 替代方差**更贴合真实偏好；分子用收益、分母只放下行（Sortino/Omega）是把"低波动逻辑"做得更精确的方向。这与第5节"低波动异象"互为呼应——风险该用"下行"而非"总波动"来量。

**核心公式速查**：
- 效用 Taylor 展开（式3.1/3.2）：E[u(x)] 按"均值 + 方差项·u'' + 偏度项·u''' + 峰度项·u'''' + …"展开，偏好方向 = 奇正偶负。
- 均值（3.3/3.4）、样本均值 x̄=Σxᵢ/n（3.5）。
- 方差 E[(X−μ)²]，样本方差分母 **n−1**（3.6，无偏）。
- 偏度 E[((X−μ)/σ)³]（3.7）；三阶中心矩无偏估计 n·Σ(xᵢ−x̄)³/[(n−2)(n−1)]（3.8）。
- 峰度 E[((X−μ)/σ)⁴]（3.9）；峰度无偏估计 n(n+1)·Σ(xᵢ−x̄)⁴/[(n−3)(n−2)(n−1)]（3.10）。
- 偏矩 LPM_n=∫_{-∞}^{t}(t−r)ⁿf(r)dr（3.11）。

**图 3.1 Mean and Variance（均值与方差）**：两条钟形曲线，横轴 "Mean"（−4 到 4），纵轴 "Frequency"（0 到 0.4）。一条 **Standard Normal（标准正态，实线蓝）**峰值约 0.4、形态**窄而高**；另一条 **Wider Normal（更宽的正态，红色点划线）**峰值约 0.26、形态**矮而宽**。一条绿色虚线竖在 0 处标出共同均值。**传达**：两条分布**均值相同（都在 0）**，但**方差不同**——均值决定"中心位置"，方差决定"围绕中心的分散程度"；方差越大曲线越扁平、越外铺。

**图 3.2 Skew Asymmetry（偏度的不对称）**：三条曲线，横轴 "Value"（−4 到 4），纵轴 "Frequency"（0 到 0.7）。
- **Skewness = Zero（橙色实线）**：对称，居中于 0，峰约 0.4（即标准正态参照）。
- **Positive Skewness（正偏，蓝色虚线）**：质量集中在**左侧**（峰位偏负、约 −0.5，峰更高约 0.7），有一条**向右延伸的细长尾**。
- **Negative Skewness（负偏，绿色点划线）**：与正偏镜像，质量集中在**右侧**（峰位偏正、约 +0.5），**向左延伸的细长尾**。
- **传达**：**正偏 = 右尾**（小概率的极端大收益，"彩票型"）；**负偏 = 左尾**（小概率的极端大亏损）。这正是投资者"偏好正偏、厌恶负偏"的图像化。

**图 3.3 Kurtosis（峰度）**：三条均居中于 0 的曲线，横轴 "Value"（−4 到 4），纵轴 "Frequency"（0 到 0.5）。
- **Mesokurtic（常峰，黑色实线）**：正态参照，峰约 0.4。
- **Leptokurtic（尖峰/厚尾，红色虚线）**：**中心更尖更高**（峰约 0.5）、中段更窄，但**尾部更厚**（极端值概率更大）。
- **Platykurtic（低峰/薄尾，橙色点划线）**：**峰更矮更平**（约 0.3）、肩部更宽、尾部更薄。
- **传达**：峰度同时刻画"中心尖峭度 + 尾部厚度"。尖峰（lepto）意味更频繁的极端事件（黑天鹅风险大），低峰（platy）意味极端事件更少。

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / A股）

- **因子构造（直接可落地）**：
  - **波动率因子**：用"过去 N 期（如 3 年）月度收益波动率"为因子值；**务必先剔除极小市值/低流动性**（如只取市值前 1000），按分位分档，季度再平衡。低波档预期跑赢——A 股低波/低 beta 异象同样常见，可直接复刻该流程。
  - **偏度因子**：优先 **月频 + 长回看窗（如过去 1 年）+ 市值加权**；避免日频+短窗+等权（结论会因估计噪音翻转）。可进一步拆 **系统性 vs 特质性偏度**（系统性偏度的横截面排名更可预测、溢价更稳健）。
  - **下行风险因子**：用 **半方差 / LPM2** 替代普通方差，阈值取无风险利率或最低可接受收益；评价用 **Sortino / Omega** 而非 Sharpe，更贴合"怕亏不怕赚"的非对称偏好。
- **因子检验（方法论纪律）**：
  - 多空收益要配 **t 统计量**（书中范例 t=3.0 / t=2.49）；
  - 用 **Fama-MacBeth 回归 + 控制 FF3/动量** 看因子是否还有独立 alpha；
  - **看全分档单调性**（低波异象"非单调"的警示）——只看两端会高估因子质量；
  - 检验信号 **持续性（persistence）**，高阶矩尤需警惕**估计误差**。
- **成本视角**：季度再平衡换手相对可控；偏度/峰度类高频构造换手高、估计噪音大，需权衡交易成本。
- **理论锚点**：效用 Taylor 展开给出"奇正偶负"的偏好方向，可作为**因子符号的先验**（均值/偏度因子取正向、波动/峰度取负向暴露），用于约束多因子模型里的系数符号合理性。

## 术语与存疑

- moments = 矩；central moment = 中心矩；standardized moment = 标准化矩。
- skewness = 偏度；kurtosis = 峰度；mesokurtic 常峰 / leptokurtic 尖峰（厚尾）/ platykurtic 低峰（薄尾）。
- systematic vs idiosyncratic skewness = 系统性 / 特质性偏度。
- LPM = Lower Partial Moments 下偏矩；UPM = Upper Partial Moments 上偏矩；semivariance 半方差。
- Sortino Ratio / Omega Ratio = 只惩罚下行风险的风险调整收益指标。
- efficient frontier 有效前沿；MPT = Modern Portfolio Theory 现代组合理论；CAPM 资本资产定价模型；beta 系统性风险敞口。
