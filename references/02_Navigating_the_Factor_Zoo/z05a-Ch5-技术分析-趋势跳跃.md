## 本块讲什么

第5章开篇到 §5.2.1。主线不是"列技术指标"，而是回答一个方法论问题：**技术分析（technical analysis, TA）这种历史上靠"看图凭经验"的主观实践，如何被改造成可计算、可回测、可作为因子检验的对象**。本块覆盖三件事：

1. **TA 的本质与三阶段演化**——从画图 → 经验法则的实证检验 → 量化指标，最终与因子投资融合。
2. **图表形态（chart patterns）如何量化**——核心难题是形态"没有固定时间窗口"，需引入时间缩放（time-scaling）与去噪；并以 Lo-Mamaysky-Wang (2000) 的核回归去噪+条件收益检验作为"把形态科学化"的范例。
3. **动量指标族（momentum indicators）的统一构造范式**——SMA/EMA 的"冲击（shock）递归"视角，及 AO / PPO / RSI / Stochastic / TSI 的构造逻辑。

## 方法论 / 机理要点（重点·展开）

### 1. TA 的立论前提与它和基本面分析的分工

TA 的两条公理：① 市场价格已反映一切已知信息；② 价格运动的形态会随时间自我重复。因此 TA 只用**过去的市场数据（主要是价、量）**预测未来，与"基于经济/财务因子评估内在价值"的基本面分析（fundamental analysis）形成对照。做技术分析的人（chartists）依赖图与工具识别形态和趋势。

> 对因子研究的含义：TA 因子天然是**价量类、纯市场数据驱动**的因子，不依赖财报，因而更新频率高、可下沉到日内/L2，是 quantamental 框架里与基本面因子互补的一支。

### 2. 文献谱系给出的"立场坐标系"（做 TA 因子前要知道自己站在哪）

作者用一组经典文献把 TA 的合法性争论摆出来，这本身是方法论态度的校准：

- **Murphy (1999)**《Technical Analysis of the Financial Markets》——实务集大成（图表类型、趋势识别、技术指标）。
- **Aronson (2006)**《Evidence-based Technical Analysis》——批判性、实证驱动，主张用**科学方法 + 统计推断验证交易信号**，反对不加检验地接受 TA。**这是本章方法论的直接前身**。
- **Malkiel (1973)**《A Random Walk Down Wall Street》——怀疑派，价格随机游走、不可由历史数据可靠预测。
- **Schwager (1989)**《Market Wizards》——访谈式轶证。
- **Lo & Hasanhodzic (2010)**《The Evolution of Technical Analysis》——历史视角，强调 TA 是"市场心理 + 行为经济学"的复杂交互，不只是启发式规则集。
- **Lo (2017) 适应性市场假说（Adaptive Market Hypothesis, AMH）**——TA 的韧性源于它能整合心理因素、随市场动态调整。

**AMH（方框）的方法论价值**：它把市场效率从 EMH 的"恒定、价格完全反映信息"改成**随时间变化的状态**——参与者用启发式/经验法则不断适应环境，效率高低随之波动。AMH 借用进化生物学思想。其推论是：**一个 TA 信号的有效性不是恒定的，可能在某些市场状态有效、另一些状态失效**——这正是后文"为何要做条件收益、要分市场状态检验"的理论依据。

### 3. Caveat：把 TA 做成因子前必须先躲开的方法论陷阱（最重要）

作者专辟 §5.1.2 警告，全是回测踩坑点：

- **确认偏误 / 选择性记忆（confirmation bias / selective recall）**：人会记住"应验的预测"、忽略失败案例，从而高估形态的有效性。形容其近似"占星术"。
- **过拟合（overfitting）**：总结形态的方法易过拟合，"类似某些深度学习模型"——在**不理解趋势背后成因、不理解投资者趋势后如何反应**的情况下硬套指标。
- **多数形态缺乏逻辑基础或严格实证支撑**，加上主观性强，导致 TA 未达到基本面分析那样的学术严谨度。
- **科学验证 TA 难，难在两点**：
  1. **形态本身有缺陷**：上涨股票或许有相似特征，但**符合该技术指标的股票里，更大比例可能并不上涨、甚至暴跌**；若据此构建组合，**整体业绩可能为负——即便其中部分个股确实涨了**。（这是典型的"信号有正例但整体期望为负"的陷阱。）
  2. **极端波动**：策略在正常市况有效，但极端波动可能造成重大损失，**侵蚀掉长期收益**。
- **正确做法**：不要因为某指标"被广泛使用且历史可靠"就采纳；要**科学评估收益来源、理解指标所反映的市场逻辑**。

> 回测启示：TA 因子检验必须做 ① 全样本而非挑出来的正例（防选择性记忆）；② 样本外/分期/分市场状态（呼应 AMH，防"只在某状态有效"被误当普适）；③ 关注尾部/极端波动下的回撤；④ 给出经济机理解释，而非纯数据挖掘。

### 4. TA 的三阶段演化——"可量化（quantifiable）"是关键跃迁

作者把 TA 发展分三段，正好对应"主观→客观→可建模"：

1. **画图阶段**：point / line / bar 图，目的是从海量数据里**视觉提取趋势与形态**。
2. **实证检验阶段**：算力与数据质量提升后，可以**实证检验这些"前计算机时代、靠经验总结"的传统规则是否真有预测力**。
3. **量化阶段（1950s–60s）**：开发**技术指标（technical indicators）**——用历史价（含日内）和量构造 RSI、MACD、Stochastic Oscillator 等，用于识别超买/超卖和潜在趋势反转。"可量化"使 TA 成为计算机技术的自然延伸。

**最新一步：与因子投资融合**——把 TA 与基本面结合，开发同时利用两者的量化方法。**本章目标即：把技术指标当作因子来分析其有效性。**

### 5. 图表形态量化的核心障碍：没有固定时间窗 → 时间缩放 + 去噪

这是本块方法论的精华，也是"形态"区别于普通因子的根本难点：

- **普通因子有固定时间窗**（如12月动量），而**图表形态没有**：一个头肩顶（HS）可能20个交易日成型，也可能200个交易日才成型。
- 因此必须用**时间缩放（time-scaling）方法**：用不同时间窗去缩放价格趋势，配合数学/模式识别技术来定位形态。
- **关键点（如 HS 的肩、三角形/矩形的趋势线）位置主观**，而股价含大量噪声尖峰，计算机直接识别这些点很难。
- 未来方向：**基于深度学习的图像识别算法**有望用于识别技术形态/指标。

### 6. 范例：Lo, Mamaysky & Wang (2000)——把"看图"变成可检验科学

这是全章"如何把主观形态做成可回测信号"的标杆方法，分三步：

1. **去噪**：用**非参数核回归（nonparametric kernel regression）**平滑价格噪声，得到光滑价格序列。
2. **自动识别**：在光滑序列上，技术形态可被**自动化识别**（绕开"尖峰让关键点难定位"的问题）。
3. **条件收益检验（construction → test）**：长样本 1962–1996，对 10 种技术形态逐一比较——**识别到形态之后的"条件收益"（conditional returns）与"无条件收益"（unconditional returns）是否存在显著差异**。结果：**10 种形态里有 7 种**条件收益与无条件收益**显著不同**；以 Nasdaq 为样本，对技术指标提供了实质性支持。

> 这套"去噪 → 自动识别 → 条件 vs 无条件收益对比"是把任何主观形态因子化、可检验化的通用配方，可直接迁移到 A 股形态因子研究。

### 7. 技术指标四大类的划分逻辑（§5.2 框架）

作者按"用什么数据 / 测什么"把技术指标分四类，这是构造因子时选料的导航：

| 类别 | 用什么数据 | 测什么 |
|---|---|---|
| 动量指标 momentum | 历史价格 | 持续性趋势的强弱 |
| 成交量指标 volume | 价 + 量 | 用量验证价的方向（更清晰的方向判断） |
| 趋势指标 trend | 不同时间跨度的价格运动 | 短/中/长期趋势 |
| 波动率指标 volatility | 价格 | 某窗口内价格运动的离散度/区间 |

**对"动量"一词的三重语义辨析（重要）**：① 学术（Carhart 1997 四因子 FF3+MOM）= 过去12个月（通常剔除最近1个月）赢家与输家的收益差（截面动量）；② 业界 = 3/6/12月区间的相对收益；③ TA 语境 = **反映"趋势延续"的一类指标，用某种移动平均（±量）刻画**。本章讲的是第③种。

### 8. 动量指标的统一构造范式：SMA / EMA 的"冲击（shock）递归"视角（核心抽象）

作者没有把指标当孤立公式，而是抽出**移动平均的递归更新结构**，所有平滑类指标都建在其上：

- **SMA（简单移动平均）递归**：来一个新点 P_{t+1}，新均值 = 旧均值 + **冲击 / (n+1)**，其中**冲击（shock）= 新数据 − 旧 SMA**。即新信息以权重 1/(n+1) 修正旧估计；**SMA 框架下，冲击与其他数据点等权**。
- **EMA（指数移动平均）递归**：新均值 = 旧均值 + **(2/(n+1)) × 冲击**。**相比 SMA，EMA 用两倍的冲击权重更新最新估计**，因此对新信息更敏感、更"跟手"。EMA 只能递归计算，初值 EMA₀ 可设为 P₀ 或 P₀…P_t 的均值来起步。

> 这个"shock 权重"抽象很有用：SMA 滞后大、稳；EMA 反应快、噪声也大。**构造平滑类因子时，平滑核的"冲击权重"本质上就是在调收益的"反应速度 vs 抗噪"权衡**——这是参数选择的统一旋钮。

### 9. 各动量指标的构造（只取构造法 + 逻辑，不堆名字）

- **Awesome Oscillator (AO)，Bill Williams**：两条 SMA 之差（常用 34日 与 5日），作用对象是**中位价（median = (当日最高 + 当日最低)/2）**。本质＝快慢均线差，度量动量、确认趋势或提示反转。
  - 方框（Bill Williams）：把混沌理论用于市场；核心概念"交易分形（Trading Fractal）"——**同一形态在不同时间尺度上重复出现**（呼应 §5 形态"时间缩放"的思想）；其指标家族还有 Alligator、Accelerator Oscillator (AC)、Market Facilitation Index (MFI，用价格变化对比量来测价格运动效率)。
- **Percent Price Oscillator (PPO)**：两条 EMA 之差（常用 12日 与 26日）**再除以 26日 EMA**。相对 MACD 的关键差别是**做了归一化（除以慢线）**，使不同价位/不同标的的读数可比。
- **Relative Strength Index (RSI)，Welles Wilder (1978)**：度量回看期（常用 9 或 14 日）内**近期涨幅相对跌幅的量级**，输出限定在 0–100，用于识别超买/超卖。构造要点是**把"平均涨/平均跌之比"压进有界振荡器**。
  - 方框（Welles Wilder，机械工程师出身→系统化）：还创了 Average True Range (ATR，波动率)、Parabolic SAR（趋势跟随的止损/反转）、Directional Movement Index (DMI = ADX + +DI + −DI，测趋势强度与方向)。
- **Stochastic Oscillator (SO)，George Lane（1950s 末）**：测**当前收盘价在回看期（常 14 日）高–低区间中的相对位置**，输出 0–100。经济直觉：**上涨时收盘趋向区间上沿，下跌时趋向下沿**。实操含两条线——%K（当前价相对高低区间）与 %D（%K 的移动平均，作信号线）；>80 超买、<20 超卖。这是"区间归一化"思路的代表。
- **True Strength Index (TSI)，William Blau**：对**价格变化（PC = 当前价 − 前一价）做双重平滑**（两层 EMA，常 13 与 25 日），再用**双重平滑的 PC ÷ 双重平滑的 |PC|** 归一。双重平滑＝在抑制噪声/假信号与保留动量方向之间取平衡。
  - 方框（William Blau）：还有 Stochastic Momentum Index (SMI，把 Lane 的 SO 改成**以零为中心的区间**以降波动、减假信号)、Double Smoothed Stochastic (DSS)。

> 跨指标的方法论共性：**(a) 平滑（SMA/EMA，单层或双层）压噪；(b) 取差或取相对位置生成动量信号；(c) 归一化（除以慢线 / 除以高低区间 / 除以绝对值）使读数有界、可跨标的比较**。做自定义动量因子时，这三步就是积木。

## 关键公式 · 图表 · 定义

### 公式

- **(5.1) SMA 定义**：`SMA_t(n) = (P_t + P_{t-1} + ... + P_{t-n+1}) / n`
- **(5.2) SMA 递归更新**（来新点 P_{t+1}）：
  `SMA_{t+1}(n+1) = SMA_t(n) + (P_{t+1} − SMA_t(n)) / (n+1)`
  — 新均值 = 旧均值 + 冲击/(n+1)，冲击 = P_{t+1} − SMA_t(n)。
- **(5.3) EMA 递归更新**：
  `EMA_{t+1}(n+1) = EMA_t(n) + (2/(n+1)) × (P_{t+1} − EMA_t(n))`
  — 冲击权重 2/(n+1)，是 SMA 的两倍。
- **(5.4) RSI**：`RSI = 100 − 100 / (1 + 平均涨幅/平均跌幅)`，涨/跌幅为回看期内正/负收益的均值。
- **(5.5) Stochastic Oscillator**：`SO = (当前收盘 − 最低价) / (最高价 − 最低价)`，最高/最低取回看期内日内极值。
- **(5.6) TSI**：`TSI = 双重平滑 PC / 双重平滑 |PC|`；PC = 当前价 − 前价；第一层 = PC 的 25 期 EMA，第二层（双重平滑）= 第一层的 13 期 EMA；分母对 |PC| 做同样两层 EMA。
  - [符号存疑] 文中"first smoothed = 25-period EMA，double smoothed = 13-period EMA"，与前文"两 EMA 常用 13日与 25日"的先后顺序一致，但快慢命名以原文为准。

### 图表转述（§5.1.3 七类形态）

- **Figure 5.1 头肩顶 HS**：纵轴 Price、横轴 Time。锯齿线先升到一个较低峰（左肩 Shoulder），回落到水平虚线（颈线 Neckline），再升到最高峰（头 Head），回落到颈线，再升到与左肩约等高的右肩，随后下跌**向下击穿颈线（向下箭头）**。信号：**牛→熊反转**。两肩应出现在相近的成交量水平。
- **Figure 5.2 头肩底 IHS（Figure 5.1 的镜像）**：价格先跌到较高的左肩谷，反弹到颈线，再跌到最深的头谷，反弹回颈线，再跌到较高的右肩谷（第三个更浅），随后**向上突破颈线（向上箭头）**。信号：**熊→牛反转**。
- **Figure 5.3 扩张顶/底 BT & BB**：两幅"喇叭口（megaphone）"。震荡**振幅逐渐放大**，由两条**发散的虚线趋势线**（上线上行、下线下行）包络。**Broadening Top**：多次小反转后**大幅下跌**（看跌）；**Broadening Bottom**：镜像，多次小反转后**大幅上冲（向上箭头）**（看涨）。
- **Figure 5.4 三角形（延续形态）**：四幅。**Ascending 上升三角**：上沿水平、下沿上行，收敛于右侧顶点，向上突破（箭头）。**Descending 下降三角**：上沿下行、下沿水平，向下突破。**Symmetrical 对称三角**：上下沿对称收敛于顶点，给出两版（一版向上突破、一版向下突破）。规则：**放量向上/向下突破 = 看涨/看跌**，趋势线在右侧顶点相交。
- **Figure 5.5 矩形（整理形态）**：两幅，价格在两条**平行水平虚线**（支撑/阻力）间横盘整理。**Uptrend**：整理后**向上突破（箭头）**，原上升趋势续行；**Downtrend**：整理后**向下突破**，下跌续行。
- **Figure 5.6 双顶/双底**：**Double Top（"M"形，看跌）**：升到一峰，温和回落到颈线（虚线），再升到约等高的第二峰，随后**跌破颈线（向下箭头）**；确认条件＝价格跌破等于两峰间最低点的支撑位。**Double Bottom（"W"形，看涨）**：镜像，两个约等深的谷后**向上突破颈线（箭头）**。

> 形态语义小结：HS/IHS、双顶/双底、扩张顶/底属**反转形态**；三角形、矩形属**延续形态**。所有形态的"确认"都落在**对颈线/区间边界的突破**这一可编码事件上——这正是把形态写成布尔信号的抓手。

## 与因子研究 & 回测的关联

- **构造**：动量指标族可直接因子化，统一积木＝平滑（SMA/EMA，单/双层）→ 取差或相对位置 → 归一化（除慢线/除高低区间/除绝对值）。"冲击权重"(SMA 1/(n+1) vs EMA 2/(n+1)) 是调"反应速度 vs 抗噪"的统一旋钮。
- **形态因子化**：照搬 Lo-Mamaysky-Wang (2000) 三步——**核回归去噪 → 光滑序列上自动识别形态 → 条件收益 vs 无条件收益显著性检验**；"突破颈线/区间边界"是把形态写成事件信号的天然触发点；无固定时间窗 → 必须做**多时间窗缩放（time-scaling）**。
- **检验纪律（来自 Caveat + AMH）**：全样本而非挑正例（防确认偏误/选择性记忆）；样本外 + 分期 + 分市场状态（AMH：有效性随状态变）；查尾部/极端波动回撤（防长期收益被极端行情侵蚀）；警惕"有正例但整体期望为负"的信号；要求经济机理解释而非纯挖掘。
- **A 股 / L2**：TA 因子纯价量驱动、不依赖财报，更新快、可下沉到日内与 L2；SO/TSI 等用到日内最高/最低，与 L2 高频数据契合；深度学习图像识别识别形态是可探索方向。

## 术语与存疑

- **chart pattern 图表形态**：HS/IHS（头肩顶/底）、Broadening Top/Bottom（扩张顶/底）、Triangle（三角，延续）、Rectangle（矩形，整理）、Double Top/Bottom（双顶/底）。
- **technical indicator 技术指标**：可量化的价量函数；四类＝momentum / volume / trend / volatility。
- **SMA / EMA**：简单/指数移动平均；**shock 冲击**＝新数据与旧均值之差。
- **AO / PPO / RSI / Stochastic Oscillator(%K,%D) / TSI / SMI / DSS / ATR / Parabolic SAR / DMI(ADX,+DI,-DI) / MFI**：本块出现的指标名（多数仅点到构造或归属，未展开公式）。
- **conditional vs unconditional returns 条件/无条件收益**；**nonparametric kernel regression 非参数核回归**；**time-scaling 时间缩放**；**AMH 适应性市场假说**；**confirmation bias / selective recall 确认偏误/选择性记忆**。
- **覆盖存疑**：文件名标"趋势/跳跃"，但本页区间（书 p.93–107）只到 §5.2.1 动量指标；**趋势因子近期进展（§5.3）与"跳跃 jump"应在后续块**，本块未涉及。
- **[符号存疑]** (5.2) 原书写作 `SMA_{t+1}(n+1)`（窗口由 n 变 n+1，即"多纳入一个点"的更新视角），与常见"固定窗口滚动"写法略不同；(5.6) TSI 快慢 EMA 的命名顺序以原文为准。
