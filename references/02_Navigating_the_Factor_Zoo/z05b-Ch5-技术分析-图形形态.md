## 本块讲什么

第 5 章前半部分（5.1 技术分析的本质 + 5.1.3 关键组件）。核心是把传统**图形形态（chart patterns）**——头肩、倒头肩、扩散顶/底、三角、矩形、双顶/双底——当作"可量化、可回测"的对象来审视：

1. **技术分析的定位与文献版图**（5.1.1）：从实务派到循证派再到随机游走质疑派的谱系。
2. **审慎告诫 / 机理批判**（5.1.2）：为什么图形形态"看着灵、其实难证"——确认偏差、过拟合、缺逻辑根基。
3. **六大形态逐图**（5.1.3，图 5.1–5.6）：每个形态的几何外形、多空含义、确认条件。
4. **从"肉眼规则"到"可计算因子"的桥**：形态没有固定时间窗（20–200 日不等）⇒ 需要时间缩放（time-scaling）；Lo-Mamaysky-Wang(2000) 用**非参数核回归（nonparametric kernel regression）平滑价格噪声后自动识别形态**，再做**条件收益 vs 无条件收益**检验。

这是全书少数"先泼冷水、再给量化方案"的一段，方法论价值高于形态本身。

## 方法论 / 机理要点（重点·展开）

### A. 技术分析的两条前提与作者立场
- **前提**：① 价格已反映全部已知信息；② 价量形态会重复出现。分析师（chartist）靠点/线/柱状图（point / line / bar charts）从海量数据里"肉眼抽取"趋势与形态。
- **作者立场是审慎、辩证的**：承认部分指标确实长期可用，但反复强调要"科学评估收益来源、理解其反映的市场逻辑"，而非把形态当占卜。

### B. 文献版图（5.1.1，谱系比清单更重要）
- Murphy (1999) *Technical Analysis of the Financial Markets*：实务全书（图表类型、趋势识别、指标），新手到老手的基础参考。
- Aronson (2006) *Evidence-based Technical Analysis*：**循证派**，用科学方法 + 统计推断去验证交易信号，是对"无批判接受 TA"的对冲。
- Malkiel (1973) *A Random Walk Down Wall Street*：**质疑派**，价格近似随机游走，TA 不可靠地预测不了。
- Schwager (1989) *Market Wizards*：访谈式轶证，侧重交易心理与方法论。
- Lo & Hasanhodzic (2010) *The Evolution of Technical Analysis*：TA 史（从巴比伦泥板到 Bloomberg；荷兰/日本蜡烛图、道氏理论 Dow Theory），把 TA 看作市场心理 + 行为经济学的复杂交互。
- Lo (2017) **适应性市场假说（Adaptive Market Hypothesis, AMH）**：市场效率不是静态的，会随参与者用启发式规则不断适应而变化（演化生物学视角）；这恰好解释了 TA 为何"打不死"。

### C. 审慎告诫 / 为什么形态"难学术化"（5.1.2，机理核心）
- **类比占卜**：新手满市场找形态，类似"日食预示厄运"的星象思维。
- **确认偏差放大错觉**：感知到的有效性常被"选择性记忆（selective recall）/ 确认偏差（confirmation bias）"夸大——记住命中、忽略落空（行为金融，第 8 章展开）。
- **过拟合风险**：把形态归纳成规则，若不理解"为什么"以及投资者在趋势后如何反应，就像某些深度学习模型一样**过拟合**。
- **缺逻辑根基**：很多被贴"看涨/看跌"标签的形态没有逻辑基础或严谨实证，所以 TA 没达到基本面分析那种学术严谨度。
- **两条难以科学证实的原因**（重点）：
  1. **形态本身有缺陷**——不保证长期盈利。即便上行股有相似特征，**更大比例"符合"该形态的股票可能并不上涨、甚至暴跌**；据此选股的组合整体可能为负（尽管个别赢家在涨）。
  2. **极端波动会吞噬收益**——常态下有效的策略，遇极端行情可致重大损失、抹平长期收益。
- **结论性提醒**：与其问"哪个指标能用"，不如**科学分析收益来源、理解形态反映的市场逻辑（供需/认知）**。→ 这一条与本目录 `_真假因子判据.md` 的"真假因子"主旨直接呼应。

### D. 关键组件（5.1.3）
- **头肩（HS）= 最被认可的形态**：一个"头"（峰）夹两个略低的"肩"（峰），且最好出现在**相近的成交量水平**；跌破**颈线（neckline，连接两肩之后那些低点的支撑线）**确认"牛转熊"。
- **支撑/阻力（support / resistance）**：因需求/供给集中而易令价格停滞或反转的阈值。
- **成交量（volume）= 基石**：为价格动作背后的"信念强度"提供旁证（放量突破才算数）。
- 形态分析常与 RSI、移动平均等**指标配合**判断动量与趋势强度。

### E. 从"肉眼规则"到"可计算因子"的桥（5.1.3 末 + p.102，★方法论重点）
- **难点 1：无固定时间窗**。一个头肩可能 20 个交易日成形，也可能拖到 ~200 日 ⇒ 不能像定窗指标那样直接算，需**时间缩放（time-scaling）**：用多种时间窗去缩放价格趋势 + 模式识别算法。
- **难点 2：关键点主观 + 价格噪声**。肩/头、三角与矩形的趋势线，其"关键点"位置很主观；价格又多尖峰（spikes），计算机算法更难定位这些点。
- **Lo-Mamaysky-Wang (2000)《Foundations of technical analysis》的桥接做法**：
  1. 用**非参数核回归（nonparametric kernel regression）平滑价格序列**，滤掉噪声；
  2. 在**平滑后的序列**上**自动**识别形态（局部极值 ⇒ 肩/头/顶/底；拟合趋势线），把"肉眼规则"算法化；
  3. **检验设计 = 条件收益 vs 无条件收益（事件研究式）**：识别到某形态之后的收益分布，与无条件收益分布做对比。
  4. **实证结果**：样本 1962–1996、用 **Nasdaq**，**10 种形态中 7 种**的条件收益**显著异于**无条件收益 ⇒ 对技术形态有"实质性支持"。
- **未来方向**：随着**深度学习图像识别**成熟，用图像算法自动识别形态有良好前景。

### F. 技术分析的演化阶段（贯穿全节）
1. **视觉形态阶段**：成形于计算机之前的"经验法则"，基于图表外形（即本篇 5.1–5.6）。
2. **可量化阶段（书中称"第三阶段"）**：1950–60 年代起用历史价量造**技术指标**（RSI、MACD、随机指标 Stochastic）识别超买超卖与趋势反转（详见 `05-technical-analysis.md` 的 5.2）。
3. **计算机化/量化桥接**：核回归等把主观形态自动化（上文 E）。
4. **关键演化 = 与因子投资（factor investing）结合**：本章的最终目标就是评估技术指标作为**因子**的有效性。

## 关键公式 · 图表 · 定义（逐图转述其内容 ★本篇核心）

> 所有图坐标统一为 y=Price（价格）、x=Time（时间）；下面"上/下箭头"指图中形态完成后画出的突破方向箭头。

### 图 5.1 — 头肩顶 Head-and-Shoulders (HS)（原书 p.98）
- **外形**：价格从左下方上行 → 升到一个中等峰（**左肩 Shoulder**）→ 回落到颈线 → 再升到**更高的峰（头 Head，全图最高点，居中）**→ 回落到颈线 → 升到**第三个峰（右肩 Shoulder，高度≈左肩、低于头）**→ 随后**下行、向下箭头跌破颈线**。一条**水平虚线"Neckline"连接两次回落的低点**（两肩之间的谷）。
- **结构口诀**：肩—头—肩三峰，中峰最高、两肩对称略低；颈线 = 两个中间谷的连线。
- **含义**：经典**牛转熊**反转，跌破颈线确认。

### 图 5.2 — 倒头肩 Inverse Head-and-Shoulders (IHS)（原书 p.99）
- **外形**：图 5.1 的上下镜像。价格从左上方下行 → 跌到第一个谷（**左肩**）→ 反弹到颈线 → 跌到**更深的谷（头 Head，全图最低点，居中）**→ 反弹到颈线 → 跌到**第三个谷（右肩，比头浅、≈左肩）**→ 随后**上行、向上箭头突破颈线**。**水平虚线"Neckline"连接两次反弹的高点**（标签 Shoulder/Head/Shoulder 在谷下方，Neckline 在上方）。
- **含义**：**熊转牛**反转；书中明确称其为预测"看跌→看涨"反转的可靠指标。

### 图 5.3 — 扩散顶 / 扩散底 Broadening Top / Bottom (BT/BB)（原书 p.99，左右两子图）
- **外形（两子图都是"喇叭口/扩音器 megaphone"形）**：价格来回锯齿，但**摆幅逐次放大**，被**两条向右发散的虚线趋势线**夹住（上轨向上斜、下轨向下斜，开口朝右）。
  - **左 "Broadening Top"**：若干次小幅反复（振幅渐扩）后 → 一次**大跌（看跌）**。
  - **右 "Broadening Bottom"**：同样的发散喇叭口、振幅渐扩 → 末端一次**大涨（看涨）**，最后一笔向上突破。
- **机理**：振幅扩大 = 多空分歧/情绪渐失控；BT 在顶部酝酿后崩、BB 在底部酝酿后冲。
- **[图中存疑]**：两子图末笔在视觉上都偏向上扬，BT 看跌 / BB 看涨的区分主要来自**正文措辞**（BT=多次反弹后"大幅下跌"，BB="大幅上涨"），并非箭头方向一望可辨。

### 图 5.4 — 三角形 Triangle Patterns（原书 p.100，四子图＝三类）
- **共性**：上轨连高点、下轨连低点，两线**在右侧收敛于顶点（apex）**；**放量向上（下）突破 = 看涨（看跌）**。属**持续形态（continuation）**。
  - **Ascending（升三角，左上）**：**上轨水平**（等高的阻力）+ **下轨上斜**（抬高的低点）→ 收敛后**向上突破（上箭头）**，看涨。
  - **Descending（降三角，右上）**：**上轨下斜**（走低的高点）+ **下轨水平**（等低的支撑）→ **向下突破（下箭头）**，看跌。
  - **Symmetrical（对称三角，下方两子图）**：上轨下斜 + 下轨上斜，对称收敛于中部顶点；书中画了**两种结局**——一子图**向上突破**、一子图**向下突破**（对称三角方向不定，延续原趋势）。

### 图 5.5 — 矩形 Rectangle Patterns（原书 p.101，左右两子图）
- **外形**：价格在**两条水平平行虚线（上=阻力、下=支撑）构成的"箱体"内来回震荡 = 盘整（consolidation）**，然后突破箱体。
  - **"Uptrend"（上升趋势中的矩形）**：箱内震荡后**向上突破上沿（上箭头）**→ 确认**原上涨趋势延续**（看涨）。
  - **"Downtrend"（下降趋势中的矩形）**：箱内震荡后**向下跌破下沿（下箭头）**→ 确认**原下跌趋势延续**（看跌）。
- **要点**：矩形是**延续**形态（区别于反转形态），方向 = 突破方向 = 原趋势方向。

### 图 5.6 — 双顶 / 双底 Double Top / Bottom（原书 p.101，左右两子图）
- **左 "Double Top"（双顶，"M"形）**：升到第一个峰 → 适度回落到一个谷 → 升到**第二个等高峰** → 下跌、**跌破"Neckline"（画在中间那个谷高度上的水平虚线）**、下箭头。**极看跌**反转。
- **右 "Double Bottom"（双底，"W"形）**：跌到第一个谷 → 适度回升到一个峰 → 跌到**第二个等深谷** → 上涨、**突破"Neckline"（画在中间那个峰高度上的水平虚线）**、上箭头。**极看涨**反转。
- **确认条件**（正文）：双顶在价格**跌破"等于两高点之间最低谷"的支撑**时确认；双底在价格**突破"等于两低点之间最高峰"的阻力**时确认。

> 另：5.1.3 列表第 6 条引出**移动平均（MA）**——平滑短期波动、显露趋势，衍生**均线交叉（crossover）**等趋势跟随法，作为通向 5.2 技术指标一节的过渡（指标细节见 `05-technical-analysis.md`）。

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **把"形态"做成因子的标准管线**（可直接落到 47 因子库 `/mnt/sda2/lichenchen/factor_library/`）：
  1. **核回归（或其他平滑）去噪** → 在平滑序列上找**局部极值**定位肩/头/顶/底，**拟合上下轨**判三角/矩形；
  2. 用**多时间窗 + 时间缩放**解决"形态无固定窗（20–200 日）"问题；
  3. 形态命中作为**事件（0/1 哑变量）**，做**条件收益 vs 无条件收益**的事件研究——这正是 Lo-Mamaysky-Wang(2000) 的检验范式，可作模板；
  4. 可探索**深度学习图像识别**直接从 K 线图识形态（书中点名的未来方向）。
- **检验设计的迁移价值**：条件 vs 无条件收益对比 = 朴素的"信号有没有 alpha"判据，和本目录 `_真假因子判据.md` 的"真假因子"思路同源；书中"7/10 形态条件收益显著"也提示**多数形态可能是噪声**，回测须配多重检验/样本外。
- **成本与 alpha 量级**：技术形态因子**编码简单、收益主要来自供需失衡（supply-demand imbalance），统计上易显著但幅度有限**（章末结论原话），适合做**择时/确认信号或与有金融逻辑的变量叠加**，不宜单吃。
- **A 股 / L2 高频落地**：形态识别天然可在**分钟级 K 线**上做（本地多数量价因子已用 1–60 分钟 K 线）；时间缩放思想恰好契合"日内 vs 多日"多尺度；A 股可像 Liu-Zhou-Zhu 的中国趋势因子那样**额外纳入成交量维度**确认突破（放量突破才算）。
- **与本地因子库的接点**：双顶/双底、头肩的"反转"逻辑 ↔ 本地 `结构化反转`/`锚定反转`/`黄金分割反转`；三角/矩形的"突破+放量"↔ `主动买入占比`/资金流类；扩散形态的"振幅渐扩"↔ 波动率类（`五大波动率因子`/`跳价期波动`）。整段指标/趋势因子细节与本地映射见 `05-technical-analysis.md` 第 66–72 行。
