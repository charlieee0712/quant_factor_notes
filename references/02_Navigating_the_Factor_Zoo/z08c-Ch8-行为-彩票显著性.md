## 方法论 / 机理要点（重点·展开）

### A. 行为三因子模型：短期 PEAD 因子 + 长期 FIN 因子（Daniel–Hirshleifer–Sun 2020）

思路是用**两个行为因子 + 市场**组成三因子模型，目标是在解释横截面上**跑赢传统多因子模型**。两个因子按"错误定价的修正期限"分工：

**短期因子 = PEAD（盈余公告后漂移）**
- 机理：盈余正惊喜的公司，随后一段时间收益高于盈余负惊喜的公司——这是市场对盈余信息**反应不足（under-reaction）**的典型证据。
- 构造（Fama-French 1993 风格）：在 **size × earnings-surprise 上做 2×3 双重排序**，取**市值加权**组合；多头=正惊喜组、空头=负惊喜组。
- 盈余惊喜的度量：取**最近一季盈余公告日附近 4 日的累计异常收益（4-day CAR）**作为代理（沿用 Chan–Jegadeesh–Lakonishok 1996）。即用"价格对公告的即时反应"反推惊喜大小，而非用分析师预测误差——这是个可直接落地的构造技巧。

**长期因子 = FIN（融资因子，Financing）**
- 成分：**1年期净股票发行 NSI**（Pontiff–Woodgate 2008）+ **5年期复合股票发行 CSI**（Daniel–Titman 2006），二者 **50/50** 合成一个"融资特征"。
- CSI 的算法：`5年市值增长（对数） − 5年股票收益（对数）`——即剔除掉价格涨幅后，市值还增长的部分，必然来自**净增发**。NSI 与 CSI 类似但**剔除现金分红、用1年窗口**。
- 构造：按 size × 融资特征排序，**多头=两个低发行（market-value-weighted）组、空头=两个高发行组**。
- 机理（核心经济逻辑）：管理层在**股价高估时增发、低估时回购**——所以"高净发行"是高估信号、"低发行/回购"是低估信号，低发行组未来收益更高。
- 为什么分长短两期：FIN 主要捕捉**1年及以上**的长期错误定价与修正，**抓不住超短期错误定价**——因为增发/回购受披露、法律、承销等成本约束，公司**没法靠超短期择时去套自家股票的错价**。这是一条很到位的"因子期限分工"论证：不同行为因子对应不同的修正速度。

**Lakonishok 人物框的两点方法论养分**：
- **LSV (1994) 反向投资**：反驳"买高成长预期的 glamour、卖低预期的 value 能赢"的主流看法；证明**买被冷落的 value、卖被追捧的 glamour 反而有超额收益**。根因是投资者对过去信息**过度反应（overreaction）**，价格偏离内在价值。
- **机构羊群（herding）**：挑战"机构总是理性、利润最大化"的假设，发现**机构交易也呈羊群倾向**，会助推市场的过度/不足反应——这是把羊群从"散户故事"上升到"专业资金也跑不掉"的关键观察。

### B. 显著性理论（Salience Theory）—— 本块方法论核心，也是"彩票偏好"的形式化

**思想源头**：显著性理论由 Bordalo–Gennaioli–Shleifer (2012) 提出，与**可得性偏差（availability bias）**同源，由 Cosemans–Frehen (2021) 做实证。核心命题：**对风险资产的需求受其收益在不同状态下"是否扎眼/突出"的影响**——投资者评估一项资产时，是拿它的各状态收益**与可比资产的平均收益做对比**。这正是"有限注意（limited attention）"：人会盯住**与众不同、不寻常**的东西。这个"扎眼"就是 *salience*。

**第1步：显著性函数（式 8.17）**

```
σ(r_{i,s}, r̄_s) = |r_{i,s} − r̄_s| / ( |r_{i,s}| + |r̄_s| + θ )
```
- `r_{i,s}` = 股票 i 在状态 s 的收益；`r̄_s = (1/N) Σ_i r_{i,s}` = 该状态下 N 只可比股票的**平均收益**；`θ>0` 是个小常数（防分母为零、并调节"递减敏感性"）。
- 满足三条公理：
  1. **排序性（ordering）**：某状态的收益**离平均越远**，越显著（分子是偏离度）。
  2. **递减敏感性（diminishing sensitivity）**：当所有股票的绝对收益水平**一起等量抬升**时，显著性下降（分母变大）——即"在高水位上，同样的差额没那么扎眼"。
  3. **反射性（reflection）**：显著性**只看收益幅度、不看正负号**（分子取绝对值）。

**第2步：彩票式决策权重——把客观概率扭曲成主观权重（式 8.18–8.19）**

显著性思考者（salient thinker）会**对每只股票的各状态收益按显著性排序**，再用 *lottery-specific decision weights*（彩票式决策权重）**替换客观状态概率**：

```
π̃_{i,s} = π_s × ω_{i,s}                                  (8.18)
ω_{i,s} = δ^{k_{is}} / ( Σ_{s=1}^S δ^{k_{is}} × π_s ),  δ∈(0,1]   (8.19)
```
- `k_{is}` = 收益 `r_{i,s}` 的**显著性排名**，从 1（最显著）到 S（最不显著）；`π_s` = 状态 s 客观概率，`Σπ_s = 1`。
- `δ` 是关键参数：**显著性扭曲权重的程度**，可代理"决策者的认知能力"。
  - `δ = 1`：无扭曲，主观权重 = 客观概率（理性人）。
  - `δ < 1`：显著性思考者，**高估显著状态、低估不显著状态**。
  - `δ → 0`：极端情形，**只看最显著的那个收益、无视其余所有状态**。
  - [疑书误] 书 p.189 原文写"overweighs salient states (ω<1) and underweights non-salient states (ω<1)"——两处都写成 ω<1 应为笔误：正确含义是**显著状态 ω>1（被高估）、不显著状态 ω<1（被低估）**。由 δ∈(0,1] 且 k=1 最显著可知 δ¹>δ²>…，归一化后显著状态权重确实 >1。

> **这就是"彩票偏好"的微观基础**：当一只股票的**极端上行**收益很扎眼，显著性思考者会过度加权这个"中彩票"状态 → 像买彩票一样**愿意为它多付钱 → 高估 → 未来收益低**；反之极端下行扎眼则被过度厌恶。经典的 MAX/lottery 因子是其**实证表亲**，显著性理论给了它一个"为什么"。

**第3步：ST 因子的构造（Cosemans–Frehen 2021，式 8.20）**

```
ST_i = cov(ω_{i,s}, r_{i,s})
     = Σ_s π_s ω_{i,s} r_{i,s} − Σ_s π_s r_{i,s}
     = E^{ST}[r_{i,s}] − r̄_{i,s}                          (8.20)
```
- 这里把"状态"取为**一个周期内的每个交易日**，`π_s = 1/S`，S = 周期内交易日数。即用**股票自身过去一段日收益序列**当作状态空间。
- 几何含义很干净：**ST = 显著性扭曲后的期望收益 E^{ST}[r] − 真实均值 r̄ = 错误定价的"楔子"**，本质是"显著性权重与收益的协方差"。
- 机理：若一只股票**过去的最高（最低）收益很显著**，投资者就**上调（下调）对它未来的预期、把价格推到公允值之上（之下）**，从而**压低（抬高）未来实现收益**。这是一条标准的"显著性 → 错误定价 → 反转"的预测链。

**实证结论（转述其传达的内容）**：
- 在美股横截面强力成立。**单变量分组**显示最高与最低 ST 十分位组的收益差**统计显著且经济上很大**，样本 1931–2015。
- 多空组合月均超额收益：**等权约 1.28%、市值加权约 0.60%**。
- 这部分收益**无法被 size/value/momentum 等常见风险因子解释**——即 ST 是个独立的 alpha 来源。
- [方向存疑] 书原文一边说"显著高收益→高估→未来收益**更低**"，一边又说"**买高 ST、卖低 ST**"取得正超额。二者要自洽，"高 ST"须对应"显著下行/被低估"一侧；按 `cov(ω,r)` 的定义，显著上行应给出**正**协方差（高 ST），机制上应是**买低 ST、卖高 ST**。这里书的多空方向标注与机制叙述有张力，落地时以**机制为准**（做空"显著上行被高估"的彩票股、做多"显著下行被低估"的）。读者复现请回看 Cosemans–Frehen (2021) 原文核对符号约定。

**第4步：A股本土化改造（Sun–Wang–Zhu 2023）——把收益换成换手率**

- 背景是 **A股的涨跌停（daily price-change limit）机制**。作者发现：把式 8.17 里的"股票收益"**替换为换手率（turnover）**，对横截面收益的解释力**更好**。
- 权重函数仍用式 8.19，只是 `k_{is}` 变成**换手率的显著性排名**；显著因子（式 8.20）相应改成**显著性权重与换手率的协方差**。
- 方法论启示：显著性理论是个**框架**，"被对比的变量"可替换——美股用收益、A股因有涨跌停而用换手率更有效。这正是"机理迁移到本土市场时要换可观测代理"的范例（A股可直接复现：日换手率序列 → 算 σ → 排名 → δ 权重 → cov）。

### C. 羊群（Herding）在本书中的落点

本书没把羊群写成独立小节，而是分散在三处，可拼出完整观点：
- **机构羊群**（Lakonishok 人物框）：专业资金也抄别人、呈羊群倾向，助推过度/不足反应。
- **附录条目**：
  - *Herding*——投资者抄别人而非靠自己分析，会催生**泡沫/崩盘**；**反向（contrarian）因子策略**在"羊群终将转向"时吃后续价格修正。
  - *Bandwagon Effect（从众）*——因为很多人这么做/这么想就跟风；**被追捧的从众股高估、被忽视的低估**。
  - *Availability Cascade（可得性级联）*——一个集体信念靠**反复传播**而越来越"可信"；对策是**靠数据驱动分析、不随大流舆论**。
- 共同的可利用逻辑：羊群制造**自我强化的反馈环（feedback loop）**→ 推动泡沫/崩盘 → 因子策略靠**纪律化、反向、等待均值回归**来收割修正。

### D. 8.8 结论：行为有效市场（Behavioral Efficient Market）

- **悖论**：从行为视角看市场"似乎无效"（认知偏差 + 套利限制），但**又出奇地难被打败**（像一种有效）。
- **Statman (2018) 的解法**："行为有效市场"概念把 EMH 的两层含义**拆开**：①"价格反映所有可得信息"（这一层行为金融否认）与 ②"市场很难被战胜"（这一层行为金融承认）。两者可以同时成立。
- **Fischer Black (1986)《Noise》** 的精炼："**噪声让市场无效，但噪声也阻止人们利用这种无效。**"——噪声交易者既制造错价，又让套利变得风险大、不可靠，于是错价**长期存在却难以稳定套出**，这恰恰构成市场对"被套利"的韧性。
- **行为金融的局限（很重要的清醒话）**：它能告诉你**错价的可能来源**，但**不直接给出"如何战胜市场"的技术**；仍依赖传统理论。书的态度：行为理论会越来越稳健、占主导，但目前是"诊断工具"而非"交易圣杯"。

### E. 附录 TABLE A8.1：偏差→因子策略 的穷举清单（按主题压缩）

附录列了约70条认知偏差，每条都配"factor strategies can exploit this by…"。不逐条抄，按机理归并，并提炼**贯穿全表的可利用范式**：

- **注意力/显著性类**：Attentional Bias（只盯情绪化、扎眼事件，忽略非显著但基本面强的）、Salience 相关 → 利用"被忽视但基本面扎实"的标的。
- **羊群/社会传染类**：Herding、Bandwagon、Availability Cascade、In-group/Affinity Bias → 反向、数据驱动、全球分散。
- **过度反应/外推类**：Overreaction、Recency Bias、Representativeness、Narrative Fallacy（"故事股"被高估）、Optimism Bias（非理性繁荣推高，做空过热）、Clustering Illusion（把随机当趋势）→ 用统计严谨区分真趋势与噪声、买被过度抛售的、卖被过度追捧的。
- **不足反应/保守类**：Conservatism（更新信念太慢）、Anchoring（锚定买入价/参考点）、Disposition Effect（赚就卖、亏死扛 → 对利好反应不足）、Confirmation Bias → 利用价格向真值的**缓慢调整**。
- **风险/损失/彩票类**：Loss Aversion（厌恶损失 → **低波动被高估、高波动被低估** → 超配被低估的风险资产、低配被高估的低风险资产）、Neglect of Probability（无视小概率/极端结果的概率量级 → 关注风险调整后收益而非潜在极端 payoff，**这条正对彩票股**）、Zero-Risk Bias、Myopic Loss Aversion → 拉长持有期、用风险调整视角。
- **过度自信类**：Overconfidence、Illusion of Control/Validity、Self-Attribution、Dunning-Kruger → 与"过度自信交易者"对做、流程化纪律。
- **框架/心理账户类**：Mental Accounting、Framing 相关、Sunk Cost、Status Quo、Endowment、Home/Familiarity Bias → 按估值而非心理账户/熟悉度决策、全球分散。

**贯穿全表的可利用范式（这才是附录真正的方法论）**：因子策略战胜偏差的"解药"高度一致——**①数据驱动而非舆论/直觉、②客观与纪律（process over outcome，重过程不重单次结果）、③反向/逆人群、④拉长视角与均值回归、⑤用更大更有代表性的样本（对抗小数定律 Law of Small Numbers）、⑥关注风险调整收益而非极端 payoff（对抗彩票偏好）**。

书末两点告诫（很关键）：
- 偏差**互相纠缠、可叠加**（例：过度自信会放大确认偏差），让利用变复杂；
- 偏差的影响**随市场环境/个体特征而变**，且市场还受经济、监管、技术等偏差之外的力量左右——所以"靠偏差套利"绝非直来直去，需要对心理驱动的细腻理解 + 对环境变化的适应力。

## 关键公式 · 图表 · 定义（图表转述）

- **式 8.17 显著性函数**：`σ = |r_{i,s}−r̄_s| / (|r_{i,s}|+|r̄_s|+θ)`，θ>0，r̄_s 为可比股票均值；三公理：排序 / 递减敏感 / 反射（只看幅度不看符号）。

- **式 8.20 ST 因子**：`ST_i = cov(ω, r) = E^{ST}[r] − r̄` = 显著性扭曲期望与真实均值之差（错价楔子）。状态=交易日，π_s=1/S。

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **可直接复现的因子（A股优先）**：
  - **ST 显著性因子**：取每只股票一个周期内的**日收益序列**当状态空间 → 算 σ 排序 → 给 δ 权重 → 求 `cov(ω, r)`；按 ST 分十分位做多空。**δ 是唯一超参**（典型设小于1，可格点搜），稳健性要对 δ、周期长度、状态定义做敏感性测试。
  - **A股换手率版（Sun–Wang–Zhu 2023）**：因涨跌停，把收益换成**日换手率**算显著性，多空收割效果更好——A股落地建议优先试这版，数据只需日换手率，门槛低。
  - **PEAD 短期因子**：用**盈余公告日附近 4 日 CAR** 作惊喜代理（绕开分析师预测数据），size×惊喜 2×3 市值加权。
  - **FIN 融资因子**：CSI=5年对数市值增长−5年对数收益；NSI=1年版剔现金分红；50/50 合成，低发行多/高发行空。
- **检验设计要点**：①单变量分组多空 + 与 size/value/momentum 等做**横截面回归剥离**，确认是独立 alpha；②长样本（书用 1931–2015）看稳健性；③区分**因子修正期限**——PEAD 抓短期（反应不足）、FIN 抓1年以上（增发择时受成本约束）。
- **成本/可交易性警示**：显著性/彩票股、羊群股往往是**高换手、高波动、散户密集**的小票，多空的**空头腿借券难、冲击成本高**；A股还有涨跌停约束实现交易。回测必须计**换手成本 + 借券/涨跌停约束**，否则纸面 1.28% 会被吃掉。
- **方法论迁移**：显著性是**框架**，被对比的变量可换（收益/换手率/…）；做 A股本土因子时，"换一个更贴合本地微观结构的可观测代理"是通用招式。
- **元认知（来自结论与附录）**：行为因子给的是**错价诊断**不是必胜技；偏差互相纠缠、随环境漂移——因子库里这类因子要持续监控失效，按"数据驱动+纪律+反向+长视角+风险调整"的范式管理，而非一次拟合长期吃。

## 术语与存疑

- **术语**：Salience Theory 显著性理论；salient thinker 显著性思考者；lottery-specific decision weights 彩票式决策权重；ST factor（Salience Theory factor）；PEAD（post-earnings announcement drift）盈余公告后漂移；FIN（Financing factor）融资因子；NSI（net-share-issuance）净股票发行；CSI（composite-share-issuance）复合股票发行；herding 羊群；bandwagon 从众；availability cascade 可得性级联；behavioral efficient market 行为有效市场；limits to arbitrage 套利限制。
- **关键文献**：Bordalo–Gennaioli–Shleifer (2012)；Cosemans–Frehen (2021)；Sun–Wang–Zhu (2023, A股换手率版)；Daniel–Hirshleifer–Sun (2020, 行为三因子)；Pontiff–Woodgate (2008, NSI)；Daniel–Titman (2006, CSI)；Chan–Jegadeesh–Lakonishok (1996, 4日CAR惊喜)；LSV (1994)；Statman (2018, 行为有效市场)；Black (1986, Noise)；An (2016, 极端赢家/输家定价—彩票相关)。
- **[疑书误]** 式 8.19 文字描述把显著与不显著状态的权重都写成 ω<1，应为"显著 ω>1、不显著 ω<1"（见上 §B）。
- **[方向存疑]** ST 因子多空方向：书说"买高ST/卖低ST"得正超额，与"显著高收益→高估→未来低收益"的机制叙述方向相反；以机制为准应是做空显著上行（高估）一侧。复现请核对 Cosemans–Frehen (2021) 的符号约定。
- **范围说明**：标题"彩票"在本页区间**无独立 MAX 小节**，本书用显著性理论作其微观基础（lottery-specific weights）；"羊群"亦无独立小节，散见于人物框+附录+结论，已整合于 §C。
