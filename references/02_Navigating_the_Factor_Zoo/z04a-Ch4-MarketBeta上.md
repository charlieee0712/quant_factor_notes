## 本块讲什么

本章核心命题：**beta 不是一个静态常数，而是随时间变化、且无法 ex-ante 精确预知的量**；因此问题不是"求出真 beta"，而是"为手头问题找一个可行（feasible）的估计方法"。本块（上半章）系统铺陈了四类 beta 估计思路，逐级解决"OLS 静态回归不够用"的问题：

1. **工具变量（IV）**——解决回归内生性（误差项与自变量相关导致 OLS 不一致），是后续"条件 beta"建模的数学基础。
2. **条件 beta（Conditional Beta）**——把 beta 显式写成一组**工具变量/状态变量的（线性）函数**，承认 beta 随宏观状态变化；并指出若忽略这一点，**无条件 alpha 会被系统性偏估**。给出一阶段与两阶段两种估计法。
3. **动态条件 beta（DCB）**——为摆脱"必须事先选对工具变量"的困境，改用 Engle 的 ARCH/GARCH→DCC 框架，直接对**时变协方差矩阵**建模，再由协方差比值反推 beta。本块花大篇幅复习 ARCH/GARCH 与 DCC 的估计机理（MLE）。
4. **连续/非连续 beta**——用高频数据把 beta 拆成连续（日内平滑共动）、非连续（日内跳跃）、隔夜三类，发现只有跳跃与隔夜 beta 被市场定价（带风险溢价），连续 beta 不被定价。这直接关系到"哪种 beta 适合做因子"。

贯穿全章的方法论主线：**"承认 beta 时变 → 选择如何刻画其时变（外生工具变量 vs 内生波动率模型 vs 高频分解）→ 估计 → 检验是否带风险溢价"**。书强调这是 insight/经验问题，没有唯一正解。

## 方法论 / 机理要点（重点·展开）

### 1. 为什么估 beta 本质上很难（4.1 Background）

- **Alpha 与 beta 的定位**：beta 度量证券/组合相对整体市场的波动性（系统性风险）；alpha 是相对基准、或"在给定 beta 下应有期望收益"之外的超额收益。alpha=0 表示收益恰好与所承担的风险（beta）相称；alpha>0 为风险调整后跑赢，alpha<0 为跑输。**alpha 被解读为基金经理"加/减"的价值**。
- **CAPM 的起点**：第1章已述 CAPM——资产收益的横截面差异**仅**取决于其对市场因子的系统性暴露（即 beta）。最朴素的估计=用历史数据把资产超额收益对市场超额收益做回归。
- **核心困难（书反复强调的"机理"）**：beta 会**随时间自然变化**，而由于金融市场的内在不确定性（第10章会展开），人们很难 ex-ante 准确预测这种变化。**推论：可能永远无法得到实时精确的 beta**；务实做法是"针对手头问题找一个可行估计法"。这句话定调了整章——不是追求真值，而是工程化逼近。

### 2. 工具变量 IV（4.2）——一致性的来源与修复

机理链条非常清晰，值得吃透，因为它是"条件 beta"的地基：

- beta 本质=回归模型的参数估计值。回归目标=估计条件期望 E[y|x]=β'x（无截距设定）。单变量：y = βx + u。
- **OLS 一致性的关键假设**：误差项 u 与 x 无关联，使 β̂ "纯粹"反映 x 对 y 的影响。数学上 dy/dx = β + du/dx；当 u 不是 x 的函数时第二项 du/dx=0，β̂ 干净。
- **内生性后果**：若 u 与 x 相关，du/dx≠0，β̂ 会把 du/dx 的影响混进来，**变成不一致估计量**。书特别给了"一致估计量"的定义——样本量趋于无穷时估计序列以概率1收敛到真参数；**不一致估计量即使收集无限样本也没希望逼近真值**（所以不能用）。这是 IV 必要性的根本理由。
- **IV 的修复逻辑**：找一个工具变量 z，**与 x 强相关、与误差 u 不相关**。则 y 对 z 的 beta 为 β_IV = (dy/dz)/(dx/dz)。直觉=用 z 这条"外生通道"分别撬动 y 和 x，相除把内生污染消掉。
- **可操作公式**：y 对 z 回归得斜率 (z'z)⁻¹z'y（分子），x 对 z 回归得 (z'z)⁻¹z'x（分母），相除得 β̂_IV = (z'x)⁻¹z'y——**对 β 一致**。（即"两个简约式之比"= Wald/IV 估计量。）

### 3. 条件 beta（4.3）——忽略时变会系统性偏估 alpha

这一节的方法论价值在于：**先证明"为什么必须做条件化"，再给"怎么做"**。

**为什么（偏误来源）**：
- Grant(1977)、Jagannathan & Wang(1996)、Lewellen & Nagel(2006)、Boguth et al.(2011) 指出：**若 beta 随市场风险溢价或市场波动率系统性变化，则无条件 alpha 是真 alpha 的有偏估计**。Cederburg & O'Doherty(2016) 实证：按过去 beta 排序的组合，其 beta 随时间大幅变化，主因是公司 beta 横截面分布在漂移。
- **偏误的解析式（式4.7）**：无条件 alpha ≈ cov(β_{i,t}, E_{t-1}[R_{m,t}]) − (E[R_{m,t}]/σ²_m)·cov(β_{i,t}, σ²_{m,t})。
  - 解读：当**条件 beta 与期望市场超额收益负相关**，和/或**与市场波动率正相关**时，无条件 alpha 出现**负偏**。Boguth et al.(2011) 强调后一项（beta 与波动率正相关）可能特别大，导致无条件 alpha 出现实质性偏差。
  - 经济直觉：如果一只股票"在市场预期收益高时反而 beta 低、在市场波动大时 beta 高"，那么用静态回归算出的 alpha 会被这种时变结构污染。

**怎么做（把 beta 写成工具变量的函数）**：
- 条件 CAPM 的定义式（式4.8）：α_{i,t} ≡ E[R_{i,t}|I_{t-1}] − β_{i,t}·E[R_{m,t}|I_{t-1}] = 0，其中 I_{t-1} 是 t−1 期末的投资者信息集。
- Shanken(1990)、Ferson & Schadt(1996)、Ferson & Harvey(1999)：把 beta 建成**工具变量的线性函数**，典型工具=总股息率（aggregate dividend yield）、违约利差（default spread）等宏观状态变量。
- Boguth et al.(2011) 的关键贡献：**把"前期已实现 beta 的滞后值"也纳入工具变量**——并指出来自前几个估计窗口的滞后 beta 通常是 beta 的良好预测变量。

**一阶段法（one-stage）**：直接把 beta 的工具变量形式代入收益回归。
- β_{i,τ}^{IV} ≡ γ_{i,0} + γ'_{i,1} Z_{i,τ−1}（式4.9 的隐含定义，τ 索引季度，Z_{i,τ−1}⊆I_{t−1}）。
- 落地回归（式4.10）：R_{i,τ} = α_i^{IV1} + γ_{i,0}R_{m,τ} + γ'_{i,1}Z_{i,τ−1}R_{m,τ} + u_{i,t}。即把 Z 与市场收益的**交互项**放进回归，交互项系数刻画 beta 如何随状态变量变动。

**两阶段法（two-stage，更直接地刻画"状态变量↔beta"关系，书认为更精确）**：
- **第0步：先用日内/日度数据估每季度的"已实现 beta"**（式4.11）：
  r_{i,j} = α_i + β_{i,0}r_{m,j} + β_{i,1}r_{m,j−1} + β_{i,2}·[(r_{m,j−2}+r_{m,j−3}+r_{m,j−4})/3] + ε_{i,j}。
  - 这是 **Dimson(1979) 式的滞后市场收益修正**：用当期、滞后1期、以及滞后2/3/4期的平均，矫正非同步交易（nonsynchronous trading）导致的 beta 低估。
  - 季度 beta = 三个系数之和：β̂_{i,τ} ≡ β̂_{i,0} + β̂_{i,1} + β̂_{i,2}（式4.12）。
- **第一阶段：把已实现 beta 对滞后工具变量回归**（式4.13）：β̂_{i,τ} = δ_{i,0} + δ'_{i,1}Z_{i,τ−1} + e_{i,τ}，得到拟合 beta：β̄_{i,τ} = δ_{i,0} + δ'_{i,1}Z_{i,τ−1}。
- **第二阶段：把拟合 beta 代入收益回归**（式4.14/4.15）：R_{i,τ} = α_i^{IV2} + φ_{i,0}R_{m,τ} + φ_{i,1}β̄_{i,τ}R_{m,τ} + v_{i,t}，其中 β_{i,τ}^{IV2} ≡ φ_{i,0} + φ_{i,1}β̄_{i,τ} 被**约束为拟合 beta 的线性函数**（故称"受限版本"）。
- **两法对比与选择**：
  - 工具变量 Z 的选择"是 insight 与经验问题"，没有公式可循。
  - 回归（4.10/4.15）里的 k+1（工具个数+1）反映"这组工具刻画 beta 动态的能力"。
  - **两阶段更优的理由**：第一阶段直接把 beta 估计值对工具变量回归（式4.13），得到的条件 beta 系数 δ'_{i,1} 比一阶段（系数来自收益回归 4.10）更精确——因为它隔离了"工具→beta"这一层关系，不被收益噪声淹没。

### 4. 动态条件 beta DCB（4.4）——绕开"必须找对工具变量"

- **动机**：条件 beta 重度依赖工具变量，而合适的工具可能**难找甚至不存在**。Engle(2016) 提出 DCB，**既不需要指定经济变量、又不做"滚动样本/状态空间"那类不现实假设**，建立在他更早的动态条件相关（DCC, Engle 2002）工作之上。
- **DCB 的定义机理**：令 z_t = (y_t, x_t')'（k+1 维，x_t 是 k 维解释变量），目标=求 y 给定 x 的条件分布。把 z_t 的协方差矩阵分块（式4.16）：
  H_{zz,t} = [[H_{yy,t}, H_{yx,t}], [H_{xy,t}, H_{xx,t}]]。
  则由 OLS：**β_t = H_{xx,t}⁻¹ H_{xy,t}（式4.17）**——这就是动态条件 beta（DCB）。
  - 关键洞见：**只要能估出时变协方差矩阵 H_{zz,t}，beta 自然就出来了**（H_{xy,t}、H_{xx,t} 是它的子块）。于是问题转化为"如何对时变协方差建模"→ ARCH/GARCH→DCC。
- **DCB 的概念定位（机理性描述）**：传统 beta 视为静态；DCB 允许资产对市场的风险暴露**随市场环境、公司财务政策、宏观因子等演化**。模型会显式考虑：波动率波动期（影响敏感度）、公司杠杆对 beta 的影响、**识别危机 vs 平稳期的 regime shift**、并可整合宏观变量。
- **估计技术**：常用 **Kalman 滤波**等计量手段，使 beta 随新信息实时更新；方法精巧但**需要大量数据**才能准确刻画 beta 动态。

#### 4.4.1 ARCH / GARCH 复习（DCC 的单变量地基）

- **ARCH(m)（式4.18）**：a_t = σ_t ε_t，σ²_t = α_0 + α_1 a²_{t−1} + … + α_m a²_{t−m}，约束 α_0>0, α_1…α_m ≥0。
  - ε_t 称"innovation（新息）"，需指定其基于信息集 F_{t−1} 的条件分布：常用**标准正态**（式4.19）或 **Student's t 分布**（式4.20，含自由度 v、Gamma 函数 Γ）。t 分布厚尾，更贴合收益的尖峰厚尾。
- **MLE 估计机理**：找一组参数使观测到样本 {a_1,…,a_T} 的似然（概率）最大。
  - 若 ε_t|F_{t−1}~N(0,1)，则 a_t|F_{t−1}~N(0,σ²_t)，密度见式4.21；联合似然=各期密度连乘（式4.22）。
  - **取对数**：因 log 单调，不改变最优解，但优化更易。对数似然（式4.23）：L = −Σ[ln(σ_t) + a²_t/(2σ²_t)]。常数 √2π 可丢弃不影响优化。**最大化 L 等价于最小化 Σ[ln(σ_t)+a²_t/(2σ²_t)]**，对 α 系列求解（σ²_t 由 ARCH 递推式给出）。t 分布下推导类似，书略。
- **ARCH 的硬伤 → 为何要 GARCH**：ARCH 的**波动率持续性只能维持 m 期**，超过 m 的冲击对 σ²_t 无影响。现实中波动率长期持续，ARCH 需要很大的 m 才能拟合 → 参数爆炸。
- **GARCH(m,s)（式4.24）**：σ²_t = α_0 + Σ_{i=1}^m α_i a²_{t−i} + Σ_{j=1}^s β_j σ²_{t−j}，约束 α_0>0, α_i≥0, β_j≥0, 且 Σ(α_i+β_i)<1（平稳性条件）。
  - **机理**：加入"滞后波动率项"σ²_{t−j}，使更久远的冲击**通过 σ²_t 的递归结构被自动吸收**，因而用很短的 m、s 就能拟合长持续性。**实务上 GARCH(1,1) 通常足够**。ε_t 同样可设正态或 t。

#### 4.4.2 动态条件协方差 DCC（4.4.2）——多变量推广

DCC 的精髓=**"分而治之 / 两步法"**：先单独建各资产波动率，再单独建相关性动态。

- **为什么先转相关矩阵**：相关系数被**限定在 [−1,1]**，比协方差矩阵好处理。故把协方差建模拆两步：① 拿到各资产波动率序列；② 对相关矩阵 ρ_t 的动态依赖建模。
- **第一步（边际波动率）**：对每个资产 j 单独拟合单变量 GARCH(1,1)，得边际方差 σ_{jj,t}。定义 D_t = diag{σ_{11,t}^{1/2},…,σ_{kk,t}^{1/2}}（式4.26），相关矩阵 ρ_t = D_t⁻¹ Σ_t D_t⁻¹（式4.25）。
- **第二步（相关性动态）**：用标准化新息 η_t（η_{it} = a_{it}/√σ_{ii,t}）。a_t 的相关矩阵 ρ_t 同时是 η_t 的协方差矩阵。Engle(2002) 定义一个类 GARCH 的 **Q_t 过程**（式4.27）：
  Q_t = (1−θ_1−θ_2)Q̄ + θ_1 Q_{t−1} + θ_2 η_{t−1}η'_{t−1}，
  其中 Q̄ 是 η_t 的无条件协方差矩阵（用样本估）；θ_1,θ_2≥0 且 0<θ_1+θ_2<1。
  - **关键技术细节**：因 η_t 不被限定在 [−1,1]，Q_t **本身不是合格的相关矩阵**（对角元非1），必须**用对角线归一化**：ρ_t = J_t Q_t J_t（式4.28），J_t = diag{q_{11,t}^{−1/2},…,q_{kk,t}^{−1/2}}（式4.29）。
  - 拿到 ρ_t 后，**左右各乘 D_t 还原动态协方差**：Σ_t = D_t ρ_t D_t（式4.30）。
- **DCC 的 MLE（以多元 Student's t 为例）**：给出 ε_t 的多元 t 密度（式4.31）、a_t=(Σ_t)^{1/2}ε_t 的密度（式4.32，含 |Σ_t|^{1/2}、a'_t Σ_t⁻¹ a_t）、对数似然（式4.33）。**把自由度 v 也作为待估参数**（因最优 v 未知），优化参数向量扩展为 {θ_1, θ_2, v}。
  - **算法五步（书原文给出）**：
    1. 对 k 维资产样本 a_1…a_T，对每个 a_{it}（i=1…k）拟合 GARCH(1,1)；
    2. 计算标准化新息 η_t（η_{it}=a_{it}/√σ_{ii,t}）；
    3. 用初值 {θ_1,θ_2,v_0} 经式4.27 估出初始 Q_1；
    4. 对 t=1…T 用式4.27 得 Q_t、式4.28 得 ρ_t、式4.30 得 Σ_t；
    5. 用第4步的 Σ_t，最大化对数似然（式4.33）优化 {θ_1,θ_2,v}。
- **从 DCC 到 DCB（收尾）**：把上述 DCC 方法应用到 z_t=(y_t, x_t')，得到 H_{zz,t}；再由分块式4.16 取出 H_{xy,t}、H_{xx,t}；最后由式4.17 算 DCB = H_{xx,t}⁻¹H_{xy,t}。**整条链路闭合：单变量 GARCH → DCC 协方差 → 分块 → DCB。**

### 5. 连续与非连续 beta（4.5）——高频视角的 beta 分解

这一节把视角从"条件化/时变"转到"**用高频数据按风险类型拆分 beta**"，对做 A股/高频因子尤其有启发。

- **核心论文**：Bollerslev, Li, and Todorov (2016)。研究个股价格如何回应市场价格变动，区分两类风险：
  - **连续风险（continuous）**：日内平滑的共动（smooth intraday co-movements）；
  - **非连续风险（discontinuous）**：日内跳跃（jumps）+ 隔夜收益（overnight returns）。
  - 据此定义**三种日内 beta**：连续 beta、非连续 beta、隔夜 beta。
- **为何对因子有吸引力（书的判断标准）**：这类 beta "**若带显著风险溢价、且彼此相关性低**，会很适合做因子"——这正是好因子的两条标准（被定价 + 低相关/正交分散）。
- **核心实证发现（机理性结论，务必记住）**：用近 1000 只股票、跨两个十年的高频数据，发现**非连续 beta 与隔夜 beta 带显著风险溢价，连续（日内）beta 不带**。
  - 含义：**市场对"大而突兀的价格跳变"与"渐进的连续价格变动"定价方式不同**——只为跳跃/隔夜风险支付溢价，不为平滑日内共动支付（或支付少得多）。对理解市场风险溢价、以及构造区分不同风险类型的策略有重要意义。

#### 4.5.1 连续 beta（Continuous Beta）

- 定义：反映个股价格与市场的**平滑日内共动**；区别于处理跳跃/突变的同类，它捕捉交易日内更规律、持续的价格移动。
- **关键结论**：投资者对这类平滑变动的看法不同于突变，风险感知与定价存在差异；**连续 beta 的风险溢价在横截面上似乎不被定价**——市场不为"连续日内价格移动"的风险给奖励（至少远不如跳跃/隔夜）。

#### 4.5.2 非连续 beta（Discontinuous Beta）——本块仅开头

- 定义：关注交易日活跃时段内的**日内价格不连续/跳跃**，即可能显著冲击组合的突兀价格变化。
- 估计**假设个股跳跃与市场跳跃正相关**，旨在捕捉这些突变带来的风险。其框架涉及"比较个股跳跃与……"（**本句被页边截断，余下并入"下"部 z04b**）。

| **历史数据依赖（非平稳）** | beta 依赖历史价格数据，但金融市场不断变化，历史未必预示未来。|
| **时间窗口选择** | 窗口长短显著影响 beta 值：太短可能错过市场全部响应，太长可能纳入过时数据。|
| **市场基准的相关性** | 准确的 beta 取决于选对相关市场指数为基准；选错指数→ beta 失真。|
| **结构性变化** | 并购、战略转向等公司层面变化会改变风险画像，进而改变 beta。|
| **统计噪声** | OLS 等统计方法估 beta 易被离群值与噪声扭曲。|
| **杠杆效应** | 公司负债水平影响 beta——财务杠杆变化改变其风险画像。|
| **流动性** | 交易频率与买卖价差（bid-ask spread）使 beta 估计复杂化。|
| **Regime 切换** | 牛熊转换等经济状态变化，削弱历史 beta 的适用性。|
| **全球化与一体化** | 国际事件冲击本地市场，使基准选择与 beta 解读更复杂。|

### Robert Engle 人物框（书内页 79，转述）

诺奖得主，美国经济学家，纽约大学 Stern 商学院教授。以**时变波动率**研究著称——提出 **ARCH** 及其推广 **GARCH**，GARCH 至今仍是学界业界建模资产收益时变波动率的标准。亦贡献于系统性风险与金融危机研究，提出 **条件 VaR（CVaR）**以修补 VaR 不满足次可加性（subadditivity）的缺陷（第7章详述）。2003 年与 Clive Granger 共享诺贝尔经济学奖（时间序列计量）。

### 关键公式索引（简化记法）

- **(4.3) OLS**：β = cov(y,x)/var(x)。
- **(4.4) 一致性条件**：dy/dx = β + du/dx；du/dx=0 时 OLS 干净，否则内生不一致。
- **(4.6) IV 估计量**：β̂_IV = (z'x)⁻¹z'y，对 β 一致；= 两个简约式斜率之比。
- **(4.7) 无条件 alpha 偏误**：α^U ≈ cov(β_{i,t}, E_{t-1}[R_m]) − (E[R_m]/σ²_m)·cov(β_{i,t}, σ²_{m,t})。beta 与期望市场收益负相关 / 与波动率正相关 → 负偏。
- **(4.8) 条件 CAPM**：E[R_{i,t}|I_{t-1}] − β_{i,t}E[R_{m,t}|I_{t-1}] = 0。
- **(4.10) 一阶段条件 beta 回归**：R = α + γ_0 R_m + γ'_1 Z_{τ-1}R_m + u（工具×市场 交互项）。
- **(4.11)–(4.12) Dimson 式已实现 beta**：当期+滞后1+（滞后2/3/4均值），三系数求和=季度 beta。
- **(4.13)–(4.15) 两阶段**：beta 对工具回归取拟合值 β̄ → 再以 β̄×R_m 进入收益回归。
- **(4.17) DCB**：β_t = H_{xx,t}⁻¹ H_{xy,t}（来自分块协方差 4.16）。
- **(4.18) ARCH(m)** / **(4.24) GARCH(m,s)**：后者加 Σβ_j σ²_{t−j} 项，平稳性 Σ(α_i+β_i)<1；实务用 GARCH(1,1)。
- **(4.23) 对数似然**：L = −Σ[ln σ_t + a²_t/(2σ²_t)]（最大化≡最小化其相反数）。
- **(4.25)–(4.30) DCC**：ρ_t=D_t⁻¹Σ_t D_t⁻¹；Q_t=(1−θ_1−θ_2)Q̄+θ_1Q_{t−1}+θ_2 η_{t−1}η'_{t−1}；ρ_t=J_tQ_tJ_t（对角归一化）；Σ_t=D_tρ_tD_t。
- **(4.31)–(4.33)** 多元 t 分布密度与 DCC 对数似然，待估 {θ_1,θ_2,v}。

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **beta 当因子时的"时变陷阱"**：式4.7 是直接警示——若用静态滚动窗口 OLS 算 beta 做因子，而真 beta 与市场状态相关，则**回测出来的 alpha 是有偏的**。做 beta 类因子（如 BAB 前置的 beta 排序）前，应先判断 beta 是否随 regime 漂移；若是，应改用条件/动态 beta。
- **工具变量的现实选择**：条件 beta 要选 Z（股息率、违约利差、宏观状态）。书反复强调"是经验/insight 问题"——在 A股可考虑用本土宏观代理（如信用利差、换手率、波动率指数代理）做工具，并把**前期已实现 beta 的滞后值**作为强预测变量（Boguth 法）。
- **两阶段 > 一阶段**：若要更干净地刻画"状态变量→beta"关系，用两阶段（先估已实现 beta 再回归到工具），系数更稳——值得在因子流水线里作为 beta 估计模块的默认选项。
- **DCB / DCC 的工程取舍**：DCB 的好处是**不必猜工具变量**，坏处是**吃数据、计算重**（需逐资产 GARCH + DCC + MLE，且 Kalman/优化收敛敏感）。做多资产组合 beta 时性价比要权衡；GARCH(1,1)+DCC 是务实默认配置。注意 DCC 的"两步法"（先单资产波动率、再相关性）也是工程上把大协方差矩阵估计降维的标准手法。
- **Dimson 滞后修正（式4.11）对 A股极重要**：A股存在涨跌停、非同步交易、流动性差的小票，**用单期回归会系统性低估 beta**；用当期+滞后市场收益（Dimson）能矫正。这是构造任何 A股 beta 因子时应内置的修正。
- **连续/非连续 beta → 高频因子方向**：4.5 的结论给出明确的因子取向——**做 beta 类因子时，跳跃 beta、隔夜 beta 更可能带风险溢价**（被市场定价），连续日内 beta 大概率拿不到溢价。若有 Level-2/高频数据，可优先尝试把 beta 分解为跳跃/隔夜/连续三类，检验各自的横截面风险溢价与相互相关性（书的"好因子=有溢价+低相关"判据）。隔夜 beta 在 A股 T+1、隔夜跳空显著的环境下尤其值得做。
- **检验设计提示**：所有这些 beta 变体最终都要回到"**是否带显著且独立的风险溢价**"来检验——这是把"一种 beta 估计法"升格为"一个因子"的门槛。

## 术语与存疑

**术语对照**
- Instrumental Variable (IV) 工具变量；consistent estimator 一致估计量；endogeneity 内生性。
- Conditional Beta 条件 beta；Conditional CAPM 条件 CAPM；instrument set / information set I_{t-1} 信息集。
- Dimson correction（非同步交易的滞后市场收益修正，对应式4.11）。
- Dynamic Conditional Beta (DCB) 动态条件 beta；Dynamic Conditional Correlation (DCC) 动态条件相关；Dynamic Conditional Covariance（动态条件协方差，4.4.2 标题）。
- ARCH / GARCH（自回归条件异方差 / 广义…）；innovation 新息 ε_t；MLE 极大似然估计；Kalman filter 卡尔曼滤波。
- standardized innovation 标准化新息 η_t；unconditional covariance matrix 无条件协方差 Q̄。
- Continuous / Discontinuous / Overnight Beta 连续 / 非连续 / 隔夜 beta；jumps 跳跃。
- BAB (Betting-Against-Beta)、BAC (Betting-Against-Correlation)——4.6 待"下"部展开。
- CVaR 条件在险价值（Engle 框；第7章）。
