# 真假因子判据 —— 因子筛查与 scorecard A/B/C 标准

> 综合 Navigating the Factor Zoo 第1章 + 《因子投资：方法与实践》第6章 + Machine Learning in Asset Pricing 第4–5章 · 生成 2026-06-30
> 用途：筛查本地 179 个已有因子 + 给每个因子定 A/B/C 评级。素材均来自深版精读笔记，只综合不杜撰。

## 一、为什么大多数"显著"因子是假的——两条独立的"假"来源

**来源一：统计层面的伪发现（p-hacking / 多重检验 / p 值误读）。** Cochrane 用"因子动物园(factor zoo)"挖苦学界挖因子的狂热：上百个"显著"因子里真能赚超额收益的极稀有。根因是功利链——低 p 值→更显著→更易发表（**发表偏差**）。Harvey-Liu-Zhu(2016) 统计 1963–2012 顶刊 316 个因子，t∈[2.00,2.57] 与 t∈[2.57,3.14] 的文章数竟差不多，而后者本应难找得多——证明 t 分布被人为推高。三个认知错误放大它：(1) **p 值误读**——p≡prob(数据|H0)≠prob(H0|数据)，低 p 值只是因子有效的**必要非充分**条件；(2) **p-hacking**——因子研究是软科学，变量、组合构建、再平衡频率、样本区间全凭研究者自由度，"一切阻碍超低 p 值的处理都会被巧妙避开"，把"先假设后验证"颠倒成"先有结果再凑假设"；(3) **多重检验**——同一批数据挖几百个因子就是一次多重假设检验，100 个独立因子里"最显著那个"t>2.0 的概率高达 99%。传统 t=2.0 必然量产**伪发现(Type I)**，316 因子按 FDR 校正后伪发现率达 27%。

**来源二：高维学习诱导的样本内可预测性（MLAP 机理，与来源一相互独立）。** Nagel(2021) 给出第三种"假"：即便投资者**风险中性、无行为偏差、用统计最优的贝叶斯学习**，只要候选预测变量数 J 相对股票数 N 不小（现实横截面正是如此），收益里就会被"印"进一项 **X(I−Γ)g**——投资者收缩学习留下的、与特征 X 相关的"反应不足"成分。研究者事后把收益对 X 回归必然"捡到"它，于是收益**样本内看似可预测**，但这既非风险溢价也非错误定价，纯是估计误差污染。高维渐近(J/N→常数)下，理性预期原假设**以概率 1 被拒**；图 5.2 数值例中投资者仅学一期、J=900，样本内调整 R² 就 >25%（不收缩的 OLS 学习更高达约 47%）。**关键转折**：把研究者拉回投资者的实时信息位置（样本外，权重只用 ≤t 数据），则 **E[r_OOS]=0，可预测性消失**。故**样本内显著只是"嫌疑"不是"证据"**；"因子动物园"文献多基于样本内检验，一旦认识到投资者也要消化同样庞大的候选集，样本内"发现新因子"对真实预期收益几乎不提供信息。

> 两条来源都指向同一铁律：**比起样本内统计显著性，更该看因子背后有无合理经济学解释，并坚持样本外检验**——这是辨真假的根本判据。

## 二、判据（逐条·可操作，每条给"怎么测 + 合格线"）

**判据1 · 统计门槛 + 多重检验校正 + 夏普 haircut**
- 怎么测：排序法/时序回归/Fama-MacBeth 算因子收益 t 值；记录搜索空间 N（从多少候选里挑出来的）做校正。
- 合格线：**单因子 t>3.0**（Harvey-Liu-Zhu 2016，3.0 已偏保守）；严格档**时序回归 t≥3.8、Fama-MacBeth t≥3.4**（Chordia et al. 2020，含未发表因子集）。校正算法：**FWER（Bonferroni / Holm）最严、抬高第二类错误，不适合金融**；金融优先 **FDR（BH/BHY）或 FDP（Romano-Wolf）**（控比例、对相关性不敏感）。夏普换算实盘要打 haircut，且与夏普**非线性**，别简单 ×0.5（Harvey-Liu 2015）。所有显著性一律用 **HAC/Newey-West 稳健标准误**（重叠收益人为制造自相关，幼稚 σ/√T 会高估 t/夏普/IR）。t<3.0 标"疑似伪发现"。

**判据2 · 经济逻辑（必要项·一票否决）**
- 怎么测：能否用**风险补偿**或**行为偏差/错误定价**二选一讲通因子来源。
- 合格线：必须有明确金融学故事。纯统计、讲不出成因 = 红牌（"聪明的股票代号"因子原始 p=0.0079 很低，但先验仅 2%，贝叶斯后验 p=0.588，任何水平都无法拒绝 H0）。"没有经济学故事的统计异象不配当定价因子。"

**判据3 · 来源三检验（风险补偿 / 错误定价 / 数据窥探）**
- 先判真假再判来源；真异象只可能是风险补偿或错误定价，假的就是数据窥探。
- **风险补偿**应满足：高收益端真的更危险（常识检验）；因子模拟组合的 **β 比变量本身更能预测**收益；衰退/极端风险期**显著更差**。
- **错误定价**特征：构建后累计收益**持续爬升**（缓慢价格发现）；业绩公告窗口收益显著更高（异象×窗口交叉项系数 > 主项，Engelberg 2018 测 97 因子达 6 倍）；能预测未来 **SUE**（非收益率指标，绕开"风险控制不足"两难）；低关注度、高套利成本公司因子收益更高（条件双重排序/交叉项回归）。

**判据4 · 样本外 & 反向 OOS（最硬关口·两道关）**
- 怎么测：留 holdout，最好含发表/上线**之前**的 pre-sample + 之后的 post-sample；并做 MLAP 的**前向 + 反向 OOS** 两关。
- 合格线：① **前向 OOS**（权重只用 ≤t 数据）仍显著——必要过滤器，能滤掉"客观先验下的纯学习污染"（E[r_OOS]=0）；从 IS 到 OOS 大幅衰减本身就是高维学习的指纹。Linnainmaa-Roberts(2018) 测 36 会计异象：样本内全显著，前样本外仅 8/8/16、后样本外仅 1/10/9。② **反向 OOS**（用 ≥t 估的权重去预测 t−1 收益）也应稳健——很多横截面异象通不过反向 OOS，恰说明它们源于高维学习而非真实溢价/偏差。**只靠提高 t 阈值作用有限，样本外全新数据才是排除虚假因子的最好办法**（Harvey 2016）。注意因子有生命周期、非普适，跨市场未必有效。

**判据5 · 增量截距（对标 FF / 已有因子）**
- 怎么测：以标准模型（CAPM→FF3→FF3+MOM→FF5）因子收益为解释变量、待检因子组合收益为被解释变量做时序回归，看截距 **α 是否显著≠0**；多因子整体用 **GRS / α 检验**。
- 合格线：对 FF3/FF5/已有因子回归后 **α 仍显著**才算有真增量。用 Harvey-Liu(2018) 正交化+自助法**按显著性逐一纳入**因子（最显著→次显著……），逐步评估每个新因子增量，避免一次塞入一堆相关因子互相"喂"显著性。

**判据6 · 与现有因子低相关**
- 怎么测：算新因子与库内已有因子收益序列的相关系数 ρ。
- 合格线：**|ρ|<0.8**（业界准入常用 <80%）。≥0.8 视为冗余，除非增量 α 检验仍显著。

**判据7 · 可投资性（净收益 / 容量 / 换手 / 拥挤）**
- 怎么测：用**有效价差**计净收益、罚换手、限再平衡频率；查 alpha 是否集中在小市值空头；监控拥挤度代理。
- 合格线：计入合理成本后净收益仍正。Chen-Velikov(2019)：120 因子计成本后月均 0.6%→0.3%、OOS 转负，分布近似 N(0)（与随机因子无异）。**OOS 衰减三机理**：①曝光（McLean-Pontiff 2016：OOS 较 IS 降 26%、发表后降 58%，曝光本身 =32%）；②拥挤（A 股可用估值价差/配对相关/因子波动/因子反转四代理，short interest spread 不适用；有 2007 Quant Quake 式踩踏肥尾）；③成本（Kok 2017：BM 策略 alpha 多来自做空微小成长股，A 股做空受限根本无法落地）。alpha 集中在小盘空头或一小撮票 = 可疑。

**判据8 · 防过拟合与先验（贝叶斯先验 + 稀疏 vs 稠密）**
- 怎么测：给每因子赋先验（无逻辑 **2%** / 可能 **20%** / 有理论 **50%**），算贝叶斯 p 值 = φ·P(H0)/[φ·P(H0)+P(H1)]，φ=exp(−t²/2)；做参数稳健、子样本/分市场状态、随机指标 placebo 模拟。
- 合格线：先验校正后后验仍显著（Harvey 2017：市场因子 0.035 站得住、盈利因子 0.053 勉强、规模因子 0.117 存疑）。**稀疏 vs 稠密（KNS/Nagel 第4章）**：characteristics 空间几无冗余，强行"挑几个真因子"（L¹/lasso）会丢定价信息、OOS 极差；对一组相关度量（多个价值/活跃度/概念热度因子）应**PC 化后差异化 L²-收缩**而非择一——极少 PC 即可总结横截面（2 个 PC≈2/3 OOS R²、10 个近最优），删因子前先看它是否对齐高方差 PC，别误删有边际贡献者。**没有合理解释的因子即便样本内 t 很高，也应被先验拉低、降级或淘汰。**

## 三、A/B/C 评级规则（门槛量化）

| 档 | 门槛（须全部满足该档条件） |
|---|---|
| **A（入库主力）** | **八条全过**：判据1 严格档（时序 t≥3.8/FM t≥3.4 且过 FDR/FDP）+ 判据2 经济逻辑硬（先验 50% 或来源明确）+ 判据4 **前向 + 反向 OOS 双稳**、跨市场不崩 + 判据5 对 FF5 回归 α 显著 + 判据6 \|ρ\|<0.8 + 判据7 计成本净收益正、拥挤健康 + 判据8 先验校正后仍显著。 |
| **B（降权/小仓/监控）** | 统计达标但有短板：t>3.0 但未到 3.8/3.4；或经济逻辑可讲但偏弱（先验 20%）；或前向 OOS 显著但衰减明显/反向 OOS 边际；或成本/拥挤侵蚀较大但净收益仍正；或 \|ρ\| 接近 0.8。可入库但**降权、小仓位、持续监控拥挤度**，不作主力。 |
| **C（淘汰/观察）** | 触发任一红线即降 C：t<3.0；或纯统计无经济逻辑（先验 2%、贝叶斯后验无法拒绝 H0）；或**样本外失效 / 通不过反向 OOS**（数据窥探或学习诱导）；或对已有因子回归后 α 不显著（无增量）；或 \|ρ\|≥0.8 冗余；或计成本后净收益转负 / 不可投资（alpha 集中在小盘空头）。 |

> 两条一票否决线：经济逻辑（判据2）配不上成因则最高只能 C；样本外/反向 OOS（判据4）失效即 C。

## 四、179 因子筛查 checklist（逐项可勾）

对每个因子逐条记录，全过→A，有短板→B，触红线→C：

- [ ] **① t 值**：时序 t≥3.8 / FM t≥3.4（或最低 ≥3.0）+ 记录 N + FDR/FDP 校正 + HAC 标准误？
- [ ] **② 经济逻辑**：风险补偿 or 行为偏差二选一讲通？非纯统计？（否 = 直接 C）
- [ ] **③ 来源**：{风险补偿 / 错误定价 / 疑似数据窥探 / 疑似学习诱导 / 未判定}？
- [ ] **④ 样本外**：**前向 OOS** 仍显著？**反向 OOS** 稳健？跨市场/子样本一致？
- [ ] **⑤ 增量**：对标 FF3/FF5/库内因子，α 仍显著（GRS/α 检验）？
- [ ] **⑥ 相关性**：与库内因子 \|ρ\|<0.8？
- [ ] **⑦ 可投资**：有效价差计净收益正？换手/容量/拥挤（估值价差·配对相关·因子波动·因子反转）健康？alpha 非集中于小盘空头？
- [ ] **⑧ 先验/稳健**：先验档（2%/20%/50%）+ 贝叶斯后验 p？相关因子 PC 化 + L²-收缩？参数/子样本/placebo 稳健？

## 五、两个深坑

**深坑一：Buffett 六因子分解——控住已知因子后 alpha 归零。** Frazzini-Kabiller-Pedersen(2018) 用六因子(MKT/SMB/HML/UMD/BAB/QMJ)分解 Buffett 的 13F 组合(1980–2017)：单因子模型下 alpha **5.8%、t=3.09** 高度显著；逐步加因子后，六因子全控住时 **alpha 仅剩 0.3%、t=0.16 完全不显著**，模型解释约 61% 变动，主要归功于 BAB（低 beta）与 QMJ（质量）。教训：**一个看似惊人的 alpha，在没控住已知因子前根本不能当真 alpha**。Buffett 的超额绝大部分是可被系统因子解释的风险暴露（便宜买优质低波公司），而非选股魔法。落到筛查——任何因子的"显著超额"都必须先对标 FF5 + 库内因子做增量回归（判据5），α 归零则无独立价值，判 C 或并入已有因子。

**深坑二：事后构造的历史变量——样本内 + 伪样本外双双假阳性。** Nagel(2021) 警示：今天的研究者能轻松构造当年投资者**根本拿不到/无力处理**的变量（如对财报做自动文本分析）。通过高维学习模型看，这等于当年定价被迫**过度稀疏**；而过度收缩/稀疏会让收益里出现一个**不仅样本内、连伪样本外也可预测**的成分（图 5.3：先验比客观先验收紧，OOS 期望收益从恒 0 升到正）。**即便前向 OOS 为正也未必是真 alpha**，须再过三关：变量当年可得性、反向 OOS 稳健性、回归外佐证（宏观风险暴露 / 投资者预期数据）。这对**另类/概念数据因子尤其要命**——本项目同花顺/东财的**概念指数、概念热度、活跃度/人气**等，当年投资者无法获取或无力处理，等价于"过度稀疏"情形，**即便伪样本外为正也不宜直接归因为 alpha**；应核查历史可得性与构造是否含前视，优先用反向 OOS + 回归外证据复核。
