# 因子投资·方法与实践 · §2.7 广义矩估计（GMM）+ §2.8 研究方法建议 [PDF p.84–96]

## 本块讲什么

两部分。

**§2.7 广义矩估计（Generalized Method of Moments, GMM）** —— 全章方法论的"进阶/收口"。GMM 由 Hansen(1982) 提出，最初为检验非线性的基于消费的资产定价模型（Consumption-based CAPM, CCAPM；Lucas 1978 / Breeden 1979）而生，但其"标准化"流程同样能用来检验**线性多因子模型**（Cochrane 2005 第13章）。本书把前面所有方法（排序、时序回归、Fama-MacBeth、GRS）背后的统计机器统一到 GMM 这一个框架里。分 5 小节：
- 2.7.1 样本均值的方差 —— GMM 的"地基"，全部公式最终都归结为它。
- 2.7.2 分析框架 —— GMM 的三部分（提出模型 / 估参 / 检验）。
- 2.7.3 数学基础 —— var(b̂) 与 var(g_T) 怎么算（答案：S/T + delta 方法）。
- 2.7.4 有效性 —— 最优权重矩阵 W=S⁻¹ 的来历与含义。
- 2.7.5 不应成为黑箱 —— 必须从经济学先验选矩/权重，不能无脑塞数据。

**§2.8 研究方法建议** —— 全章收尾，把"研究异象 / 研究因子 / 比较多因子模型"该用哪套方法写成了**惯例清单**。这正是可以直接抄进研究流程 / scorecard 的方法论清单。

作者反复强调的一句话："GMM 的核心最终能够归结为计算**样本均值的方差**（the variance of the sample mean）"。看懂这句，整章公式都不可怕。

## 方法论要点（重点·展开）

### A. 一句话抓住 GMM：一切归结为"样本均值的方差"

Cochrane(2005) 说学 GMM 最大障碍是符号多。本书的"破局"方式：先把"样本均值的方差"讲透（2.7.1），后面所有看似吓人的转置 / 求逆公式（2.115、2.116、2.122、2.125…）**本质都只是 (样本均值的方差 S/T) × (delta 方法)**。

为什么因子检验需要 GMM？因为它提供了一个**统一、稳健**的框架来同时回答三件事：模型对不对？参数怎么估？误差是运气还是模型错了？——而且它给出的标准误天然可以处理**异方差 + 自相关**（这正是金融时间序列的标配毛病）。

### B. 样本均值的方差：从 iid 到自相关（2.7.1）—— 稳健标准误的地基

考虑随机变量 u_t（在一个样本里取 0,−1,3,3,−3，样本均值 ū=0.4）。关键认识：**样本均值本身也是随机变量**（换个样本就变），所以要问它的方差。

- 最简单情形——u_t **独立同分布（iid）**：
  - var(ū) = σ²(u_t) / T  …(2.101)
  - s.e.(ū) = σ(u_t) / √T  …(2.102) ← 这就是统计课上"印象最深"的那个 √T 标准误。
- **但金融收益率序列有非零自相关 cov(u_t, u_{t−j}) ≠ 0，不满足 iid。** 必须用一般式：
  - var(ū) = (1/T)·[ var(u_t) + Σ_{j=1}^{T} ((T−j)/T)·( cov(u_t,u_{t−j}) + cov(u_t,u_{t+j}) ) ]  …(2.103)
  - T→∞ 时 (T−j)/T → 1，渐进式：var(ū) → (1/T)·Σ_{j=−∞}^{∞} cov(u_t, u_{t−j})  …(2.104)
  - 再设总体均值 E[u_t]=0，用 var(X,Y)=E[XY]−E[X]E[Y]：
    **var(ū) → (1/T)·Σ_{j=−∞}^{∞} E[u_t u_{t−j}] ≡ S / T**  …(2.105)

**S 称作 u_t 的"谱密度矩阵"（spectral density matrix）**，等于所有阶自协方差之和（即"长期方差/long-run variance"）。书里直说："千万不要小看 var(ū) → S/T 这个式子，它在 GMM 的数学推导中起着至关重要的作用。"

> 这条就是稳健标准误（Newey-West 一类）的全部理论来源：有自相关时，样本均值的方差不是 σ²/T，而是 S/T，S 把所有自协方差都加进来。详见下文「踩坑对照」。

### C. GMM 三部分框架（2.7.2）

GMM = 提出模型 → 把模型与数据连起来估参 → 检验。

**第一部分：总体矩条件（提出模型 = 写下原假设）。**
令 x_t 为数据、b 为参数（均为向量）。把猜想的模型写成函数 f(x_t, b)，使得当 b=b_0（真值）时：
- **E[ f(x_t, b_0) ] = 0**  …(2.106) ← "总体矩条件"，就是待检验的原假设。
- （注意符号：**E = 对总体求期望；E_T = 对样本求均值**。）

资产定价里这一步特别干净。Cochrane(2005)：所有资产定价理论都归结为一个式子
- **p_t = E[ m_{t+1} · x_{t+1} ]**  …(2.107)，m=随机折现因子（SDF，由 b_0 决定），x=未来回报，p=现价。
- 无风险资产（投 1 得总回报 R_f^g）：E[ m(b_0)·R_f^g − 1 ] = 0  …(2.108)
- **资金中性多空对冲组合**（某因子/异象组合，靠卖空融资买多，成本 p_t=0，超额收益 R^e）：**E[ m(b_0)·R^e ] = 0**  …(2.109) ← 因子/异象检验就长这样。

**第二部分：样本矩替总体矩，估参（把模型连到数据）。**
- 样本矩 g_T(b_0) ≡ E_T[f(x_t,b_0)] = (1/T)Σ_{t=1}^{T} f(x_t,b_0)  …(2.111)。**g_T 也是一种样本均值**（呼应 A 段）。
- 找 b̂_0 使所有样本矩尽量为零：E_T[f(x_t, b̂_0)] ≈ 0  …(2.112)。用"≈"是因为实务中**矩个数 n > 参数个数 p（过度识别 overidentification）**，无法让 n 个矩同时精确为零。
- 于是取其中 p 个线性组合令其为零，得 GMM 估计量：**选 b̂_0 使 a·g_T(b̂_0) = 0**  …(2.113)。a 是 p×n 矩阵，每行=样本矩的一个线性组合。a 可任选，但有个特殊的 a 会让估计有效（见 E 段）。

**第三部分：算方差、做检验。**
- 关心两个方差（都是方差-协方差矩阵）：var(b̂_0)（p 阶，参数估得准不准）与 **var(g_T(b̂_0))（n 阶，样本矩偏离零是运气还是模型错）**。后者是检验模型的关键。
- 渐进分布（Hansen 1982 定理3.1 / 引理4.1）：
  - √T·(b̂_0 − b_0) →d N( 0, (ad)⁻¹ a S a' (ad)⁻¹' )  …(2.115)
  - √T·g_T(b̂_0) →d N( 0, (I − d(ad)⁻¹a) S (I − d(ad)⁻¹a)' )  …(2.116)
- **关键前提假设：数据 x 须满足弱平稳性（weak stationarity）**——因为 GMM 的根基是 T→∞ 时样本均值收敛到总体均值。
- 用 g_T(b̂_0) 的分布构造检验统计量；以资产定价为例，矩条件 = 各资产/组合的定价误差 α，检验"α 联合是否为零"。统计量超阈值 → 拒绝模型。

### D. 数学基础：全靠 S/T + delta 方法（2.7.3）

把第三部分的方差算出来，只用两块积木。

1. **var(g_T(b_0)) = S/T**：因 g_T 就是 f 的样本均值，直接套 2.7.1 的 (2.105)：
   - var(g_T(b_0)) → (1/T)·Σ_{j=−∞}^{∞} E[ f(x_t,b_0) f(x_{t−j},b_0)' ]  …(2.117)
   - **S ≡ Σ_{j=−∞}^{∞} E[ f(x_t,b_0) f(x_{t−j},b_0)' ]**（n 阶矩阵，实务用样本估计）  …(2.118)
2. **delta 方法（一阶泰勒展开）**：定义雅可比矩阵
   - **d ≡ E[ ∂f(x_t,b_0)/∂b' ] = ∂g_T(b̂_0)/∂b'**（n×p，"分子布局/雅可比布局"）  …(2.120)
   - 对 a·g_T(b̂_0)=0 在 b_0 处展开 → b̂_0 − b_0 = −(ad)⁻¹ a g_T(b_0)  …(2.121)
   - 两边求方差：**var(b̂_0) = (1/T)(ad)⁻¹ a S a' (ad)⁻¹'**  …(2.122)
   - 同法对 g_T 展开：g_T(b̂_0) = [I − d(ad)⁻¹a] g_T(b_0)  …(2.124)
   - **var(g_T(b̂_0)) = (1/T)·[I − d(ad)⁻¹a] S [I − d(ad)⁻¹a]'**  …(2.125)

直觉一条：**var(g_T(b̂_0)) < var(g_T(b_0))** ——因为估 b̂_0 时用掉了 p 个"令其为零"的约束，这些约束"消耗"了样本矩的部分变动。

检验统计量（χ²）：
- g_T(b̂_0)' · var(g_T(b̂_0))⁻¹ · g_T(b̂_0) ~ χ²_{n−p}  …(2.126)/(2.127)
- **自由度 = n − p**（估参用掉 p 个自由度）。也因此 var(g_T(b̂_0)) **不满秩**，(2.127) 里的"求逆"实际是**伪逆（pseudo-inverse）**。

作者总结：公式看着复杂，但实质只是"样本均值的方差 S/T + delta 方法"。用 GMM 时只需喂给它 a、g_T、d、S 四样东西，剩下交给统计软件。

### E. 有效性：最优权重矩阵 W = S⁻¹（2.7.4）

如何选 a？两条路。

- **统计最优**：Hansen(1982) 证明特殊的 **a = d'·S⁻¹**  …(2.128) 使 var(b̂_0) 最小（有效估计量, efficient GMM estimator）。
- **等价的权重矩阵写法**：GMM 也可写成
  - **b̂_0 = argmin g_T(b)' · W · g_T(b)**  …(2.129)，W=半正定权重矩阵（让 n 个矩的加权范数尽量近零）。
  - **最优权重 W = S⁻¹**。直觉极好懂：S/T 是矩的方差，**用 S⁻¹ 加权 = 给"误差大"的矩低权重、"误差小"的矩高权重**——更信赖准的矩 → var(b̂_0) 最低，这就是"有效"。
- 两种写法等价：对 (2.129) 取 W=S⁻¹ 求一阶条件，括号里恰好凑出 d'·S⁻¹ = a（2.130→2.135）。一般地，任一 W 都对应一个 a = d'W。

当取最优（a=d'S⁻¹ 或 W=S⁻¹）时，公式大幅化简（Hansen 1982）：
- var(b̂_0) = (1/T)·(d' S⁻¹ d)⁻¹  …(2.136)
- var(g_T(b̂_0)) = (1/T)·( S − d (d' S⁻¹ d)⁻¹ d' )  …(2.137)
- **T · g_T(b̂_0)' · S⁻¹ · g_T(b̂_0) ~ χ²_{n−p}**  …(2.138)
- ⚠️ 这些简化式**只在 W=S⁻¹ 时成立**。很多资料默认 W=S⁻¹ 却不写前提，照抄要当心。

**实务两阶段法**（解"先有 b̂ 才能估 S、却又要 S⁻¹ 才能得有效 b̂"的鸡生蛋）：
1. 第一阶段取 W=I（单位阵）估出初始 b̂_0；
2. 第二阶段用它估 S，令 W=S⁻¹ 再估一次。可迭代多次。

### F. 不应成为黑箱（2.7.5）—— 最重要的态度

GMM 太强、软件太方便，于是有人"把问题写成矩条件后一股脑塞进 GMM，再无脑选 W=S⁻¹ 求有效估计"。**书里直说：这么做十分危险。**

- GMM 真正的强大不在于能做检验，而在于**灵活**——能让你**从经济学/金融学先验出发，定义最适合该问题的 a（或 W）**，而不是无条件 W=S⁻¹。
- CCAPM 例子（4 资产 = 市场、无风险、HML、SMB；2 参数）：**从业务出发选 a**，令市场超额收益与无风险资产两个矩条件被**完美满足**（用来估两个参数 b̂_1, b̂_2），用**另两个资产（HML、SMB）的矩来检验** CCAPM。χ² 自由度 = 4−2 = 2，**检验的实质就是 HML、SMB 在 CCAPM 下的定价误差联合是否显著≠0**。
- 有效性 W=S⁻¹ 是"**给定这组矩**"才成立的；换矩/加矩，有效估计就变。市场上有无数资产→无数矩，一味追求有效就该把成千上万资产全塞进去——但那毫无业务意义。
- 结论金句：**"不要让 GMM 成为计量经济学的黑箱。"** 资产定价研究应**用业务含义最重要的资产去检验定价模型**，而不是做一个"统计上正式、模型上无意义"的检验。

> 花絮（脚注）：Hansen 因 GMM 及其系列扩展，2013 年与 Eugene Fama、Robert Shiller 同获诺贝尔经济学奖。

### G. §2.8 研究方法建议 —— 可直接抄的研究流程清单 ★

全章方法在此收成"惯例"。分两类问题。

**① 研究异象（anomaly）—— 三步递进，标准越来越严：**
1. **排序法（§2.1）测绝对收益**：先用异象变量做单变量排序（构造异象组合、看收益）；为控市值常再与市值**双重排序**。⚠️ **排序法只能检验异象的"绝对收益"，不能证明它是超额收益。**
2. **时序回归（§2.4.1）测 α（超额收益）**：以某多因子模型的**因子收益为解释变量**、**异象组合收益为被解释变量**，看截距 α 是否显著≠0。α 显著 ⇒ 现有多因子模型解释不了 ⇒ "发现一个异象"。
3. **Fama-MacBeth 回归（§2.4.4）控制其他变量**：排序法难控住其他变量对预测性的干扰。把**异象变量 + 常见控制变量当解释变量、个股超额收益当被解释变量**，每期截面回归 → 得到异象变量的系数（=异象收益率）时间序列 → 其**均值显著≠0** ⇒ 控制其他变量后仍能预测收益 ⇒ 确认异象。

**② 研究因子 / 多因子模型 —— 两件事：模型含哪些因子；如何比模型。**

选因子（"哪些因子"）的克制原则：发表的异象 >400 个，但选定价因子很克制：
- **必要条件：有明确的金融学依据。** 要么是风险解释（如 Fama-French 2015 / Hou et al. 2015 的盈利、投资因子源自金融学原理），要么是行为金融的错误定价解释。**没有经济学故事的统计异象不配当定价因子。**
- **因子自身必须"被定价"**：因子是收益驱动力，它本身要能获得显著超额收益。检验法：排序法构因子组合看收益显著性 / **Fama-MacBeth 回归（§2.3）** 看因子收益率是否显著。

模型能否解释截面 & 模型比较：
- **单模型解释力**：选**已发表异象当测试资产**，用 **GRS 检验（§2.5.1）** 看模型能否解释其预期收益。
- **比较两模型，同一组测试资产**：**GRS 检验 + α 检验（§2.5.4）**——看哪个模型下测试资产超额收益更接近零。
- **比较两模型，因子互为测试资产**：**GRS 检验 + 均值-方差张成检验（§2.5.2）**。

（第3~5章即用以上方法对中国 A 股做实证：第3章常见因子、第4章七个主流多因子模型、第5章异象。）

## 关键公式 · 定义 · 判定标准

- iid 样本均值标准误：s.e.(ū) = σ(u_t)/√T  …(2.102)【仅 iid 成立，金融数据慎用】
- 一般样本均值方差（含自相关）→ **S/T**：var(ū) → (1/T)Σ_{j=−∞}^{∞} E[u_t u_{t−j}] ≡ S/T  …(2.105)
- 谱密度矩阵 / 长期方差：**S ≡ Σ_{j=−∞}^{∞} E[f(x_t,b_0) f(x_{t−j},b_0)']**  …(2.118)
- 总体矩条件（原假设）：E[f(x_t,b_0)] = 0  …(2.106)
- 资产定价基本式：p_t = E[m_{t+1} x_{t+1}]  …(2.107)；多空组合：E[m·R^e]=0  …(2.109)
- 样本矩：g_T(b_0) = (1/T)Σ f(x_t,b_0)  …(2.111)
- GMM 估计量两写法：a·g_T(b̂_0)=0（a 为 p×n）…(2.113)　⇔　argmin g_T(b)'W g_T(b) …(2.129)
- 雅可比：d ≡ E[∂f/∂b']（n×p）…(2.120)
- 有效权重 / 有效 a：**W=S⁻¹**，**a=d'S⁻¹**  …(2.128)
- 有效情形方差与检验：var(b̂_0)=(1/T)(d'S⁻¹d)⁻¹ …(2.136)；**T·g_T(b̂_0)'S⁻¹g_T(b̂_0) ~ χ²_{n−p}** …(2.138)
- **判定标准（检验定价模型）**：构造 χ² 统计量（定价误差 g_T 的加权平方），自由度 **n−p**（n=矩数, p=参数数）；超显著性阈值 → 拒绝模型。
- **前提核对**：① 数据弱平稳；② 过度识别需 n>p（n=p 时无需权重，可令所有矩=0 精确求解）；③ 简化公式仅在 W=S⁻¹ 下成立。

## 踩坑对照

- **√252 / 重叠收益年化** —— ★强相关，本块给出根本原理。
  - 现象：把日/周收益的均值、夏普、IR、t 值简单乘 √252 或 √T 来年化。
  - 书里为什么不对：(2.102) 的 s.e.=σ/√T **只在 iid 下成立**。金融收益率自相关 cov(u_t,u_{t−j})≠0（重叠样本 overlapping returns 会**人为制造强自相关**），此时样本均值的真实方差是 (2.103)/(2.105) 的 **S/T**——要把**所有阶自协方差都加进来**（S=谱密度矩阵/长期方差）。忽略自相关 ⇒ 方差被**低估** ⇒ 标准误偏小、t 值/夏普/IR 被**高估**，把噪声当显著。
  - 正确做法：用稳健的"长期方差"S 来算标准误，即 Newey-West（HAC）一类对自相关-异方差稳健的估计——本质就是用样本估 (2.118) 的 S。GMM 框架天然内置这一点（var=S/T），所以一切建立在均值/回归系数显著性上的检验，标准误都应走 S 而非 σ/√T。年化时也应在"已正确处理自相关的方差"基础上换算，而不是对幼稚 t 值再乘 √T。
- **单调性判定** —— 本块未涉及。
- **合成稀释尾部** —— 本块未涉及。

## 对 scorecard / 回测框架的直接启示

1. **§2.8 清单直接落地为研究/打分流程**：
   - 异象 = 排序法（绝对收益，必经但不充分）→ 时序回归 α 显著（超额收益）→ Fama-MacBeth 控变量后系数均值仍显著。三关递进，**少一关不能下"发现异象"的结论**。scorecard 可设三档置信度。
   - 因子入库门槛 = **(a) 明确金融学依据（风险 or 错误定价，必要条件，一票否决纯数据挖掘）+ (b) 因子自身被定价（排序/FM 看收益显著）**。
   - 模型评估 = GRS（以已发表异象为测试资产）；模型对比 = 同测试资产用 GRS+α 检验、因子互为测试资产用 GRS+均值-方差张成检验。
2. **所有显著性检验都要用稳健（HAC/S 矩阵）标准误**，尤其涉及重叠收益、长 horizon 收益、自相关明显的策略；幼稚 σ/√T 一律视为下限不可信。
3. **GMM 当统一引擎**：排序、时序回归、Fama-MacBeth、GRS 都可写成矩条件，纳入 GMM → 标准误口径统一、可同时容纳异方差+自相关。框架内 χ²(n−p) 给出"定价误差联合是否为零"的总检验。
4. **用前先查弱平稳性**：因子收益/特征若非平稳，GMM（及大多数显著性结论）地基不稳。
5. **反黑箱纪律**：检验定价模型时，从经济学先验选"业务含义最重要的资产/矩"，不要为追求统计有效性而堆砌成千上万个矩；宁要"有经济含义"的检验，不要"统计上漂亮但模型无意义"的检验。
6. 实务跑 GMM 用**两阶段法**（先 W=I 估初值，再 W=S⁻¹ 重估，可迭代）。

## 术语与存疑

**术语：**
- GMM（Generalized Method of Moments, 广义矩估计）；CCAPM（Consumption-based CAPM, 基于消费的资产定价模型）。
- 总体矩 / 样本矩（population / sample moments）；矩条件（moment conditions）。
- 过度识别（overidentification, n>p）。
- 谱密度矩阵（spectral density matrix）= 长期方差（long-run variance）S = 全阶自协方差之和。
- SDF（stochastic discount factor, 随机折现因子 m）。
- 权重矩阵（weighting matrix W）；有效估计量（efficient estimator）。
- delta 方法（delta method, 一阶泰勒展开求渐进方差）。
- 两阶段 GMM（two-stage GMM）。
- 雅可比 / 分子布局（Jacobian / numerator layout）vs 黑塞 / 分母布局（Hessian / denominator layout）；伪逆（pseudo-inverse）。
- 弱平稳性（weak stationarity）。
- GRS 检验、均值-方差张成检验（mean-variance spanning test）、Fama-MacBeth 回归（均为前面小节方法，§2.8 处汇总引用）。
- HAC / Newey-West（书中未点名，但 (2.103)/(2.118) 的 S 即其理论对象，本人补注）。
