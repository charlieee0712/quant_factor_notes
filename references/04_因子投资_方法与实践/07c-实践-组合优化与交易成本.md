# 因子投资·方法与实践 · 7.3 投资组合优化（错位/目标函数/约束/交易成本）[PDF p.317–327]

> 石川/刘洋溢/连祥斌《因子投资：方法与实践》· 视觉精读(PDF 物理页 p.317–327) · 生成 2026-06-30

## 本块讲什么

主动管理三件套的最后一步：把**收益模型**（7.1 节，预测 μ）和**风险模型**（7.2 节，估 Σ）拼到一起做**投资组合优化**，求最优权重 ω。本块五个小节：

- **7.3.1 错位的收益与风险模型（misalignment）** —— 当收益模型的预测变量与风险模型的因子变量不一致时，优化会被系统性带偏；机理 + 两条解决思路。
- **7.3.2 目标函数** —— 均值-方差（MVO）、最小方差、最大多样化、风险平价四种，及它们的解析解。
- **7.3.3 不同目标函数的比较** —— 各目标函数在什么参数条件下等价于 MVO；用三资产数值实验展示放松条件后的差异。
- **7.3.4 约束条件** —— 七类约束（卖空/杠杆/上下限/换手率/持仓数量/因子暴露/跟踪误差）。
- **7.3.5 交易成本模型** —— 线性/二次成本函数，如何作为惩罚项进目标函数。★ 本块对回测框架成本假设最直接。

主线判断（书的态度）：**μ 极难预测且 MVO 对 μ 高度敏感**，所以实务里常"退而求其次"，仅用 Σ 构造目标函数；但放弃区分 μ 不等于放弃收益模型——收益模型仍用于"先筛出预测值高的股票"，只是不再细分它们之间的收益差。

## 方法论要点（重点·展开）

### 一、错位问题（7.3.1）——为什么"独门 alpha + 第三方 Barra 风险模型"会出事

**场景**：基金经理普遍有独门收益模型，但风险模型多采购第三方商用多因子模型（如 Barra）。收益模型里的预测变量 ↔ 风险模型里的因子变量**无法完全对齐**，即式(7.50)的暴露 β_α ≠ 式(7.51)的暴露 β_R。这就是 misalignment。Lee & Stefek (2008)、Bender et al. (2009)、Karels & Sun (2013) 是这方向的代表研究。

**机理（核心推导）**：把收益模型给出的超额收益 R^α，按"能否被风险模型的 K 个因子张成的超平面解释"做正交分解（式7.54）：
- **R^α_∥**（平行分量）= R^α 在风险因子超平面内的投影；
- **R^α_⊥**（垂直分量）= 残差，与风险因子超平面正交，满足 **β'_R · R^α_⊥ = 0**。

关键后果：因为 β'_R·R^α_⊥=0，**从风险模型的角度看 R^α_⊥"没有任何风险"**。把无约束最优解 ω*=(1/ζ)Σ⁻¹R^α 用分解代入（并设各股特质风险相同），得到 ω* 是两部分的加权平均（式7.55）：
- 对应 R^α_⊥ 的部分：优化**只惩罚了特质风险 σ_s²**；
- 对应 R^α_∥ 的部分：优化**同时惩罚了因子风险和特质风险**。

→ 错位让优化器**误以为 R^α_⊥ 没有因子风险，于是给它过高权重**。但"盈亏同源"：R^α_⊥ 是风险模型解释不了的收益，**绝不等于没有风险**。结果是这部分收益的风险被系统性低估，最优组合在因子暴露上偏离使用者本意。**要解决的就是"正确评估 R^α_⊥ 的风险并对它惩罚"。**

**两条解决思路：**

**思路 A — 调整风险模型**（直接，但实务受限）。由简到繁三种：
1. 对 β_α 与 β_R 中"相似的列"（相似暴露≈相近因子），从风险模型(7.51)中**剔除**对应因子——操作上把 β_R 的某些列整列置 0。
2. 对相似因子，用**收益模型的因子暴露替换**风险模型对应暴露（β_α 某列 → β_R 对应列），仅换暴露、不动因子收益率。
3. 用收益模型的因子 + 风险模型中与之无关的因子，**联合组成一个全新的风险因子模型**（与方法2区别：方法2只换暴露，方法3用有差异的因子重建模型）。
   - **局限**：商用第三方模型难改动；"如何定义因子相近"本身很模糊，强行识别会引入新的不确定性 → 实际作用有限。

**思路 B — 直接改进优化过程**（书认为更好）。Bender et al. (2009) 在目标函数里对 R^α_⊥ 加额外惩罚（式7.56）：

```
max_ω  (R^α)'ω − (ζ/2)·ω'Σω − θ·(ω'·R^α_⊥)²
```

- θ ≥ 0 是新系数，−θ(ω'R^α_⊥)² 即对 R^α_⊥ 暴露的惩罚。
- **怎么定 θ**：若 R^α_⊥ 是纯噪声（当收益/风险因子非常接近时可能出现）→ 不希望给它任何权重 → θ 取足够大；更一般情形 R^α_⊥ 绝非纯噪声（它是经理独门变量带来的、风险模型解释不了的**正收益**，但仍有风险）→ **θ 应与 R^α_⊥ 的风险成正比**。

**结论提醒**：若收益模型与风险模型**相同**，则不存在错位（7.3.2 起的讨论都默认这一前提）。

### 二、四种目标函数（7.3.2）

输入：μ=股票相对无风险利率的超额收益，Σ=协方差矩阵。

1. **均值-方差 MVO（Markowitz 1952）**：`max_ω μ'ω − (ζ/2)ω'Σω`(7.57)。无约束解 ω_mvo=(ζΣ)⁻¹μ(7.58)；加 Σωᵢ=1 约束后只差一个缩放常数，ω_mvo ∝ Σ⁻¹μ(7.59)。
   - 优点：同时考虑收益与风险（给定风险最大化收益 / 给定收益最小化风险，有效前沿性质）。
   - **致命缺点：对输入 μ、Σ 极度敏感**，参数细微变化→最优权重剧烈波动；而 μ 又最难预测 → 实务诟病。

2. **最小方差 minimum variance**：`min_ω ω'Σω`(7.60)，ω_mv ∝ Σ⁻¹·1(7.61)。**完全不用 μ**。

3. **最大多样化 maximum diversification**：令 σ=[σ₁..σ_N]' 为各股标准差向量；最大化"线性加权波动率 / 组合波动率"之比（比值越大越分散）：`max_ω (ω'σ)/√(ω'Σω)`(7.62)，ω_md ∝ Σ⁻¹σ(7.63)（Clarke et al. 2013）。

4. **风险平价 risk parity / 等风险贡献 ERC**（Qian 2005，PanAgora 首提；桥水"全天候"发扬）：让每个资产对组合风险的贡献相等。等风险贡献条件 ωᵢ·∂σ_p/∂ωᵢ = ωⱼ·∂σ_p/∂ωⱼ ∀i,j(7.64)，转成优化问题 `min_ω Σᵢ[ωᵢ − σ_p²/((Σω)ᵢ·N)]²`(7.65)。
   - **一般无解析解，需数值求解**；仅当所有资产**两两相关系数相等**时，在 Σωᵢ=1 下有闭式 ω_rp,i = σᵢ⁻¹ / Σⱼσⱼ⁻¹(7.66)（Maillard et al. 2010，即按波动率倒数配权）。
   - 配**个股**时"两两相关系数相同"难成立 → 该闭式不太现实；但作为**大类资产配置**最重要方法之一，该假设常可接受，且比直接用历史数据估相关系数**更稳健**。

### 三、目标函数等价关系（7.3.3）

各 Σ-only 目标函数在特定参数条件下退化为 MVO（表7.5）：

| 配置方法 | 等价于 MVO 的条件 |
|---|---|
| 最小方差 | μᵢ = μⱼ, ∀i,j（所有收益率相等） |
| 最大多样化 | μᵢ/σᵢ = μⱼ/σⱼ, ∀i,j（收益波动比相同） |
| 风险平价 | μᵢ/σᵢ 相同 **且** ρᵢⱼ=ρ 两两相关相等（Leote de Carvalho et al. 2012） |
| 等权重(1/N) | μᵢ/σᵢ 相同 **且** ρᵢⱼ=ρ **且** σᵢ=σⱼ（波动率也都相同） |

- **等权重 equal weight**：完全不用 μ、Σ，ω_ew=[1/N,...,1/N]'，无需任何优化；但要满足三个条件才等价 MVO（条件最苛刻）。
- 数值实验（表7.6，三个假想资产，均加 Σωᵢ=1 约束）：从最苛刻条件出发逐一放松，**条件越松，等权重→风险平价→最大多样化依次在与 MVO 的夏普比较中败下阵**（实验四全不满足时夏普：EW 0.50 < RP 0.61 < MD 0.63 < MVO 0.67）。
- **判断**：参数准确估计至关重要；等权重只是"无法准确估参时的无奈之举"。一旦有任何关于 μ/σ/ρ 的靠谱信息都应充分利用，并对未知参数可能满足的条件做合理假设，再选最合适的目标函数。
- 最小方差 vs MVO（表7.7）：只改 μ、协方差不变时最小方差给出**相同**权重——**仅当所有 μᵢ 相同时最小方差才=MVO**；μ 不同时最小方差夏普明显落后（0.67 vs MVO 0.77）。

### 四、七类约束条件（7.3.4）

实务中目标函数必须配约束以体现机构/资金的风险偏好：

1. **卖空约束**：ω ≥ 0(7.67)——多数机构受限不能做空。
2. **杠杆约束**：ω'1 = 1(7.68)——权重和为 1，资金利用率最大化（满仓）。
3. **上下限约束**：L ≤ ω ≤ U(7.69)。动机：因子代表一揽子股票的共性收益，持仓个数不能太少——只有足够分散，**各股特质收益才能相互抵消，组合才反映出共性（因子）收益**；过度集中则难剔除特质收益、遇尾部风险事件易大亏。行业/板块层面：l_p ≤ Σ_{i∈I_p}ωᵢ ≤ u_p(7.70)。**行业中性化**：Hω=h(7.71)，H 为 P×N 哑变量矩阵（股 i 属行业 p 则第 p 行 i 列=1），h 为基准各行业权重——令组合行业分布=基准，规避行业风险（相对收益选股策略中行业常是重要风险源）。
4. **换手率约束**：单股 |ωᵢ−ωᵢ⁰| ≤ φᵢ(7.72)，全组合 Σ|ωᵢ−ωᵢ⁰| ≤ Φ(7.73)，ωᵢ⁰ 为再平衡前权重。动机：信息衰减→需提高再平衡频率→更高换手→更高成本，故必须约束换手，**资金量大的策略尤其重要（"船大难掉头"）**。
5. **持仓数量约束**：哑变量 δᵢ=1(ωᵢ>0)/0(ωᵢ=0)(7.74)，N^L ≤ Σδᵢ ≤ N^U(7.75)。太少→不分散、容量有限；太多→损失灵活性、因子暴露被稀释。
6. **因子暴露约束**：β_k^L ≤ Σᵢβ_ik·ωᵢ ≤ β_k^U(7.76)，组合在因子 k 的暴露限定在上下限内。动机：有些**风险因子**长期溢价不高、波动大，截面上解释不了预期收益差异，却能在时序上解释收益波动（如 Barra 的国家、行业因子）→ 控暴露以降组合波动。**主动权重版**：用 ωᵢ−ωᵢ^B（相对基准）替代 ωᵢ，β_k^L ≤ Σ(ωᵢ−ωᵢ^B)β_ik ≤ β_k^U(7.77)；令 **β_k^L=β_k^U=0 即"风格中性组合"**（在因子 k 上的暴露=基准），业界应用极广。
7. **跟踪误差约束**：((ω−ω^B)'Σ(ω−ω^B))^(1/2) ≤ σ_TE(7.78)。跟踪误差=策略相对基准的主动收益波动率，**指数增强策略**的关键考核指标；σ_TE 为可容忍的最大 TE。

**重要工程提醒**：加入这些线性/非线性约束后，**优化问题通常不再有解析解，需数值求解**；股票多时找全局最优极具挑战；约束参数选得不当甚至会**无解**——此时按约束的优先级适当放松调整。

### 五、★ 交易成本模型（7.3.5）——回测成本假设的直接依据

**成本分两类**（务必在框架里分开建模）：
- **显性成本（explicit）**：交易费用（佣金/规费/印花税等）+ **买卖价差（bid-ask spread）**。
- **隐性成本（implicit）**：主要是**市场冲击成本（market impact）**——大单推动价格不利于自己的部分。

**控制方式两层**：
- 粗：用**换手率约束**间接控成本（主要压低交易"费用"那部分）；
- 细：**直接对交易成本建模，作为惩罚项加进目标函数**，实现"收益 vs 成本"的均衡。

两种常见成本函数（Δωᵢ := ωᵢ−ωᵢ⁰，相对调仓前权重的变化）：

**1. 线性成本函数**（最简单，对应"按成交额固定比例"）：
```
TC(ω) = Σᵢ cᵢ·|ωᵢ − ωᵢ⁰|        (7.79)
```
cᵢ 为股 i 的成本占权重变化的**固定比例**。缺陷：假设每股每笔交易成本比例都一样，**不含市场冲击**——大单小单一个价，不够精细。

**2. 二次成本函数**（线性基础上加二次项刻画冲击）：
```
TC(ω) = Σᵢ cᵢ·|ωᵢ − ωᵢ⁰| + Σᵢ dᵢ·(ωᵢ − ωᵢ⁰)²    (7.80)
```
二次项 dᵢ·(Δωᵢ)² 表示**交易权重变化越大、冲击成本越高**（凸成本）。cᵢ 项≈显性成本/价差，dᵢ 项≈市场冲击。

**纳入目标函数**：设原始目标为 max_ω f(μ,Σ,θ,ω)，则含成本后：
```
max_ω  f(μ,Σ,θ,ω) − γ_TC·TC(ω)        (7.81)
```
γ_TC 为交易成本惩罚系数。用(7.81)再配必要约束，求解即得最优权重。

更详细成本模型见 Lobo et al. 2007、Kopman & Liu 2011、Hedayati et al. 2018、Chen et al. 2019。

## 关键公式 · 定义 · 判定标准

- **错位定义**：β_α（收益模型暴露）≠ β_R（风险模型暴露）。
- **正交分解**(7.54)：R^α = β_R(β'_Rβ_R)⁻¹β'_R·R^α + [I − β_R(β'_Rβ_R)⁻¹β'_R]·R^α = R^α_∥ + R^α_⊥；性质 **β'_R·R^α_⊥ = 0**（残差对风险因子零暴露）。
- **MVO 解**(7.59)：ω_mvo ∝ Σ⁻¹μ ；**最小方差**(7.61) ω_mv ∝ Σ⁻¹·1 ；**最大多样化**(7.63) ω_md ∝ Σ⁻¹σ ；**风险平价（等相关闭式）**(7.66) ω_rp,i ∝ σᵢ⁻¹。
  - 三者形式都是 Σ⁻¹·(某向量)：MVO 用 μ、最小方差用 1（全 1 向量）、最大多样化用 σ、风险平价用 σ⁻¹ 归一。一眼可看出等价条件来源。
- **风险贡献相等**(7.64)：ωᵢ·∂σ_p/∂ωᵢ 对所有 i 相等。
- **等价条件判定**：见上表7.5（最小方差条件最松，等权重最严）。
- **约束式速查**：卖空 ω≥0(7.67)；杠杆 ω'1=1(7.68)；上下限 L≤ω≤U(7.69)；行业 l_p≤Σωᵢ≤u_p(7.70)；行业中性 Hω=h(7.71)；换手 Σ|Δωᵢ|≤Φ(7.73)；持仓数 N^L≤Σδᵢ≤N^U(7.75)；因子暴露 β_k^L≤Σβ_ikωᵢ≤β_k^U(7.76)，主动版(7.77)，风格中性=两限取 0；跟踪误差 √((ω−ω^B)'Σ(ω−ω^B))≤σ_TE(7.78)。
- **成本式速查**：线性 TC=Σcᵢ|Δωᵢ|(7.79)；二次 TC=Σcᵢ|Δωᵢ|+Σdᵢ(Δωᵢ)²(7.80)；带成本目标 f−γ_TC·TC(7.81)。

## A股特殊性 / 实证结论

- **国内研究普遍忽视错位**（书中明确点名）：国内券商金融工程报告里"用不同的收益和风险模型"十分常见，却**对错位问题不做任何处理**——说明该问题在国内尚未受到足够重视。这是直接针对 A 股研究生态的批评，做组合优化时务必自查 β_α vs β_R 的对齐。
- **卖空约束 ω≥0 在 A 股几乎是硬约束**［笔者补充，非书中明说］：A 股融券难、贵、券源少，做空基本不可行，所以 7.3.4 的卖空约束在 A 股是常态，多因子组合实际多为纯多头——这会改变最优权重结构、削弱空头侧 alpha 的可实现性。
- **换手率约束对 T+1/容量更敏感**［笔者补充］：A 股 T+1 + 涨跌停会让"计划换手"与"可成交换手"产生缺口；大资金（"船大难掉头"）叠加流动性分层，换手约束(7.73)与二次冲击成本项 dᵢ(Δωᵢ)² 的标定要按个股流动性分档，不能一刀切。
- **行业中性化(7.71)/风格中性(7.77)** 是 A 股指数增强与中性策略的标配做法；跟踪误差约束(7.78)对应指数增强考核。
- 数值实验结论：在能拿到靠谱 μ/σ/ρ 的前提下 MVO 系最优；信息缺失时才退到风险平价/最小方差/等权重——A 股 μ 预测尤其不稳，故"用收益模型筛股 + 仅用 Σ 定权"的折中范式很常见。

## 踩坑对照

本块未涉及"√252 重叠收益年化""单调性判定""合成稀释尾部"三个指定坑。但本块自带一个**同等量级的踩坑——收益/风险模型错位（misalignment）**，按"现象→为什么→正确做法"记录：

- **现象**：独门 alpha 模型 + 第三方商用风险模型（Barra），两套因子变量不对齐；优化出的最优组合在因子暴露上偏离本意，事后风险收益特征走样。
- **书里为什么**：超额收益 R^α 可分解为风险因子超平面内的投影 R^α_∥ 和正交残差 R^α_⊥。因 β'_R·R^α_⊥=0，风险模型**误判 R^α_⊥ 无风险**，优化只惩罚其特质风险却不惩罚因子风险 → **给 R^α_⊥ 过高权重**。但"盈亏同源"，R^α_⊥ 虽不被风险模型解释，并非真的无风险，其风险被系统性低估。
- **正确做法**：①（次优）调整风险模型——剔除相似因子整列置 0 / 用 β_α 列替换 β_R 列 / 重建风险模型；但商用模型难改、"相近因子"定义模糊，作用有限。②（更优）直接改优化目标，对 R^α_⊥ 加惩罚项 −θ(ω'R^α_⊥)²(式7.56)：纯噪声则 θ 取大，一般情形令 **θ ∝ R^α_⊥ 的真实风险**。

附带可归入"合成/暴露被稀释"思想的一点：**持仓数量约束(7.75)** 提醒——持仓太多会"损失灵活性、因子暴露被稀释"；太少又无法靠特质收益相互抵消来逼出共性（因子）收益。这与"合成稀释尾部"同源：分散过度或合成过度都会摊薄目标暴露，需在分散与暴露强度间取舍。

## 对 scorecard / 回测框架的直接启示

**1. 交易成本必须双层建模，别只扣固定费率。** 框架里把成本拆成 (a) 显性=佣金+规费+印花税+买卖价差 → 线性项 Σcᵢ|Δωᵢ|；(b) 隐性=市场冲击 → 二次项 Σdᵢ(Δωᵢ)²。最小可用版先上线性(7.79)，进阶上二次(7.80)。**cᵢ 反映价差/费率、dᵢ 反映冲击**，dᵢ 应随个股流动性（如 ADV、换手率分位）分档标定，大单/小单不同价。

**2. 成本进目标而非只在事后扣减。** 回测优化器用 max f(μ,Σ,·,ω) − γ_TC·TC(ω)(7.81)，让换手与 alpha 在求解时就权衡，避免"先满仓再扣成本"的高估。γ_TC 作为可调超参，回测里对其做敏感性扫描（与净夏普曲线对照）。

**3. 换手率约束 + 成本惩罚双管。** 换手约束(7.72/7.73)是硬上限（控容量/控费用、应对信息衰减带来的高频再平衡），成本惩罚项是软权衡，两者并用；资金量越大上限收得越紧。

**4. 默认约束清单（A 股实盘版）：** 卖空 ω≥0（A 股几乎必加）、杠杆 ω'1=1、个股上下限 L≤ω≤U（防集中/尾部）、行业中性 Hω=h、风格中性 Σ(ωᵢ−ωᵢ^B)β_ik=0、跟踪误差 ≤σ_TE（指数增强）、持仓数 N^L≤Σδᵢ≤N^U。加约束后无解析解→上数值求解器（QP/凸优化）；**约束要分优先级**，参数过紧导致 infeasible 时按优先级松弛，框架要有可行性回退逻辑。

**5. 错位自检作为 scorecard 一项。** 当收益模型变量与风险模型（如 Barra）因子重叠时，记录 R^α_⊥ 占比与其历史波动；若错位明显，目标函数加 −θ(ω'R^α_⊥)² 惩罚（θ∝R^α_⊥ 风险）。这是国内普遍漏掉的一环，可作为差异化检查点。

**6. 目标函数选择与参数置信度挂钩。** μ 可信→MVO；μ 不可信但 Σ 可信→最小方差/最大多样化/风险平价；什么都不可信→等权重兜底。注意 MVO 对 μ、Σ 极敏感，回测须做输入扰动的稳健性测试（参数小变→权重剧变是危险信号）。持仓权重四式都是 Σ⁻¹·(向量) 形态，实现时复用同一套 Σ 求逆/正则化（Σ 病态时需 shrinkage）。

## 术语与存疑

- **misalignment（错位）**、**R^α_∥/R^α_⊥**（收益在风险因子超平面内的投影分量/正交残差）、**MVO**（mean-variance optimization 均值-方差优化）、**minimum variance**（最小方差）、**maximum diversification**（最大多样化）、**risk parity / equal risk contribution**（风险平价/等风险贡献）、**equal weight**（等权重 1/N）、**industry neutralization**（行业中性）、**style-neutral portfolio**（风格中性组合）、**tracking error**（跟踪误差）、**explicit/implicit cost**（显性/隐性成本）、**market impact**（市场冲击成本）、**bid-ask spread**（买卖价差）。
- **符号约定**：ζ=风险厌恶系数；σ_s=股票特质波动率；ωᵢ⁰=再平衡前权重；ω^B=基准权重；γ_TC=成本惩罚系数；θ=R^α_⊥ 惩罚系数；H(P×N 行业哑变量阵)、h(基准行业权重)。
- **存疑**：式(7.55) 第二项中与 R^α_∥ 相关的那一项，PDF 渲染中末尾的收益符号略糊，按正文文字描述"第二部分和 R^α_∥ 有关"理解为 R^α_∥［符号存疑］；不影响"两部分加权平均、R^α_⊥ 仅被特质风险惩罚"的结论。其余公式(7.50–7.81)与表7.5/7.6/7.7 数值均清晰可读。
