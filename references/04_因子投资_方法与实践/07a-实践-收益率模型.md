## 方法论要点（重点·展开）

### 一、术语辨析：为什么不叫"阿尔法因子"，而叫"(收益)预测变量"(§7.1.1)

业界把能预测收益的量价/财务指标(海外叫 return predictors)常称"阿尔法因子"。本书在式(1.3) `E[R_i] = α_i + β_i'λ` 的统一视角下认为该叫法有**两处错误**，关键是要把"**构建因子的变量**"和"**因子/异象这个投资组合本身**"分开：

- **错误一(因子 ≠ 变量)**：学界惯例里"因子""阿尔法"指的是**投资组合**(异象组合/定价因子组合)，不是背后的变量。例：BM 是变量→用它构建的组合叫价值因子(定价因子)；IVOL 是变量→用它构建的组合叫特质性波动率异象(异象因子)。BM/IVOL 是"构建因子的变量"，价值/IVOL异象才是"因子"。业界却用"阿尔法因子"直接指代变量。
- **错误二(强调α，漏掉β'λ)**：预测变量**既可对应式(1.3)右侧 β'λ 部分(如 BM，是因子变量/"风险因子")，也可对应 α 部分(如 IVOL，是异象变量)**。"阿尔法因子"因强调"阿尔法"似乎只认 α 部分，会漏掉 BM 这类同样能预测收益的变量——但业界其实关心**所有能区分高/低收益股票的变量**，不管它在学界算因子变量还是异象变量。

→ 结论：全书用"**(收益)预测变量**"统称收益率模型的研究对象。**实务启示**：做收益模型时不要被"阿尔法/风险"的标签框住——只要某变量能在截面上区分未来高低收益股票，就有选股价值，纳不纳入取决于下面的六大标准而非它的学术出身。

### 二、到哪里找预测变量(§7.1.2)：三个来源方向

寻找预测变量是主动管理的核心。三条路，难度递增：

1. **从传统量价/财务数据挖新异象**：二三十年前易，如今越来越难(新异象论文逐年减少即是信号)。
2. **改进已有预测变量**：两类改进——(a)**分解**，如 ROA 拆成效率×规模两部分以提升收益(见 6.7.4)；(b)**修正定义以适配经济变迁**，最典型是 BM——科技公司大量研发支出、广告支出未计入净资产，故把**研发+广告支出加回净资产作分子**重算 BM，改进版超额收益更高(Liu et al. 2019)。
3. **另类数据(alternative data)**：传统量价/财务已被挖到"山穷水尽"。只要与传统数据不同、可能含新信息即受重视——交易社区、微观、ESG、电商、卫星气象等(详见 §7.8.1)。

### 三、★挑选预测变量：六大标准 + 检验方法(§7.1.3)——因子 scorecard 实战版

变量不是越多/越复杂越好。是否纳入已有模型，需一套明确标准。**理想预测变量须同时满足六条**，从想法到落地缺一不可：

**① 逻辑性(intuitiveness)——最重要的一条。**
- 因多重假设检验问题(6.1.5)，光靠运气也能在样本内挖出"看似有效"的变量，样本外即失效。逻辑性是排除这种数据窥探(data snooping)干扰的首道闸。"没有底层逻辑，再漂亮的结果都没有灵魂。"
- 逻辑只能来自两个方向：**风险补偿**(经典金融：异象超额收益是对某种未知系统性风险的补偿) 或 **错误定价**(行为金融：投资者偏差 + 套利限制共同导致定价错误且不会迅速消失)。两个方向都解释不了 → 很可能纯属样本内数据窥探。如何区分风险补偿/错误定价/数据窥探见 §6.5。

**② 持续性(persistence)——逻辑要有实证支撑、且发表后不失效。**
- 评价有效性的常用三步：**IC 测试、投资组合排序法、发表前后检验**。
- **IC(Information Coefficient，信息系数)**：主动管理最爱用的指标。定义为 **t 时刻预测变量与 t+1 时刻收益率在截面上的相关系数**：`IC = corr(z_it, R_i,t+1)` (式7.1)。取值[-1,1]，绝对值越大预测力越强；经验上**按日频评价 IC>2% 即优秀变量**。配套指标见下表(式7.2)。
- **投资组合排序法**：用预测变量作排序变量做单/多重排序构组合，再做**时序回归看能否获得相对基准模型的超额收益**(详见 §2.4)，能则称有效。
- **警惕样本外失效**：每个有效变量都代表市场某方面的非有效性；用的人越多，市场越有效，变量越会失效(拥挤交易)。

**③ 信息增量性(information increment)——不能被已有变量解释。**
- 变量不在多，在于**每个变量对预测收益是否有增量信息**。新变量逻辑通、有效后，下一关就是增量。三种方法：
  - **变量相关性**：与已有变量相关性低→可能特立独行、有增量；相关性高→很可能只是已有变量的"影子"。
  - **条件排序(双重排序)**：先按已有变量分组，组内再按新变量分组，看**控制第一变量后**新变量还能否预测收益(见 §2.1.3)。缺点：一次只能控制一个变量。
  - **Fama–MacBeth 回归**：可**同时控制多个**已有变量，检验新变量增量贡献——新变量回归系数显著则是好变量(见 §2.2.4)。
- **正交化/中性化(orthogonalization/neutralization)**：A、B 相关，A 是好预测变量，B 能解释收益波动但选股无超额收益(B 即"风险因子")。为剔除 B 对 A 的干扰，**在股票截面上以 B 为解释变量、A 为被解释变量回归，取残差代替原始 A 当预测变量**——残差与 B 正交，预测时不再受 B 影响。业界常用**行业哑变量 + 市值变量**做 B，得到的残差即"**行业和市值中性的预测变量**"。

**④ 稳健性(robustness)——对构造方法/参数不敏感、跨区间表现一致。** 三个角度：
- **参数**：定义中的设定值。如动量用"过去12个月收益(剔除最近1个月)"，12 是参数；换成 11/13 若表现差异不大则稳健；参数微调就大变→怀疑过拟合。
- **算法**：同一变量的不同算法。如动量用原始收益 vs 风险调整后收益；稳健变量在不同算法下应有相似表现。
- **样本**：把研究区间切成子区间分别测(10年窗 / 商业周期 / 发表前后=样本内外)。优秀变量各区间虽有差异但总体都不俗。

**⑤ 可投资性(investability)——纸面收益能否变实盘收益。** 三个考量：
- **信息衰减(information horizon/信息时间尺度)**：变量预测力随时间衰减(像食物保质期)。**半衰期(Grinold & Kahn 1999)** = 信息衰减一半所需时间；实操用 IC 度量：计算 t 时刻变量值 `z_it` 与**滞后 s 期收益** `R_i,t+s+1` 的 IC，滞后 IC 降到原始 IC 一半所用的期数即半衰期。**高频量价变量保质期短、财务基本面变量保质期长**。衰减快→需高换仓频率才能追上→高换手率/高成本+低容量；需在"信息利用效率 vs 调仓频率"间权衡。
- **市场冲击成本(market impact)**：选出股票流动性差→隐形冲击成本高，超额收益"看得到拿不着"。典型反例 Amihud(2002) 非流动性指标 **ILLIQ**(单位成交额对应的涨跌幅，值越大流动性越差、非流动性溢价越高)——看着诱人实则难交易。
- (信息衰减直接决定**换仓频率与策略容量**。)

**⑥ 普适性(pervasiveness)——跨市场、跨资产都成立。** 优秀代表是价值和动量：Asness et al.(2013) 在美/英/日/欧成熟股市，以及股票/债券/商品期货/外汇等资产中均发现明显价值与动量效应。

### 四、★收益率预测：从变量到选股的完整流水线(§7.1.4)

流水线四模块：**确定投资范围 → 剔除变量异常值 → 预测收益(非参数 / 参数化)**。

**模块1：确定投资范围 = 原始股票池 + 优化股票池**
- **原始股票池**(限定可选范围)：普通投资者=全市场；行业基金=本行业内；指数增强(如中证500增强)=天然限定在该指数成分股，买入沪深300成分即视为"越线"。三类范围见下方表7.3。
- **优化股票池(黑名单)**：在原始池上"防踩雷"，剔三类——
  - **低流动性股票**：控制其他因素后收益随流动性降低而升(非流动性溢价，Amihud 2002 / Pástor-Stambaugh 2003)；但冲击成本高、容量有限，不适合机构资金，故剔(中证指数多数编制方案含流动性剔除规则)。
  - **高风险股票**(见 §3.1)：待退市股、风险警示股(ST/*ST)、净资产为负、次新股；基金合同明令禁投。可自定义扩展如"高商誉""高负债"。
  - **大概率跑输大盘股票**(负超额收益)，三类应预先剔除：①量价上=高短期动量+高换手+高波动的**投机股**；②基本面上=高估值低盈利的**低性价比股**；③事件驱动上=持续负向事件(股东减持、财务造假等)。

**模块2：剔除预测变量异常值**(变量只是特征的代理，应落在合理分布区间；数据质量/算法会产生异常值)。三法对比：
- **缩尾法(winsorization，固定比例法)**：排序后把 <p% 分位 和 >1-p% 分位的值剔除或用临界点替换。最常见，但缺陷多：①异常值本不常见的指标(如异常换手率)强行缩尾会误删合理数据；②异常值多时临界点取小则剔不干净；③分布不对称时两尾用同一百分比不恰当。
- **三倍标准差法(均值方差法)**：正态下 ±1σ/±2σ/±3σ 覆盖 68.3%/95.4%/99.7%，落在 3σ 外概率<0.3%，故距均值 >3σ 视为异常。直观、考虑波动；但**均值/方差本身被异常值污染**——极端异常值会撑大 σ 使 ±3σ 区间过宽，造成"是异常值却识别不到"的盲区。
- **绝对中位差法(MAD, median absolute deviation)**：用中位数替均值、用 MAD 替标准差(两者更抗异常值)。`MAD = median(|X_i − X_m|)`(式7.2，X_m 为中位数)。因正态下 1 个标准差≈1.4826×MAD，故乘系数得 `MAD_e = 1.4826×MAD`；偏离中位数(左右)超过 `3×MAD_e` 即异常，剔除或用 `X_m ± 3×MAD_e` 替换。更稳健、少量异常值不干扰识别；但异常值很多时仍有盲区。

**模块3a：非参数化预测**(不假设变量与收益的具体函数关系，只用二者**单调相关性**)：
- **条件选股法**：两步——(1)选指标(如 BM、ROE、EPS变化率)；(2)各设阈值筛选(如 BM>1/3 且 ROE(TTM)>10%)，选出全部满足者等权持有、定期调仓。**四大不足**：①条件过多→满足者太少甚至为空，过少→太多难交易；②持股数时序上极不稳定；③每变量设阈值→过拟合概率高；④结果难与组合优化模型融合，无法做精细风险控制。
- **排序打分法**：各预测变量分别对个股排序(遵循"排名与预期收益正相关"，排名靠前得分高)，**各变量得分加总取总分最高的 N_h 支**等权配置。优点：可同时考虑多指标、选股数稳定可控、过拟合嫌疑更低，**较好解决了条件选股的缺陷**。缺点：**基础版假定各变量等权**，不考虑各变量预测力差异及其变化(这属因子择时范畴，见 §7.5)。

**模块3b：参数化预测**(线性回归：解释变量=预测变量，被解释变量=收益)：
- **标准化(Z-Score)**：因不同变量量纲不一(如 BM vs 对数市值)，先按截面标准化：`Z-Score = (变量 − 截面均值)/截面标准差`(式7.3，均值与标准差用 t 期股票池所有股票计算)。
- **合成综合 Z-Score**：`综合Z-Score = Σ_{k=1}^K ω_k × Z-Score_k`(式7.4)。理论上权重应正比于各变量预测力；**实践中等权常常就是很好的选择**，偏离等权≈对变量做择时。**关键细节**：合成后截面均值仍为 0，但**截面标准差通常 <1(不再为1)**，故须对综合 Z-Score **再标准化一次**使截面标准差=1。(见踩坑对照：这就是"合成稀释"的数学体现。)
- **★层级处理(hierarchical，学界业界通行的正确合成法)**：当从估值/盈利/成长/安全等**多个维度**找变量、每维度内又有多个细分变量时，**不要把所有细分变量一股脑塞进式(7.4)**。正确做法是分层：**先在维度内合成细分变量→得维度 Z-Score；再把各维度 Z-Score 汇总→得最终综合 Z-Score**。范例：Asness et al.(2019) 质量因子——盈利维度(GPOA/ROE/ROA)、成长维度(毛利润/ROE/ROA 的5年增长率)、安全维度(BAB[Frazzini-Pedersen 2014]/杠杆[总负债比总资产]/ROE标准差)各自合成后再汇总。
- **一元截面回归预测**：`R_it = c_t + b_t·z_{i,t−1} + ε_it`(式7.5，`z_{i,t−1}` 为 t 期期初综合 Z-Score)。每期 OLS 估出 `ĉ_t, b̂_t`，得时间序列后用**最近 T 期均值**作下期参数估计(月频常取 T=12/24/36)：`c̃_{t+1}=(1/T)Σĉ_i`(式7.6)、`b̃_{t+1}=(1/T)Σb̂_i`(式7.7)；代回得预测 `R̃_{i,t+1} = c̃_{t+1} + b̃_{t+1}·z_it`(式7.8)。**另一常见做法**：用最近 T 期收益 + 对应滞后一期综合 Z-Score 做**面板回归(panel regression)**，系数直接当 `c̃, b̃` 代入(7.8)。
- **★α 三要素公式(Grinold 1994 著名经验法则)**：模型(7.5)两侧减基准收益 `R_Bt` → `R_it − R_Bt = (c_t − R_Bt) + b_t·z_{i,t−1} + ε`(式7.9)，b 不变；令超额收益 `α_it ≡ R_it − R_Bt`、`c'_t ≡ c_t − R_Bt` → `α_it = c'_t + b_t·z_{i,t−1} + ε`(式7.10)。OLS：`b̂_t = cov(z,α)/var(z)`(7.11)、`ĉ'_t = ᾱ_t − b̂_t·z̄_{t−1}`(7.12)；因 Z-Score 截面均值为 0(`z̄=0`) → `ĉ'_t = ᾱ_t`。代回得 `α̂_it = ᾱ_t + [cov(z,α)/var(z)]·z_{i,t−1}`(7.13)。第一项是截面均值(**基准为等权指数时严格=0**)，个股预测只取决于第二项；用 `cov=ρ·σ(z)·σ(α)` 化简、又因 Z-Score 的 `σ(z)=1`，得：

  **`α̂_it ≈ ρ(z_{i,t−1}, α_it) × σ(α_it) × z_{i,t−1}`(式7.17) = IC × 超额收益截面波动率 × Z-Score**

  即"**预期超额收益 = IC × 波动率 × 得分**"的 α 三要素法则。这是把"预测变量得分"翻译成"个股预期超额收益"的桥梁，直接喂给组合优化。

## 关键公式 · 定义 · 判定标准

**统一视角(式1.3)**：`E[R_i] = α_i + β_i'λ`。预测变量既可落在 α(异象变量)也可落在 β'λ(因子变量/"风险因子")。

**六大筛选标准(表7.1，即 scorecard 维度)**：
| 标准 | 定义/判定 |
|---|---|
| 逻辑性 intuitiveness | 须有合理逻辑(风险补偿 或 行为金融错误定价)；否则疑为数据窥探。最重要 |
| 持续性 persistence | 有效且长期有效，公开发表后未失效 |
| 信息增量性 information increment | 有差异性，不被已有变量解释，带来独立增量信息 |
| 稳健性 robustness | 对参数/算法/样本不敏感，经得起稳健性检验 |
| 可投资性 investability | 收益不止于纸面，可投资性高、交易成本可控 |
| 普适性 pervasiveness | 广泛存在于不同资产类别/不同国家的同类资产中 |

**IC 及配套评价指标(表7.2)**：
| 指标 | 计算 | 含义 |
|---|---|---|
| IC | `corr(z_it, R_i,t+1)`(式7.1) | 变量信息含量；日频 >2% 为优秀 |
| IR | IC 均值 / IC 标准差 | 预测能力的稳定性 |
| \|IC\|≥2% 比例 | IC 序列中绝对值>2% 的占比 | 预测能力强弱 |
| t 值 | IC 均值的 t 值 | IC 是否显著 |
- **rank IC**：用 Spearman 秩相关代 Pearson(消除分布不均影响)时，IC 称 rank IC(秩信息系数)。

**原始股票池范围(表7.3)**：全部A股(沪深主板/中小板/创业板/科创板) | 指数成分股(中证500、沪深300) | 行业成分股(医药/房地产/国防军工)。

**异常值处理**：MAD 法 `MAD = median(|X_i − X_m|)`(7.2)，`MAD_e = 1.4826×MAD`，越界阈值 `±3×MAD_e`。

**收益预测核心式**：Z-Score(7.3)；综合 Z-Score `Σω_k·Z-Score_k`(7.4，合成后须再标准化使 σ=1)；一元截面回归(7.5) + 参数取最近 T 期均值(7.6/7.7) + 预测(7.8)；α 三要素 `α̂ ≈ IC×σ(α)×Z-Score`(7.17)。

**半衰期**：滞后 IC 降到原始 IC 一半所需期数(Grinold & Kahn 1999)。

## A股特殊性 / 实证结论

- **美股变量到A股"水土不服"**：很多美股研究出的预测变量在A股失效，**很大程度源于两国投资者结构不同**(A股散户主导)。→ 检验持续性时，A股必须用本土数据重做 IC/排序/前后检验，不能照搬美股结论。
- **投资范围的A股制度约束**：指数增强产品(中证500增强/沪深300等)投资范围**天然锁定在对应指数成分股**，买出范围算"越线"；行业基金锁定本行业。
- **A股黑名单(高风险股)落地清单**：待退市股、**风险警示股(ST/*ST)**、净资产为负、**次新股**——基金合同明令禁投；可自定义扩展"高商誉""高负债"等。这是A股(退市制度/风险警示制度/打新次新炒作)特有的 universe 清洗项。
- **中证指数流动性剔除规则(表7.4，A股编制实例)**：
  - 中证沪港深红利低波动指数：非 ST/*ST 且非暂停上市A股中，按近一年(新股自上市以来)A股日均成交额由高到低排序，**剔除排名后 20%**。

## 踩坑对照

### 合成稀释尾部 —— ★本块正面涉及，且给出标准做法
- **现象**：把多个变量的 Z-Score 直接加权求和(式7.4)合成综合得分，会稀释暴露、淹没单变量信息。
- **书里为什么**：合成后综合 Z-Score 的**截面标准差通常 <1(不再为1)**——因为各单变量 Z-Score 间相关性<1，求和后离散度被"摊薄"，这正是合成稀释在数学上的直接体现(截面均值仍=0，但 spread 缩小)。
- **正确做法(书中三条)**：
  1. **合成后再标准化一次**，强制把综合 Z-Score 的截面标准差拉回 1，恢复被稀释的暴露口径。
  2. **层级处理(hierarchical)而非"一股脑塞入"**：先维度内合成→维度 Z-Score→维度间汇总。避免某维度细分变量数量多就主导综合分、把别的维度信息淹没(Asness 2019 质量因子范式)。
  3. **正交化/中性化**先剔除冗余(用行业/市值哑变量回归取残差)，避免相关变量重复计入造成的隐性加权失衡。
- 另：**排序打分/等权合成**默认各变量等权，"不考虑预测力差异及其变化"本身也是一种信息损失，权重优化属因子择时(§7.5)。

### 单调性判定 —— 部分涉及，严谨检验在他节
- 本块明确：非参数化预测(条件选股/排序打分)**"对变量与收益的具体关系不做假设，仅利用二者的单调相关性"**，持续性检验也用投资组合排序法 + 时序回归。
- **但本块未展开严谨的单调性检验方法**(如分组单调秩检验/单调性统计量)，书把投资组合排序法的细节放在 **§2.4**。→ 实操时不要只看多空两端价差，需回到 §2.4 的排序检验框架判定逐档单调。[本块未展开，指向 §2.4]

### √252 / 重叠收益年化 —— 本块未直接展开
- 本块用日频 IC 评价、并用 **IR=IC均值/IC标准差** 与 **IC 的 t 值** 衡量稳定性/显著性——这些**正是需要正确年化与显著性口径**的地方，但本块未讨论重叠样本(overlapping returns)下自相关导致方差低估、Newey–West 调整、不能简单乘 √T 等问题(属其他章节范畴)。→ 落地算 IR/t 值年化时仍须套用书中别处的重叠样本修正。[本块未涉及具体修正]

## 对 scorecard / 回测框架的直接启示

1. **§7.1.3 六大标准 = 现成的因子 scorecard 维度**。可直接做成打分卡：逻辑性(风险/行为可解释?是否过6.5的数据窥探检验) → 持续性(IC>2%? IR? t值? rank IC? 排序法时序α显著? 发表后仍有效?) → 增量性(与现有因子相关性? 条件双重排序通过? FM回归增量系数显著?) → 稳健性(参数±1/算法替换/子区间三测) → 可投资性(半衰期长短→换仓频率/容量；ILLIQ 流动性 / 冲击成本) → 普适性(跨市场跨资产)。**逻辑性权重最高，作一票否决项**。
2. **评价指标面板**：IC、IR、|IC|≥2% 比例、IC 的 t 值、rank IC、半衰期——应作为因子库每个因子的标配输出字段。
3. **回测框架流水线对应模块**：universe(原始池 + 黑名单过滤：低流动/高风险/大概率跑输) → winsorize/MAD 去异常(默认优先 MAD，对偏态/重尾比缩尾稳健) → 截面标准化(Z-Score) → **分层合成(维度内→维度间) + 合成后再标准化** → 一元截面回归(滚动 T 期取均值) 或 面板回归 → 输出个股预期超额收益。
4. **去异常值默认值**：MAD 法 `1.4826×MAD`、阈值 `±3×MAD_e` 可设为框架默认；缩尾仅在分布对称且异常值确实存在时用，并对左右尾分别设分位以应对偏态。
5. **合成模块必须内置"再标准化"步骤**(综合 Z-Score 截面 σ 会<1)，否则下游 α 三要素/组合优化的暴露口径被系统性低估。
6. **α 三要素 `α̂ ≈ IC×σ(α)×Z-Score`** 是把因子得分翻译成"个股预期超额收益"的标准接口，直接作为组合优化器(§7.3)的 expected return 输入；其中 IC 来自因子评价模块、σ(α) 来自风险模块、Z-Score 来自打分模块——三模块在此公式处汇合。
7. **半衰期 → 换仓频率/容量**应作为策略层的硬约束：高频量价因子短半衰期=高换手高成本低容量，需在框架里把"信息利用效率 vs 调仓频率/成本"显式做成可配置权衡。
8. **A股 universe 过滤项**(ST/*ST、净资产为负、次新、暂停上市、流动性后 N% 剔除)应硬编码进回测的股票池清洗层，并对齐目标指数成分(指增产品防越线)。
