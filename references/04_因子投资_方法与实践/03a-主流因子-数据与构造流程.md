## 本块讲什么

第3章正文开篇定调：本书的特色是对 A 股做"严谨、可复制"的实证，分布在第3、4、5章（因子 / 多因子模型 / 异象）。但"巧妇难为无米之炊"——实证最大的坑在数据处理（未来数据迷雾、不切实际的设定）。因此在所有实证之前，§3.1 把**数据来源、量价处理、财务处理、因子构造流程、历史回溯约定**一次讲清，**全书所有实证都遵循这一节**。

本块几乎全是"踩坑→标准做法"的构造方法论，对回测框架最直接。核心五件事：

1. **量价数据清洗**：复权口径、复牌首日异常收益压缩、停牌填充值、最少交易日门槛。
2. **财务数据对齐（重中之重）**：报告期 / 四表一附注 / 基准报告期 / 调整与更正 / **PIT（point-in-time）原则** / 底层表四种记录类型 / 取数时点逻辑 / 单季度数据 / TTM 数据。
3. **股票池构造**：是否剔金融股、黑名单、异常值缩尾。
4. **排序分组与调仓**：单变量 + 独立双重排序、调仓频率、不可交易股处理、交易成本设定。
5. **实证设定总表（表3.11）**：一张表锁死全书回测口径。

## 方法论要点（重点·展开）

### A. 数据来源（§3.1.1）
- 主用 **Wind + Tushare**。基本信息、行情数据、指数成分、宏观数据来自 **Wind**；**财务数据来自 Tushare**。
- 选择依据：数据质量 + 可得性。
- 启示：行情/财务分别来自两个数据商 → 代码/复权/报告期口径可能不一致，跨源拼接时要对齐主键（股票代码、交易日、报告期）。

### B. 量价数据处理（§3.1.2）
全书实证均为**日线级别**，不涉及高频。量价数据 = 开/高/低/收、成交量、换手率、交易状态、涨跌停状态。四个常见坑：

**1. 前复权 vs 后复权**
- 复权的原因：配股、分拆、合并、分红会造成股价跳空缺口；用不复权价计算收益会出错、失去连续性。
- **前复权**：保持"当前价"不变，反向增减历史价 → 历史价**时变**（每次除权除息都要重算历史价，不同时点看到的历史前复权价不同）；对持续分红公司，前复权价**可能为负**。便于看盘/叠加技术指标，是行情软件默认。
- **后复权**：保持历史价不变，每次权益事件后调整当前价 → 可视为投资者**长期财富增长曲线**，反映真实收益率。不适合看盘。
- **本书铁律：无论算收益率还是回溯历史表现，一律用后复权价。**（图3.1：不复权=a+b 段；后复权=a+d 段；前复权=c+b 段。）

**2. 长期停牌股复牌 → 收益率压缩**
- 重大重组/暂停上市后复牌，复牌首日常**不受 10% 涨跌停限制**，产生异常巨大收益（例：★ST 超日 002506.SZ，2014/4/29 停牌，重组更名协鑫集成，2015/8/12 复牌当日涨 593.72%）。
- **标准做法（压缩处理 winsorize-by-rule）**：涨跌停板制度 1996/12/16 实施，故对 **1996/12/15 之后**数据，**日收益率 >10% 用 10% 代替，<−10% 用 −10% 代替**。
- 注意（脚注[3]）：10%/−10% 只是理论值。实际涨跌停价 = 上一交易日收盘价 ×(1±10%) 后**四舍五入到 0.01 元**，故实际涨跌幅会微偏离（例：前收 17.85，涨停理论 19.635→实际 19.64=+10.03%；跌停理论 16.065→实际 16.07=−9.97%）。处理异常值时要考虑四舍五入。**首日上市股不受 10% 限制；ST 股限制 ±5%。**

**3. 填充值（停牌日是填充还是设缺失，按场景定）**
- A 股有停牌制度，停牌日无交易数据。**是否用上一个非停牌日数据填充，必须看应用场景**：
  - **算动量 → 填充 OK**：动量 = 当前价相对 T 日前（基期）涨跌幅，基期若停牌，用上一个非停牌价作价格水平逻辑通顺。
  - **算波动率 → 必须设缺失**：填充会让停牌日收益率=0（无波动），而"无波动"是停牌造成的假象 → **填充会系统性低估历史波动率**。
  - **算 β（个股超额收益对市场超额收益的暴露）→ 必须设缺失**：市场超额收益每天都有，个股停牌日用 0 填 → **β 估计出现较大偏差**。
- 准则：凡是"0 收益会被误当成真实信息"的指标（波动、β、相关性类），停牌日设缺失、不参与计算。

**4. 最少交易日（2/3 窗口门槛）**
- 部分公司滥用停牌、交易断续，小样本会让指标失真。
- **本书规则：用时间窗口计算的指标，只有当有效样本数 > 窗口的 2/3 时才计算，否则置缺失。**（例：月波动率，一个月约 21 个交易日 → 样本数 ≥14 才算。）
- 制度背景：2018/12/28 沪深交易所发布停复牌指引，停牌家数近年大降（图3.2，2005–2019 每日停牌占比）。

### C. 财务数据处理（§3.1.3）——本块最厚、坑最多
财务数据反映企业一段时期经营成果与财务状况变动，比量价复杂得多。

**1. 报告期**
- 《会计法》：会计年度 1/1–12/31。一年披露四次：一季报、半年报、三季报、年报。披露日无严格规定，只需在截止日前。
- 标识用"年份+季度末"：2018 三季报=20180930，年报=20181231。
- 最晚披露日（表3.2）：一季报同年 4/30、半年报同年 8/31、三季报同年 10/31、**年报次年 4/30**。
- 脚注[4]：季度资产负债表+利润表从 **2002 年**起披露，季度现金流量表从 **2003 年**起。

**2. 财务报表 = "四表一附注"**（表3.3）

| 表 | 含义 | 数据性质 |
|---|---|---|
| 资产负债表 | 某时点资产/负债/所有者权益 | **时点数据**（存量） |
| 利润表 | 某区间经营成果（盈/亏）与利润分配 | **区间数据**（流量） |
| 现金流量表 | 各项目对现金流影响（经营/投资/筹资三类） | **区间数据** |
| 所有者权益变动表 | 期初到期末所有者权益增减 | **区间数据** |
| 附注 | 会计政策/变更/非经常项/明细/或有事项/期后事项等 | 关键解释信息 |

- **"时点 vs 区间"是后续单季度、TTM 处理差异的根因**：资产负债表科目不能简单相减/累加，利润&现金流可以。

**3. 基准报告期（披露最新报表时同时披露的历史对照）**
- 规律（表3.4/3.5）：
  - **资产负债表的基准报表 = 永远是上年年报**（因要披露"年初余额"=上年末）。
  - **利润表 / 现金流量表的基准报表 = 上年同期**（同比报告期）。
- 例：白云机场披露 2018 三季报时，资产负债表同时给 2017 年报（年初余额）；现金流量表同时给 2017 三季报（上年同期）。

**4. 调整与更正（financial data 两类"事后变动"）**
- **首次披露的值 = 原始报表（原始值）**。
- **调整（adjustment）**：后续定期报告再次披露某基准报表时，若科目值与原始报表不同 → 发生调整（例：中铁工业 600528.SH，2016 年报原始值 vs 一年后随 2017 年报披露的 2016 基准值，货币资金/应收票据/应收账款/预付款项均被调整）。
- **不定期更正（correction）**：因会计政策变更或统计差错，对以前期报告更正，**需发更正公告、无固定时间规律**（例：柳工 000528.SZ，2018/8/30 披露半年报，2018/9/29 更正——见表3.6，营业总成本/投资收益/营业利润/净利润/EPS 均变）。

**5. 基本原则 —— PIT（point-in-time），全书财务取数的总纲**
- 两种偷懒做法都被否决：
  - 只用首次披露值 → 简单、不引入未来数据，但无法吸收调整/更正的新信息，**信息利用不充分**。
  - 始终用调整/更正后的最新值 → 简单，但历史回溯时**用了未来数据**，偏差大。
- **正确原则：在当前时点利用"可得的最新信息"（point-in-time）。** 既充分利用最新数据（含调整、更正，以及业绩预告、业绩快报），又避免使用未来数据。
- 实证资产定价/因子投资是极精细的过程，必须遵循 PIT。

**6. 数据存储 —— 底层表的四种记录类型**（表3.7/3.8）
同一报告期在底层表可能有多条记录，分四类（以更正前/后、初始/基准交叉）：

| 类型 | 简称 | Wind | Tushare |
|---|---|---|---|
| 类型1 | 初始报表（最新值）| 合并报表 | 合并报表 |
| 类型2 | 基准报表（最新值）| 合并报表(调整) | 调整合并报表 |
| 类型3 | 初始报表（原始值）| 合并报表(更正前) | 调整前合并报表 |
| 类型4 | 基准报表（原始值）| 合并调整(更正前) | 调整前合并报表 |

- 类型1 = 首次披露财报，如有更正则为**更正后**值；类型2 = 最新报告期披露的基准报告期报表（上年同期/上年年报），如有更正则更正后值。
- 类型3/4 = 更正公告发出后，被修改科目的**更正前**旧记录被另存为"原始值"。
- 四条注意：(1) 若无任何科目调整，初始报表与基准报表记录相同；(2) **1998 年之前的数据无调整与更正，每个报告期只有 1 条记录**；(3) 若更正发生在基准报告**之前**，只有 3 条记录（缺类型4）；(4) 若更正发生在基准报告**之后**，则有全部 4 条。

**7. 如何使用（取数时点逻辑，图3.7 四情形）** —— PIT 的落地
总原则：**在你的决策日，取"当时可见的、对应该报告期的最新记录"**。四种情形枚举落到哪个类型：

| 情形 | 时间线 | 取数规则 |
|---|---|---|
| (a) 无调整无更正 | t1 披露 x 年报 | t1 后取**类型1** |
| (b) 有调整无更正 | t1 披露；t2 随 x+1 年报再披露 x 年报基准 | t1–t2 取**类型1**；t2 后取**类型2** |
| (c) 有调整+更正在基准前 | t1 披露；t2 更正；t3 披露调整(基准) | t1–t2 取**类型3**；t2–t3 取**类型1**；t3 后取**类型2** |
| (d) 有调整+更正在基准后 | t1 披露；t2 披露基准；t3 更正 | t1–t2 取**类型3**；t2–t3 取**类型4**；t3 后取**类型2** |

- 脚注[6]：极特殊情形——更正公告与基准报表同日披露（小差错常在基准报告里顺手改），取数时可忽略更正记录，因新基准报表已体现该更正（例 600087.SH 2010 年报）。

**8. 单季度数据**（表3.9 + 图3.8 流程）
- **资产负债表是时点数据 → 单季度数据 = 对应报告期数据**（不相减）。
- **利润表/现金流量表（区间）**：一季度 = 一季报；其余三季 = 当前报告期 − **环比报告期**（上一个报告期）：
  - 二季度 = 半年报 − 一季报
  - 三季度 = 三季报 − 半年报
  - 四季度 = 年报 − 三季报
- **若环比报告期缺失 → 无法计算 → 单季度数据为空（NaN）。**

**9. TTM 数据（Trailing Twelve Months，滚动 12 个月，消除季节性、横向可比）**（图3.9 流程）
- **资产负债表科目（时点）三种 TTM 算法**：(1) 当前最新数据；(2) 最近四个报告期平均值；(3) 当前最新报告期与其同比报告期的平均值。
- **利润表/现金流量表 TTM**：
  - 若最新报告期是**年报** → TTM = 年报数据。
  - 否则 → **TTM = 当前最新报告期 + 上年年报 − 上年同比报告期**。
  - 若上年年报或同比报告期不存在 → **年化算法**：TTM = 当前报告期 × 年化系数（一季报 4、半年报 2、三季报 4/3）。

### D. 因子构造流程（§3.1.4）
有了干净数据，用**投资组合排序法**构造因子。实务要回答三问：股票池怎么定、异常值怎么处理、分组参数与调仓频率。

**1. 是否剔除金融股 → 本书"保留"**
- 学术惯例剔除金融业（银行/证券/保险，经营模式与会计科目差异大），但**本书不剔**，理由三：(1) 本书多数因子有普适性，不涉特殊会计科目；(2) 定位理论+实践，金融业有优质公司，投资角度宜保留；(3) 经典 Fama-French (1993, 2015) 也未刻意剔金融。

**2. 剔除"黑名单"（从股票池过滤）**
- 黑名单 = **待退市股、风险警示股(ST)、净资产为负股、次新股**。
- **风险警示股(ST)**：1998/4/22 沪深所对财务/其他异常股施行特别处理(special treatment)。两种主要异常：(1) 经审计连续两个会计年度净利润为负；(2) 最近一个会计年度经审计每股净资产**低于股票面值**。ST 股面临暂停上市/退市、停牌流动性丧失、剧烈波动；决定退市者复牌后常一连串跌停，是"地雷"。
- **净资产为负**：年报净资产为负可能被列风险警示；故对最近报告期净资产为负者列入黑名单。
- **次新股**：上市时间短（各机构定义不一，通常不超一年；本书定义**上市不足一年**）。高弹性、高波动、无套牢盘、无解禁压力、有高送转预期，易被资金追捧，风险高。

**3. 异常值 → 缩尾法**
- 指标只是股票特征的代理变量，应有合理分布区间；数据质量/算法问题会产生异常值（例：2019/5/28 全市场 ROE(TTM) 最小值 −1227.85%，源于株治集团 600961.SH 亏损过大且净资产太小；表3.10：3365 只，均值 4.16%、标准差 34.53%、中位数 7.03%）。
- 方法有缩尾法(winsorization)、均值标准差法、中位数法、Boxplot 法；**缩尾法最简单常见**（详见 §7.1.4）。
- **本书采用左右各 1% 缩尾：小于 1% 分位数、大于 99% 分位数的样本剔除。**

**4. 排序和分组**（书中此处编号疑为笔误，与"异常值"重号，按内容应为第4点）
- **单变量排序**：按某指标**十分位数**把股票分 **10 组**，统计各组特征与预期收益，检验显著性。
- **独立双重排序**：用**市值**与目标因子两个变量**各自独立**排序、各分 **5 组**，相交得 **5×5=25** 个组合，按式(2.6)算因子收益并检验。**因双重排序默认含市值变量，对规模因子不适用 → 规模因子只用市值单变量排序。**
- **加权**：无论 10 组还是 25 组，组合内股票按**等权重**或**总市值加权**算组合收益。

**5. 因子投资组合调仓频率**
- 惯例：量价类因子（动量）月度调仓；财务类因子（盈利）年度调仓。
- **本书统一每月末换仓**（更快利用因子信息、减少滞后损耗）。
- 调仓日**剔除当天停牌股、一字涨停股**（无法买入成交）。
- **一字跌停股也列为不可交易**：各组合本身是多头组合；因子组合需"挑一部分做多、一部分做空"做多空对冲，故"做多时禁买一字涨停"对应"做空时禁卖一字跌停"，使空头收益更贴近现实。
- **交易成本设为 0**（不考虑）；通常成本需精确建模，或粗估**单边 3‰**。

**6. 研究范围**
- 股票池 = 上交所主板 + 深交所主板 + 中小板 + 创业板的全部股票，即**除去科创板**的全部 A 股。
- 时间：**2000/1/1 – 2019/12/31，整 20 年**。从 2000 起的两个理由：(1) **会计规则 1999 年才成型标准化**，保证财务口径前后一致；(2) 保证样本量、避免统计误差。
- 规模增长（图3.10）：2000 年初 925 只 → 2019/12/31 达 3692 只。

### E. 实证设定总表（§3.1.5，表3.11）—— 全书回测口径
| 项目 | 设定 |
|---|---|
| 数据来源 | Wind 和 Tushare |
| 收益计算 | **后复权收盘价** |
| 长期停牌股复牌首日收益率 | 压缩处理（±10% 封顶） |
| 缺失值处理 | 向前填充 **或者不填充**（按场景） |
| 最少交易日 | 不少于计算窗口 **2/3** 的数据 |
| 财务数据处理 | 遵循 **point-in-time 原则** |
| 股票范围 | **全部 A 股，不剔除金融股** |
| 时间区间 | 2000/1/1 – 2019/12/31 |
| 黑名单 | 待退市股、净资产为负、次新股、风险警示股 |
| 不可交易股 | 停牌股、一字涨停股、一字跌停股 |
| 异常值处理 | 缩尾法 |
| 单变量排序 | 按变量从小到大分 **10 组** |
| 双重排序 | 独立双重排序，两变量各分 5 组，交叉得 **25** 组合 |
| 组合内权重 | 等权重 或 按总市值加权 |
| 调仓频率 | 每月末 |
| 交易成本 | 设为 0 |

- 表3.11 脚注：双重排序分**独立**与**条件**两种。**独立双重排序**=两变量分别排序再相乘得组合，形象称"**交叉法**"（Fama-French 1993 用此）；**条件双重排序**=先用第一变量划分、再在每组内用第二变量划分，形象称"**分层法**"。

## 关键公式 · 定义 · 判定标准

- **复权口径**：收益/回溯一律用**后复权**（后复权 = 历史价不变、调当前价 = 财富增长曲线）。
- **复牌/异常收益压缩**：1996/12/15 后，日收益率 ∈ [−10%, +10%]（理论值；实际值含四舍五入、ST 为 ±5%、上市首日不限）。
- **最少交易日门槛**：有效样本数 > 窗口长度 × 2/3 才计算（月窗 21 日 → ≥14）。
- **基准报告期**：资产负债表→上年年报；利润表/现金流量表→上年同期。
- **PIT 原则**：在当前时点取"可得的最新值"（含调整、更正、业绩预告/快报），禁用未来数据。
- **单季度（区间表）**：Q1 = 一季报；Qn = 当前报告期 − 环比报告期；环比缺失则 NaN。资产负债表单季度 = 报告期值本身。
- **TTM（区间表）**：年报期 → = 年报；否则 = 当前 + 上年年报 − 上年同期；缺失则年化（×4 / ×2 / ×4/3）。TTM（资产负债表）三选一：最新值 / 近四期均值 / 当前与同比均值。
- **异常值**：左右各 1% 缩尾（< P1、> P99 处理）。
- **排序判定**：单变量 10 分组、独立双重 5×5=25 组；规模因子只用市值单变量。

## A股特殊性 / 实证结论

- **涨跌停**：±10% 限制是制度核心 → 复牌首日/异常收益必须压缩；一字涨停（不可买入）、一字跌停（不可卖出/不可交易）在调仓时剔除，否则高估多空收益。理论 ±10% 因四舍五入实际微偏（10.03%/−9.97%）；ST ±5%；上市首日不限。
- **停牌**：停牌日无数据 → 填充 vs 缺失按指标性质区分（动量可填、波动/β 必缺）；滥用停牌 → 2/3 最少交易日门槛过滤；2018 停复牌指引后停牌大降。
- **ST/退市制度**：连续两年净利润为负 或 每股净资产 < 面值 → ST；退市股复牌后连环跌停 → 列黑名单剔除。
- **注册制/科创板**：股票池**剔除科创板**（科创板涨跌幅 ±20% 等制度不同，且上市晚样本短）。
- **会计制度**：1999 年会计规则才标准化 → 研究区间从 2000 年起；1998 年前财务数据无调整/更正、每报告期单条记录。
- **次新股**：上市不足一年，高弹性/无套牢盘/有高送转预期/易被炒 → 黑名单剔除。
- **样本结论**：A 股上市公司数 2000 年初 925 → 2019 年末 3692，逐年增长。

## 踩坑对照

> 本块是全书数据方法论的"踩坑总集"。spec 点名的三坑（√252 重叠年化 / 单调性判定 / 合成稀释尾部）**本块未涉及**（属后续排序检验与多因子合成章节）；但本块自带一批更基础、对回测框架更致命的坑，逐一列出：

| 现象（坑） | 书里为什么 | 正确做法 |
|---|---|---|
| 用前复权价算收益/回测 | 前复权价**时变**（每次除权重算历史价），不同时点看到的历史价不同；持续分红公司前复权价**可能为负** | **一律用后复权价**算收益与回溯 |
| 复牌首日 593% 收益污染 | 重组复牌不受涨跌停限制 → 异常巨值拉偏均值/方差 | 1996/12/15 后日收益**压缩到 ±10%** |
| 停牌日用 0 或填充值算波动/β | 填充 → 停牌日收益=0=假"无波动" → **系统性低估波动率**；β 用 0 填 → **大偏差** | 波动/β/相关类指标停牌日**设缺失、不参与计算**；动量类才可填充 |
| 断续交易小样本算指标 | 滥用停牌 → 有效样本太少 → 指标受小样本影响、偏差大 | **有效样本 > 窗口 2/3 才计算**，否则置缺失 |
| **回测用了"未来才知道"的财务值** | 只用首次值→信息不足；只用最终修正值→**用未来数据**，回测虚高 | **PIT 原则**：每个决策日取"当时可见的最新记录"（四类型 + 图3.7 取数逻辑） |
| 资产负债表科目直接相减做单季度 | 资产负债表是**时点数据**，相减无意义 | 时点表单季度=报告期值本身；只有区间表（利润/现金流）才"当前 − 环比" |
| TTM 直接累加四个季度 | 缺同比/上年年报时无法配平；季节性未消 | 区间表 TTM = 当前 + 上年年报 − 上年同期；缺失则年化(×4/×2/×4/3) |
| 异常值（ROE −1227%）入回归 | 个别公司净资产极小 → 比率爆表，是噪声非信息 | **左右 1% 缩尾** |
| 调仓买入一字涨停 / 卖出一字跌停 | 一字板无法成交，记入会高估多空收益 | 调仓日剔除停牌、一字涨停（买不进）、一字跌停（卖不出） |

## 对 scorecard / 回测框架的直接启示

1. **价格管线**：行情统一存/取**后复权**收盘价；前复权仅供画图。复权因子要随除权除息事件更新。
2. **收益清洗层**：内置"复牌首日/异常收益压缩"（1996/12/15 后 ±10%，ST ±5%，首日不限），并对一字涨停/跌停/停牌打"不可交易"标记，调仓时强制过滤。
3. **停牌处理要按指标分流**：框架需支持"填充 / 置缺失"两套策略——动量类填充、波动/β/相关类置缺失；并实现"窗口有效样本 > 2/3 才出值"的门槛函数。
4. **财务数据必须 PIT 化**：底层财务表存四种记录类型（初始/基准 × 最新/原始值）并带披露日；取数函数输入"决策日"，按图3.7 逻辑返回该日可见的最新记录。**这是避免未来函数的核心，回测正确性的命门。**
5. **派生财务指标**：单季度（区分时点表/区间表、当前−环比、缺失置 NaN）与 TTM（区间表当前+上年年报−上年同期，缺失年化；时点表三选一）应做成可复用算子。
6. **股票池服务**：可配置剔金融股开关（默认不剔）、黑名单过滤（待退市/ST/净资产为负/次新股<1 年）、科创板剔除、起始日 2000-01-01。
7. **排序/分组引擎**：支持单变量 10 分组 + 独立双重 5×5；组合权重等权/总市值两选；规模因子走单变量分支；交易成本可配（默认 0，可切单边 3‰）。
8. **可复现性**：把表3.11 这套设定固化成一份"实证配置"（config），保证所有因子/异象实证口径一致、可被他人复现。

## 术语与存疑

- **后复权 / 前复权**（backward / forward adjustment）；**压缩处理**（rule-based winsorize of returns）。
- **PIT（point-in-time）原则**：当前时点只用可得最新信息，禁用未来数据。
- **四表一附注**：资产负债表(时点)、利润表(区间)、现金流量表(区间)、所有者权益变动表(区间)、附注。
- **基准报告期**：披露最新报表时同时给出的历史对照（资产负债表→上年年报；利润/现金流→上年同期）。
- **调整(adjustment) / 更正(correction)**；**底层四记录类型**：初始/基准 × 最新值/原始值。
- **TTM**（Trailing Twelve Months，滚动十二月）；**环比报告期**（sequential 上一报告期）；**同比报告期**（YoY 上年同期）。
- **缩尾法(winsorization)**：[书中表述存疑] 书中文字写"将小于 1% 分位、大于 99% 分位的样本点**剔除**"——严格意义"缩尾"是把超界值**替换为分位边界值**，"剔除"更接近截尾(trimming)；按本书 §7.1.4 应以缩尾(替换)为准，此处文字疑为简化。
- **独立双重排序=交叉法**；**条件双重排序=分层法**。
- **黑名单**：待退市股、风险警示股(ST)、净资产为负股、次新股。
- **不可交易股**：停牌股、一字涨停股、一字跌停股。
- **科创板剔除**：本书股票池不含科创板（制度差异 + 样本期短）。
- [图中存疑] §3.1.4 中"排序和分组"小节编号与"异常值"重号（均印作"3."），按内容顺序应为第 4 点；不影响内容理解。
