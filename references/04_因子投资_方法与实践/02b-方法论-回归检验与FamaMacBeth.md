# 因子投资·方法与实践 · §2.2 多因子模型的回归检验（时序 / 截面 / Fama–MacBeth） [PDF p.40–54]

> 石川/刘洋溢/连祥斌《因子投资：方法与实践》· 视觉精读(PDF 物理页 p.40–54) · 生成 2026-06-30

## 本块讲什么

本节回答因子研究的核心问题：**拿到一个多因子模型，如何定量判断它"好不好"**。判断标准统一为——能否在统计上证明"所有资产的定价误差 α_i 联合为零（模型能解释截面收益差异）"，同时"每个因子的预期收益 λ_k 是否显著"。围绕这一目标，本节系统对比三种回归检验法：

- §2.2.1 时间序列回归（time-series regression）
- §2.2.2 截面回归（cross-sectional regression，又称两步回归 two-pass）
- §2.2.3 时序 vs 截面（直观图 + 数学对比，引出"纯因子组合" Ω）
- §2.2.4 Fama–MacBeth 回归（"先回归、再均值"）
- §2.2.5 三法比较与作者两点体会

**统一视角（全书反复强调）**：多因子模型 `E[Rᵢᵉ] = αᵢ + βᵢ'λ`（式 2.12）研究的是**预期收益率在截面（不同资产之间）上的差异**——只关心收益均值的结构，**不关心收益在时序上如何波动**。一旦因子结构给定，任意个股/组合的预期收益就完全由它在这些因子上的暴露 β 决定（暴露高→预期收益高，是 β 的线性函数）。"找最好的因子结构"＝找哪些因子能让个股在截面上的预期收益区分度高、且无法被解释的定价误差 α 尽可能低。

## 方法论要点（重点·展开）

### 0. 检验的三要素 + 通用三步（表2.3）

任何多因子模型的回归检验都拆成三个产出：
| 任务 | 内容 |
|---|---|
| 估计值 estimate | α̂ᵢ, β̂ᵢ, λ̂ |
| 标准误 standard error | σ(α̂ᵢ), σ(β̂ᵢ), σ(λ̂) |
| 检验 test | ① **联合检验所有 N 个资产的定价误差** α 是否=0；② 检验每个因子的预期收益 λ_k |

通用三步：**(1) 算每个资产在所有因子上的暴露 βᵢ →(2) 回归估计模型 →(3) 联合检验 αᵢ 与各 λ_k**。无论用风格因子还是宏观因子、无论时序还是截面回归，都按这三步走。其中**最重要的检验是"所有 α 联合是否足够接近 0"**——这是衡量模型解释力的关键。

---

### 1. 时间序列回归（§2.2.1）——最简单直接，但只能用于风格因子

**适用场景**：因子收益率时序**已知**的因子，即**风格因子**（可用 §2.1 排序法构建因子模拟组合并算出收益率序列，如 FF1993 用独立双重排序构建 HML/SMB）。**对 GDP/CPI/利率等宏观因子无能为力**（无法用排序法造组合算收益）。Black et al.(1972) 最早用它检验 CAPM。

**模型与步骤**：
- 自变量＝因子收益率 λ_t，因变量＝资产超额收益 Rᵢᵉ。对**每个资产 i 单独**做 OLS 时序回归：
  `Rᵢₜᵉ = αᵢ + βᵢ'λ_t + εᵢₜ,  t=1..T`（式 2.13）
- 得到该资产的暴露 β̂ᵢ、截距 α̂ᵢ、残差。再对两边取时序均值得截面关系：
  `E_T[Rᵢᵉ] = α̂ᵢ + β̂ᵢ'λ̄,  i=1..N`（式 2.14），其中 E_T[·]=(1/T)Σ(·)
- **因子预期收益＝因子收益率的时序均值**：`λ̂_k = E_T[λ_kt]`（式 2.15）——这是时序回归的便利之处。

**两个核心判定/直觉（图2.4）**：
- **时序回归的截距 α̂ᵢ 就是资产 i 的定价误差**（直接可读）。
- 单因子下截面直线 `E[Rᵢᵉ]=βᵢλ` **必过 (0,0) 与 (1,λ) 两点**：βᵢ=0 时收益 0；把因子组合自身当资产，其对自己的暴露=1，故收益=λ。
- **关键差异**：时序回归对**每个资产独立**立模型回归 N 次，**直线不是以最小化 Σα̂ᵢ² 为目标求出的**（这是与截面回归的最大区别——截面回归才以最小化 Σα̂ᵢ² 为目标）。

**标准误与联合检验**：
- ε_it **无自相关、无异方差**时，OLS 标准误公式可用。
- 进一步假设 **ε_it ~ IID 正态**，Gibbons-Ross-Shanken(1989) 给出检验"α 是否联合为零"的方法，即著名的 **GRS 检验**（至今仍是学界检验/比较因子模型的首选）。GRS 统计量服从自由度 (N, T−N−K) 的 F 分布（式 2.16），算 p 值判接受/拒绝。
- **陷阱**：一旦 ε_it 存在相关性或异方差，**OLS 标准误公式就是错的，GRS 统计量也有问题**→需用更强工具 GMM（广义矩估计，§2.7）。
- 因子 λ_k 的检验：因子收益序列已知，直接对其做 t 检验即可。

### 2. 截面回归（§2.2.2）——两步回归，应用更广（含宏观因子）

**适用场景**：**因子收益率时序未知也能用**，故能处理 GDP/CPI/利率等宏观因子，应用更广。又称 **two-pass regression（两步回归）**。

**第一步（时序回归定暴露）**：
  `Rᵢₜᵉ = aᵢ + βᵢ'f_t + εᵢₜ,  ∀i`（式 2.17），f_t 是因子取值向量。OLS 估出 β̂ᵢ。
- **★截距符号陷阱**：这里截距用 **aᵢ 而非 αᵢ**。因为当解释变量是因子**取值** f_t（而非因子**收益率**）时，**第一步时序回归的截距 aᵢ ≠ 定价误差**。定价误差留到第二步产生。

**第二步（截面回归）**：用第一步的 β̂ᵢ 作解释变量、资产的时序平均收益 E_T[Rᵢᵉ] 作被解释变量，做一次截面回归：
  `E_T[Rᵢᵉ] = β̂ᵢ'λ + αᵢ,  i=1..N`（式 2.18）
- 估计量（矩阵形式，β̂ 为 N×K 暴露矩阵）：
  `λ̂ = (β̂'β̂)⁻¹β̂' E_T[Rᵉ]`（式 2.20）
  `α̂ = E_T[Rᵉ] − β̂λ̂`（式 2.21）
- **默认不含截距**，直接用回归残差当定价误差 α̂ᵢ。原因：模型设定无偏误时，预期收益应**仅**由暴露与因子预期收益决定。但 Cochrane(2005) 指出"公式是死的、应用是活的"，可加截距 γ 做稳健性：`E_T[Rᵢᵉ]=γ+β̂ᵢ'λ+αᵢ`（式 2.19）。
- **几何（图2.5）**：OLS 截面回归过原点、**以最小化所有资产定价误差 Σα̂ᵢ² 为目标**。

**标准误的两大坑（本节最实操的部分）**：
- **坑 A — 生成回归变量（generated regressors）**：第二步用的 β̂ 来自第一步回归，是**估计值非真值**，对模型(2.18) 而言 β̂ 是"生成的回归变量"。直接算标准误会忽略这层误差。**Shanken(1992) 修正**：在 cov(λ̂)、cov(α̂) 表达式上乘系数 `(1 + λ'Σ_f⁻¹λ)`（式 2.24/2.25，实操用 λ̂、Σ̂_f 代入）。
- **坑 B — α 的截面相关性**：截面上各 αᵢ 存在相关。这不影响 OLS 估计的性质，但**会使 OLS 算出的标准误产生巨大误差、严重低估标准误**→改用 **GLS（广义最小二乘）替代 OLS**（式 2.26–2.29，GLS 用 Σ⁻¹ 加权）。
- 协方差矩阵基础式（假设 f_t 与 ε_t 互独立且各自时序 IID，Cochrane 2005）：cov(λ̂) 式 2.22、cov(α̂) 式 2.23；实操用残差样本 Σ̂、Σ̂_f 代替真值。

**联合检验**：
- 定价误差联合检验，自由度 N−K 的 χ²：`OLS: α̂'cov(α̂)⁻¹α̂ ~ χ²_{N−K}`（式 2.30）；`GLS:` 同形（式 2.31）。
- 因子收益检验：从 cov(λ̂) 对角线取元素开方得 λ_k 的标准误，配合 λ̂_k 算 t 统计量（自由度 T−1）。

> 作者注：本方法在学界因子研究中其实不常用，但理解它有助于体会不同方法的差异。当 f 与 ε 不独立时仍可用 GMM；GMM 还能修正"β̂ 是估计值"造成的误差。

### 3. 时序回归 vs 截面回归（§2.2.3）——本质区别与"纯因子组合"

对**风格因子**（两法都能用），区别在第二步：
- **时序回归**：每个资产独立时序回归 + **时序取均值** → 隐含截面直线**必过 (0,0)(1,λ)**；因子收益 = 因子模拟组合收益在全部 T 期的**简单均值**。
- **截面回归**：第一步与时序回归相同；第二步**不取时序均值**，而是以 β̂ 为解释变量、E_T[Rᵢᵉ] 为被解释变量做截面回归，OLS **最小化全部 N 个 α̂ᵢ²** → **同时用上所有资产数据**，某种意义上更合理；因子收益由 OLS/GLS 解出，**与时序均值不同**。

**★关键洞见——截面回归暗含"纯因子组合"（pure factor portfolio）**：
考察 `λ̂=(β̂'β̂)⁻¹β̂'E_T[Rᵉ]`（式 2.32）中的 `Ω ≡ (β̂'β̂)⁻¹β̂'`，这是 K×N 矩阵，**第 k 行可视作因子 k 的一个投资组合**（第 k 行第 i 列＝资产 i 在该组合中的权重）。由代数恒等式：
  `Ω·β̂ = (β̂'β̂)⁻¹(β̂'β̂) = I`（式 2.33）
展开为：因子 k 组合对其他因子 j(≠k) 的暴露 `Σᵢ ω_ki β̂_ij = 0`（式 2.34）、对自身暴露 `Σᵢ ω_ki β̂_ik = 1`（式 2.35）。
→ 即**截面回归在求因子收益的同时，自动构造出一组"纯因子组合"：只对目标因子有暴露(=1)、对其他因子零暴露**。因此截面回归的因子收益**控制了在其他因子上的暴露**，比时序回归（排序法组合，未必正交）**更客观地评价因子的风险溢价**。这也解释了为什么两法的因子收益数值会不同。

**两法并用做诊断（图2.7）**：若对同一因子，时序回归斜率为正（λ>0）、而带截距 OLS 截面回归 `E[Rᵢᵉ]=γ+βᵢλ+αᵢ` 斜率为负（λ<0），**两模型背离 → 说明所选因子可能有问题，需进一步分析**。

### 4. Fama–MacBeth 回归（§2.2.4）——"先回归、再均值"，巧排截面相关

Fama & MacBeth(1973) 提出的两步法，原为检验 CAPM，是计量经济学被引最频繁的文献之一。**巧妙地排除了随机扰动在截面上的相关性对标准误的影响**，业界广泛使用。

**步骤**：
- **第一步**：与截面回归相同，N 个时序回归得每个资产的 β̂ᵢ。
- **第二步（与传统截面回归的最大不同）**：在**每个时间点 t**，以 **t 期收益 Rᵢₜᵉ（注意是当期收益、不是 T 期均值）** 为因变量、β̂ᵢ 为自变量做截面回归，**共做 T 次**：
  `Rᵢₜᵉ = β̂ᵢ'λ_t + αᵢₜ`（式 2.36）；带截距：`Rᵢₜᵉ = γ_t + β̂ᵢ'λ_t + αᵢₜ`（式 2.37）
- 得到 λ̂_t、α̂_it 的**时间序列**，再取均值：`λ̂=(1/T)Σλ̂_t`（式 2.38）、`α̂ᵢ=(1/T)Σα̂_it`（式 2.39）。

**巧妙之处**：把 T 期回归结果当作 **T 个独立样本**。传统截面回归只回归一次、只得一个样本估计；FM 得到 {λ̂_t}、{α̂_it} 两条序列 → **直接用样本方差算标准误**：
  `σ(λ̂_k) = [ (1/T²) Σ_t (λ̂_kt − λ̂_k)² ]^{1/2}`（式 2.40）
  `σ(α̂ᵢ) = [ (1/T²) Σ_t (α̂_it − α̂ᵢ)² ]^{1/2}`（式 2.41）
  `cov(α̂) = (1/T²) Σ_t (α̂_t − α̂)(α̂_t − α̂)'`（式 2.43）
检验：λ_k 用 t 统计量；定价误差联合用 `α̂'cov(α̂)⁻¹α̂ ~ χ²_{N−K}`（式 2.44）。

**与传统截面回归的本质区别（一句话）**：
- **Fama–MacBeth ＝"先估计、再均值"**（每期先回归出 λ̂_t/α̂_it，再对 T 期取均值）；
- **传统截面 ＝"先均值、再估计"**（先把 Rᵢₜᵉ 时序平均成 E_T[Rᵢᵉ]，再一次性回归）。
- 故 FM 可理解为一种特殊的截面回归；**当解释变量 β̂ᵢ 在全部 T 期不变时，两者估计相同**，但 FM 在应对 α_it 截面相关性上仍有优势。

**时变暴露（实务常用）**：FM 不必固定 β。Fama-MacBeth(1973) 原文即用**滚动窗口**：用截至 t−1 的历史数据估 β̂_{i,t−1}，作为 t 期截面回归的解释变量：
  `Rᵢₜᵉ = β̂_{i,t−1}'λ_t + αᵢₜ`（式 2.45）；带截距式 2.46。比全局用同一 β 更贴近研究与投资实践。

**不足（陷阱）**：
- ① 对 **α_it 的时序相关性无能为力**（只解决了截面相关）。
- ② β̂ 是第一步时序回归的**估计值、有误差**，FM 对此也无能为力，**仍需 Shanken 修正**。

**学界惯例**：FM 主目的本是检验 α 是否联合为零，但学者更常用它**检验因子预期收益 λ_k**（因为能方便地得到因子收益序列→算均值与标准误）。检验 λ 时通常用**带截距模型(2.46)**，以排除模型设定偏误。至今约 1/3 面板数据论文用 FM（Petersen 2009）。

### 5. 不同回归方法比较（§2.2.5）——作者两点体会

- **① "所有模型都不完美"**：把足够多的资产放进回归左侧，**任何多因子模型都会被拒绝**（α 联合≠0）。所以研究动机不应追求统计上多"完美"，而应关注**每个因子背后到底有多少逻辑**。学界最常见做法：**不用个股、而是按规则把股票"打包"成投资组合**当作资产去检验模型（否则模型必被拒）。
- **② "殊途同归"**：检验多因子模型时，不同方法的差异也许没有它们名字的差异大；**特定假设下常等价**（如 β̂ᵢ 时序不变时，传统截面 = FM 截面）。应用中通过**比较不同方法的结果**来加深对模型的认知——这才是学多种方法的最大价值。

## 关键公式 · 定义 · 判定标准

| 公式 | 含义 |
|---|---|
| `E[Rᵢᵉ]=αᵢ+βᵢ'λ` (2.12) | 多因子模型截面关系；αᵢ＝定价误差 |
| `Rᵢₜᵉ=αᵢ+βᵢ'λ_t+εᵢₜ` (2.13) | 时序回归（因子收益作自变量）；截距 αᵢ=定价误差 |
| `λ̂_k=E_T[λ_kt]` (2.15) | 时序回归：因子预期收益=因子收益时序均值 |
| GRS F 统计量 (2.16) | ε~IID正态下联合检验 α=0，F(N, T−N−K) |
| `Rᵢₜᵉ=aᵢ+βᵢ'f_t+εᵢₜ` (2.17) | 截面回归第一步；截距 **aᵢ≠定价误差**（解释变量是因子取值非收益） |
| `E_T[Rᵢᵉ]=β̂ᵢ'λ+αᵢ` (2.18) | 截面回归第二步；默认无截距，残差即 α̂ |
| `λ̂=(β̂'β̂)⁻¹β̂'E_T[Rᵉ]` (2.20) | 截面 OLS 因子收益估计 |
| Shanken 修正系数 `(1+λ'Σ_f⁻¹λ)` (2.24/2.25) | 修正"生成回归变量 β̂"造成的标准误偏差 |
| GLS 估计 (2.26–2.29) | 解决 α 截面相关导致的标准误低估 |
| `α̂'cov(α̂)⁻¹α̂~χ²_{N−K}` (2.30/2.31/2.44) | 定价误差联合检验 |
| `Ω≡(β̂'β̂)⁻¹β̂'`, `Ωβ̂=I` (2.33) | 截面回归隐含"纯因子组合"：对自身暴露1、对他因子0 |
| `Rᵢₜᵉ=β̂ᵢ'λ_t+αᵢₜ` (2.36) | FM 第二步：每期 t 一次截面回归 |
| `λ̂=(1/T)Σλ̂_t`, `σ(λ̂_k)` (2.38/2.40) | FM："先估计、再均值"，由序列算标准误 |
| `Rᵢₜᵉ=β̂_{i,t−1}'λ_t+αᵢₜ` (2.45) | 时变暴露 FM（滚动窗口估 β） |

**三法对照表（★必抓·适用场景/步骤/陷阱）**：

| 维度 | 时序回归 | 截面回归（两步） | Fama–MacBeth |
|---|---|---|---|
| 适用因子 | 仅风格因子（需因子收益时序） | 风格+宏观（因子收益时序可未知） | 风格+宏观 |
| 第一步 | 各资产时序回归→β̂ᵢ，**截距=定价误差** | 时序回归→β̂ᵢ，**截距 aᵢ≠定价误差** | 时序回归→β̂ᵢ（可滚动窗口时变） |
| 第二步 | 对 R、λ **取时序均值** | **一次**截面回归（β̂ vs E_T[R]） | **T 次**截面回归（β̂ vs 当期 R_t）再均值 |
| 思路口诀 | 时序平均 | **先均值、再估计** | **先估计、再均值** |
| 因子收益 λ | 因子收益时序均值 | OLS/GLS 解出（Ω 纯因子组合，控制他因子暴露） | T 期 λ̂_t 的均值 |
| 是否最小化 Σα² | 否（各资产独立回归 N 次） | 是（OLS 最小化 Σα̂²） | 每期最小化、再均值 |
| 截距过点 | 必过 (0,0)(1,λ) | 默认过原点（可加 γ） | 默认过原点（可加 γ_t） |
| α 联合检验 | **GRS（F 检验）** | χ²(N−K) | χ²(N−K) |
| 标准误主要坑 | ε 自相关/异方差→GRS失效→GMM | ① β̂ 生成回归变量→Shanken修正；② α 截面相关→标准误低估→GLS | 已排除 α **截面**相关；**时序**相关无解；β̂ 误差仍需 Shanken |

## A股特殊性 / 实证结论

- **实证脉络（为什么要多因子）**：CAPM 单一市场因子 → 实证发现 E[Rᵢᵉ] 与市场 β 的截面关系拟合差 → Black et al.(1972) 加 zero-beta 因子（后称 Black CAPM）→ Fama-French(1993) 三因子加入价值(HML)、规模(SMB)。一路都是为**更好解释截面收益差异**。
- **本块为纯方法论，未直接讨论 A 股涨跌停/停牌/T+1/壳价值/注册制等制度**。但方法层面的 A 股映射值得记：
  - **"个股必被拒→打包成组合"** 对 A 股尤其关键：A 股散户主导、个股噪声大，用个股做左侧资产几乎必然拒绝模型；按市值/行业/因子分位打包成组合再检验是标准做法。
  - **α 的截面相关性在 A 股可能更强**（同涨同跌、板块联动、散户情绪驱动）→ 传统截面 OLS 标准误低估更严重 → **Fama–MacBeth / GLS 在 A 股更稳健**。
  - **时变暴露（滚动窗口 β）** 契合 A 股结构快变（注册制、风格轮动）的现实，比全局固定 β 更可取。
- 一般认为股票收益**时序相关弱、截面相关强**（脚注8），故 FM 排除截面相关即可解决大部分问题；但若研究重叠/低频信号则时序相关不可忽视（本节未展开，见 §2.3 与 Petersen 2009）。

## 踩坑对照（√252重叠年化 / 单调性判定 / 合成稀释尾部 / 其他）

- **√252 重叠收益年化**：本块未涉及（属收益/夏普/IR 年化口径问题，留待相关章节）。但相关线索：脚注8 与 Petersen(2009) 指出，忽略随机扰动的**时序相关性**会**低估标准误**——这与重叠样本下自相关导致方差低估同源，FM 只解决截面相关、对时序相关无能为力，故低频/重叠信号仍需 Newey–West 类时序调整。
- **单调性判定**：本块未涉及（属排序法分组检验，见 §2.1 相关笔记）。
- **合成稀释尾部**：本块未直接涉及，但**截面回归的"纯因子组合" Ω（式 2.33–2.35）是正交化思想的源头**——它构造出只对目标因子暴露=1、对其他因子=0 的组合，正是"控制其他因子暴露、避免暴露被稀释"的正确做法，可迁移到多因子正交合成。
- **其他·本块核心坑（★必抓）**：
  1. **截距 α 的含义随模型而变**：时序回归(2.13) 截距 α̂ᵢ = 定价误差（可直读）；但截面回归第一步(2.17) 截距 **aᵢ ≠ 定价误差**（因解释变量是因子取值 f_t 而非因子收益）；截面回归第二步默认不要截距、用残差当 α̂，加截距 γ 只为排查模型设定偏误。**写检验代码时务必分清当前截距是不是定价误差。**
  2. **生成回归变量（generated regressors）**：截面/FM 第二步用的 β̂ 是估计值，标准误必须做 **Shanken(1992) 修正** `×(1+λ'Σ_f⁻¹λ)`，否则低估标准误、高估显著性。
  3. **α 截面相关→OLS 标准误巨幅低估**：截面 OLS 不可直接信标准误，须用 **GLS** 或 **Fama–MacBeth**。
  4. **GRS 前提**：仅在 ε IID 正态下成立；存在自相关/异方差时 OLS 标准误与 GRS 统计量均失效，须 GMM。
  5. **"所有模型都不完美"**：资产足够多时任何模型被拒是常态，**不要把"被拒/未被拒"当唯一标准**，重在因子逻辑 + 用打包组合检验。

## 对 scorecard / 回测框架的直接启示

- **检验因子是否有效＝Fama–MacBeth 是默认武器**：对每个候选因子，第一步估 β̂（或直接用因子值），第二步每期截面回归得 λ̂_t 序列，取均值算 λ̂ 与 t 值（`t = λ̂ / σ(λ̂)`）。scorecard 的"因子显著性"一栏应输出 FM 的 λ̂、t 值、以及是否带截距(2.46) 两版结果。
- **标准误必须修正，别用裸 OLS 标准误**：① 截面/FM 用了估计的 β̂ → 加 **Shanken 修正**；② 提防 α 截面相关 → 优先 FM 或 GLS；③ 若信号有时序自相关（重叠/低频）→ 还需 Newey–West（本节未展开，框架里留接口）。
- **模型层面联合检验**：评估一组因子能否解释一篮子组合，用 **GRS（F 检验，ε IID 正态）** 或 **χ²(N−K)**（α̂'cov(α̂)⁻¹α̂）；前提不满足时退回 GMM。
- **构造"纯因子组合" Ω=(β̂'β̂)⁻¹β̂'** 作为正交化/多因子合成的基础设施：得到只暴露目标因子的组合，既能客观评估单因子风险溢价，又能避免合成时暴露被稀释。
- **测试资产用组合而非个股**：回测/检验框架默认把股票按规则打包成 N 个组合作为左侧资产，A 股尤其如此（个股噪声大、必被拒）。
- **β 用滚动窗口（时变）**：估计 β̂_{t−1} 时严格只用 t−1 及之前数据，**杜绝前视偏差**，并贴合风格轮动。
- **多方法交叉验证**：同一因子同时跑时序回归（看 GRS）、FM（看 λ 序列 t 值）、带/不带截距截面回归；若时序斜率与截面斜率**符号背离**（图2.7），标红该因子"逻辑存疑"。

## 术语与存疑

- **time-series regression 时序回归** / **cross-sectional regression 截面回归（two-pass 两步回归）** / **Fama–MacBeth regression（FM 回归）**
- **GRS test**（Gibbons-Ross-Shanken 1989）：检验 α 联合为零的 F 检验，自由度 (N, T−N−K)。
- **Shanken 修正（Shanken 1992）**：修正生成回归变量（generated regressors）β̂ 造成的标准误偏差，系数 `(1+λ'Σ_f⁻¹λ)`。
- **generated regressors 生成的回归变量**：第二步回归中作为解释变量的 β̂ 来自第一步估计、非真值。
- **GLS 广义最小二乘**：替代 OLS 以应对 α 截面相关导致的标准误低估。
- **GMM 广义矩估计**：ε 非 IID（自相关/异方差）时的更强工具（§2.7 展开）。
- **纯因子组合 / pure factor portfolio**：Ω=(β̂'β̂)⁻¹β̂' 各行，对自身因子暴露=1、对其他因子=0。
- **zero-beta factor**：Black et al.(1972) 在 CAPM 上加的因子（Black CAPM）。
- 符号约定：Rᵢᵉ=资产 i 超额收益，βᵢ=K 维暴露，λ=K 维因子预期收益，αᵢ=定价误差，Σ=cov(ε_t)，Σ_f=cov(f_t)，E_T[·]=时序均值。
- 存疑：式 2.16/2.22–2.29 在 PDF 渲染图中为小字公式，矩阵转置/求逆位置已按 Cochrane(2005) 标准式核对誊写，个别下标若与原书印刷有出入以原书为准 `[公式小字存疑]`。
