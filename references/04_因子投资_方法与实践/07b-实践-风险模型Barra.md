## 本块讲什么

主动管理两大模块——收益率模型（解释/预测截面收益）与**风险模型**（准确估计资产协方差矩阵 Σ）——本块讲后者，以业界标杆 **Barra** 的中国 A 股模型 **CNE5** 为例。

主线一句话：风险模型的目的**不是解释预期收益的截面差异，而是为了通过多因子结构对协方差矩阵降维、给出 Σ 的准确"事前(ex-ante)估计"**，因为 Σ 是组合优化（最小方差、均值-方差）必不可少的输入。

四节递进：
- **7.2.1 模型结构**：CNE5 = 1 国家因子 + P 行业因子 + Q 风格因子；如何解决国家/行业共线性、如何标准化风格暴露、国家因子≈市场组合。
- **7.2.2 模型求解**：每期 WLS 截面回归（本质 Fama–MacBeth），带约束最小二乘解出**纯因子组合(pure factor portfolio)**权重矩阵 Ω。
- **7.2.3 纯因子组合性质**：Ωβ 的块结构证明"风格纯因子组合只在自己身上有 1 单位暴露、其余全 0"；行业纯因子组合 = 100% 多该行业 + 100% 空国家因子（≈市场）的资金中性对冲。
- **7.2.4 协方差矩阵估计与调整**：样本协方差不可信 → 偏差统计量(Bias statistic)作为检验标尺；对因子协方差 Σ̂_λ 用**特征因子调整法(eigenfactor adjustment, bootstrap)**，对特质协方差 Σ̂_ε 用**贝叶斯收缩(Bayesian shrinkage)**；脚注另提 **Newey–West** 消时序自相关。

风险模型是回测/组合优化的"地基"，这一块价值密度高。

## 方法论要点（重点·展开）

### 1. 风险模型的定位：降维估 Σ，而非选股
- 核心恒等式（式 7.18，即第 1 章式 1.6）：
  **Σ = β Σ_λ β' + Σ_ε**
  - β：N×K 因子暴露矩阵（N 支股票、K 个因子）；
  - Σ_λ：K 阶**因子协方差矩阵**；Σ_ε：N 阶**特质收益协方差矩阵**，因假设特质收益相互独立 → **Σ_ε 是对角阵**。
- 降维意义：直接估 N×N 的 Σ 需 N(N+1)/2 个参数（A 股 N 数千 → 上千万参数，样本不够、估计极不稳）。借因子结构只需估 K 阶 Σ_λ（K 几十）+ N 个特质方差，参数量骤降、可估。
- 与 alpha 模型的根本区别：同一套多因子框架，**目标函数不同**——风险模型只求 Σ 准，不要求因子能解释预期收益。业界因此称这类模型为"风险模型"。

### 2. CNE5 模型结构（7.2.1）
**因子集合**：K = 1 + P + Q。1 个国家因子(country)、P 个行业因子(industry，记 I₁…I_P)、Q 个风格因子(style，记 S₁…S_Q)。

**截面模型（式 7.19）**，t 时刻个股 i 超额收益：
  R^e_{it} = λ_{Ct}·1 + Σ_{p} β^{I_p}_{i,t-1}·λ_{I_pt} + Σ_{q} β^{S_q}_{i,t-1}·λ_{S_qt} + u_{it}
- **国家因子暴露恒为 1**（所有股票）；行业暴露 β^{I_p} ∈ {0,1}（一个公司任一时刻只属于一个行业，哑变量)；风格暴露 β^{S_q} 由公司特征标准化得到。
- λ_{Ct}/λ_{I_pt}/λ_{S_qt}：国家/行业/风格因子在 t 期的**因子收益率**；u_{it}：特质性收益率(idiosyncratic return)。
- **时间错位**：t−1 期初暴露已知 + t 期收益已知 → 用 **WLS 截面回归**估 t 期因子收益率。**本质即 Fama–MacBeth 截面回归**。

**共线性与约束（式 7.20）**：因每股恰属一个行业，P 个行业暴露行和恒 =1 = 国家因子暴露 → 国家因子暴露 = 行业暴露的线性组合，**回归解不唯一**。施加约束去掉冗余自由度：
  s_{I₁}λ_{I₁t} + s_{I₂}λ_{I₂t} + … + s_{I_P}λ_{I_Pt} = 0
  其中 s_{I_p} = 行业 I_p 内所有股票的市值权重之和。即**行业因子收益率的市值加权和 = 0**（行业收益是相对"全市场"的，自然加总为零）。

**风格因子暴露的确定（Barra 特色）**：
1. **直接用公司特征做原始暴露值，而非时序回归系数**（区别于 Fama–French 的时序 β）。例：价值因子直接取 t−1 期 BM 指标值作原始暴露。
2. **标准化两步**：
   - (a) 去均值：原始值 − **市值加权**均值。理由：假设市场组合在任何风格因子上**中性**（暴露 = 0）。去均值后满足 Σ_i s_i·β^{S_q}_{i,t-1} = 0（式 7.21，s_i 为市值权重）。
   - (b) 除以**截面标准差**。
   — 注意去均值用的是市值加权而非等权，与"市场组合风格中性"的设定一致。

**国家因子 ≈ 市场组合（式 7.22 推导）**：把市值加权市场组合 R^e_{Mt}=Σ_i s_i R^e_{it} 当作一个"资产"代入式 7.19，
  - 行业项：Σ_p (Σ_i s_i β^{I_p}_{i,t-1}) λ_{I_pt} = Σ_p s_{I_p} λ_{I_pt} = 0（用约束式 7.20）；
  - 风格项：Σ_q (Σ_i s_i β^{S_q}_{i,t-1}) λ_{S_qt} = Σ_q 0·λ_{S_qt} = 0（用式 7.21）；
  - 特质项：Σ_i s_i u_{it} ≈ 0（市值加权特质收益≈0）；
  ⟹ **R^e_{Mt} ≈ λ_{Ct}**。国家因子收益 ≈ 市场超额收益 → **国家因子组合 ≈ 市场组合**。
  数学上：因所有股票国家因子暴露恒 = 1，**国家因子在截面回归中起截距项作用**。

### 3. 模型求解 → 纯因子组合（7.2.2）
矩阵形式（略去时间下标，Menchero & Lee 2015）：
  **R^e = β λ + u**（式 7.23），β 为 N×K，第一列全 1（式 7.24）。

待求：**Ω = 全部因子的纯因子组合权重矩阵（K×N）**，第 k 行 = 因子 k 的纯因子组合在 N 支股票上的权重。两个技术难点：

**(a) WLS 回归权重 W（式 7.25）**：对角阵，
  W_ii = √s_i / Σ_{j} √s_j
  来源假设：**个股特质收益率方差 ∝ 市值平方根的倒数**（市值越大特质波动越小）→ 大市值股给更高回归权重。这是 GLS 的体现：按噪声大小反比加权。

**(b) 约束矩阵 C（式 7.26）**：把约束式 7.20 编码为 K×(K−1) 矩阵（K 个因子收益 + 1 个约束 → (K−1) 个自由参数）。构造法是把 λ_{I_P} 用其余行业收益的线性组合表达，从而消去 1 个自由度。

**带约束 WLS 解（式 7.27，核心）**：
  **Ω = C (C'β'Wβ C)^{-1} C'β'W**
- Ω 第 k 行 = 因子 k 纯因子组合权重；
- 因子收益率（式 7.28）：**λ_{kt} = Σ_i ω_{ki} R^e_{it}** —— 因子收益就是用纯因子组合权重对个股收益加权；
- 把 λ_{kt}(k=1…K) 反代式 7.19 → 解出每股特质收益 u_{it}。
- **关键认知**：多元回归求因子收益率的同时，**天然得到了每个因子的投资组合**（2.2.3 节）。多元回归（控制其余因子）得到的组合，相比单变量排序法构建的组合，暴露"干净"，故称**纯因子组合**。

### 4. 纯因子组合的性质（7.2.3）—— 本块最重要的"思想"
令 **Ωβ**（K×N · N×K = K 阶方阵），第 k 行 = 因子 k 纯因子组合在全部 K 个因子上的暴露。按 [国家, 行业, 风格] 顺序排列，**块结构（式 7.29）**：

  Ωβ = ⎡ D_{(1+P)×(1+P)}   0_{(1+P)×Q} ⎤
        ⎣ 0_{Q×(1+P)}      I_{Q×Q}     ⎦

逐块读懂：
- **右上 0_{(1+P)×Q}**：国家、行业纯因子组合在所有风格因子上暴露 = 0；
- **左下 0_{Q×(1+P)}**：风格纯因子组合在国家、行业因子上暴露 = 0；
- **右下 I_{Q×Q}**：每个风格纯因子组合对自己有 **1 单位**暴露、对其他风格 = 0。
- ⟹ **任一风格因子的纯因子组合，除自己外所有因子上零暴露**。这就是"纯"的含义：**单位的目标暴露 + 其余全部因子零暴露**，无污染。

**左上 D 的含义（示例：1 国家 + 27 申万一级行业 + 11 风格 = 39 因子，式 7.30）**：
- **第 1 行（国家因子组合）**：对自身暴露 = 1；因 ≈ 市场组合而市场含全行业，故对每个行业有正暴露（如 0.03、0.02…）；对风格暴露 = 0。是**纯多头**组合。
- **行业因子组合（以行业 1 为例，式 7.31–7.33）**：构造多空组合 PI₁ = [100% 市值配多行业 1 全部股票] − [等额做空国家因子组合]。
  - 多头暴露：[1, 1, 0, …]（国家 1 + 行业 1 为 1）（式 7.31）；
  - 空头(国家因子组合)暴露：[1, 0.03, 0.02, …]（式 7.32）；
  - 净暴露 = 多 − 空 = [0, 0.97, −0.02, …]（式 7.33），与式 7.30 第 2 行**完全一致**。
  ⟹ **行业纯因子组合 = 100% 做多该行业 + 100% 做空国家因子组合(≈市场)** 的对冲组合 → **资金中性**，业务含义 = 该行业相对市场的超额收益。
- **资金中性证明（通用）**：因子 k 组合在国家因子上暴露 = Σ_i 1·ω_{ki} = Σ_i ω_{ki}。由 D 第一列结合 0_{Q×(1+P)} 知行业/风格组合该暴露 = 0 → **Σ_i ω_{ki}=0**（权重和为零）→ dollar-neutral。

**三类纯因子组合特征小结**：
- **国家因子组合**：≈ 市场组合，纯多头；国家暴露 = 1，所有行业正暴露，风格暴露 = 0。
- **行业因子组合**：资金中性；100% 多该行业、100% 空国家因子组合；反映行业相对市场超额；风格暴露 = 0。
- **风格因子组合**：资金中性；仅目标风格 1 单位暴露、其余行业/风格全 0；靠这"一单位暴露"赚该风险因子的超额收益。

### 5. 协方差矩阵估计与调整（7.2.4）—— 风险模型的硬骨头
得到因子收益序列 {λ} 与特质收益序列 {u} 后，由式 7.18 算 Σ。需估 **Σ_λ（因子协方差）** 与 **Σ_ε（特质协方差，对角）**。给定窗口的**样本协方差** Σ̂_λ、Σ̂_ε 是最简单估计，但**实证表明历史样本协方差估不准**，必须调整。两大调整 + 一个标尺：

#### 5a. 偏差统计量 Bias statistic（标尺，Menchero et al. 2011）
衡量"事前风险预测 vs 事后实际风险"的误差。令 σ_{it} = t 期初对资产 i 风险（标准差）的预测，R_{it} = 实际收益：
  **b_{it} = R_{it}/σ_{it}**（式 7.34，用预测风险标准化收益）
  **B_i = √( 1/(T−1) Σ_t (b_{it} − b̄_i)² )**（式 7.35，标准化收益的样本标准差）
- 若 σ_{it} 预测准确 → b_{it} 标准差应为 1；
- 收益正态 + 预测准 + T 大时，95% 置信区间：
  **B_i ∈ [ 1 − √(2/T), 1 + √(2/T) ]**（式 7.36）
- 判读：**B_i > 1 ⟹ 低估了风险**；**B_i < 1 ⟹ 高估了风险**；超出区间 ⟹ 事前风险估计不可信。
- 这是验证任何风险模型的**标准 KPI**，下面两种调整都以"把 B 拉回 1.0"为目标。

#### 5b. 特征因子调整法 Eigenfactor adjustment（调 Σ̂_λ，Menchero et al. 2011）
**动机**：最优组合方差 σ_p² = ω'Σω，其偏差统计量是否偏离 1 决定组合事前风险是否可信。由 Σ=βΣ_λβ'+Σ_ε 且 Σ_ε 对角 → σ_p² 准确性主要看 **ω'βΣ_λβ'ω**。令 **ω_λ ≡ (ω'β)'** → 简化为 **ω_λ'Σ_λω_λ**：把纯因子组合当"资产"、按某最优权重 ω_λ 配置的方差。于是**把对个股最优组合偏差的关注，转移到对"因子最优组合"偏差的关注**。

**为何样本 Σ̂_λ 估不准**——特征分解（Σ̂_λ 对称）：
  **Σ̂_λ = U D U'**（式 7.37），U 正交(U⁻¹=U')，D 对角=特征值；**U'Σ̂_λU = D**（式 7.38）。
- 以特征向量第 k 元素作因子 k 权重 → **特征因子组合(eigenfactor portfolio)**，共 K 个；正交 ⟹ 相互独立、协方差 0；**D 对角元 = 各特征因子组合方差**。
- K 个特征因子组合构成正交基，任意 ω_λ 是其线性组合 → ω_λ'Σ̂_λω_λ 准不准取决于这 K 个组合方差估得准不准。
- **实证结论**：特征因子组合的偏差统计量**并非都接近 1，而与组合方差成反比**——**事前方差越小的特征因子组合，事后偏差统计量越大且显著 > 1.0**（即低方差方向的风险被系统性低估）。故必须调整。

**Bootstrap 步骤（核心思想：用模拟量出偏差，再反向修正）**：把 Σ̂_λ 视为"总体"/数据生成过程(DGP)的已知参数，模拟出"样本"协方差 Σ̃_λ；D̃ 与 D 的偏差 ≈ D 与"真实未知"特征因子协方差的偏差。
1. Σ̂_λ 特征分解 → U, D。
2. 第 m 次模拟：特征因子 k 收益 ~ N(0, D(k))，生成 T 期 → K×T 矩阵 b_m；因子收益矩阵 **λ̃_m = U·b_m**（式 7.39）。
3. **Σ̃_{λm} = cov(λ̃_m, λ̃_m)**（式 7.40）。
4. Σ̃_{λm} 特征分解 → Ũ_m, D̃_m。
5. "真实"特征因子协方差 **D_m = Ũ_m' Σ̂_λ Ũ_m**（式 7.41，用已知"总体"Σ̂_λ 投影到模拟特征基上）。
6. 重复 M 次，平均偏差 **ν(k) = (1/M) Σ_m √( D_m(k)/D̃_m(k) )**（式 7.42）。
7. **经验缩放**（真实收益非正态/非平稳，需应对经验波动率偏误）：**ν_s(k) = a(ν(k)−1) + 1**（式 7.43），a 为经验系数，通常 ∈ [1,2]。
8. 修正：**D₀ = ν² D**（式 7.44），**Σ̂_{λ0} = U D₀ U'**（式 7.45），ν² 对角元 = ν_s(k)²。Σ̂_{λ0} 即调整后的因子协方差矩阵。

#### 5c. 贝叶斯收缩 Bayesian shrinkage（调 Σ̂_ε）
**框架**：先验 + 新息(样本) → 后验。业界用例：Black–Litterman（用于预期收益，Black & Litterman 1992）、Ledoit & Wolf (2003)（用于协方差）。
**问题**：特质波动率样本外**持续性差**——按特质波动分十档，**小波动档 B > 1.0**（样本低估了样本外事后波动）、**大波动档 B < 1.0**（样本高估）。典型均值回归 → 应**向先验收缩**。
**先验构造（按市值分档）**：把全部股票按市值分十档。市值档 g 内权重 ω_i = s_i / Σ_{j∈M_g} s_j（式 7.46）；档内市值加权特质波动 **σ̄^i_g = Σ_{i∈M_g} ω_i σ̂^i_i**（式 7.47）作为该档所有股票的先验。
**后验收缩（式 7.48）**：
  **σ̂^{bs}_i = η_i·σ̄^i_g + (1 − η_i)·σ̂_i**
  η_i = 赋予先验的权重（**收缩强度系数**）。
**收缩强度（式 7.49）**：
  η_i = q|σ̂_i − σ̄^i_g| / [ ( (1/N_g) Σ_{j∈M_g}(σ̂_j − σ̄^i_g)² )^{1/2} + q|σ̂_i − σ̄^i_g| ]
  - 分子 & 分母第二项 |σ̂_i − σ̄^i_g|：股票 i 样本波动**偏离先验**的程度；
  - 分母第一项：档内所有股票偏离先验的**标准差**（平均偏离的度量）；
  - q：经验压缩系数。
  判读：**偏离越大 → 越像噪声 → 越信先验 → η_i 越大（收缩越强）**；偏离越小 → 越信样本 → η_i 越小。q 通过"综合考虑收缩前后所有个股偏差统计量的改进"来标定。

**收尾态度**：Barra 业界广用，但任何模型都有假设、**模型无好坏之分**，CNE5 只是众多风险模型之一（CNE6 已于 2018 年推出）；践行者应保持开放、持续吸收新协方差估计方法。

| 因子收益(7.28) | λ_{kt}=Σ_i ω_{ki}R^e_{it} | 用纯因子组合权重加权个股收益 |
| 纯因子暴露结构(7.29) | Ωβ = [[D,0],[0,I]] | 风格组合只对自己 1 单位暴露 |
| 资金中性判据 | Σ_i ω_{ki}=0 ⟺ 国家因子暴露=0 | 行业/风格纯因子组合皆 dollar-neutral |
| 偏差统计量(7.34/7.35) | b_{it}=R_{it}/σ_{it}；B_i=std(b_{it}) | 风险模型事前/事后误差标尺 |
| 偏差置信区间(7.36) | B_i∈[1−√(2/T), 1+√(2/T)] | 出区间=风险估计不准；>1低估、<1高估 |
| 特征分解(7.37/7.38) | Σ̂_λ=UDU'；U'Σ̂_λU=D | 构造特征因子组合(正交基) |
| 特征因子偏差(7.42) | ν(k)=(1/M)Σ_m√(D_m(k)/D̃_m(k)) | bootstrap 量出方差估计偏差 |
| 经验缩放(7.43) | ν_s(k)=a(ν(k)−1)+1，a∈[1,2] | 应对非正态/非平稳 |
| 因子协方差修正(7.44/7.45) | D₀=ν²D；Σ̂_{λ0}=UD₀U' | 特征因子调整后的 Σ_λ |
| 贝叶斯收缩(7.48) | σ̂^{bs}_i=η_i σ̄^i_g+(1−η_i)σ̂_i | 特质波动向市值档先验收缩 |
| 收缩强度(7.49) | η_i=q·偏离 / (档内偏离std + q·偏离) | 偏离先验越大→收缩越强 |

**判定标准速记**：风险模型是否可信，看组合层面的 **B_p**（用最优组合权重 ω 与估计 Σ 算 σ_p，再做偏差统计量）是否落在 [1−√(2/T), 1+√(2/T)] 内；落不进就说明 Σ 需要上面三种调整。

## A股特殊性 / 实证结论

- **行业分类与哑变量暴露**：CNE5 中行业暴露是 0/1 哑变量，"一个公司任一时刻只属于一个行业"。示例用 **27 个申万一级行业 + 11 个风格 + 1 国家 = 39 因子**（A 股本土化口径）。
- **国家因子 = A 股市场组合**：CNE5 相对 Barra 早期中国模型最大改动是**加入国家因子**，其纯因子组合 ≈ 市值加权 A 股全市场组合；行业/风格纯因子组合都被"剥离"了市场 β，给出相对市场的纯超额。
- **WLS 权重 ∝ √市值**：隐含"大市值特质波动小、小市值噪声大"，与 A 股小市值股波动大、信息噪声高的现实一致；回归中自动压低小市值噪声影响。
- **特质波动按市值分档收缩**：贝叶斯收缩的先验**按市值分十档**取档内市值加权特质波动，照顾了 A 股不同市值层特质波动差异显著的结构。
- **实证（样本外不稳定）**：① 特征因子组合事前方差越小、事后偏差统计量越偏大（低估）；② 特质波动小的股票样本外被低估、大的被高估——两条都说明 A 股（及一般市场）历史样本协方差不能直接用，必须调整。
- 模型迭代：Barra 被 MSCI(晨明)收购，**CNE6 于 2018 年 8 月底推出**。

## 踩坑对照（√252重叠年化 / 单调性判定 / 合成稀释尾部 / 其他）

### √252 / 重叠收益年化 → 本块的对应坑：时序自相关导致样本协方差有偏（Newey–West）
- **现象**：直接用历史收益序列算样本协方差 Σ̂_λ、Σ̂_ε，默认各期收益时序独立；但因子收益率与特质收益率存在**时序自相关(serial correlation)**，忽略它会使方差/协方差估计有偏（与"重叠收益下 √252、√T 简单年化低估方差"同源——都是自相关被忽略导致的方差估计偏误）。
- **书里为什么 / 正确做法**：脚注[3]明确——Barra 模型在计算样本协方差矩阵 Σ̂_λ 和 Σ̂_ε 时进行了 **Newey–West 调整**，以消除因子收益率与特质收益率的时序相关性。即估协方差时用 HAC(异方差自相关稳健)型校正，而非朴素样本协方差。这与本书前文"重叠收益/年化 t 值须用 Newey–West"的处理是同一思想在协方差矩阵上的落地。
- **启示**：我们自建风险模型/因子收益序列时，凡涉及由日频聚合到月频、或暴露持有期重叠，年化方差与协方差都不能简单 ×√频率，须做 Newey–West（HAC）。

### 单调性判定 → 本块未涉及
（分组单调性检验属收益模型/排序法范畴，本风险模型块不涉及。）

### 合成稀释尾部 → 纯因子组合 = "合成/排序污染"的标准解药
- **现象**：用单变量**排序法/打分合成**构建因子组合时，目标因子的暴露会被其他相关因子"污染"——例如按市值排序的小市值组合天然携带价值、低流动性等暴露，导致目标因子暴露被稀释、且相关因子在尾部纠缠、信息损失。
- **书里为什么 / 正确做法**：CNE5 用**多元截面回归**同时控制全部因子，解出的**纯因子组合(式 7.27)**满足 Ωβ=[[D,0],[0,I]]（式 7.29）——**目标(风格)因子上恰好 1 单位暴露、其余所有因子上暴露为 0**。这正是"正交化/控制变量"对"简单合成"的纠偏：纯因子组合给出**干净、不被其他因子稀释**的单位暴露，因子收益(式 7.28)是控制了其他因子后的纯净溢价。
- **启示**：要量一个因子的"纯"溢价或做因子对冲组合，应走多元回归的纯因子组合路线，而非排序/线性合成；后者得到的暴露是被污染的，尾部尤甚。

## 对 scorecard / 回测框架的直接启示

1. **风险模型是组合优化的地基**：任何 min-variance / mean-variance / 风险预算都要 Σ=βΣ_λβ'+Σ_ε。回测框架应内置一个"风险模块"产出 Σ，而非临时用样本协方差。
2. **纯因子组合权重 Ω=C(C'β'WβC)⁻¹C'β'W 是可复用模板**：① 直接拿来生成因子收益序列 λ_{kt}=Σω_{ki}R^e_{it}；② 作为"单因子纯多空组合"的构造方法（资金中性、单位暴露），优于排序法。
3. **WLS 权重默认 W_ii∝√s_i**：因子回归/纯因子组合可把"权重 ∝ √市值"设为默认加权方案（GLS 思想，压低小市值噪声）。
4. **协方差必须三段调整，缺一不可**：
   - 时序自相关 → **Newey–West**（对 Σ̂_λ、Σ̂_ε）；
   - 因子协方差 Σ̂_λ → **特征因子调整(bootstrap, M 次模拟)**，修正低方差方向被低估的问题；
   - 特质协方差 Σ̂_ε → **贝叶斯收缩**（按市值档先验、强度 η_i 由偏离度定）。
   建议在框架里做成可插拔的"协方差调整流水线"。
5. **偏差统计量 B 作为风险模型的验收 KPI**：每个回测/组合都应计算 B_p 并检验是否落在 [1−√(2/T), 1+√(2/T)]；B>1 系统性低估风险、B<1 高估，据此回调 Σ。这是把"风险预测准不准"变成可量化、可回归校准的指标。
6. **特征因子调整里的经验系数 a∈[1,2]、收缩里的 q**：都是需在自有 A 股样本上以"偏差统计量改进"为目标拟合的超参，应纳入框架的标定流程。

## 术语与存疑

- **CNE5 / CNE6**：Barra(MSCI) 中国 A 股多因子风险模型，CNE5 为本节范例，CNE6 为 2018 年新一代。
- **纯因子组合(pure factor portfolio)** / **特征因子组合(eigenfactor portfolio)**：前者=多元回归解出的、对目标因子单位暴露其余零暴露的组合；后者=对 Σ̂_λ 特征分解后、以特征向量为权重的正交组合。
- **偏差统计量(Bias statistic)**：事前风险预测准确性的检验量，理想值 1.0。
- **特征因子调整法(eigenfactor adjustment)** / **贝叶斯收缩(Bayesian shrinkage)** / **Newey–West 调整**：三种协方差矩阵调整手段。
- **WLS / GLS / Fama–MacBeth / DGP(data generating process) / bootstrap**：求解与模拟手段。
- 参考文献：Menchero & Lee (2015)、Menchero et al. (2011)、Ruud (2000)、Ledoit & Wolf (2003)、Black & Litterman (1992)。
