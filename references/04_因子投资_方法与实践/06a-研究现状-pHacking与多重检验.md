# 因子投资·方法与实践 · §6.1 p-hacking与"因子动物园" + §6.2 从"因子动物园"到"因子大战" [PDF p.224–238]

> 石川/刘洋溢/连祥斌《因子投资：方法与实践》· 视觉精读(PDF 物理页 p.224–238) · 生成 2026-06-30

## 本块讲什么

本块是全书"**辨真假因子**"的方法论核心。它不讲新因子，而是回答一个元问题：**学术界挖出的上百个"显著"因子，绝大多数是不是样本内过拟合的产物？怎么把真因子从假因子里挑出来？**

- **§6.1 p-hacking 与"因子动物园"**：拆解"为什么会有这么多超低 p 值的因子"。链条是 *发表偏差(publication bias) → 追逐超低 p 值 → 数据操纵(p-hacking) → 因子动物园(factor zoo)*。给出三件可操作的武器：(1) **正确理解 p 值**（p 值≠"因子为真的概率"）；(2) **多重假设检验校正**（提高 t 门槛、控制 FWER/FDR/FDP）；(3) **先验**（把贝叶斯思想嵌进显著性判断）。
- **§6.2 从"因子动物园"到"因子大战"(factor war)**：当挖因子的热潮退去，学术界转而比拼"谁的多因子模型能解释更多异象"。本块剖析这场比拼在比什么、为什么"解释异象个数"不该是唯一标准，并以 Fama–French 投资因子之争、q⁵ 模型、Fama–French(2018) 的"冷静"为例。

一句话主线：**比起因子收益率在样本内的统计显著性，更应关注因子背后是否有合理的经济学解释**——这是把真因子和数据挖掘产物区分开的根本判据。

## 方法论要点（重点·展开）

### 0. "因子动物园"的由来：发表偏差的因果链

John Cochrane（曾任美国金融协会主席）用 **"factor zoo / 因子动物园"** 挖苦学术界挖因子的狂热，原话已成金句：

> "We thought 100% of the cross-sectional variation in expected returns come from the CAPM, now we think that's about zero and a zoo of new factors describes the cross section." —— John Cochrane

含义：从前认为 CAPM 能解释资产预期收益率在**截面**上的全部差异；如今人们抛弃 CAPM，挖出众多因子来解释。真有这么多能解释收益率的因子吗？答案是否定的。Campbell Harvey（杜克大学，曾任美国金融协会主席）为此大声疾呼学术界应"后退一步"，重审学术氛围与文化(Harvey 2017)。

**为什么会挖出这么多假因子？一条功利驱动的因果链：**

```
p 值越低 → 因子越"显著" → 成果越吸引眼球 → 引用越高
→ 期刊影响因子越高 → 期刊声望越高 → 编辑越倾向接收低 p 值文章
→ 学者越倾向只挖、只发"最显著"的因子（顶级期刊发文≈终身教职）
```

结果是 **发表偏差(publication bias)**：学者只愿发"最显著"的结果，不愿研究"无效因子"。但从推动学科看，**"无效因子"和"有效因子"同样重要**——能确切证明某因子无法带来超额收益，对实务同样有价值（可以放心避开它），可惜顶级期刊鲜有它们的容身之处。

**发表偏差的实证证据（Harvey, Liu & Zhu 2016）**：统计 1963–2012 年顶级期刊上 **316 个因子**的 t 值分布。t 值在 2.00~2.57 的文章数，与 2.57~3.14 的文章数**十分接近**。但 t=2.57 对应（双尾）p≈0.01，t=3.14 对应 p≈0.002——理论上找到 p=0.002 的因子要比 p=0.01 难得多，文章数却大致相同。这只能说明：**在顶级期刊发文时，学者系统性地倾向于更低的 p 值**（分布被人为"推"向高 t 值一侧）。

### 1. p 值的正确含义：低 p 值是"必要非充分"

**p 值定义(Fisher 1925)**：令原假设 H0 为"随机变量 X=0"，样本中观测到 X 取值 x，则 p 值 = 在 H0 下 X 取到比 x **更极端**数值的条件概率。对因子，X 代表因子预期收益。

- 右尾：p = prob(X≥x | H0)
- 左尾：p = prob(X≤x | H0)
- 双尾：p = 2·min{ prob(X≥x|H0), prob(X≤x|H0) }

p 越小→在 H0 下观测到 x 越不可能→越倾向拒绝 H0。**但关键的逻辑陷阱在于**：一旦"因子预期收益为零"被拒绝，人们就顺水推舟地认为"显著不为零"，并误以为"**p 越低→因子收益越显著**"，从而追逐更低的 p。

> **铁律：足够低的 p 值仅是"因子预期收益显著不为零"的必要条件，而非充分条件。** 数据窥探/操纵 + 多重假设检验，会让低 p 值无法准确传递"因子是否真有效"。

**p 值最常见的误解（Harvey 2017 的"董事会规模"思想实验）**：H0=董事会规模与超额收益无关，H1=董事会规模小的公司收益更高；构建多空组合检验，得到 p<0.01（非常显著）。下面四句**全错**：

1. ✗ 我们证明了原假设是错误的
2. ✗ 我们找到了原假设为真的概率 prob(H0|D)
3. ✗ 我们证明了"规模小的公司有更高超额收益"
4. ✗ 我们推断出"规模小的公司有更高超额收益"为真的概率 prob(H1|D)

**为什么全错**：以 D 代表观测到的极端事件，按定义 **p 值 ≡ prob(D|H0)**——它是"数据与假设之间关系"的陈述，不是"假设本身"的陈述，故 ①③ 错（p 值不证明假设真伪）。更要命的混淆是把它当成 prob(H0|D)：

> **式(6.1)：p 值 ≡ prob(D|H0) ≠ prob(H0|D)**

站在研究因子的角度，真正有意义的是 **prob(H0|D)**（看到数据后，"因子无效"为真的概率），但 p 值**不等于**它，混淆二者是严重错误（②错）。同理 p 值≠prob(H1|D)（④错）。

**美国统计协会(ASA)关于 p 值的六准则(Wasserstein & Lazar 2016)——可直接当作 checklist：**

1. p 值衡量"数据与给定统计模型的**不兼容**程度"。
2. p 值**不**表示原假设为真的概率；也**不**表示数据仅由随机因素产生的概率。
3. 科学结论、商业/政策决策**不应只**根据 p 值是否低于某阈值来定。
4. 完整的分析报告 + 完全的透明度，是适当统计推断的前提（即必须**规避 p-hacking**）。
5. p 值或统计显著性，**不衡量效应大小或结果的重要性**（研究因子时还须看因子收益率在**经济上**是否显著，即量级够不够大）。
6. 关于模型/假设是否有效，**p 值本身证据不足**。

### 2. 硬科学 vs 软科学：p-hacking 为何在因子研究里防不胜防

Auguste Comte 把科学分等级：数学、物理这类 **"硬科学"** 在上，社会学（含经济学）这类 **"软科学"** 在下（"硬/软"非"好/坏"之分）。

- **硬科学**：成果不受研究者个人偏好影响，从数据**直接**得到结论，结论高度可归纳（四色定理一旦证明即成立；引力波一旦发现即存在）。
- **软科学**：成果**依赖于"提什么假设、怎么处理数据、怎么解释结果"**，与研究者的声望/利益/偏好挂钩，结论往往**无法归纳**。

**因子研究无疑是软科学**：因子变量选择、组合构建、收益率计算**都因人而异**。研究者面对一连串选择，每一个都是"自由度"：

> 用过去 50 年还是 30 年数据？用美股还是别国股票？每月还是每年再平衡？用百分比收益率还是对数收益率？是否剔除异常值？用截面回归还是时序回归？某因子只对部分股票有效，发论文时是否就只用这部分股票？……

在追逐超低 p 值的暗示下，做这些选择时"心理非常微妙"，**一切阻碍超低 p 值诞生的处理都会被巧妙避开**。这种"为追求低 p 值而千方百计操纵数据、直到结果显著"的行为，就叫 **p-hacking**。

**p-hacking 的本质 = 把科学方法颠倒**：正常是"**先有假设、再用数据验证**"（先观察→提假设→验证）；p-hacking 变成"**先有结果、再凑假设**"（换变量、挑数据范围，直到显著）。这已经偏离科学，沦为对历史数据的过拟合。其根源是那个错误信念："**p 值越低 ≡ 因子预期收益越显著**"。

### 3. ★多重假设检验(multiple hypothesis testing)：本块最硬的可操作内核

**直觉（两个例子）**：
- *硬币例子*：街上随机遇一人，他猜 20 次扔硬币全对→可能真有"天生神力"；但若让**三亿人同时**玩猜 20 次的游戏，20 轮后全对的还有约 250 人——这时只能认为他们**运气好**。
- *因子例子*：只检验 1 个因子、t=2.0(p=0.05)，可认为它在 0.05 水平显著；但若**同时检验 100 个**因子，发现第 55 个 t 最高且 t=2.0，**就不能**说它显著——因为仅靠运气，100 个独立因子里"最显著那个" t>2.0 的概率高达 **99%**。

> **核心结论：用同样的历史数据挖成百上千个因子，就是一次多重假设检验。多重检验会让单个检验的显著性被严重高估。** 若仍用传统 t=2.0 当门槛，必然产生大量**伪发现(false discovery / false rejection)**。**校正的核心 = 控制伪发现的概率 → 单因子 t 值必须远高于 2.0 才可信。**

**表6.1 假设检验结果（混淆矩阵，全块的记号基石）**：设用同样数据挖出 S 个因子，

| | H0 被拒绝（判为"显著因子"/有发现） | H0 被接受（判为"无效"/未发现） | 合计 |
|---|---|---|---|
| **H0 为真**（因子**实际无效**，不能获显著收益） | **F1 = 伪发现** false discovery（**第一类错误 Type I**） | T0（正确接受） | S0 |
| **H0 为假**（因子**实际有效**） | **T1 = 真发现** true discovery | F0 = 漏发现（**第二类错误 Type II**） | S1 |
| 合计 | **R = F1 + T1**（被拒绝/被宣称显著的总数） | S − R | S |

> 书中正文显式定义：S0=H0为真的因子数、S1=H0为假的因子数、R=被拒绝总数=F1+T1、T1=真发现、F1=伪发现；T0、F0 由补集得到。**所有多重检验算法，都是以"控制某个关于 F1 的统计量"为目标。** 这些统计量都刻画**第一类错误**（错误地拒绝 H0）。

**三类控制目标 + 对应算法（按从严到松排列）：**

| 控制目标 | 定义 | 公式 | 严苛度/特点 | 代表算法 |
|---|---|---|---|---|
| **FWER**（族错误率, family-wise error rate） | 出现**至少一个**伪发现的概率 prob(F1≥1) | **prob(F1≥1) ≤ αs**（式6.2） | **最严**：对单假设极苛刻，会大幅抬高第二类错误；**太严苛、不太适合金融领域**(Chordia et al. 2020) | **Bonferroni(1936)**、**Holm(1979)**（直接修正单假设 p 值来压显著性）；White(2000) bootstrap reality check；Romano & Wolf(2005) StepM |
| **FDR**（伪发现率, false discovery rate） | 伪发现**比例**的期望 E[F1/R] | **E[F1/R] ≤ δ**（式6.3） | **较温和**：允许 F1 随 R 增加，控制的是比例；比 FWER 拒绝更多原假设 | **Benjamini–Hochberg(1995)**、**Benjamini–Yekutieli(2001, BHY)**；**对统计量间的相关性不敏感、适应性强** |
| **FDP**（伪发现比例, false discovery proportion） | 限制 F1/R 超过阈值 γ 的概率 | **prob(F1/R ≥ γ) ≤ αs**（式6.4） | 与 FDR 类似 | Romano & Wolf(2007)、Romano et al.(2008) |

> **金融领域的选择**：一般认为以 **控制 FDP / FDR 为目标更好**（因为 FWER 太严、不适合金融）。综述见 Farcomeni(2008)、Romano et al.(2008)。

**★把校正落到一个数：t 值门槛应该提到多少？**

| 研究 | 检验/对象 | 结论（可直接用作筛因子门槛） |
|---|---|---|
| **Harvey, Liu & Zhu (2016)** | 以控制 FDR 校正已发表的 316 个因子 | **单假设原始 t 值需 > 3.0**（而非传统 0.05 对应的 2.0）才算有效；按此，已发表 316 因子中**伪发现比例达 27%**。作者强调 **3.0 其实已经相当保守**（见下） |
| **Chordia, Goyal & Saretto (2020)** | 控制 FDP，并用**模拟(simulation)** 推断"研究过的因子集"（远大于"已发表因子集"） | 关键洞见：Harvey(2016) 只看**已发表**因子，低估了运气成分，故 3.0 偏保守。校正后——**时序回归 t≥3.8、Fama–MacBeth 回归 t≥3.4** 才可能真有效（均高于 3.0） |
| **Harvey & Liu (2015)** | 多重检验对**夏普比率**的影响 | 量化了夏普比率在实盘外的"打折(haircut)"程度；经验常说"实盘夏普≈回测的 50%"，但该文发现**打折程度与夏普比率之间是非线性关系**（不是简单乘 0.5） |
| **Harvey & Liu (2018)** | 基于回归 + 正交化 + 自助法(bootstrap) 的框架 | 用正交化+bootstrap 估出"**仅靠运气**能得到的显著性的经验分布"，由此定出"运气阈值"；某因子显著性超过该阈值才算排除运气后仍显著。**优点：可按显著性顺序逐一挑因子**（最显著、第二显著……直到再无显著因子），从而评价**每个新增因子的增量贡献**。同样可用于"从一堆基金经理/策略里挑出真能战胜市场的" |

> 国内落地：公众号"川总写量化"《出色不如走运(II)》介绍 Harvey & Liu(2018) 框架，《实证研究——使用正交化和自助法寻找显著因子》将其**应用于 A 股**；《出色不如走运(IV)》解读 Chordia et al.(2020)。

### 4. ★先验(prior)的重要性：把贝叶斯思想嵌进显著性判断

发表偏差 + 多重检验之下，很多挖出的因子都是样本内过拟合。**怎么从上百个因子里找真的？关键靠先验。** 先验来自人们对市场的客观认知、来自经济/金融学规律——它允许给每个候选因子一个**先验概率**。例：有人说"按股票代码首字母选股能获超额收益"，你会嗤之以鼻，因为没有科学逻辑，**先验概率极低**。把先验概率与检验的 p 值结合，就能得到"因子是否显著"的**后验概率**——这正是贝叶斯思想。

**三个故事（同样 10 次全对、p 远低于 0.001，认知却截然不同）：**
1. 音乐家声称能完美区分莫扎特和海顿乐谱，连对 10 张 → **先验就信他能行**，实验只是确认。
2. 老妇人声称能尝出奶茶是先加奶还是先加茶，连对 10 杯 → 人们**本有怀疑（先验）**，但 10 次全对让人**倾向推翻先验、拒绝 H0**，承认她有这能力。
3. 酒馆老板号称"酒精赐他预知未来"，连对 10 次扔硬币 → "酒精预测未来"的**先验概率太低**，即便 10 次全对也**不会推翻 H0**，只当他**运气好**。

> 结论：**同样的 p 值，先验不同，该不该拒绝 H0 的结论就不同。** 这就是贝叶斯思想的力量。

**贝叶斯 p 值(Harvey 2017)**：把传统 p 值嵌入贝叶斯框架得到的**后验概率**。它与先验概率、贝叶斯因素有关；贝叶斯因素=（H0 下观测到因子收益的似然）/（H1 下的似然）。因 H1 下的概率分布未知，贝叶斯因素有取值范围，其**下限**称**最小贝叶斯因素 φ**——φ 越小越倾向拒绝 H0。两种算法：

> **式(6.5)：φ = −e · p · ln(p)**（e=exp(1)，用 p 值算）
> **式(6.6)：φ = exp(−t²/2)**（用 t 值算）
> **式(6.7)：贝叶斯 p 值 = (φ × 先验) / (1 + φ × 先验)**

作为后验概率，贝叶斯 p 值的强大在于它直接回答了人们真正关心的 **prob(H0|D)**（看到极端事件后，"因子无效"为真的条件概率）。

> **[符号/可操作还原]** 式(6.7) 里的"先验"按书中三个实证例子反推，应为**先验赔率** P(H0)/P(H1)（而非直接代入"因子为真的先验概率"）。等价的、可直接计算的形式是：
> **贝叶斯 p 值 = prob(H0|D) = φ·P(H0) / [ φ·P(H0) + P(H1) ]**。
> 验证（聪明股票代号因子）：原始双尾 p=0.0079 → t≈2.66 → 由(6.6) φ=exp(−2.66²/2)≈0.029；先验 P(H1)=2%、P(H0)=98% → 0.029×0.98/(0.029×0.98+0.02)≈**0.588**，与书中一致。注意(6.5)与(6.6)给的 φ 不同（同例 (6.5) 得 0.104），书中实证例子用的是 t 版(6.6)。

**Harvey(2017) 的三类先验 + 三个实证结论（直接给出"哪些因子经得起贝叶斯检验"）：**

| 先验类别 | 先验概率 | 代表因子 | 原始 p 值 | **后验贝叶斯 p 值** | 判定 |
|---|---|---|---|---|---|
| 非常不可能（无逻辑支撑） | **2%** | "聪明的股票代号"因子(Head et al. 2009)——代号好听的股票有超额收益 | 0.0079（很低！） | **0.588** | 任何常见水平都**无法拒绝 H0** → 贝叶斯框架**扭转了"显著"的错误结论** |
| 可能 | **20%** | FF 五因子中的**盈利因子** | 超低 | **0.053** | 后验仍较低（虽比原始高很多），勉强 |
| 可能 | **20%** | FF 五因子中的**规模因子** | 超低 | **0.117** | 后验偏高，存疑 |
| 非常可能（有金融/经济理论支撑） | **50%** | **市场因子** | 超低 | **0.035** | 仍在 0.05 下显著，站得住 |

> **总判据**：能获超额收益的因子**不是不存在，但非常稀有**，数量远少于学术界挖出的上百个伪因子。**实践告诫——研究因子时，比起样本内显著性，更应关注因子背后是否有合理的经济学解释；只有有合理解释，它才可能是真因子而非过拟合产物**（书 §6.5 专讲因子背后的解释）。可复现性也在变成主流（顶级期刊中仅 Journal of Finance 要求提供代码，至今无期刊要求提供数据，导致 p-hacking 难被复现）。

### 5. §6.2 因子大战：在比什么、为什么"解释异象个数"不该是唯一标准

**好的多因子模型应回答两个问题**（也是评判模型优劣的根本，而非数异象个数）：
1. 资产收益率背后的**驱动有哪些**（模型该含哪些因子）；
2. 每个驱动**背后的原因**（因子的经济学解释）。

最早的 Fama–French 三因子确实朝此努力：FF(1996) 从风险角度把规模(SMB)、价值(HML)与**财务困境风险**联系起来。但**自三因子之后，新模型越来越不回答这两问，而是比拼"谁解释更多异象"**。作者把"用解释异象能力来评判模型高下"称为 **"因子大战(factor war)"**，并强调：解释异象固然重要，但**不该是研究/比较多因子模型的唯一目标**。

**§6.2.1 形同意不同的投资因子（FF 投资因子被"抨击"）**：FF 五因子与 Hou–Xue–Zhang(HXZ) 四因子**都含投资因子、变量都用过去一年总资产变化率、都用排序法**，但出发点截然相反：
- **HXZ 来自实体投资经济学理论(q-theory)**：数学模型直接推出"预期收益率与**历史**投资成反比"，故用最近财年总资产变化率构建(Hou et al. 2015)。
- **FF 来自股息贴现模型(DDM)**：推出"预期收益率与**未来预期**投资成反比"，再用**历史投资作为未来投资的朴素估计**(FF2015，依据 FF2006)。

Hou et al.(2019b)《Which factors?》从理论+实证两面质疑 FF 投资因子：
- *理论*：DDM 里的"长期平均收益率"≠多因子模型真正关心的"未来单期预期收益率"。把前者换成后者，可得式(6.8)（见下）。因市净率 M/B 通常>1，在其他变量固定时，**更高的预期账面价值变化率 → 隐含更高的预期收益率**，即**正相关**（再用总资产变化率替换账面价值变化率，结论不变）。这与 FF(2015) 主张的**负相关**相反。
- *实证*：用历史投资当未来投资的朴素估计**并不靠谱**。
- *结论*：FF(2015) 的投资因子其实是"**负负得正**"——它阴差阳错地利用了 q-theory 支持的"过去投资 vs 收益率"负相关，而非 DDM 推出的"预期投资 vs 收益率"负相关。这撼动了 FF 五因子中投资因子的根基。

> **作者的警觉（很重要的方法论态度）**：作者倾向认为 HXZ 的解读更合理，但对这种"因子竞赛"感到不安——**"人们总能用不同的金融/经济模型，推出截然相反的结果"**。式(6.8) 与 DDM 式(4.9) 一比，无非是把"长期平均收益率"换成"单期预期收益率"，**巧妙移除了账面价值变化前的负号，就把"负相关"改成了"正相关"**。"对着历史数据讲最适合数据的故事，真的让人离真相更近了吗？"

实证上 HXZ 投资因子的因子溢价也**完胜** FF：因为 q-theory 里"投资 vs 收益率负相关"是**在控制 ROE 之后**才成立。HXZ 用市值、ROE、总资产变化率做 **2×3×3 三重排序**（控制了 ROE）；FF 只用市值、总资产变化率做 **2×3 双重排序**（没控制 ROE），故效果不如 HXZ。

**§6.2.2 q⁵ 模型**：Hou et al.(2020) 在 HXZ 四因子上加一个**预期投资增长因子**，得 Hou–Mo–Xue–Zhang 五因子（即 **q⁵**）。动机来自 Cochrane(1991)：在控制当期投资与预期盈利后，**预期投资增长率与预期收益率成正比**。数学上 q⁵ 是 HXZ 的直接延伸——HXZ(2015) 是两期（单期投资）框架不涉及投资增长；q⁵ 扩展到**多期**（折旧率 δ，HXZ 中 δ=1），由投资第一性原理得投资收益式(6.9)。其中第一项≈分红，第二项（平方/边际调整费用，高阶小量忽略），**第三项≈预期投资增长**（即 I/A 之比的预期增长率）。实务构造时用历史数据做**截面回归预测模型**，再用最新财务数据预测下期 I/A 增长率作为因子变量。**作者提醒：该构造复杂度高，可能有模型设定与参数估计误差。**

**§6.2.3 因子大战的硝烟（按发表时间梳理）**：
- HXZ(2015) 用 **80 个异象**当测试资产，胜过 FF 三因子、Carhart 四因子；
- Stambaugh–Yuan(2017) 四因子、Daniel–Hirshleifer–Sun(2018/2020) 三因子相继问世，各自"赢"过旧模型；
- q⁵(Hou et al. 2019b) **解释最多异象**，成为现阶段"**最强**"的实证资产定价模型（但作者断言：不久必有新模型来打败它）。

> **作者对 DHS 三因子"不含规模因子"被质疑的反驳，是理解因子大战之弊的范例**：DHS 从行为金融理论出发提出长/短两个时间尺度的行为因子。**行为金融理论本就推不出规模因子**，故不含规模因子无可厚非；硬加进去反而更有数据窥探之嫌。"只有当人们把目光局限于'谁能解释更多异象'，才会得出该加规模因子的结论。"

**Fama–French(2018)《Choosing factors》的"冷静"（全块的收束）**：作为多因子模型的开创者，FF 没有去和别人的模型比，而是"**自己和自己比**"（比 CAPM、FF3、FF5、及加动量的六因子）：
- 方法：拓展 **GRS 检验**，用"模型所含因子能构成的**最大夏普比率的平方**"来比较不同模型；
- 担忧：CAPM/CCAPM 虽被实证拒绝，但它们在**收益与风险之间建立了有效联系**；而各种多因子模型更多是"对历史收益率数据中某种模式的挖掘"。**模型背后必须有足够的理论支撑**，否则就退化成"对着成百上千因子、以提升事后有效前沿切点组合夏普比率为目标的数据挖掘"；
- 一条因子模型的本职判据：**因子必须与个股的协方差矩阵密切相关**(Pukthuanthong et al. 2019)——因为模型是为了解释股票收益率**如何共同运动**。仅以"谁解释异象更多"比模型，偏离了多因子模型的出发点，意义有限。FF(2018) 因此呼吁"**高质量、有意义**的因子模型比较"。

## 关键公式 · 定义 · 判定标准

**p 值与误解（§6.1.1、§6.1.4）**
- p 值（双尾）= 2·min{ prob(X≥x|H0), prob(X≤x|H0) }
- **式(6.1)：p 值 ≡ prob(D|H0) ≠ prob(H0|D)**（D=观测到的极端事件）。低 p 值是因子有效的**必要非充分**条件。

**多重假设检验的三类控制目标（§6.1.5）**
- 式(6.2) FWER：prob(F1≥1) ≤ αs ——最严，不适合金融。算法 Bonferroni / Holm / White / Romano–Wolf StepM。
- 式(6.3) FDR：E[F1/R] ≤ δ ——温和，控比例。算法 Benjamini–Hochberg / Benjamini–Yekutieli(BHY)，对相关性不敏感。
- 式(6.4) FDP：prob(F1/R ≥ γ) ≤ αs ——与 FDR 类似。算法 Romano–Wolf。
- 记号（表6.1）：S 总因子数；S0=H0真(无效)；S1=H0假(有效)；R=F1+T1=被拒绝(宣称显著)总数；F1=伪发现(Type I)；T1=真发现；F0=漏发现(Type II)。

**★t 值门槛判定标准（直接可用于筛因子/scorecard）**
- 传统：t≥2.0（p≤0.05）——**单因子时勉强，多重检验下已失效**。
- **Harvey-Liu-Zhu(2016)：t > 3.0**（控 FDR；已发表 316 因子伪发现率 27%；3.0 仍偏保守）。
- **Chordia et al.(2020)：时序回归 t≥3.8、Fama–MacBeth t≥3.4**（含未发表因子集，更严）。
- 夏普比率：实盘"打折"与夏普之间**非线性**（不是简单×0.5）——Harvey-Liu(2015)。

**贝叶斯化（§6.1.6）**
- 式(6.5) φ = −e·p·ln(p)；式(6.6) φ = exp(−t²/2)（φ=最小贝叶斯因素，越小越倾向拒绝 H0）。
- 式(6.7) 贝叶斯 p 值 =(φ×先验)/(1+φ×先验)；可操作形式 = φ·P(H0)/[φ·P(H0)+P(H1)] = prob(H0|D)。
- 先验三档：无逻辑 2% / 可能 20% / 有理论支撑 50%。

**因子大战相关（§6.2）**
- 式(6.8)：M_t/B_t = { E[Y_{t+1}/B_t] + E[(dB_{t+1}/B_t)(M_{t+1}/B_{t+1}−1)] + E[M_{t+1}/B_t] } / (1+E[r_{t+1}])（DDM 换成单期收益率的版本；推出预期投资与预期收益率**正相关**）。
- 式(6.9)：r^I_{it+1} ≡ { Π_{it+1} + (a/2)(I_{it+1}/A_{it+1})² + (1−δ)[1+a(I_{it+1}/A_{it+1})] } / (1+a(I_{it}/A_{it}))（q⁵ 多期投资收益；第三项≈预期投资增长）。
- 模型比较判据(FF 2018)：**最大夏普比率的平方**（拓展 GRS），且因子须与个股协方差矩阵密切相关。

## A股特殊性 / 实证结论

- **本块以美国学术界的方法论为主，非 A 股制度细节**；但教训对 A 股**全部适用，且更紧迫**——A 股历史更短、数据窥探空间更大，"因子动物园 + p-hacking"风险只多不少。**在 A 股做因子筛选时应主动套用提高后的 t 门槛（≥3.0，时序检验趋近 3.8 / FM 回归趋近 3.4）与多重检验校正**，不能沿用 t=2.0。
- **已有 A 股落地**：公众号"川总写量化"《实证研究——使用正交化和自助法寻找显著因子》把 Harvey & Liu(2018) 的正交化+自助法框架**实证应用于 A 股**，可作为本地复现的现成参考。
- **实证排序结论（贝叶斯检验后哪些因子真站得住）**：市场因子(后验 p=0.035)站得住；盈利因子(0.053)勉强、规模因子(0.117)存疑；"聪明股票代号"等无逻辑因子(0.588)被否。**有经济学解释的因子才扛得过先验校正。**
- **模型层面结论**：q⁵（实体投资经济学/q-theory）是现阶段最强实证资产定价模型；FF 五因子的投资因子根基（"负负得正"）被质疑。但作者立场是：**别为"打赢因子大战"而比，应为"加深对收益率驱动力的理解"而比。**

## 踩坑对照（√252重叠年化 / 单调性判定 / 合成稀释尾部 / 其他）

- **√252 重叠收益年化**：本块未直接涉及（属 §2 检验章）。但**强相关的同类坑**——夏普比率的实盘"打折"：经验常说"实盘夏普≈回测 50%"，**Harvey & Liu(2015) 证明打折与夏普之间是非线性关系，不能简单乘 0.5**；这与"重叠样本下不能简单 √T 年化"同属"别用朴素缩放糊弄统计量"的大类。
- **单调性判定**：本块未涉及（见 §2.1 排序法笔记）。
- **合成稀释尾部**：本块未涉及（见多因子合成章）。
- **★本块自身的头号踩坑 ①——用 t=2.0 当因子显著门槛**：
  - *现象*：用历史数据挖几百个因子，挑出 t 最高的一个，t>2.0 就当显著。
  - *书里为什么*：多重检验下，纯靠运气，100 个独立因子中"最显著那个" t>2.0 的概率≈99%；已发表 316 因子按 FDR 校正后伪发现率达 27%。
  - *正确做法*：**提高门槛**——t>3.0（Harvey 2016），时序检验趋近 3.8、Fama–MacBeth 趋近 3.4（Chordia 2020）；并用 FDR/FDP 算法(BH、BHY、Romano–Wolf)或 Harvey–Liu(2018) 正交化+自助法定"运气阈值"。
- **★本块头号踩坑 ②——把 p 值当成"因子为真的概率"**：
  - *现象*：看到 p<0.01 就说"证明了因子有效""原假设为假的概率 99%"。
  - *书里为什么*：p 值≡prob(D|H0)≠prob(H0|D)；它是数据与假设的关系，不是假设真伪的概率。
  - *正确做法*：要回答 prob(H0|D)，须引入**先验**走贝叶斯（贝叶斯 p 值 式6.5–6.7）；并对照 ASA 六准则自查。
- **★本块头号踩坑 ③——p-hacking / 先有结果再凑假设**：
  - *现象*：换变量、挑数据范围、只用对自己有利的股票子集，直到结果显著。
  - *书里为什么*：因子研究是软科学，研究者自由度大，"一切阻碍超低 p 值的处理都会被巧妙避开"，结果只是过拟合。
  - *正确做法*：**先有经济学假设、再验证**；预先登记处理口径、全程透明、保证可复现（ASA 准则 4）；事后还要看因子收益率在**经济上**是否显著（准则 5），不只看统计显著。

## 对 scorecard / 回测框架的直接启示

1. **t 门槛硬性上调**：scorecard 里"因子显著性"一项的及格线**不要用 t=2.0**。建议——时序回归用 **t≥3.0（严格 3.8）**、Fama–MacBeth 用 **t≥3.4**，作为"单因子可信"的最低线；低于此一律标记为"疑似伪发现"。
2. **加一项"多重检验惩罚"**：凡是"从 N 个候选里挑出来的"因子，必须记录 **N（搜索空间大小）** 并做 FDR/FDP 校正（BH/BHY 或 Harvey–Liu(2018) 自助法阈值），把"挑选过程"本身计入风险；单看入选因子的 p 值是系统性高估。
3. **强制"先验/经济学解释"打分项**：在 scorecard 中为每个因子赋一个**先验概率**（无逻辑 2% / 可能 20% / 有理论 50%），并计算**贝叶斯 p 值 = φ·P(H0)/[φ·P(H0)+P(H1)]**（φ=exp(−t²/2)）。**没有合理经济学解释的因子，即便样本内 t 很高，也应被先验拉低、降级或淘汰**——这是区分真因子与过拟合的根本判据。
4. **顺序增量评估**：用 Harvey–Liu(2018) 思路——按显著性**逐一**纳入因子（最显著→第二显著……直到再无显著者），每步评估**新因子的增量贡献**，避免一次性塞入一堆相关因子互相"喂"显著性。该框架也可直接用于"从一堆策略/子模型里挑真正跑赢的"。
5. **夏普比率打折非线性**：回测夏普换算到实盘预期时，**别用固定 ×0.5**，参照 Harvey–Liu(2015) 的非线性 haircut；高回测夏普≠高实盘夏普。
6. **模型比较用"最大夏普²"而非"数异象个数"**：若要比多个多因子模型，采用 FF(2018) 的 GRS 拓展（最大夏普比率平方），并核查因子是否与个股**协方差矩阵**相关；不要把"解释异象个数"当唯一 KPI。
7. **流程纪律**：研究先写假设、固定数据处理口径、留存代码与数据保证可复现；区分"统计显著"与"经济显著"（量级），两者都进 scorecard。

## 术语与存疑

- **术语**：p-hacking（数据操纵到显著）/ 发表偏差(publication bias) / 因子动物园(factor zoo) / 因子大战(factor war) / 多重假设检验(multiple hypothesis testing) / 伪发现(false discovery) / 第一类错误(Type I)·第二类错误(Type II) / FWER(族错误率) / FDR(伪发现率) / FDP(伪发现比例) / 最小贝叶斯因素(minimum Bayes factor, φ) / 贝叶斯 p 值 / 先验(prior)·后验。
- **人名/模型**：Cochrane / Campbell Harvey / Fisher / Wasserstein & Lazar(ASA) / Bonferroni / Holm / White / Romano–Wolf / Benjamini–Hochberg / Benjamini–Yekutieli / Harvey-Liu-Zhu(2016) / Chordia-Goyal-Saretto(2020) / Harvey-Liu(2015,2018) / Hou–Xue–Zhang(HXZ) 四因子 / q⁵ (Hou–Mo–Xue–Zhang) / Stambaugh–Yuan / Daniel–Hirshleifer–Sun / Fama–French(1996/2006/2015/2018) / Carhart / Pukthuanthong et al. / GRS 检验。
- **αs**：书中用 α 表示超额收益，故显著性水平改记 αs（下标 s=significance）。
- **[符号存疑] 式(6.7)**：书面写作"贝叶斯 p 值=(φ×先验)/(1+φ×先验)"，但按书中三个实证例子反推，分子/分母里的"先验"应为**先验赔率 P(H0)/P(H1)**，而非直接代入"因子为真的先验概率"。等价可算形式 = φ·P(H0)/[φ·P(H0)+P(H1)] = prob(H0|D)，已用"聪明股票代号"例(p=0.0079→t≈2.66→φ≈0.029，先验2%→0.588)验证一致。
