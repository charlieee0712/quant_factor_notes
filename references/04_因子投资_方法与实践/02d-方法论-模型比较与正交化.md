## 本块讲什么

两大主题，互为表里：

1. **§2.5 怎样科学地"比较两个多因子模型谁更好"**。所有多因子模型都"不完美"，但有些"有用"（因子有可靠经济学/金融学依据 + 能解释大量资产预期收益）。比较的逻辑主线是 **"两个目标、两个切入点、多种方法"**：
   - **两个目标**（Barillas-Shanken 2017）：① 比较不同模型对**同一组测试资产(test assets)** 的解释程度；② 不同模型**两两互检**——能否解释对方的因子。
   - **两个切入点**：被解释资产总是多个 → ① **联合检验**所有定价误差是否同时为零；② **独立看待**每个资产的定价误差。
   - **多种方法**：联合检验 → **GRS 检验** + **均值—方差张成(mean-variance spanning)检验**；独立看待 → **α 检验**；外加 **贝叶斯方法**。
   - 关键认知：一旦选定方法，"测试资产"和"对方模型的因子"都只是被解释变量，对统计方法本身无差异 → 同一套工具既测资产、也做因子互检。实证中两个切入点的方法**经常同时用**，让结论更稳。

2. **§2.6 因子正交化的原理、几何意义与数学后果**。"正交"= 两个因子代表的收益驱动力不同（贡献超额收益里不同的部分）；正交化目的 = **降低资产对各因子暴露在截面上的相关性**。核心结论：因子暴露截面高相关 → Fama–MacBeth 截面回归求因子收益率时**标准误暴涨、估计不稳**；用 **Gram–Schmidt 连续正交化** 可把"某因子的回归系数"诠释为"该因子相对其他因子的**增量贡献**"。

## 方法论要点（重点·展开）

### A. GRS 检验（2.5.1）——联合检验 α，"切点组合夏普比能否被显著抬高"

- **设定**：N 个待检验资产、待检模型有 K 个因子。对每个资产做时序回归 `R^e = α + βλ_t + ε`，得定价误差向量 α̂（N×1）、残差向量 ε̂_t、因子收益向量 λ_t（K×1）。
- **统计量（矩阵形式，式2.16/正文）**：
  ```
  [(T−N−K)/N] · (1 + E[λ_t]' Σ̂_λ⁻¹ E[λ_t])⁻¹ · α̂' Σ̂⁻¹ α̂  ~  F_{N, T−N−K}
     其中 Σ̂_λ = (1/T)Σ_t (λ_t−E[λ_t])(λ_t−E[λ_t])'    ← 因子协方差
           Σ̂   = (1/T)Σ_t ε̂_t ε̂_t'                       ← 残差协方差
  ```
- **两个优点**：① **有限样本(finite sample)精确分布**——直接给出给定样本 T 下 α̂ 应满足的联合分布，是 F 分布而非渐近 χ²（大样本下 α̂ 联合分布才渐近 χ²，但有限样本用 χ² 不可靠，这正是 GRS 的价值）；② **检验效力(power)高**。
- **两个代价**：① 精确性**高度依赖正态分布假设**，现实中常被违背 → 实践吸引力打折；② 要求 **样本数 T > 资产数 N** → 资产很多时必须拉长历史窗口。
- **★ 判优用的直观形式（式2.68）**——把统计量改写成**夏普比之比**：
  ```
  [(T−N−K)/N] · ( [ √(1+θ̂²_{N+K}) / √(1+θ̂²_K) ]² − 1 )  ~  F_{N, T−N−K}
  ```
  θ̂_{N+K} = 由全部 N 资产 + K 因子构成的事后(ex post)最大夏普比组合的夏普比；θ̂_K = 仅由 K 因子构成的事后最大夏普比。**判优逻辑**：在 K 因子之外加入 N 资产后，可达到的最大夏普比是否**显著高于**只用 K 因子的最大夏普比。显著提高 → 模型**解释不了**这 N 资产 → 拒绝 H0（模型差）。
- **联合检验的局限**：即便拒绝 H0，也只能说"这 N 个资产作为整体无法被模型解释"，**无法定位**是哪个/哪几个资产在起作用。

### B. 均值—方差张成检验（2.5.2）——"加入新资产能否让最小方差前沿整体更优"

- **思想来源**：Huberman-Kandel(1987)，与 Markowitz(1952)现代投资组合理论(MPT)/均值—方差分析血脉相连；Kan-Zhou(2012)做了系统介绍。
- **"张成(spanning)"的含义**：K 个因子组合按各种权重配置 → 对每个目标预期收益 μ̂ 都有唯一的最小方差配置 → 所有 μ̂ 的最小方差组合连成 **最小方差前沿(minimum-variance frontier)**，形状为抛物线（图2.8），称"这条前沿由 K 因子张成"。
- **★ 检验关注**：加入 N 个新资产后，N+K 资产张成的新前沿能否**"优于"** 仅 K 因子张成的前沿。"优于"= 对每个给定 μ̂，N+K 前沿上的点都比 K 前沿上的点**方差更低**。
- **原假设的充要条件（式2.69–2.71）**：令 R_t=[R_{1t},R_{2t}]'（R_1=K 因子收益、R_2=N 资产收益），由多因子模型 `R_{2t} = α + βR_{1t} + ε_t`，可解出 `α = μ_2 − βμ_1`、`β = V_{21}V_{11}⁻¹`；定义 `δ = 1_N − β1_K`，则
  ```
  H0:  α = 0_N   且   δ = 0_N
  ```
  - α=0 + δ=0（即 β1_K=1_N）→ 总能用原始 K 因子搭一个组合，其预期收益与测试资产**相同**；
  - 式2.70 中 R_1 与 ε 不相关且 var(ε)>0 → 该组合方差**更低**。
  - 两条合起来：N 资产无法在 K 基础上张成更优前沿 → 接受 H0。
- **几何直觉（Kan-Zhou 2012）**：N+K 前沿上有两个特殊组合——**全局最小方差组合(GMV)** 与 **从原点(μ̂=0,σ̂=0)向前沿做切线的切点组合**。H0 成立时，δ=0 ⇒ GMV 里 N 资产权重全为 0；α=0 ⇒ 切点组合里 N 资产权重全为 0 → 这两个特殊组合**纯由 K 因子构成**。再由 **两基金分离定理(two-fund separation theorem，Merton 1972)**：前沿上任意两组合即可线性组合出整条前沿 → 若这两个组合都不含 N 资产，则整条前沿都不含 N 资产 → 这就是 "α=0_N 且 δ=0_N 为充要条件" 的来历。
- **三个统计量（式2.72–2.77）**：先在纵轴取 (0,r)，向 K 因子前沿做切线、切点 (μ̂_tp, σ̂_tp)，定义切线斜率 `θ̂_K(r) = (μ̂_tp − r)/σ̂_tp`（r=R_f 时即夏普比，Sharpe 1966a）；对 N+K 前沿同法定义 θ̂_{N+K}(r)。于是
  ```
  s1 = max_r [ (1+θ̂²_{N+K}(r)) / (1+θ̂²_K(r)) ] − 1      ← 两前沿差异"最大"的那个 r
  s2 = min_r [ (1+θ̂²_{N+K}(r)) / (1+θ̂²_K(r)) ] − 1      ← 两前沿差异"最小"的那个 r
  LR = T·( ln(1+s1) + ln(1+s2) )            ~ χ²_{2N}   (似然比)
  W  = T·( s1 + s2 )                         ~ χ²_{2N}   (Wald)
  LM = T·( s1/(1+s1) + s2/(1+s2) )           ~ χ²_{2N}   (拉格朗日乘数)
  ```
  三者都是把 s1、s2"加权相加"成一个综合分数；本质是在纵轴上搜两个特殊 r（差异最大/最小），用两前沿的综合差异检验是否统计显著。无显著差异 → 接受 H0。注：以上为**大样本渐近**；T 相对 N+K 不够大时不准，需像 GRS 一样上**有限样本**统计量（2.5.3 给几何解释）。
- **GRS vs 张成 的关键差别**：GRS **假设存在无风险利率 R_f 且可无约束借贷**，只关心**切点组合**；张成**不假设 R_f**，从两条前沿上各取特殊点比较 → **适用面更广**。

### C. 从几何角度比较（2.5.3）——"几何之美、数学之美"

- 把式2.75–2.77（代入 s1,s2）与式2.68 并排看，**都含 θ̂**：GRS 的 θ̂ 默认用 R_f 算夏普比；张成的 θ̂ 用一般 r 算 → 注定相关。两者本质同一目标：**新加 N 资产能否在 K 因子上改善组合的风险—收益特征**；若能 → 拒绝 H0。
- **存在 R_f 时**（图2.10a）：有效前沿(efficient frontier)= 过 (0,R_f) 与切点组合的直线（斜率最高=夏普比最高）。
- **★ GRS 的几何意义（图2.11）**：纵轴取相对 R_f 的超额收益，从横轴 σ̂=1 处作竖线，交两条切线于 A、B 两点 → A、B 纵坐标恰为 θ̂_K、θ̂_{N+K} → 线段 **OA=√(1+θ̂²_K)**、**OB=√(1+θ̂²_{N+K})**。GRS（式2.68）正是 OB/OA 之比 → **几何意义 = 看 OB 是否显著长于 OA**（即切点组合夏普比是否被显著抬高）。**GRS 只看切点，不管前沿其它点**。
- **不存在 R_f 时**（图2.10b）：有效前沿 = 最小方差前沿的**上半部分**，没有传统切点组合 → 只比切点不够 → 改为从两前沿各取**特殊点**比较，这就是张成检验的有限样本几何含义；三种统计量的差别仅在于**各自选的特殊点不同**。
- **图2.12 的六条线段**：g_K、g_{N+K} 为两前沿的事后最小方差组合，标准差由 OD、OC 表示（OD≥OC）。以两组合向纵轴投影得 A、B，分别作切线/渐近线交 σ̂=1 于 G、H、E、F，对应长度：
  ```
  AG=√(1+θ̂²_K(μ̂_{g_{N+K}}))   AH=√(1+θ̂²_{N+K}(μ̂_{g_{N+K}}))
  BE=√(1+θ̂²_{N+K}(μ̂_{g_K}))    BF=√(1+θ̂²_K(μ̂_{g_K}))
  ```
  - **似然比（有限样本，~F_{2N,2(T-K-N)}）**：`LR = [(T−K−N)/N]·[ (OD/OC)(AH/BF) − 1 ]`。OD/OC≥1（K 的 GMV 标准差 ≥ N+K 的）、AH/BF≥1（N+K 前沿必"优于"K 前沿）；H0 成立（事前两前沿相同）→ 两比值都不显著偏离 1；任一显著>1 → 拒绝。
  - **Wald 与 LM 的"完美对称"（式2.79/2.80）**：
    ```
    W:  s1+s2          = (OD/OC)² − 1  +  (BE/BF)² − 1
    LM: s1/(1+s1)+s2/(1+s2) = 1 − (OC/OD)²  +  1 − (AG/AH)²
    ```
    第一项都是两 GMV 标准差的偏离（W 用 (OD/OC)²−1，LM 因分子分母互换用 1−(OC/OD)²）；第二项 W 的 (BE/BF)²−1 衡量"加入 N 资产使切线斜率²的提升"，LM 的 1−(AG/AH)² 衡量"剔除 N 资产使切线斜率²的降低"——一进一出，对称。

### D. α 检验（2.5.4）——独立看待每个 α，取绝对值均值

- 与 GRS/张成（联合）不同，α 检验**逐个**看 α_i 是否为零，最后**取平均**评价模型。
- **实操（极简）**：对每个测试资产/对方因子，超额收益作被解释变量、待检模型作解释变量，**时序回归**估 α̂_i 及其标准误（**标准误通常用 Newey–West 调整**），算 t = α̂_i / s.e.(α̂_i)。H0 下 α_i=0。
- **★ 两个判优指标**：**|α̂_i| 的均值** 与 **|t 值| 的均值**（取绝对值因只关心偏离 0 的程度、不关心方向）。两指标**越低 → 模型越能解释资产 → 越好**。
- 应用极广（Hou et al. 2015、Fama-French 2020），且**常与 GRS 同时使用**；本书第4章实证多因子模型时即两者并用。

### E. 贝叶斯方法（2.5.5）——边际似然/贝叶斯因子，及其争议

- Barillas-Shanken(2018，*Journal of Finance*)：模型 `R^e_t = α + βλ_t + ε_t`，令 Σ=cov(ε_t)。先验设定：β、Σ 服从特定**非正常先验(improper prior)**；α 在 H0 下为 0、在备择下服从 `f(α|β,Σ)=N(0, τΣ)`（τ>0）。
- **边际似然度(marginal likelihood，式2.81)**：`prob(D|M_i)=∬ f(D|M_i,α,β,Σ) f(α|β,Σ) f(β,Σ) dα dβ dΣ` —— 给定模型 M_i 下观测到数据 D 的条件概率。
- **后验概率比（式2.82）**：`prob(M_i|D)/prob(M_j|D) = [prob(M_i)/prob(M_j)] · [prob(D|M_i)/prob(D|M_j)]`。第二项 = 边际似然之比 = **贝叶斯因子(Bayes factor)**；通常假设两模型先验概率相等 → **边际似然高低主导模型选择**。
- 应用：Stambaugh-Yuan(2017) 比较其四因子 vs FF5 vs Hou et al.(2015) 四因子。学术界用得比 GRS/张成少。
- **★ 争议（Chib et al. 2020，同登 JoF，标题加 "On" 评价前文，"火药味十足"）**：直指 BS(2018) **有误**。问题在 β、Σ 的非正常先验里需定一个常数，只有当所有待比模型满足 ①β_i、Σ_i 同一非正常先验、②常数对所有模型相同、③参数空间一样，用式2.82 比较才合理；BS(2018) 不满足 → 比较不正确。Chib 修正了先验、给出改进法。**本书因贝叶斯法尚存疑而不深入**。

### F. 因子正交化的动机（2.6 引言）

- "正交" 的经济学含义：两因子代表**不同的收益驱动力**，贡献超额收益里不同的部分；一组相互正交的因子往往能**更好地解释预期收益的截面差异**。
- **极端例子（冗余）**：用市盈率(P/E)、市净率(P/B)分别造两个价值因子。P/E 高(低)的股票通常 P/B 也高(低) → 两个多头组合高度相似、两个空头也相似 → 两个价差(因子)组合高度相似 → 模型名义上 2 个因子，实则**只解释了价值这一维**。**两因子是否相似，体现在"资产对它们的暴露在截面上相关性的高低"**；正交化目的就是**降低这种截面相关性**。
- **数学后果（一句话）**：用 **Fama–MacBeth 截面回归** 求因子收益率时，因子暴露的**截面高相关会放大因子收益率的标准误** → 影响对因子收益率的检验。一般线性模型 `y = Xb + ε`（式2.83；多因子语境：y=N 资产超额收益，X 为 N×(K+1)，首列全 1=截距、后 K 列=因子暴露，b=因子收益率），OLS `b̂=(X'X)⁻¹X'y`（式2.84）。**X 各列（不同因子的暴露向量）不正交时，列间相关会污染 X'X 与 b̂ 的计算——这不是好事**。

### G. 一元回归与"正交即解耦"（2.6.1）

- 无截距一元回归 `y=bx+ε` 的 OLS：`b̂ = ⟨x,y⟩/⟨x,x⟩`（式2.85；⟨x,y⟩=Σxᵢyᵢ 内积）。
- **★ 关键推论（式2.86）**：若多元模型里**所有解释变量两两正交** ⟨x_i,x_j⟩=0 (i≠j)，则每个系数 `b̂_i = ⟨x_i,y⟩/⟨x_i,x_i⟩` —— 与一元回归**一模一样**。原因：⟨x_i,x_j⟩=0 ⇒ X'X 为**对角阵** ⇒ 逆也是对角阵（对角元取倒数）⇒ b̂_i 只由 x_i 与 y 决定。
- **含义**：解释变量相互正交时，**不同因子暴露对彼此的因子收益率没有影响**——这正是因子正交的好处（解耦、各管各）。

### H. 回归的几何意义（2.6.2）

- **残差与解释变量正交（式2.87）**：`X'ε̂ = X'(y−Xb̂) = X'y − X'y = 0`。OLS 残差 ε̂ ⊥ X。
- **一元（图2.13）**：把 y **垂直投影**到 x 上，使 y 到投影的距离（‖ε̂‖）最短；投影 = (⟨x,y⟩/⟨x,x⟩)·x。
- **二元正交（图2.14，x_1⊥x_2）**：ŷ = y 到 x_1,x_2 张成平面的投影；因 x_1⊥x_2，ŷ **恰等于** y 单独投到 x_1 的分量 + 单独投到 x_2 的分量之和 → b̂_i 仅由 x_i、y 决定，其他 x_j 无影响。
- **二元非正交（图2.15，x_1 与 x_2 不正交）**：ŷ ≠ 两个单独投影之和 → 解释变量之间对各自系数的 OLS 估计**有相互作用**，b̂_i **不再等于** ⟨x_i,y⟩/⟨x_i,x_i⟩。怎么量化这种作用？→ 连续正交化。

### I. 用 Gram–Schmidt 正交化求解多元回归（2.6.3）★ 核心

- **铺垫（含截距的二元，式2.88–2.91）**：`y=ax_0+bx_1+ε`，x_0=[1,…,1]'、x_1=[x_1,…,x_n]' 非正交。令 z_0=x_0=1，把 x_1 对 z_0 做无截距一元回归取残差得正交向量 `z_1 = x_1 − (⟨z_0,x_1⟩/⟨z_0,z_0⟩)z_0 = x_1 − x̄·1`（x̄=x 的均值）。再把 z_1、y 代入式2.85，**恰好**得到式2.89 的 b̂——**没有直接 OLS，而是用正交化算出了 b̂**。
- **三步 Gram–Schmidt（多元，Hastie et al. 2016）**：
  1. **初始化**：任选一个解释变量作 x_0（含截距时截距列也是解释变量），z_0=x_0。
  2. **依次正交化**：其余变量任意排序 x_1,…,x_K，对 x_k：
     ```
     z_k = x_k − Σ_{j=0}^{k-1} (⟨z_j,x_k⟩/⟨z_j,z_j⟩)·z_j        (式2.92)
     ```
     z_0…z_{k-1} 是已处理好的正交向量。[书中求和下标印作 j=1，但据正文"z_0…z_{k-1}已正交"及标准 Gram–Schmidt，下界应为 j=0 — 符号存疑]
  3. **取系数**：得到 z_K 后，`b̂_K = ⟨z_K,y⟩/⟨z_K,z_K⟩`（式2.93）——它**正是**多元回归中 x_K 的 OLS 系数。
- **★ 易踩的"坑感"与澄清**：式2.93 **只对"最后一个被正交化的变量"成立**；对 k<K，⟨z_k,y⟩/⟨z_k,z_k⟩ **≠** b̂_k。看似"白算一堆 z_k"，但**正交化顺序任意** → 想求哪个变量的系数，就把它排成最后一个再走一遍即可。本质洞见：**b̂_k = x_k 在被其他所有解释变量正交化（剔除其他 x_j 对它的影响）之后，仍能对 y 产生的"增量贡献"**。
- **★★ 系数不稳的根源（式2.94）**：b̂_K 与正交化残差 z_K 的范数挂钩。x_K 与其他变量**高度相关 → z_K 很小**；当 y_i iid 时
  ```
  var(b̂_K) = var(y_i)/⟨z_K,z_K⟩ = var(y_i)/‖z_K‖²
  ```
  ‖z_K‖² 越小 → var(b̂_K) 越大 → b̂_K **极不稳定**（样本微小变动→系数大幅变化）。在 FM 截面回归里 b̂_K 就是因子 K 的收益率 → **为让因子收益率估计稳定，要求因子之间尽量正交**。例：**Barra 中国市场多因子模型把"非线性市值因子"对"市值因子"做了正交化**。
- **一次正交化求全部系数（倒序，式2.95–2.98，Drygas 2011）**：给定顺序正交化得 z_0…z_K 后，最后一个 `b̂_K=⟨z_K,y⟩/⟨z_K,z_K⟩`；把 x_K 解释掉的部分 b̂_K·x_K 从 y 剔除，`y−b̂_K x_K` 作新被解释变量 → x_{K-1} 变成"最后一个"，`b̂_{K-1}=⟨z_{K-1}, y−b̂_K x_K⟩/⟨z_{K-1},z_{K-1}⟩`（式2.95）；以此倒推：
  ```
  b̂_K = ⟨z_K,y⟩/⟨z_K,z_K⟩                                   (式2.96)[分母书中印作⟨z_p,z_p⟩，疑笔误，应为 z_K — 符号存疑]
  b̂_j = ⟨z_j, y − Σ_{i=j+1}^K b̂_i x_i⟩ / ⟨z_j,z_j⟩,  j=K-1,…,0  (式2.97)
  ```
  用二元模型2.88 验证（z_0=1）：`â=⟨1,y−b̂x⟩/⟨1,1⟩ = (Σyᵢ−b̂Σxᵢ)/n = ȳ − b̂·x̄`（式2.98）= 式2.89 的 â。✓

## 关键公式 · 定义 · 判定标准

| 方法 | 切入点 | 统计量/核心量 | 判优标准 |
|---|---|---|---|
| **GRS**（2.5.1） | 联合 α | 式2.68：[(T−N−K)/N]·((√(1+θ̂²_{N+K})/√(1+θ̂²_K))²−1) ~ F_{N,T−N−K} | 加 N 资产后最大夏普比 θ̂_{N+K} 是否**显著>**θ̂_K；显著→拒绝 H0→模型解释不了。要 T>N、依赖正态 |
| **均值—方差张成**（2.5.2） | 联合 α | H0: α=0_N 且 δ=0_N；s1,s2 → LR/W/LM ~ χ²_{2N}（大样本） | N+K 前沿是否**显著优于** K 前沿；无显著差异→接受 H0（模型够用）。不需 R_f，适用更广 |
| **几何解释**（2.5.3） | — | GRS=OB/OA；LR/W/LM 用 OD,OC,AG,AH,BE,BF 六线段 | OB 显著>OA（GRS）；OD/OC、AH/BF 显著>1（LR）等 |
| **α 检验**（2.5.4） | 独立 α | 时序回归取 α̂_i、t=α̂_i/s.e.（**NW 调整**） | **mean(\|α̂_i\|)** 与 **mean(\|t\|)** 越低越好 |
| **贝叶斯**（2.5.5） | 模型后验 | 边际似然 prob(D\|M_i)（式2.81）；贝叶斯因子（式2.82） | 边际似然高者胜（需先验设定可比，否则不成立——Chib 2020） |

- **正交的定义**：⟨x_i,x_j⟩=0；正交 ⇒ X'X 对角 ⇒ b̂_i=⟨x_i,y⟩/⟨x_i,x_i⟩（式2.86），因子系数互不干扰。
- **增量贡献诠释**：多元 OLS 的 b̂_k = x_k 被其余变量正交化后的残差 z_k 对 y 的回归（式2.93），即"剔除共线部分后的净贡献"。
- **稳定性判据（式2.94）**：var(b̂_K)=var(y)/‖z_K‖²；因子越共线 ‖z_K‖² 越小、系数方差越大、估计越不稳。

## A股特殊性 / 实证结论

- **中国版三因子 vs FF3（Liu et al. 2019）**：用 **GRS 检验**，将两模型的因子**互为解释/被解释变量**（即 §2.5 "目标②因子互检"的范式）。结果：**中国版三因子能解释 FF3 因子，FF3 无法解释中国版三因子 → 中国版三因子更适用于 A 股**。本书第4章会对该模型做具体实证（脚注[2]）。
  - 方法论提示：A 股做模型互检时，构造测试资产/因子要先处理 **小市值壳价值、ST/退市、涨跌停与停牌导致的不可交易** 等问题（本节为市场中性的统计方法论，未展开 A 股制度细节，但这些会直接影响 α̂、夏普比 θ̂ 的估计与 GRS/张成的结论可靠性——属下游构造环节须把关）。
- **Barra 中国市场模型的正交化实践**：**非线性市值因子(non-linear size) 对 市值因子(size) 做正交化**，正是 §2.6 "因子高相关→FM 系数不稳→须正交"的工程落地。这对 A 股尤为重要：A 股市值因子极强且与众多风格因子高相关，不正交化会让其他风格因子的收益率估计被市值"带偏"、标准误失真。
- **应用案例（美股，方法论可迁移）**：Han et al.(2016) 用**张成检验**证明其"趋势因子"（综合多时间尺度的动量+反转）无法被传统短期反转/中期动量/长期反转解释——加入后最小方差前沿显著抬升。该"用张成检验论证新因子有增量"的范式可直接搬到 A 股新因子论证。

## 踩坑对照

### ★ 合成稀释尾部（与 §2.6 强相关，重点展开）

- **现象**：把多个相关子信号（如 P/E、P/B、P/S 等多个价值代理）**标准化后简单平均/等权合成**成一个复合因子，复合因子的**尾部（最高/最低分位）暴露被稀释**——单信号上极端、但其他信号上平庸的股票，合成后被"拉回"中庸；而因子溢价**往往集中在尾部**（多空价差由极端股驱动），稀释尾部=损失最值钱的信息。
- **书里为什么（§2.6 给的机理）**：
  1. **冗余维度不增信息**：书中 P/E+P/B 极端例子说明——高度相关的子信号**只代表同一维**（价值），叠在一起名义因子多了、有效维度没增。简单平均把"同一维的噪声"平均掉的同时，也把"该维的真实极端信号"一起平均掉了（相关<1 时，平均的方差 < 各自方差的平均，分布向中心收缩、尾部变薄）。
  2. **共线放大标准误**：式2.94 `var(b̂_K)=var(y)/‖z_K‖²`——子信号越相关，正交残差 ‖z_K‖² 越小，合成/回归出的暴露与系数**越不稳**，尾部端点最先失真。
- **★ 正确做法（综合书中正交化原理）**：
  - **别对高相关子信号无脑等权平均**——先认清它们是不是**同一维**。是同一维 → 当一个因子用，别假装两个；是不同维 → 用 **Gram–Schmidt 正交化** 提取每个因子相对其他因子的**增量残差 z_k**（式2.92–2.93），保留各自的独立（含尾部）信息再组合。
  - 正交化使各因子暴露解耦（式2.86），FM 因子收益率估计更稳（式2.94），尾部不被"互相抵消"。
  - **正交化本身也要小心方向/顺序**：z_k 把"共同方差"全部归给了"先正交化的那些变量"，是**非对称选择**（书中强调顺序任意但结果对"最后一个"才等于 OLS 系数）。若把承载真实信号的因子当成"被正交化掉的对象"，会**误删真信号**——所以"对谁正交、谁留增量"要由经济学含义决定（如 Barra 让非线性市值对市值正交，是因市值是更基础的维度）。
  - 工程上优先**保留极端暴露的构造**（如分位/排序时不过度平滑、合成时考虑对尾部加权或先正交再合成），避免"平均—稀释—丢尾部"。

### √252 / 重叠收益年化

- 本块**间接涉及**：α 检验在估 α̂_i 标准误时**明确采用 Newey–West 调整**（2.5.4）。NW 正是处理**残差自相关/异方差**导致的标准误低估——与"重叠收益(overlapping returns)下不能简单 √T 年化、须 NW 修正"同源。涉及夏普比 θ̂、t 值时同理：自相关会使简单年化的 t 偏大、夸大显著性。（详细年化口径见本书其他章节笔记。）

## 对 scorecard / 回测框架的直接启示

- **模型/因子打分要"两个切入点都上"**：scorecard 里同时算 **联合检验**（GRS 的 F 值/对应 p、张成的 LR/W/LM）与 **独立指标**（mean|α|、mean|t|），并做**因子互检**（新因子 vs 已有模型、已有模型 vs 新因子双向跑），单一指标易误判。判优口径写死：GRS 看"夏普比是否被显著抬高"、α 检验看"mean|α|、mean|t| 越低越好"。
- **新因子"有没有增量"用张成检验背书**：把候选新因子当测试资产、现有因子库当解释变量跑张成检验（不依赖 R_f，A 股友好），最小方差前沿显著抬升才算真增量——比单看 IC/单因子收益更严谨。
- **因子库构建强制正交化体检**：入库前查因子暴露的**截面相关矩阵**；对高相关簇（尤其价值类、市值衍生类）做 Gram–Schmidt 正交化，监控每个因子的 **‖z_k‖²/var(b̂_k)**，对"被解释后残差过小"的因子降权或合并——直接防"合成稀释尾部"和"FM 系数爆方差"。
- **合成模块默认"先正交后合成、护尾部"**：复合因子不要等权平均高相关子信号；正交化保留增量、必要时对尾部样本加权；正交化的"对谁正交"由经济学优先级（基础维度优先保留）决定，并固化到 pipeline 配置。
- **标准误一律 NW**：所有 α/因子收益率 t 检验默认 Newey–West，杜绝自相关导致的假显著（与重叠收益年化坑同根）。
- **GRS 的硬约束记进框架**：需 T>N 且依赖正态——测试资产数量大时要么拉长窗口、要么改用对正态不敏感的方法（张成/自助法），避免盲目套 GRS。
