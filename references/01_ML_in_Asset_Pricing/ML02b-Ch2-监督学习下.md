## 本块讲什么

- **神经网络怎么训练、怎么防过拟合**：最小化平方误差目标(2.14)，数值优化(SGD/拟牛顿)；加 L2 惩罚(2.15)收缩权重——并揭示一个机理：收缩权重会把网络"拉回线性回归"。
- **超参数为何不能用样本内误差选**：样本内 MSE 是预测误差的**向下偏估**（模型把噪声也拟合进去了）；要为"模型复杂度"打折。
- **两条选超参的路线**：①解析路线——用**有效参数个数** d(γ)=帽子矩阵的迹，构造 **AIC**；②数据驱动路线——**交叉验证(CV)**，含 k 折 CV、折数的偏差-方差权衡、以及"验证集选完还要再留测试集"的三段式纪律。
- **贝叶斯解释（全章统一视角，也是全书主线）**：正则化 = 把先验知识写成先验分布。正态先验→岭回归；拉普拉斯先验→近似 lasso（且 lasso = 后验众数 MAP，不是后验均值）；缩放 = 隐含先验；CV ≈ 经验贝叶斯估先验离散度。落点：纯数据驱动不可能（无免费午餐），必须注入领域先验。

## 方法论 / 机理要点（重点·展开）

### 1. 神经网络的拟合与正则化（书 p.21–22）

- **训练目标**：把网络全部参数收进向量 θ，用平方误差损失(2.14) `min_θ Σ_i [y_i − f(x_i,θ)]²`。
- **数值优化**：用随机梯度下降(SGD)或拟牛顿法(quasi-Newton)（指向 Hastie-Tibshirani-Friedman 2009）。**为便于优化，倾向用光滑激活函数（sigmoid）而非不可导的 ReLU**——这是个工程取舍：ReLU 表达力/稀疏激活好，但在 max(0,z) 拐点不可导，给基于梯度的最优化添麻烦。
- **过拟合与 L2 收缩**：节点多、层数深的网络能把训练数据拟合到极好却样本外很差。类比岭回归的惩罚项，给目标加平方 L2 范数惩罚(2.15) `min_θ Σ_i [y_i − f(x_i,θ)]² + γ·θ'θ`。
- **关键机理——收缩=把网络拉向线性模型**：惩罚把权重收向零；在 sigmoid 激活下，**权重很小 ⇒ 激活函数工作在它近似线性的区段**。于是收缩有"双重效果"：不仅输出被收向零，**整个网络也被收向一个线性回归模型**。这把"正则化强度"和"模型非线性程度"直接挂钩——γ 大则更线性、更简单。
- **缩放即隐含收缩差异**：某输入的绝对量级越大，与它相连的权重量级就越小，从而在惩罚项 γθ'θ 里占比越小、受到的收缩越弱。把输入标准化到零均值、单位标准差是常规做法，**但未必最优**；基于对预测问题与数据性质的先验，可以**故意给不同输入不同尺度**，以对 θ 的某些分量加大收缩、对另一些减小收缩。（此点在 §2.4 被贝叶斯化为"缩放=先验"。）

### 2. 为什么不能用样本内误差选超参（书 p.22–23）

- **超参数 = 训练前必须先定死的量**：岭回归的 γ；lasso 要 1 个惩罚参数；弹性网要 2 个；神经网络目标里也可有惩罚参数；随机森林的"每次分裂候选特征子集大小"和"树最大深度"也是超参。
- **目标是预测误差最小**，所以要能**估计**给定超参下的预测误差——这并不直接。**不能直接用训练集的样本内误差**，因为它是"新数据上预测误差"的**向下偏估**。
- 岭回归样本内误差(2.16) `mse(γ) = (1/N)[y − Xĝ(γ)]'[y − Xĝ(γ)]`。机理：训练集里 ĝ 不仅拟合了 y 中真正与 X 相关的变化，**也拟合了残差 ε 里的一部分噪声**；能(过)拟合多少噪声取决于**模型复杂度**。
- 所以选最优 γ **不是**简单地最小化 mse(γ)，而要把"高 γ 带来的复杂度下降、从而样本外更好"的好处计入。

### 3. 有效参数个数 + AIC（解析路线，书 p.23–24）

- 对**线性光滑器**（拟合值可写成 ŷ = Hy 的模型），复杂度 = **帽子矩阵 H 的迹**（=有效参数个数）。岭回归情形(2.17) `d(γ) = tr[ X(X'X + γI_K)⁻¹ X' ]`。
- 极限与直觉：OLS（γ=0）时 d=K（=协变量个数）；γ>0 时 d<K，**收缩降低了有效参数数**；当 X 列正交时 d = K/(1+γ)。脚注用迹的循环性证明 OLS 下 `tr[X(X'X)⁻¹X'] = tr(I_K) = K`。
- **AIC**：在 ε 为不相关高斯噪声假设下(2.18) `AIC(γ) = N·log mse(γ) + 2d(γ)`。第一项随 γ 增大而升（拟合变差），第二项随 γ 增大而降（复杂度下降）；取使两项之和最小的 γ̂。
- **AIC 的短板**：①非线性模型的复杂度/有效参数数**难算**；②即便线性，AIC 也依赖**强假设**——(2.18) 要求 ε **独立同分布且正态**；一旦偏离就得显式写出似然形式。金融数据通常违背 iid 正态，这是后面偏向 CV 的理由。

### 4. 交叉验证 CV（数据驱动路线，书 p.24–26）—— 重点

- **思想**：用训练集估出模型，到一个**独立的验证集**上做预测，验证集误差就是预测误差的估计。假设要求少、纯数据驱动，故 ML 实务更常用。Stone(1977) 证明基于 CV 与基于 AIC 的模型比较**渐近等价**——可把 CV 看作"经验地逼近 AIC"。
- **单次划分**(2.19)：用训练集的 ĝ(γ) 到独立验证集 (X_v, y_v) 上算预测误差，取 `γ̂ = arg min_γ [y_v − X_v ĝ(γ)]'[y_v − X_v ĝ(γ)]`。
- **k 折 CV**（更省数据，2.20）：把数据均分成 k 折；每次留 1 折做验证、其余 k−1 折做估计；轮换到每折都恰好当过一次验证；取 k 个验证折预测误差的**平均**作为期望预测误差估计，再据此选超参：
  `γ̂ = arg min_γ (1/k) Σ_{j=1}^k [ y_{v(j)} − X_{v(j)} ĝ_{−v(j)}(γ) ]'[ y_{v(j)} − X_{v(j)} ĝ_{−v(j)}(γ) ]`，其中 v(j) 是第 j 折索引、−v(j) 指不在第 j 折的其余数据。
- **折数 k 怎么选？文献无定论，是个权衡**：
  - **k 小**：每次训练集小 ⇒ 模型受限（估参数据少）⇒ 对预测误差给出**悲观（偏大）**的估计；但各折训练集**重叠少、彼此差异大**，更像独立重抽，能防"整份数据恰好特殊"的风险。
  - **k 大**：各次训练集**高度重叠** ⇒ 担心这份数据在估计与验证上都"特殊"，估出的预测误差也特殊；这是**偏差-方差权衡**(HTF 2009)：k 大 ⇒ 预测误差估计**近似无偏但方差高**（对"换一份新数据会怎样"高度不确定）。
  - **k 大还很贵**：要重复 k 次训练+验证，大数据上可能不可行。**实务常取 k 远小于样本量 N。**
- **三段式纪律**：用验证折选出 γ̂ 之后，那份验证折平均误差对 γ̂ 而言是**乐观（偏小）**的预测误差估计（因为 γ̂ 正是被挑来最小化它的）。所以应**另留一份测试集**——既不用于训练、也不用于调超参——在其上做最终预测误差评估。即 **训练 / 验证 / 测试** 三分。

### 5. 贝叶斯解释（统一视角，书 p.26–30）—— 全书主线

- **动机**：ML 充满"无客观答案"的选择——惩罚用 L1（产生稀疏）还是 L2（不稀疏）？输入怎么缩放？想"让数据自己说话"（拿 CV 去挑哪种设定验证误差更低）最终会撞上**无免费午餐定理**(Wolpert 1996)：**不存在对所有数据生成分布都更优的算法**。因此必须引入先验假设，**纯数据驱动不可能**——必须带入关于"目标"和"数据性质"的先验知识。
- **贝叶斯框架**：把先验知识用概率（先验分布）表达。设 y = Xg + ε，ε ~ N(0, Σ)，Σ 已知、g 未知；对 g 给先验。
- **正态先验 ⇒ 岭回归**：设 g ~ N(0, Σ_g)。由贝叶斯法则(2.21) `p(g|y) ∝ p(y|g)p(g)`，正态先验×正态似然 ⇒ 后验也正态。后验均值形如 GLS、但逆里多一项(2.22, Lindley-Smith 1972) `ĝ = (X'Σ⁻¹X + Σ_g⁻¹)⁻¹ X'Σ⁻¹y`；多出的 **Σ_g⁻¹** 把估计从 GLS 拉向先验均值（这里是零向量）= 收缩。特化到不相关同方差残差 Σ=σ²I_N、且先验 Σ_g=σ_g²I_K，得(2.23) `ĝ = (X'X + (σ²/σ_g²)I_K)⁻¹ X'y`，**与岭回归(2.7) 完全相同，且 γ = σ²/σ_g²**。解释：σ_g² 大（对系数量级无确切先验）⇒ γ 小、几乎不收缩；先验紧紧集中在均值附近、σ_g² 小 ⇒ 强收缩。
- **"过拟合"被重新定义**：过拟合 = **没有给"关于参数的先验信息"应有的权重**。若先验弥散 σ_g²→∞，本就没有收缩的动机，不收缩也谈不上过拟合。只有当你确有先验理由（如基于经济合理性）相信系数量级不会极端大时，忽视它、不收缩，才叫过拟合。
- **贝叶斯视角 vs 偏差-方差框架**：ML 文献更常用偏差-方差权衡来谈正则化（不正则=低偏差高方差；正则化以偏差换方差下降）。但作者强调：贝叶斯视角下，频率派的"无偏"参照点**并无特殊意义**；给定先验，**后验本身就是把先验信念与数据信息相结合的最优方式**。要把正则化与**经济约束**挂钩（全书核心），用"先验如何塑造正则化"来表述比"偏差-方差"更有用。
- **缩放 = 先验**（岭）：用岭回归隐含假定"对每个 g 分量赋等方差的先验是合理的"。若你认为某些系数先验上就比别的小，岭回归会**收缩失当**——对那些系数本该远离零的协变量收缩过度。应**重新缩放协变量**使"等方差先验"变得可信——**这正是需要领域先验知识介入之处**；现成的"统一标准化"未必对。
- **拉普拉斯先验 ⇒ 近似 lasso，且 lasso = MAP**：先验从正态换成拉普拉斯(Laplace)就近似得到 lasso。拉普拉斯相对正态**在零点质量更集中、尾更厚**（图2.3），故"对大系数收缩更少、对小系数收缩更多"，像 lasso。**但要点**：拉普拉斯先验下的**后验均值并不稀疏**；产生稀疏解的 lasso **不是后验均值、而是后验众数**——即 lasso = g 的**最大后验估计 MAP**(Tibshirani 1996)，lasso 的惩罚参数实际控制先验的离散度。
- **MAP 与 MLE**(2.24) `ĝ = arg max_g p(g|y)`，由贝叶斯法则等价于最大化 `p(y|g)p(g)`（似然×先验）；极大似然 MLE 只最大化 p(y|g)、不乘先验；**平坦先验下 MAP 与 MLE 重合**。提醒：基于 MAP 的预测**不是完整贝叶斯**（完整做法用后验构造预测分布、且要计入对 σ² 的先验不确定性），但 MAP 是把先验信念简单注入的途径，也让我们看清 lasso 是如何隐含地编码对 g 的先验的。
- **缩放 = 先验**（lasso）：同理，用 lasso 即隐含假定"对每个 g 分量等离散度的拉普拉斯先验合适"；不合适就得相应缩放协变量；现成的标准化未必对。
- **CV ≈ 经验贝叶斯**：CV 本质上是**从数据里估先验离散度 σ_g²**。George-Foster(2000) 指出"优化 AIC（或带复杂度惩罚的拟合度量）的估计"与**经验贝叶斯(empirical Bayes)**密切相关——经验贝叶斯给先验参数再加一个无信息的超先验，从数据里挑使后验概率最大的先验参数值。故可把 CV 看作经验贝叶斯的近似。
- **小结（作者原话精神）**：正则化 = 关于模型参数的**先验知识的表达**。即便不走完整贝叶斯，也可用这一解释把领域知识带进学习问题：**数据预处理（缩放）与惩罚设定（用哪种范数）都在隐含地表达先验观点**。下一章就用这把"贝叶斯透镜"去审视资产定价里典型预测问题的性质，据此判断合适的预处理与设定选择。

**公式（编号同原书）**
- (2.14) 神经网络平方误差目标：`min_θ Σ_{i=1}^N [y_i − f(x_i, θ)]²`。
- (2.15) 加 L2 惩罚：`min_θ Σ_{i=1}^N [y_i − f(x_i, θ)]² + γ·θ'θ`（收缩权重 ⇒ sigmoid 下趋近线性模型）。

- (2.18) AIC：`AIC(γ) = N·log mse(γ) + 2d(γ)`（第一项随 γ↑，第二项随 γ↓；取和最小者）。
- (2.19) 单次划分 CV：`γ̂ = arg min_γ [y_v − X_v ĝ(γ)]'[y_v − X_v ĝ(γ)]`。
- (2.20) k 折 CV：`γ̂ = arg min_γ (1/k) Σ_{j=1}^k [y_{v(j)} − X_{v(j)} ĝ_{−v(j)}(γ)]'[y_{v(j)} − X_{v(j)} ĝ_{−v(j)}(γ)]`。
- (2.21) 贝叶斯法则：`p(g|y) ∝ p(y|g)·p(g)`。
- (2.22) 后验均值（GLS+收缩，Lindley-Smith 1972）：`ĝ = (X'Σ⁻¹X + Σ_g⁻¹)⁻¹ X'Σ⁻¹y`。
- (2.23) 特化为岭回归：`ĝ = (X'X + (σ²/σ_g²)I_K)⁻¹ X'y`，即 **γ = σ²/σ_g²**。
- (2.24) MAP 估计：`ĝ = arg max_g p(g|y) = arg max_g p(y|g)p(g)`；平坦先验下 = MLE。

## 与因子研究 & 回测的关联（构造/检验/成本/L2/A股 视情况）

- **三段式划分是回测铁律**：训练/验证/测试。验证集上选出的最优惩罚（或任意超参）对应的误差是**乐观偏估**——把它当样本外表现汇报就是自欺。最终必须在一份**从未参与训练与调参**的测试集（真 OOS）上评估。这直接对应通用回测库的样本外评估环节。
- **时序数据的 CV 要改造（实务补充，超出本书此处）**：本书 §2.3 的 k 折 CV 默认可随机分折；但因子/收益是时间序列且 AIC 的 iid 正态假设本就被金融数据违背（作者也以此为偏向 CV 的理由）。随机分折会造成**未来信息泄漏**，实务应改用**按时间前向滚动 / 分块 CV（purged + embargo）**。本书此节未展开，做因子调参时须自行补上。
- **有效参数数 = 诚实的复杂度账**：一个被重度正则化的高维因子模型，其有效自由度 d(γ) 远小于名义因子数 K。报告模型复杂度/做模型比较时，应按 d(γ) 而非"用了几个因子"来算，避免高估模型容量。
- **贝叶斯透镜 = 把经济先验装进因子模型的通道**（最可操作的一条）：
  - 给每个因子**怎么缩放**，等价于设定其系数量级的先验——别无脑统一标准化，应按"哪些因子系数先验上更接近零"来差异化缩放（岭=等方差正态先验须可信；lasso=等离散度拉普拉斯先验须可信）。
  - 选**岭 vs lasso** = 选对因子载荷用**正态 vs 拉普拉斯先验**；γ = σ²/σ_g² 把惩罚强度直接绑到"你对该因子预期量级的把握"。
- **慎读 lasso 选出的因子集**：lasso 的稀疏来自**后验众数(MAP)**，而完整贝叶斯的**后验均值并不稀疏**。所以"lasso 选中了这 N 个因子"是众数点的产物、对数据扰动不稳健，不应过度解读为"这就是真因子集"（与上半笔记"强相关因子下 lasso 选谁不稳定"互为印证）。
- **CV ≈ 经验贝叶斯估收缩强度**：用 CV 调因子模型惩罚 ≈ 让数据来定收缩力度，可行；但无免费午餐意味着**仍需先验**——不能把全部设定都外包给 CV，缩放与范数选择仍要靠经济判断。
