## 本块讲什么

本块是第3章的核心方法论段，回答一个对实战至关重要的问题：**用什么标准评估一个收益预测模型？样本外 R²（OOS R²）和组合（Sharpe）表现是不是一回事？** 全书在此给出明确的反直觉结论——**不是**。涵盖三节：

- **§3.2 预测表现评估**：在简化设定（截面回归、残差截面不相关、协方差 Σ=I_Nσ²）下，推导 OOS R² 的解析分解，揭示"估计误差"如何系统性拉低 OOS R²、甚至在训练样本短时把它拉成负数；再构造均值-方差有效（MVE）组合，推导其样本外期望收益、方差与平方 Sharpe。
- **§3.3 正则化与投资表现**：用岭回归（ridge）当例子，解析地说明**改善 OOS R² ≠ 改善组合表现**。关键机理：常数比例收缩（X'X=I_K 时）会同等收缩 μ̂ 的所有分量，从而**改变 R² 却不改变（归一化后的）组合权重和 Sharpe**。要让正则化对组合有用，必须让协变量在截面离散度上**不同质**（X'X 非比例于单位阵），收缩才会差异化作用。由此引出"协变量缩放（scaling）不是中性预处理"这一深刻结论，以及如何用先验知识缩放。
- **§3.4 期望收益与协方差的经济联系（开头）**：用经济学论证（无近似套利 → 高均值的多空组合必然高波动）给出缩放/正则化的先验来源；引入"协变量=Σ 的特征向量"这一极简假设，得到主成分（PC）组合，并设定让 Sharpe 集中在高方差 PC 上的先验。

一句话主线：**R² 与组合 Sharpe 会给出冲突信号；正则化与缩放的效果对协变量缩放高度敏感；要让正则化有效，必须注入"哪些协变量更重要"的经济先验。**

## 方法论 / 机理要点（重点·展开）

### 1. 评估的统一设定（简化但可解析）
- 收益取**超额收益**（相对某市场指数）。
- 用 t=1…τ 的**平均收益** r̄ = (1/τ)Σ_t r_t 对 K 个协变量 X 做 OLS：μ̂ = X(X'X)⁻¹X'r̄。
- 真值结构 μ = Xg（真实期望收益由协变量线性张成）。预测可分解为"真信号 + 估计误差"：
  - **(3.5)** μ̂ = μ + u，u = X(X'X)⁻¹X'ε̄，其中 ε̄=(1/τ)Σε_t；E[u]=0，E[uu']=(1/τ)X(X'X)⁻¹X'σ²。
- 关键简化假设（本节通篇）：残差**截面不相关、同方差**，Σ=I_Nσ²；X **不随时间变**（见存疑）。这些假设让一切量都能写成闭式，便于看清"信号 vs 噪声"的拉锯。

### 2. 样本外 R²（OOS R²）的解析分解——估计误差是隐形税
- 验证期 t=τ+1…T 的总收益与残差均值 **(3.6)**：r̄_v=(1/(T-τ))Σr_t，ε̄_v=(1/(T-τ))Σε_t；故 r̄_v=μ+ε̄_v，预测误差 r̄_v−μ̂ = ε̄_v−u。
- **(3.7)** OOS R² = 1 − (ε̄_v−u)'(ε̄_v−u) / (ε̄_v+μ)'(ε̄_v+μ)，用期望值替换二次型（(1/N)u'u 等）后近似为：

  R²_OOS ≈ 1 − [σ²/(T−τ)] / D − [σ²/τ] / D，  其中 D = (1/N)μ'μ + σ²/(T−τ)

- **机理（核心）**：
  - 第二项 σ²/(T−τ) 来自**验证期自身的不可预测噪声**（再大的样本也躲不掉，反映收益本就大部分不可测）。
  - 第三项 **σ²/τ 来自训练期的估计误差**，**随训练样本 τ 增大而消失**。
  - **若 τ 太小，估计误差项可把 OOS R² 拉成负数**——尽管训练集内的 in-sample R² 永远为正。这是"过拟合在样本外暴露"的解析写照：样本内拟合优度毫无保证。
- **转折**：投资者/经济学家未必关心 R²，更关心**利用截面期望收益差异的组合策略的收益**——于是转向组合评估。

### 3. MVE / Sharpe 评估的构造
- 组合权重用与预测成比例、但按"平方和=1"归一化的缩放 **(3.8)**：ω̂ = μ̂ / √(μ̂'μ̂)。
  - 这样权重有可解释的尺度（∑权重²=1）。比按"绝对权重之和"归一更便于计算，代价是失去"恰好半美元多、半美元空"的解释。
- 在 Σ 对角的假设下，**MVE 组合（最大化 Sharpe）的权重正比于期望收益向量**，故 ω̂ 也就是（估计的）MVE 组合权重。
  - （脚注3）一般情形 MVE 权重 ∝ Σ⁻¹μ；按平方和归一为 ω̂ = Σ⁻¹μ / √(μ'Σ⁻²μ)。此组合贯穿全书。
- 该组合样本外（t>τ）单期收益的期望与方差 **(3.9)**：
  - E[ω̂'r̄_v | ω̂] ≈ μ'μ / √(μ'μ + (N/τ)σ²)，  var(ω̂'r̄_v | ω̂) = σ²/(T−τ)。
- 平方 Sharpe **(3.10)**：(E[·])²/var(·) ≈ [(T−τ)/σ²] · (μ'μ)² / (μ'μ + (N/τ)σ²)。
  - Σ 对角时，**平方 Sharpe 近似正比于 (3.9) 的平方期望收益**，故后续只需盯着期望收益看。
- **与 (3.7) 对照**：组合期望收益同样随 τ 增大而上升（估计误差越小越好）。
  - **机理（很重要的直觉）**：训练样本越小、估计误差越大 → μ̂ 相对真信号 μ 噪声越多 → μ̂ 各元素绝对值被噪声"撑大" → 归一化分母 √(μ̂'μ̂) 变大 → 组合期望收益被压低。**直觉**：在"多空总敞口固定"的组合里，权重里的噪声等于把一部分投资容量浪费在"交易噪声"而非"交易信号"上。
  - （脚注4）若不归一、直接用 ω̂=μ̂：期望收益几乎不受估计误差影响，但方差升高，对平方 Sharpe 同样有害。

### 4. 为什么 R² ≠ 组合 Sharpe（本块的灵魂结论）
- 表面看：R² 和组合均值收益**都**被估计误差拖累，似乎 R² 是组合表现的好指标。**但一旦离开 Σ=I_Nσ²，就不成立**——组合 Sharpe 依赖 Σ 的性质，而 R² 对 Σ 的性质完全沉默。
- **更深一层**：即便撇开协方差矩阵问题，R² 与组合表现仍会冲突。实证中（Table 3.2）从 OLS 换到岭回归，对 CV R² 影响很大，对组合均值收益却只有微小改善。由此引出两问：
  1. 正则化如何影响各预测表现度量？
  2. 现成 ML 的正则化（默认目标=最大化交叉验证 R²）在资产定价里合适吗？

### 5. 正则化的解析剖析（§3.3，用 ridge 当显微镜）
- 岭估计 **(3.11)**：ĝ = (X'X + γI_K)⁻¹ X'r̄；γ>0 即开始收缩。
- **情形 A：X'X=I_K（协变量已正交且单位化）**——收缩退化为常数比例：
  - **(3.12)** ĝ = X'r̄/(1+γ)；**(3.13)** μ̂ = XX'r̄/(1+γ) = μ/(1+γ) + u/(1+γ)；**(3.14)** u=XX'ε̄。
  - OOS R² **(3.15)** 在 (3.7) 基础上多出两个收缩项：
    R²_OOS ≈ 1 − [σ²/(T−τ)]/D − [γ²/(1+γ)²]·[(μ'μ)/N]/D − [1/(1+γ)²]·[σ²/τ]/D。
    - 末项（估计误差）被收缩**减小** → 抬高 R²；中间新项是**偏差**（μ̂ 被拉离 μ）→ 收缩越多越**降低** R²。
    - 二者净效果取决于 γ、(1/N)μ'μ（信号方差）、(1/τ)σ²（噪声方差）的相对大小：**噪声相对信号越大，使 R² 最大的 γ 越大**（噪声大时该多收缩）。
  - **但对组合却完全无效**：常数因子 1/(1+γ) 收缩 μ̂，**归一化权重 ω̂=μ̂/√(μ̂'μ̂) 丝毫不变** → 期望收益、方差、平方 Sharpe 全不变！
    - **铁律结论**：**OOS R² 改善，不保证 OOS 组合表现改善。**
    - （脚注5）若用未归一权重 ω̂=μ̂，期望收益按比例下降、波动同比例下降，平方 Sharpe 仍不变。
  - 此结论**与 Σ 是否对角无关**：一般 Σ 下 MVE 权重 ω̂=Σ⁻¹μ̂/√(μ̂'Σ⁻²μ̂)，标量收缩同样被约掉。**故 R² 与组合表现脱节，不是"R² 忽略了预测误差协方差"造成的**，而是更根本的结构问题。
  - 这正解释了 Table 3.2 现象：那里协变量被**标准化**（截面离散度同质）且彼此不高度相关 → X'X ≈ 比例于单位阵 → ridge 近似等比例收缩所有系数 → 帮了 R²，没帮 Sharpe。

- **情形 B：X'X 非比例于单位阵（收缩才可能有用）**——令 **(3.16)** X = Q_K Λ_K^{1/2}，Q_K'Q_K=I_K（正交但不等方差），X'X=Λ_K，对角元 λ_j 即各协变量的**截面离散度**。
  - 岭预测 **(3.17)**：μ̂ = X(Λ_K+γI_K)⁻¹Λ_K^{1/2}Q_K'r̄ = X(I_K+γΛ_K⁻¹)⁻¹ ĝ_OLS。
  - **差异化收缩**：低 λ_j 的预测变量（截面离散度小）→ Λ_K⁻¹ 中对应元素大 → 系数被收缩得**更狠**。经济含义：截面离散度小的协变量，其对期望收益的效应**难以估准、对估计误差贡献大**，因此**下调它是有益的**——只要"省下的估计误差"大于"丢掉的信号"，收缩就能改善组合表现。
  - 此时组合期望收益与方差 **(3.18)**：
    E[ω̂'r̄_v|ω̂] ≈ g'Λ_K(I_K+γΛ_K⁻¹)⁻¹g / √( g'Λ_K(I_K+γΛ_K⁻¹)⁻²g + (σ²/τ)·tr[(I_K+γΛ_K⁻¹)⁻²] )，var = σ²/(T−τ)。
  - 写成逐分量求和 **(3.19)**：
    E ≈ Σ_j [g_j²λ_j/(1+γ/λ_j)] / √( Σ_j[g_j²λ_j/(1+γ/λ_j)²] + (σ²/τ)Σ_j[1/(1+γ/λ_j)²] )。
    - 分母第二项=估计误差，γ>0 收缩它；但 γ 也收缩分子。**孰胜取决于 g 与 Λ_K**：若有些 λ_j 极小，岭回归把这些分量收得最狠 → 估计误差大降；代价（分子里丢的预测力）若 g_j²λ_j（该协变量的"可预测收益方差"）不高，则代价很小、值得。
  - 因方差不受收缩影响，期望收益的改善**直接转化为 Sharpe 改善**。
  - **总规律**：在"截面同方差、不相关、协变量正交"的设定下，**收缩要想改善组合，协变量必须在截面离散度上不同**；更一般地，**只要协变量在"对期望收益 / 估计误差 / 风险的相对贡献"上存在异质性，收缩就可能有用——前提是收缩对"坏贡献（估计误差、风险）"的打压强于对"好贡献（期望收益）"的打压。**
  - **lasso 类比**：除离散度异质外，**真系数 g_j 量级的异质**本身也是 lasso 能影响组合的额外理由——即使协变量离散度都一样，lasso 也会把 OLS 估计接近 0 的系数直接归零，从而产生有益的差异化收缩。

### 6. 协变量缩放（scaling）不是中性预处理——一个被严重低估的坑
- **核心警示**：典型应用里预测变量常被任意缩放（排名 rank、截面标准化到单位标准差，如 Table 3.2）。**标准化使每个预测变量截面离散度相同**；若它们还截面不相关，则 ridge 等比例收缩所有系数 → **对 Sharpe 零作用**。
- 统计软件包默认在估计前标准化变量。**本书强调：这绝非无害**——它**决定了"在收缩什么"**。若协变量二阶矩矩阵对角，标准化等于保证 ridge 在组合表现上**退化为 OLS**。
- **缩放可被任意改写而改变 ridge 结果（却不改变资产真实期望收益）**：把 (3.16) 中系数重定义为 Λ_K^{1/2}g、协变量重定义为 XΛ_K^{-1/2}=Q_K，则 **(3.20)** μ̂ = Q_K(I_K+γI_K)⁻¹Q_K'r̄ = Xĝ_OLS/(1+γ) → 又回到常数收缩、改 γ 对 Sharpe 无效。**同一份数据、不同缩放，ridge 给出本质不同的收缩行为。**

### 7. 如何缩放？——把先验知识接进来（Bayesian 视角）
- 回到 §2.4：**ridge = 系数取正态先验的 Bayesian 回归**（当先验协方差与残差协方差都比例于单位阵时）。标准 ridge **隐含假设：g 的每个分量来自同一分布**（先验等量级）。
- 因此**应当反向缩放协变量，使"各系数量级大致相等"这一隐含先验变得可信**：若先验认为某协变量不重要（更多贡献噪声而非信号），就把它缩放到**更低的截面离散度**，使它仍能拿到与重要协变量"量级相当"的系数（即被收得更狠）。
- 这把"如何缩放"从任意的预处理，升级为**注入经济先验**的建模决策。

### 8. 实证验证（Table 3.2，转述其传达内容）
> Table 3.2 是收益预测回归（含一阶/二阶/三阶过去收益项，见式(3.2)）在不同正则化/缩放/调参目标下的对照表，列含：样本内 R²(IS R²)、交叉验证 R²(CV R²)、组合均值收益、Sharpe、估计的惩罚参数 γ。本块文字给出各行数值：
- **第2行（标准化协变量 + ridge + 以最大化 CV R² 调 γ）**：IS R²=2.63%，CV R²=0.84%，均值收益=4.20%，Sharpe=0.30。
- **第3行"unequal"（先标准化，再把平方项÷2、三阶项÷4，降低非线性项离散度；仍以 CV R² 调 γ）**：γ 明显更低（因协变量已不等离散度，需要更少收缩）；IS R² 微升到 2.69%；**CV R² 升逾三分之一，0.84%→1.18%**；均值收益 4.20%→4.55%；Sharpe 0.30→0.37。说明"非线性项不太重要"这一先验在数据里**确有效力**。
- **第4行（unequal 缩放 + 直接以最大化"组合均值收益"调 γ，而非 R²）**：估得 γ=3.11（远高于第3行），故 IS/CV R² 都明显更低；但组合均值收益 4.58%（vs 4.55%）仅略高；Sharpe 0.35（vs 0.37）反而略低（若想最大化 Sharpe 应直接以 Sharpe 为目标）。
- **结论**：在此设定里，**"直接拿组合表现当调参目标"相比"拿 R²"，收益相当有限**；但这再次证明 **R² 作为组合表现指标可能严重误导**。

- **§3.3 总结**：正则化对 R² 和组合表现的影响，**对协变量缩放高度敏感**。要让正则化有效，必须注入"各协变量相对重要性"的先验。**先验从哪来？→ §3.4：金融经济学给出"协方差 ↔ 期望收益"的联系，用它来定缩放/正则化。**

### 9. 期望收益与协方差的经济联系（§3.4 开头）
- 框架（承 §2.4）：截面模型 r̄=Xg+ε̄，先验 g~N(0,Σ_g)，Σ=var(ε_t)，后验均值 **(3.21)**：ĝ = (X'Σ⁻¹X + (1/τ)Σ_g⁻¹)⁻¹ X'Σ⁻¹r̄。此处放弃 Σ=I_Nσ²，允许**一般 Σ**（仍假设 Σ 已知、不需估计）。
- **经济论证（关键直觉）**：能预测收益的协变量**应当与股票收益协方差相关**。具体：若按某协变量构造的多空组合有可观均值收益，它**也应有可观波动**。
  - 例：若"小市值"预测高收益，则"多小空大"组合应有大波动 → 要求**组内同向协动**（小股彼此协动 > 与大股协动；大股彼此协动 > 与小股协动），使风险无法被分散掉、长短组合保有较高收益方差。
  - **若这种"均值-协方差联系"缺失，就会出现经济上不合理的近似套利机会**（near-arbitrage）。Kozak, Nagel, Santosh (2018) 指出：理性投资者模型与"情绪/行为偏差驱动需求"的模型**都**预测这种联系。
- **极简假设（为得清晰结果）**：令用于预测的 K 个协变量**恰等于 Σ 的 K 个特征向量**。设 Σ=QΛQ'，取 Q 的 K 列为 Q_K，令 X=Q_K，则 X'X=I_K。由此组合权重基于协方差矩阵的特征向量/主成分 → 称**主成分（PC）组合**。
  - PC 组合的期望收益与方差 **(3.22)**：E[Q_K'r_t]=g，var(Q_K'r_t)=Λ_K（Λ_K 对角线=对应 K 个特征值）。
- **经济动机的先验**：各 PC 组合的 Sharpe 向量为 Λ_K^{-1/2}g。有经济合理性的假设是：**高 Sharpe 集中在高方差（高 λ_j）的 PC 上**（下一章详述）。于是设先验 **(3.23)**：g ~ N(0, γ⁻¹Λ_K²)，0<γ<1 → 蕴含 Sharpe 向量 Λ_K^{-1/2}g ~ N(0, γ⁻¹Λ_K)，即允许高 λ_j 的 PC 组合的 Sharpe 取更大量级。
  - **超参 γ 由此获得经济解释**：它控制**先验下的（最大）期望平方 Sharpe**——把纯统计的"惩罚强度"翻译成了"对市场可达 Sharpe 上限的先验信念"。（推导在下一块继续）

**关键定义**：
- **OOS R²（样本外 R²）**：用训练期估的模型在独立验证期度量的拟合优度；与样本内 R² 可符号相反。
- **MVE 组合（mean-variance efficient）**：最大化 Sharpe 的组合；Σ 对角时权重 ∝ 期望收益，一般情形 ∝ Σ⁻¹μ。
- **PC 组合**：以 Σ 的特征向量（主成分）为权重的组合；其收益方差=对应特征值 λ_j。

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **评估口径要双轨**：评一个因子/预测模型，**别只看 OOS R² 或 IC**。本章证明：你可以把 OOS R² 调高却对组合 Sharpe 毫无帮助（甚至相反）。**回测里务必并行报告"统计拟合（R²/IC）"与"组合表现（多空收益、Sharpe、净值）"**，二者背离时以组合表现为准。
- **调参目标的选择**：CV 调参默认最大化 R²（sklearn/glmnet 默认），但**这不是组合最优目标**。可以、且本书示范了**直接以组合均值收益 / Sharpe 为 CV 目标**调惩罚参数——尽管本例增量有限，但这是正确的方法论姿态；当协变量异质性强时增量会更大。
- **标准化是有后果的决策，不是清洗步骤**：很多因子流水线习惯把每个因子截面 z-score（单位方差）。本章指出：**若因子间近似正交，标准化会让 ridge/lasso 的收缩对组合 Sharpe 失效**（等比例收缩被归一化抵消）。要让正则化对组合真正有用，应**按"先验重要性"差异化缩放**（重要因子留大离散度、可疑因子压小），而非一刀切标准化。
- **非线性项的处理**：示例把"过去收益的平方、三阶项"分别 ÷2、÷4 来表达"怀疑非线性"的先验，结果 CV R² 与 Sharpe 双升。**实务启示**：对你不太信的高阶/交互/另类因子，可用更小的截面尺度（或更强的单独惩罚）注入怀疑，而不是直接删除或等权纳入。
- **协方差先验 → 因子合理性筛查**：本块的经济论证（高均值多空 ⇒ 高波动，否则近似套利）给了一把**因子证伪尺**：若某因子的多空组合"高收益却低波动"，要高度警惕（可能是数据窥探、微观结构、或交易不可实现）。这对 A股"高换手、易拥挤"的另类/概念类因子尤其适用——把"是否伴随真实风险暴露"作为留用门槛。
- **PC/特征组合视角**：把因子空间投到收益协方差的主成分上、并相信"高 Sharpe 集中于高方差 PC"，等价于一种**有经济依据的收缩**——抑制低方差 PC（往往是难估、易过拟合的方向）。这与"对小特征值方向重收缩"的岭回归一致，可作为 A股多因子组合的稳健化手段。
- **样本量纪律**：(3.7)/(3.9) 表明 OOS R² 与组合期望收益都随训练期 τ 增大而改善、随 τ 小而恶化（甚至 R²<0）。**短样本 + 多因子（K 大）= 估计误差税重**——回测中要警惕"训练窗口过短/横截面过窄"导致样本外塌方，必要时增窗、降维或加正则。

## 术语与存疑

- **术语对照**：OOS R²（样本外 R²）/ CV R²（交叉验证 R²）/ IS R²（样本内 R²）；MVE = mean-variance efficient（均值-方差有效）组合；shrinkage（收缩）；ridge / lasso（岭/套索回归）；PC portfolio（主成分组合）；cross-sectional dispersion（截面离散度）；eigendecomposition Σ=QΛQ'（特征分解）。
- **符号约定**：r̄=训练期平均收益向量，r̄_v=验证期平均收益；μ=真期望收益(=Xg)，μ̂=预测；u=权重中的估计误差；σ²=残差方差（Σ=I_Nσ²）；τ=训练期长度，T−τ=验证期长度；N=资产数，K=协变量数；γ=惩罚/超参；Λ_K=协方差特征值对角阵（=X'X，在(3.16)设定下）；Q_K=正交协变量矩阵；g=真系数；Σ_g=系数先验协方差。
- **(3.7)/(3.9)/(3.15)/(3.18)–(3.19) 均为"用期望值替换二次型后的近似式"**（书中明言对 (1/N)u'u、(1/N)ε̄_v'ε̄_v 等取期望），非精确等式；定量套用时注意这是大数近似。
- **(3.19) 中分母记号**：原书写作 (1+γλ_j⁻¹)，本笔记按 1+γ/λ_j 表述（等价）；low λ_j ⇒ 该项大 ⇒ 收缩强，逻辑无歧义。
- **X 不随时间变的假设（脚注2）**：仅当预测仅依赖"截面排名"时成立（每期可对排名后的预测向量同样重排）；若期望收益还依赖协变量**水平值**（μ_{t-1}=X_{t-1}g 随时间变），常数 X 无法刻画。本块所有解析结果建立在此简化上，迁移到时变 X 的真实数据需谨慎。
- **Table 3.2 行1（OLS 基准）的具体数值**：本块文字未直接给出（仅以"从 OLS 换 ridge 对 CV R² 影响大、对组合小"定性引用）；表体本身在前一块/前页，本笔记只转录了正文明示的行2–4 数字。[图中存疑：行1精确数值未在 p.55–65 正文出现]
