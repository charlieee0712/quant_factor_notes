## 本块讲什么

第3章的核心命题：**资产定价中的预测问题本质是高维的**(海量可能有用的预测变量)，监督学习理应是利器；但**直接照搬科技界现成的 ML 方法会失败**，因为资产定价应用在五个维度上与典型 ML 应用根本不同(Table 3.1)。本块先系统论证这五点差异——尤其反复强调的**极低信噪比**——然后用一个"只用个股自身过去收益"的极简横截面预测示例(滞后收益的一/二/三次幂 + 岭回归 + 留一年交叉验证)把这些抽象差异落到实处，并由此引出全章要解决的开放问题：如何度量预测表现、如何预处理、如何正则化、如何应对结构性变化。本块末尾(§3.2 开头)抛出一个关键悖论：**个股层面 R² 更高的方法，组合层面 Sharpe 未必更高**(OLS 的 CV R² 为负，组合表现却不输甚至略胜岭回归)。

## 方法论 / 机理要点（重点·展开）

### 1. 为什么资产定价是"高维"问题，以及"武断稀疏"为何错

- 收益预测的潜在预测变量极多：会计科目刻画的公司特征、公司披露文本里抽取的信号、价格与成交量历史、媒体报道信息等，**都可能含预测信息**。
- 传统文献用**武断稀疏 (ad hoc sparsity)** 回避高维：不同时考虑大量预测器，而是**孤立地**逐个考察小集合。典型做法——检验一个新变量相对一小撮"标准"公司特征(Hou-Xue-Zhang 2015、Fama-French 2015 的规模/账面市值比/投资/盈利)是否还有**边际**预测力。
- 作者的批判(机理)：这种孤立小集合的做法，**忽略了文献里已发现的预测器之间存在大量冗余**；更关键的是，**武断稀疏没有任何令人信服的经济学动机**。因此需要一种能直面高维、又不强加武断稀疏的方法——ML 适配此任务，但**不能开箱即用**。

### 2. Table 3.1：典型 ML 与资产定价的五大差异(全章纲领)

这张表是理解全书"为什么不能照搬 ML"的总纲。逐项展开其机理：

**(a) 信噪比：高 vs 极低 —— 本章反复强调的头号障碍。**
- 典型 ML(如图像分类 hot dog/not hot dog)：训练数据里**真实标签已知**，算法直接学已知的正确答案。
- 收益预测：训练只能喂给算法**已实现收益 r_{t+1}**，而真正想要的**期望收益 E_t[r_{t+1}] 不可观测**。已实现收益只是期望收益的**含噪信号**。
- 致命之处：在典型收益数据里，E_t[r_{t+1}] 的横截面/时序变动，**只占已实现收益总方差的极小一部分**——所以信噪比极低。
- **信噪比问题被数据量限制进一步放大**：历史收益数据库只覆盖几十年。一个诱人的错觉是"用高频数据把测量区间切细来增加样本量"，但作者明确指出**这对估 E_t[r_{t+1}] 基本无用**：除非可预测成分本身在高频上变动(除非是想捕捉市场微结构诱发的可预测性，否则不现实)，否则提高测量频率在统计功效上**得不到什么**。这呼应 Merton(1980)：提高测量频率**不会**让期望收益估得更准(方差/协方差可以靠高频估准，但**均值/期望收益不行**)。结论：只能接受收益数据有限这一现实。

**(b) 数据维度：多预测器+多样本 vs 多预测器+少样本。** 资产定价是"宽而短"——预测器多但有效观测(独立时间截面)少，这与(a)的数据限制互为表里。

**(c) 关心的聚合层级：个体结果 vs 组合结果。**
- 资产定价**未必关心个股预测的准**，真正目标常是**构造一个风险性质好的组合**。
- 机理性的开放问题：在个股层面给出最准预测的方法，聚合成组合后是否自动给出表现最好的组合？**答案并不显然**(本章后面用例子证明二者会背离)。

**(d) 预测误差协方差：统计上的麻烦 vs 组合风险的重要决定因素。**
- 数据科学家眼里预测误差协方差只是 nuisance；但在多资产组合里，**收益预测误差的协方差决定了组合波动的大部分**，而组合波动又决定组合的风险-收益性质。
- 推论：误差协方差**在 ML 应用的每一环都重要**——选什么算法、怎么正则化、怎么评估预测表现、怎么用输出构造组合。

**(e) 稀疏性：常稀疏 vs 不明。**
- ML 在很多领域(基因组学、图像分类、文本分析)里稀疏模型表现好(Hastie-Tibshirani-Wainwright 2015)，且**有先验理由**预期稀疏(图像中大片区域与分类无关；候选基因里大量与疾病毫无关系)。
- 资产定价**没有这种先验**：凭什么在看数据之前就断定"一大批资产负债表变量对收益预测**恰好完全无关**(系数恰为零)"？更合理的猜测是——**它们都是某些不可观测的底层因子的、含噪程度不同的信号**，有的更相关有的更不相关，但**先验上恰为零的极少**。这是后文反对 lasso 式硬稀疏、偏好岭回归式收缩的伏笔。

**(f) 结构性变化：无 vs 投资者学习并适应(非平稳)。**
- 这是资产定价独有的**内生非平稳**机理：喂进算法的资产价格，本身是(人或机器的)投资决策的**结果**，而这些决策又被过去数据影响。
- 自我消解的可预测性：设变量 x 在 t 之前能预测收益；t 时刻投资者发现并激进交易，导致该可预测性在随后时期**消失**。一个在 T>t、用横跨 t 前后数据的分析师，若假设存在一个稳定的底层 DGP 供 ML 逼近，**训练出的模型在 T 及以后大概率失效**——因为训练数据里混入了已被"发现并交易掉"而**作废**的观测。
- 对比 ML：图像里的 hot dog 不会因为你分析了它的图片就改变形状或颜色——**没有这种"数据被使用→数据性质随之改变"的内生机制**。作者坦言 ML 文献目前对如何应对资产价格的这类结构性变化**指导甚少**。

> 作者口径："这些区分并非总是泾渭分明，但多数情况下都在不同程度上成立。"全章定位是**勾勒待解决的挑战**，而非给出定论——如何度量预测表现、如何预处理、如何正则化、如何应对结构性变化，"在很大程度上仍是未决问题"。

### 3. §3.1 横截面收益预测示例：模型构造逻辑

- **示例定位(很重要的免责声明)**：故意只用"个股自身过去收益"作预测器。作者明说这**不是**今天能赚钱的策略模板——用如此易得的输入还能赚钱的"低垂果实"**早就被摘光了**。这个极简模型纯粹用来**演示采用 ML 时要面对的各种问题**。
- **目标**：逼近条件期望收益 f(·)，即式(3.1) E[r_{i,t+1}|x_{i,t}]=f(x_{i,t})，把可观测的资产层面特征 x_{i,t} 映射到条件期望收益。
- **横截面口径**：只预测收益的**横截面差异**，故可把个股收益 r_i 理解为**相对市场指数收益**的(市场调整后)收益。
- **预测器构造**：f 是个股自身过去月度收益的**120 期滞后**的线性函数，且同时纳入收益的**一次、二次、三次幂**。二/三次幂项用于容纳过去收益与未来收益之间可能的**非线性**。
- **跳过第1滞后**：为避免被**微结构诱发的偏误**(bid-ask bounce 等)和极短期效应污染，回归从滞后2开始。
- 回归式(3.2)：r_{i,t+1} = Σ_{k=1}^{119} b_k r_{i,t-k} + Σ_{k=1}^{119} c_k r²_{i,t-k} + Σ_{k=1}^{119} d_k r³_{i,t-k} + e_{i,t+1}，共 **3 × 119 = 357 个预测变量**。即便只用过去收益，也轻易得到一个相当高维的问题——正好用来说明"不想强加武断稀疏"时的处境。

### 4. 样本、过滤与预处理(方法论细节，决定结果可信度)

- **样本**：CRSP 全样本，但**剔除**市值低于 NYSE 20 分位、且 t−1 月末价格低于 1 美元的小盘股——确保结果不被极小且不流动的股票驱动。因变量 r_{i,t+1} 用 1970-01 至 2019-06 的收益。
- **逐月去均值 (demean)**：因变量和所有解释变量**逐月**去均值，从而**纯聚焦横截面变动**——这也是回归式(3.2)**不含截距项**的原因。
- **逐月横截面标准化 (standardize to unit s.d.)**：所有预测变量每月横截面标准化到单位标准差。好处之一是**系数量级可直接解读**(见下节)。
- **逐月等权 (equal weight per month)**：加权使回归对样本中**每个月等权**。机理：在非平衡面板里这是自然权重——组合策略的平均收益(每月股票数可能不同)本来也是对每个时期等权，故估计权重应与之一致。
- **正则化**：用上一章的**岭回归(ridge)估计量(2.7)**。
- **惩罚超参的交叉验证：留一年 CV (leave-one-year-out CV)**。设计要点：
  - CV 折 = **连续的一个日历年**的若干月度横截面(contiguous blocks)，而非随机打散。
  - 流程：留出一年作验证、用其余所有年作训练折估计、在留出年算预测收益并记录 R²；轮换留出年重复，直到每一年都被留出过一次；最后**对所有留出年的 R² 取平均**，搜索使该 CV R² 最大的惩罚值。
  - **为什么用"年块"而非随机折**：与"每个日历期等权"的数据加权方案一致(内在机理：时序数据不能随机打散，连续块更贴近真实的时序外推；这是资产定价 CV 设计的关键 know-how)。

### 5. Figure 3.1 解读：一次岭回归同时"复活"多个经典异象(机理亮点)

作者用系数图证明**岭回归确实抽取出有用信息**——一次估计就自动复现了文献里分别记录的多个过去收益相关异象。三个面板(y 轴为系数，约 −0.0002 ~ 0.0003；x 轴为滞后 0~120):

- **Panel (a) 一次项 b_k(主信号)**：
  - **动量 (momentum, Jegadeesh-Titman 1993)**：滞后 ~12 个月内为正系数(近期收益延续)。
  - **近期延续集中在滞后 6–12(Novy-Marx 2012)**：图中这段正系数最突出。
  - **长期反转 (long-term reversal, DeBondt-Thaler 1985)**：滞后 12 个月以上系数**大多为负**。
  - **动量季节性 (momentum seasonality, Heston-Sadka 2008)**：在**12 的整数倍**滞后处出现**显著正峰**，且这一模式**一直延伸到滞后 120**。
  - 价值(机理)：这些效应在不同研究里**分别**记录，而这里**一次回归就全部抓到**——展示了高维、不强加稀疏的 ML 式做法的整合力。
- **Panel (b) 二次项 c_k**：量级明显更小,故二次项(平方收益)的可预测性看似更弱。但作者警示**别只看单系数量级**：平方收益**正自相关**——一只股票更可能持续落在分布尾部(一连串大幅波动)，而非持续为正或持续为负；因此众多小的正系数**累计起来可达不可忽略的效应**。系数**以正为主**意味着：滞后收益(尤其长滞后)**两侧尾部**都多的股票、即**历史高波动**的股票，未来收益**更高**。
- **Panel (c) 三次项 d_k**：系数**倾向为负**,提示"历史上有负向尾部实现"与"未来更高收益"相关；但系数很小。且与二次项不同，**三次幂收益几乎无自相关**，故其累计效应相对 (a) 的一次项**很可能很小**。

### 6. 两个贯穿全章的核心观察(由系数图直接引出)

- **(I) 信噪比极低**：图中系数量级隐含的可预测性虽与文献一致，但**可预测成分只占总收益方差极小一份**。量化对照：与滞后12个月收益一个标准差变动相关的预测收益价差约**年化 2%**(见下节系数加总)，而个股市场调整后收益的年化标准差常在 **40% 以上**——信号相对噪声极小。
- **(II) 非线性很微弱**：二/三次幂只是处理非线性的**粗糙手段**，还有许多未探索的非线性(如协变量之间的**交互项**)；但**强非线性并不会从数据里明显跳出来**。言下之意：若真有非线性，需要**更精细的工具**去找(为后文铺垫)。

### 7. 系数量级的经济解读(标准化带来的便利)

- 因协变量已标准化，系数可直接读：滞后 t−2 到 t−12 的"动量"系数**加总约 0.15%**——即过去12个月每月各正向移动一个标准差，会使 t 月预测收益增加略超 0.15%，**年化接近 2%**。
- 与传统十分位排序法对照：动量常用最高/最低十分位平均收益之差衡量；高减低十分位价差大致对应滞后收益约**3 个标准差**的离散度，平均年化价差常在 **6–8%**(随样本期)。把本文 ~2%(对应1个标准差)放大到 3 个标准差量级，**与传统估计一致**——交叉印证模型抓到的是真信号。

### 8. Table 3.2 + §3.2 开头：R² 与组合表现的悖论(本块的高潮)

- **Table 3.2 五行**(OLS / Ridge-等权-R² / Ridge-非等权-R² / Ridge-非等权-E[r_p] / Lasso-非等权-R²)，所有数字**年化**(月度数 ×12，在可预测成分占比小时是好近似)。本块只聚焦 R² 两列：
  - **OLS**：IS R² = **5.22**，但 CV R² = **−1.18**(为负)。说明 OLS 看似很好的样本内拟合**大部分来自对噪声的过拟合**。
  - **Ridge(等权,R²)**：IS R² = **2.63**(约 OLS 的一半)，CV R² = **+0.84**。
  - 结论:**正则化对获得好的预测表现至关重要**。
- **R² 再次印证低信噪比**：CV R² 约 1% 看似很小，但**经济上可观**——隐含的年度可预测成分标准差约 **4%**;然而可预测成分仍只占总收益方差极小一部分。
- **CV R² 并非完全样本外**：因为超参 γ 是在验证折上**最大化 CV R²**选出来的,真正的测试集(既不参与训练也不参与调参)上的 OOS R² 还会更低些(本章后文再看)。
- **§3.2 抛出方法论质疑**：ML 惯于最小化平方预测误差之和(及其函数如 R²)，并以之为预测表现度量。但在资产定价里**这未必对**：组合管理的终极目标常不是预测个股，而是构造**样本外风险调整后收益高**的组合;研究风险溢价/市场有效性时也更关心**聚合组合**的风险-收益性质。**个股 R² 更高 ≠ 组合 Sharpe 更高**。
- **Table 3.2 第(iv)列直接证伪"R² 越高组合越好"**：用预测收益给个股加权构造组合(见 eq 3.3)，岭回归组合的 CV 年化均值 **4.20%** 仅**微胜** OLS 的 **4.12%**——尽管二者 CV R² 差距很大(0.84 vs −1.18)。换言之，**OLS 的 CV R² 为负，其组合的 CV 均值收益却出奇地高**。更甚者第(v)列:OLS 组合的收益标准差**更低**，故 OLS 组合的 **CV Sharpe 反而更高**(0.35 vs 0.30)。这个反直觉的背离正是 §3.3 要解释的核心(本块未展开)。

## 关键公式 · 图表 · 定义（图表要转述其内容）

- **式(3.1) 条件期望收益**：E[r_{i,t+1} | x_{i,t}] = f(x_{i,t})。目标是逼近把资产特征映射到条件期望收益的函数 f。
- **式(3.2) 多项式滞后收益回归**：
  r_{i,t+1} = Σ_{k=1}^{119} b_k r_{i,t-k} + Σ_{k=1}^{119} c_k r²_{i,t-k} + Σ_{k=1}^{119} d_k r³_{i,t-k} + e_{i,t+1}
  共 357 个预测变量;无截距(因逐月去均值);跳过第1滞后(避微结构)。
- **式(3.3) 组合权重(预测收益加权,美元中性)**：
  ŵ_{t−1} = ( 1 / Σ_{i=1}^N |μ̂_{i,t−1}| ) · μ̂_{t−1}
  μ̂_{t−1} 是用训练折模型得到的预测收益向量;正元素=多头、负元素=空头，且**权重加总为零**(零成本)。分母的绝对值之和把组合规模**固定为半美元空、半美元多**(总杠杆1美元)。验证折组合收益 r_{p,t} = ŵ'_{t−1} r_t，再对时间取均值得 CV 平均组合收益(调参仍暂以最大化 CV R² 为准)。
- **Table 3.1**：典型 ML vs 资产定价的六行差异(信噪比 高/极低、维度 多样本/少样本、关心 个体/组合、误差协方差 麻烦/风险来源、稀疏性 常稀疏/不明、结构变化 无/投资者学习适应)。详见上文方法论第2节。
- **Figure 3.1（三面板系数图，岭回归估计）**：
  - (a) 一次项：滞后≤12正(动量)、6–12最突出(近期延续)、>12转负(长期反转)、12的倍数处正峰且延伸至滞后120(动量季节性)。
  - (b) 二次项：量级小、以正为主→高波动史股票未来收益更高(借平方收益的正自相关累计放大)。

## 与因子研究 & 回测的关联（构造/检验/成本/L2/A股 视情况）

- **极低信噪比是第一性约束**：本块最该内化的一条。做收益预测因子时，已实现收益只是期望收益的含噪信号，可预测成分占总方差极小(年化预测价差~2% vs 个股波动~40%+)。这解释了为什么因子 R²/IC 天然很低、为什么**强正则化(收缩)几乎必需**、为什么"全自动让数据说话"在因子研究里不现实。
- **横截面口径 = 逐月去均值 + 逐月横截面标准化 + 无截距**：这正是构造横截面因子(market-neutral)的标准预处理。本书把"去均值故无截距""标准化故系数可解读"的逻辑讲透，可直接套用到自建因子库的预处理层。
- **CV 折必须用连续时间块(留一年),不能随机打散**：对时序面板，随机 K 折会泄露未来信息、低估样本外误差。回测调参/选超参时应优先**连续年块/滚动**，这与"每期等权"的组合口径一致。A股回测同理(且 A股样本更短，留一年更要珍惜每折)。
- **小盘/低价过滤**：剔除 NYSE 20 分位以下且价<\$1，避免结果被极小不流动股驱动。A股对应做法：剔除 ST、次新、低价、流动性尾部，否则因子收益常是"流动性幻觉"。注意美股用 NYSE 断点而非全样本断点(避免小盘扎堆扭曲分位)，A股可类比用主板/特定池定断点。
- **一次回归整合多异象**：滞后收益的一/二/三次幂 + 岭回归，一次性复现动量、近期延续(6–12)、长期反转、动量季节性(12 倍数)。提示：**动量类因子族高度冗余**，与其堆叠多个孤立动量因子，不如用一个高维收缩模型统一刻画。动量季节性(年度倍数滞后)是 A股可直接尝试的 alpha 来源。
- **二次项=波动记忆、三次项=尾部偏度**:平方收益正自相关→"历史高波动→未来高收益"可由众多小系数累计;三次项→"负尾历史→未来更高收益"。构造特征时，**波动/下行尾部的多期滞后聚合**比单期更有信息(因自相关)。
- **评估指标要对齐目标(关键回测教训)**：**别只用预测 R²/IC 调参选模型**。本块的 OLS-vs-Ridge 悖论证明:个股预测 R² 更高的模型，组合 Sharpe 可能更低。回测应直接以**组合层面指标(Sharpe/净值)**为最终评判,R² 仅作中间诊断。式(3.3)的"预测收益加权、绝对值归一、零成本、固定总杠杆"是一个干净的多空组合权重模板，可直接用于因子组合回测。
- **非平稳/结构性变化 = 因子衰减**：可预测性被发现后会被交易抹平(投资者学习)。这是 A股因子拥挤、alpha 衰减的根本机理。回测必须看**滚动 OOS** 而非全样本 IS，并预期因子随时间趋弱;调参可考虑指数加权近期数据。
- **非线性优先找交互而非单变量高次幂**：单变量二/三次幂贡献甚微;若要挖非线性，应优先**协变量交互**(如动量×市值、动量×波动)，并用更精细的工具(树/网络)而非简单多项式。
