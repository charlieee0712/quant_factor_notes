## 本块讲什么

本块收尾第3章，把前文"用监督学习做截面收益预测"的讨论推向一个核心论点：**现成（off-the-shelf）的 ML 方法不能直接搬进资产定价，必须用经济推理来约束**。具体分五段：

1. **3.4 续**：把贝叶斯回归里的先验参数 γ 解释为"对最大平方夏普比（max squared Sharpe ratio）的先验信念"——这正是收缩（shrinkage）的经济含义；并给出 PC 组合上贝叶斯回归的收缩公式 (3.25)。
2. **3.5 协方差与组合聚合（Return Covariances and Portfolio Aggregation）**：为什么预测误差的协方差矩阵在资产定价里远比一般 ML 重要；把个股聚合成"协变量加权组合"在什么条件下不损失投资机会（公式 3.26–3.27）。
3. **3.6 非线性（Nonlinearity）**：加性高阶项几乎不带来预测力，但**协变量之间的交互（interactions）**可能是更可信的非线性来源。
4. **3.7 稀疏（Sparsity）**：lasso 的稀疏性在资产定价里**没有**带来优势——稀疏先验缺乏经济正当性。
5. **3.8 结构变化（Structural Change / concept drift）**：金融数据生成过程持续漂移，滚动窗/指数加权/递归更新如何应对；以及 CV 在时间方向上的隐患。
6. **3.9 结语**：全章总纲——需要一个能"注入有限经济推理"的分析框架（即下一章的贝叶斯框架）。

> 关于"SDF（随机贴现因子）结构"：本块没有逐字出现 SDF，但**"最大平方夏普比"就是 SDF 结构的对偶量**——见下节机理展开。这是把本块与全书 SDF 主线接上的关键。

## 方法论 / 机理要点（重点·展开）

### 一、收缩 = 对"最大平方夏普比 / SDF 波动率"的经济先验（3.4 续）

**核心机理（与 SDF 的连接）。** 一组资产能达到的最大平方夏普比，等于 Hansen–Jagannathan 边界的平方，也就是（最小方差）随机贴现因子（SDF, stochastic discount factor）波动率与均值之比的平方：max SR² ≈ Var(M)/E[M]²。所以**"对最大平方夏普比设一个不太大的先验"⇔"相信 SDF 不会太波动"⇔"相信市场上不存在好得离谱的投资机会"**。本书此处用"最大平方夏普比"这一可观测对偶量来表述，但其经济内核就是对 SDF 结构的约束。这是后续章节把回归/收缩搭在 SDF 框架上的伏笔。[符号存疑：M 为本笔记按惯例补的 SDF 记号，原文未给]

**γ 的角色。** 在 PC 组合（principal-component portfolios，由 Σ 的前 K 个特征向量 Q_K 张成）这一"透明特例"里，先验下最大平方夏普比的期望为 (1/γ)·tr(Λ_K)（式 3.24，Λ_K 为对应特征值对角阵）。于是：

- **γ 大 ⇒ 先验认为可达夏普比很小**。其经济解读：实证里估出来的夏普比往往因"过拟合噪声"而系统性偏高（upward biased），所以贝叶斯回归会把它们**朝 0 收缩**。
- 给定 γ，若 K 个 PC 组合本身**收益方差高**（tr(Λ_K) 大），先验允许的最大平方夏普比就更高。即"风险大的方向才被允许承载高夏普比"。

**收缩的方向性（关键直觉）。** 在 PC 组合上的贝叶斯回归（式 3.25）等价于一个 ridge 型收缩：
预测收益 ĝ = (I_K + (γ/τ)Λ_K^{-1})^{-1} · (Q'_K r̄)，其中 Q'_K r̄ 是 OLS 估计。
收缩强度由 Λ_K^{-1} 决定——**低方差的 PC 组合（λ 小 ⇒ Λ_K^{-1} 项大）被收缩得最狠**。经济含义：**低方差组合不太可能是高夏普比的来源**，所以先验把它们的预测收益压向 0。这与"高夏普比应来自高风险方向"一脉相承，也正是把统计收缩翻译成经济判断的范例。

**贝叶斯框架顺带解决"协变量缩放任意性"。** 把回归放进贝叶斯框架后，协变量怎么缩放不再是任意的：若把第 j 个协变量除以常数 c，只要把先验里对应系数 g_j 乘以 c（即其先验方差乘 c²），则 μ = Xg 的后验均值**对这种缩放保持不变**。这把第3章前面反复出现的"缩放敏感"问题在原理层面收口——下一章会在更一般框架里用"经济上可信的方式"设定先验。

> 注意：以上 PC 特例假设协变量恰好等于 Σ 的特征向量，并不现实。现实中收益预测变量与协方差矩阵特征向量只有"大致相关"（一个基于协变量构造的多空组合会沾上协方差风险，但没这么紧）。该特例的价值在于**透明地演示如何把经济先验注入收缩参数并给出经济解释**。

### 二、协方差与组合聚合：为什么先聚合成组合（3.5）

**为什么协方差在资产定价里格外重要。** 目标是用预测模型构造"相对风险而言高收益"的组合，所以**收益中不可预测部分的协方差矩阵**直接进入组合构造。而一般 ML 只关心点预测，协方差是次要的——这是资产定价与通用 ML 的一大分野。

**直接估个股协方差的四个困难。**
1. 维数爆炸：上千只个股 ⇒ 协方差矩阵上百万个元素，不加结构约束或收缩几乎估不动。
2. 非平衡面板（unbalanced panel）：个股进出市场，数据难用。
3. 时变：个股的协方差性质随公司特征变化而漂移。
4. （综合后果）估计误差会严重污染组合构造。

**解法：先按预测模型的协变量把个股聚合成组合。** 若用来形成组合的特征**与个股的协方差暴露相关**，则组合层面的协方差比个股层面**稳定得多**；且只要协变量个数 K < 资产数 N，协方差矩阵估计就更可行、更抗误差。这就是"组合聚合（portfolio aggregation）"的动机。

**聚合不损失投资机会的充要条件（公式机理）。** 设收益 r = Xg + ε，可预测部分 μ = Xg；协变量加权组合的实现收益 r_p = X'r，于是组合期望收益 μ_p = X'Xg、组合协方差 Σ_p = X'ΣX。要求"个股可达的最大平方夏普比 = 聚合组合可达的最大平方夏普比"，即 μ'Σ^{-1}μ = μ_p'Σ_p^{-1}μ_p（式 3.26）。**充要条件**（Amemiya 1985, Thm 6.1.1）是协方差矩阵取如下形式（式 3.27）：

  Σ = X Ψ X' + U Φ U' + σ² I_N，其中 U'X = 0。

**这三项的经济解读（重要）：**
- **第一项 XΨX'**：协变量 X 不仅要捕捉期望收益的截面差异，还要捕捉资产对"产生系统性时序波动的因子"的载荷（loadings）——即 X 同时解释收益的横截面差异与系统性风险暴露。
- **第二项 UΦU'**：任何"剩余系统性因子"的载荷 U 必须与 X **正交**（U'X = 0）。
- **第三项 σ²I_N**：除上述两类系统性风险外，剩下的风险必须是**纯特质（idiosyncratic）**。

**为何在大维协变量下近似成立。** 若 Σ 有 L-因子结构 Σ = GΩG' + σ²I_N（G 为 N×L 因子载荷，Ω 非奇异；实证中 L ≤ 20 即可解释绝大部分个股协方差），且 X 含有大量"能信息化地刻画因子载荷"的特征，使得 **G ≈ XB**，则 Σ ≈ X(BΩB')X' + σ²I_N，恰是 (3.27) 的特例。结论：**当协方差的主要来源数目有限、而用于聚合的特征数相对较多且关乎载荷时，聚合带来的投资机会损失很小**。

**双重收益。** 以上还只是总体矩（population moments）层面的"无损"。一旦把"矩必须被估计"也算进来，聚合的真正好处更大：组合数 K < 个股数 N，使协方差矩阵估计**可行且更不易受估计误差污染**。下一章正是沿这条聚合路线落地贝叶斯回归。

### 三、非线性：加性高阶项无用，交互才可信（3.6）

- **经验事实（反直觉）**：在"用过去收益预测未来收益"的简单例子里，加入二阶、三阶项几乎不增加预测力；甚至调整缩放、让 ridge **下调**这些非线性项（Table 3.2 第3→第4行）反而**改善**了样本外表现。即非线性不像在其他 ML 领域那样从资产价格数据里"跳出来"。
- **更可信的非线性来源 = 交互（interactions）而非个体预测变量的加性非线性**：例如收益可预测性在**小盘、流动性差**的股票里更强（规模/流动性 × 信号），这是加性模型抓不到的交互；又如"是否强暴露于投资者情绪/宏观风险"这类哑变量与信号的交互。
- **支持证据**：Chen–Pelger–Zhu (2019) 的深度网络发现非线性主要以协变量间交互形式出现；Gu–Kelly–Xiu (2020a) 同样发现非线性"主要是交互，而非个体变量的加性非线性"；Bryzgalova–Pelger–Zhu (2019) 的"按特征逐步切分股票池"的树型组合能捕捉高阶交互、对刻画风险与收益差异很重要；Moritz–Zimmermann (2016) 发现即便只用"过去收益的各阶滞后"，**不同滞后之间的交互**也带来增量预测力。
- **方法论取向**：下一章的实证将允许公司特征之间的交互。

### 四、稀疏性：在资产定价里没占到便宜（3.7）

- **背景**：稀疏诱导方法（lasso 等）在很多 ML 应用里很成功（少数协变量即可获得稳健预测）。资产定价里也有大量跟进（Chinco et al. 2019；DeMiguel et al. 2019；Feng–Giglio–Xiu 2020；Freyberger–Neuhierl–Weber 2020）。但**稀疏先验在资产定价里是否有正当性，并不显然**。
- **lasso vs ridge 的实证对照（Table 3.2 末行 + Figure 3.2）**：
  - lasso 的最优惩罚值比 ridge **低得多**——因为系数绝对值都远小于 1，绝对值之和（lasso 罚项）远大于平方和（ridge 罚项），要达到同等正则化强度只需更小的惩罚超参。
  - 样本内 R²（IS R²）lasso = 3.55%，高于 ridge（第3行）；但**交叉验证 R²（CV R²）lasso = 0.84% 明显差于 ridge 的 1.18%**。即按"被最大化的预测准则"看，lasso 不如 ridge。
  - 组合表现：lasso 的均值收益略低，但波动率也更低，**夏普比反而略高**——说明"R² 准则"与"组合表现准则"的结论会分叉（diverge）。即便如此，lasso 的稀疏性也没换来实质优势。
- **结论**：不能想当然地认为稀疏性对资产定价的预测有帮助；在**低信噪比 + 无先验理由预期特定加性非线性**的场景，"把高阶项设为恰好 0（lasso）"还是"压到接近 0（ridge）"对预测**几乎没区别**。

### 五、结构变化 / 概念漂移（3.8）——资产定价与通用 ML 最大分野之一

- **为何漂移（concept drift）**：① 经济、技术、监管、制度持续变化，"公司特征↔未来收益"的关系不太可能保持不变；② **投资者会从数据中学习**——历史上被发现的可预测性会诱使他们改变策略，从而**自我毁灭**早先的模式。所以历史可预测性未必在未来重演。
- **应对手段**：给近期数据更高权重的加权方案；滚动窗（rolling window，丢弃某 vintage 之前的数据）；指数加权（exponential weighting，渐进下调旧数据）。但把这些方法系统地引入资产定价 ML、并判别哪种最好，目前研究很少。
- **递归更新机理（公式）**：当回归系数服从随机游走时，稳态 Kalman 滤波（Hamilton 1994）隐含指数加权。把 (3.11) 的 r̄ 换成指数加权平均得 (3.28)；其计算优势在于可写成**递归更新**式 (3.29)：ĝ_t = (1−φ)·ĝ_{t-1} + φ·(X'X+γI_K)^{-1}X'r_t——只用最新一期数据更新上期估计，无需在全样本上重估。递归更新对 ridge 同样适用。
- **正则化下的新难题**：高维 + 需正则化时，不只要追踪**模型参数**的结构变化，还要问**惩罚超参 γ 是否/如何随时间变**。在 (3.29) 里 γ 被固定，但没有充分理由固定；若 γ 时变，需要数据驱动地估计其变化。每期在重叠滚动/扩张窗里重估超参可能算力上太贵；Monti–Anagnostopoulos–Montana (2018) 提出了对惩罚超参的递归更新方案，是潜在可行方向。
- **结构变化对 CV 的警示（重要）**：标准 k 折 CV 默认"验证折相对训练折的时间位置无关"——验证折可以取自比训练数据更老的时段。**平稳时无妨；但存在结构变化时，用早于训练期的数据做验证未必恰当：时间方向有意义**。一个在"向后验证折"上表现好的模型，未必在"向前预测"上表现好。这是资产定价 ML 的重要且研究稀缺的问题，第5章再回来谈。

### 六、第3章结语（3.9）——全章总纲

- ML 在资产定价/量化投资里有用，但**直接套用现成方法、不针对资产定价的特殊条件做适配，难有好结果**。看似琐碎的问题（如**如何缩放预测变量**）会显著影响监督学习算法的表现。
- **选择某种 ML 方法 = 隐含地预设了数据里的模式**（稀疏程度、非线性的种类与强度）能否被检测到。在**低信噪比**下，"让数据自己说话、全自动地解决这些问题而不施加结构"是不现实的。
- **总结论 → 下一章纲领**：需要一个能"注入有限经济推理"的分析框架。下一章将（i）超越"仅用过去收益"的简化例子，改用更广的**公司特征**作候选预测变量；(ii) 把**协方差矩阵在组合表现中的核心作用**纳入；(iii) 允许**公司特征之间交互**形式的非线性；(iv) 搭在**贝叶斯基础**上，作为注入经济推理、设定估计与正则化的通道。
- 监督 ML 在资产定价里还有别的用武之地（本章未展开）：预测**现金流**而非收益、信用风险预测、近似对冲策略搜索。

## 关键公式 · 图表 · 定义（图表已转述）

**公式**

- (3.24) 先验下最大平方夏普比的期望：E[g'Q'_K Σ^{-1} Q_K g] = E[g'Λ_K^{-1} g] = (1/γ)·tr(Λ_K)。γ↑ ⇒ 先验认为可达夏普比越小 ⇒ 收缩越强。
- (3.25) PC 组合上的贝叶斯/ridge 收缩：
  ĝ = (Λ_K^{-1} + (γ/τ)Λ_K^{-2})^{-1} Λ_K^{-1} Q'_K r̄ = (I_K + (γ/τ)Λ_K^{-1})^{-1} Q'_K r̄；
  Q'_K r̄ 为 OLS。低方差 PC（λ 小）收缩最强。[符号：τ 为先验里控制 g 方差量级的尺度参数，承自上一块的式 (3.23)]
- (3.26) 聚合无损的等式条件：g'X'Σ^{-1}Xg = g'X'X(X'ΣX)^{-1}X'Xg。
- (3.27) 充要条件（Amemiya 1985, Thm 6.1.1）：Σ = XΨX' + UΦU' + σ²I_N，且 U'X = 0。三项 = 被 X 捕捉的系统性风险 / 与 X 正交的剩余系统性风险 / 纯特质风险。
- L-因子特例：Σ = GΩG' + σ²I_N；若 G ≈ XB 则 Σ ≈ X(BΩB')X' + σ²I_N（(3.27) 的特例）。
- (3.28) 指数加权 ridge：ĝ_t = (X'X + γI_K)^{-1} X' · (Σ_{s=1}^t r_s (1−φ)^{t-s} φ)，0<φ<1，且 Σ_{s=1}^t (1−φ)^{t-s}φ ≈ 1。
- (3.29) 递归更新：ĝ_t = (1−φ)·ĝ_{t-1} + φ·(X'X + γI_K)^{-1} X'r_t。

**Figure 3.2（lasso vs ridge 系数对比，三面板，x 轴 = 滞后 Lag 0–120，y 轴 = 系数 −0.00075~0.00100；× = ridge，· = lasso）转述：**
- (a) 一阶项（过去收益）：ridge 与 lasso 系数幅度相当、散布相似；lasso 略更分散，**只有极少数被设为 0**——说明一阶项没有强先验理由稀疏化。
- (b) 二阶项（过去收益平方）：ridge 系数普遍贴近 0、紧凑成带；lasso 出现若干尖峰（如滞后≈90 处达 ~0.00075），但**把大量系数设为恰好 0**。
- (c) 三阶项（过去收益立方）：ridge、lasso 都几乎压平在 0 附近；lasso **几乎全部归零**，ridge 保留极小的非零值。
- 整体信息：lasso 在系数不接近 0 时收缩**弱于** ridge；但对二/三阶非线性项，lasso 近乎"整组丢弃"，而 ridge 只是压到接近 0。对预测表现而言，"恰好 0"与"接近 0"差别不大。

**Figure 3.3（滚动向前样本外预测 R²，x 轴 ≈ 1960–2020，y 轴 R² −0.015~0.020）转述：**
- 实线 = 样本内（In-sample）R²，虚线 = 12 个月滑动平均的样本外（out-of-sample）R²，灰带为置信区间。
- 形态：IS R² 在 1960s–70s 最高（约 0.010–0.015），随后**单调衰减**，到 2010–2020 已贴近 0；OOS R² 几乎处处**低于** IS，更嘈杂、时常为负。
- 传达的两个事实：① 即便 ridge 正则化+CV 调参，**CV 的 IS R² 仍系统性高估 OOS R²**（部分因 CV 选惩罚时仍会过拟合，但结构变化也贡献了大半 R² 衰减）；② IS 与 OOS R² 都**随时间朝 0 衰减**，最近 10–15 年平均 OOS R² 接近 0——与"结构变化使旧的可预测关系不再延续"一致。

**实验设置（Figure 3.3 对应）**：CRSP 全样本起自 1926，NYSE 规模门槛 + 最低滞后价格要求（同 Figure 3.1）；ridge 含一/二/三阶项，在 **20 年滚动窗**内估计，惩罚超参用"窗内留一年 CV（leave-one-year-out）"调；因需 20 年估计窗 + 至多 10 年滞后预测变量，首个预测月为 1956-01（CRSP 起点后 30 年）；逐月前移窗口重复。

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **收缩要带方向，不要均匀压。** (3.25) 的核心可迁移：对**低方差/低风险方向**的因子组合，预测收益应收缩更狠（它们不该是高夏普比来源）；对高波动主因子方向给更多空间。落地时可按因子组合协方差/特征值给差异化的 ridge 收缩，而非对所有系数同一 γ。等价地，把"对最大平方夏普比（≈SDF 波动率）的上限信念"显式设为先验，再反推各方向的收缩量。
- **先聚合再估协方差。** (3.27) 给出"按特征聚合不损失投资机会"的判据：**用来形成组合的特征要既解释截面预期收益、又解释因子载荷**。A股实操中个股协方差矩阵又大又时变又是非平衡面板，先把个股按特征（规模、行业、风格、流动性等关乎载荷的变量）聚合成 K(<N) 个组合，再估 K×K 协方差，远比直接估个股协方差稳健——这是构造均值-方差/最大夏普组合的现实路径。
- **非线性优先做交互，而非堆高阶项。** 实证（含本书与 Gu–Kelly–Xiu、Chen–Pelger–Zhu）一致指向：加性二/三阶项基本无用，**交互**（信号 × 规模/流动性/情绪/宏观暴露）才是可信非线性。建模时优先引入交互项或用树/网络去抓交互；对纯加性高阶项保持怀疑并强收缩。这与 A股"小盘、流动性差处效应更强"的常识吻合。
- **别迷信稀疏。** lasso 的变量选择在本书实证里 CV R² 反而更差、夏普仅打平。低信噪比下"恰好 0 vs 接近 0"对预测无实质差异，因子筛选不必强求稀疏；ridge 型整体收缩往往更稳。
- **结构变化是回测设计的第一性约束。** ① 用滚动窗/指数加权/递归更新 (3.28)–(3.29) 让参数跟随漂移；② 警惕 IS（含 CV 的 IS）R² 系统性高估 OOS——评估必须**严格样本外、向前**；③ **k 折随机 CV 在时变市场里有方向性偏误**（用"未来验证过去"不合理），应改用**按时间切分的前向（walk-forward）CV**；④ 投资者学习会自毁信号，历史 alpha 的持续性要打折，回测要纳入"可预测性随时间朝 0 衰减"的预期。
- **超参也会漂移。** 不止系数，惩罚强度 γ 本身可能时变；每期全样本重调算力太贵，可考虑对超参的递归更新（Monti et al. 2018）。
- **L2/A股延伸**：高频/Level-2 因子的协方差结构更不稳、维数更高，"先按特征聚合再估协方差"的思路尤其关键；同时高频环境结构变化更快，前向 CV + 指数加权更新几乎是必需。

## 术语与存疑

- **SDF（stochastic discount factor，随机贴现因子）**：本块未逐字出现，但"最大平方夏普比（max squared Sharpe ratio）"即其对偶量（≈ Hansen–Jagannathan 边界平方 ≈ SDF 波动率平方）。笔记中 M 与该等价关系为按通用资产定价惯例所补，原文此处只用"最大平方夏普比"表述。[此连接为本笔记补充，非原文逐字]
- **PC 组合 / Q_K, Λ_K**：协方差矩阵 Σ 的前 K 个特征向量构成 Q_K，对应特征值对角阵 Λ_K；Q'_K Σ^{-1} Q_K = Λ_K^{-1}。
- **τ（式 3.25）**：先验中控制系数 g 方差量级的尺度参数，承自上一阅读块的先验式 (3.23)，本块未重述其确切定义。[符号存疑：需回看 3.4 前半]
- **concept drift（概念漂移）= 结构变化（structural change）**：ML 文献术语，对应本书"数据生成过程持续变化"。
- **covariate-weighted portfolios（协变量加权组合）**：r_p = X'r，把个股按特征矩阵 X 线性聚合成 K 个组合。
- **(3.27) 中 Ψ, Φ, U**：Ψ、Φ 为相容维度矩阵；U 为满足 U'X=0 的"与 X 正交的剩余系统性载荷"矩阵；σ²I_N 为纯特质风险。Amemiya (1985, Thm 6.1.1) 给出该形式为聚合无损的充要条件。
