## 全书定位（高维预测 / 为何样本内≠真因子 / ML 需经济先验）

全书母题：**资产定价的核心是高维预测，而 ML 不能开箱即用，必须用经济先验约束。**

- **高维预测**：给股票定价要预测现金流、找 alpha 要预测收益，本质都是"给定预测变量求条件期望"的监督学习；但候选变量数 J(公司特征×文本×价量×另类)极易超过观测数 N(~5000 股)。J≥N 时 OLS 无唯一解、J≈N 时过拟合噪声 → 样本内漂亮、样本外崩。
- **传统稀疏之弊**：学界用"人为稀疏(ad hoc sparsity)"回避高维——只用 3–6 个手挑因子(FF3/FF5/HXZ4/BS6)，把上千候选系数强设为零，既无经济依据，也可能正是现有模型实证表现差的根源。
- **样本内 ≠ 真因子**：低信噪比下样本内拟合极易是过拟合假象；Ch5 更证明——高维下即使无风险溢价、无行为偏差，投资者实时学习本身就"印"出样本内可预测性。故样本内显著只是"嫌疑"非"证据"。
- **ML 需经济先验**：no-free-lunch 定理(Wolpert 1996)说没有放之四海皆准的学习器；资产定价数据(极低信噪比、目标是组合非个股、DGP 持续漂移)与图像/医学根本不同，照搬必败。桥梁是 Bayesian——**正则化 = 施加先验 = 注入经济约束的通道。**

## 方法论主线（监督学习 → 资产定价 SL → KNS 横截面 → 投资者信念 → 研究议程）

一条"收缩(shrinkage)"线索贯穿全书：

1. **监督学习工具箱(Ch2)**：OLS→ridge(L2 比例收缩)→lasso(L1 软阈值出稀疏，相关变量下不稳)→elastic net；树/随机森林(bagging 降方差)；神经网络(非线性靠激活函数、交互是命门)。超参用交叉验证(训练/验证/测试三段式)。落点：**正则化 = Bayesian 先验**(正态先验⇒ridge；Laplace 先验⇒lasso=MAP)。
2. **资产定价 SL(Ch3)**：列与典型 ML 的五大差异(信噪比极低、宽而短、组合非个股、预测误差协方差即风险源、稀疏性不明、结构变化)。两条反直觉结论：**缩放(scaling)不是中性预处理而是隐含先验**；**OOS R² 改善 ≠ 组合 Sharpe 改善**(等比收缩被组合归一化抵消)。收缩要有用须差异化(按特征值)。
3. **KNS 横截面(Ch4)**：Kozak-Nagel-Santosh 框架——每个特征做多空组合当因子，联合估单一 SDF(风险价格 b 即 MVE 权重)，以"高夏普来自高方差主成分(先验方差∝Σ²)"差异化收缩。实证(50 anomaly/80 WFR/上千交互)：**特征空间几无冗余、稀疏 SDF 不存在；要稀疏须到 PC 空间；L2 相对收缩(twist)是 OOS 主引擎。**
4. **投资者信念(Ch5)**：视角翻转——投资者本身是高维 Bayesian 学习者。学习的估计误差被印进收益：**样本内可预测(高维渐近下理性预期原假设以概率 1 被拒)、前向样本外为零**；而过度收缩/事后变量又能让样本外可预测复活。
5. **研究议程(Ch6)**：把经济推理注入算法——经济约束(用摩擦特征调收缩)、非线性(交互优先)、结构变化(递归更新+时序 CV)、需求体系与预期形成估计、有限理性与异质性理论。

## 对本项目最有价值的点（OOS 评估 / 收缩=经济先验 / 稀疏 vs 稠密 / 学习诱导伪可预测性 → 真假因子两道关）

- **OOS 评估双轨**：别只看 OOS R²/IC，须并行报告组合 Sharpe/净值，背离以组合为准；CV 选罚参后另留测试集做真 OOS；时序面板用留一年/walk-forward，不可随机分折(泄漏未来)。
- **收缩 = 经济先验**：标准化方式直接决定"收缩什么"，应按"哪些因子先验上更接近零"差异化缩放，勿无脑统一 z-score。先验内核 = 无近似套利 ⇒ 高均值多空必高波动、SDF 方差集中于高方差 PC；用先验最大平方夏普 κ 当收缩旋钮。
- **稀疏 vs 稠密**：特征空间稀疏 SDF 不存在(各因子有边际定价贡献)，"挑几个真因子"徒劳；正确做法是**稠密保留 + 差异化 L2 收缩**，确需稀疏到 PC 空间做。相关因子群(多个价值/活跃度/概念热度度量)宜加权(PC 化收缩)而非 lasso 择一。
- **学习诱导伪可预测性 → 真假因子两道关**：第一关(前向伪样本外)——高维学习会造样本内可预测但前向 OOS=0，仅样本内显著者标"疑似学习诱导，非风险/错误定价"；第二关(反向 OOS + 当年可得性 + 回归外佐证)——前向 OOS 为正仍可能假(过度收缩/有限注意/事后变量同样造 OOS 可预测)。**对本项目另类数据(东财活跃度/人气、同花顺概念热度、AlterData attention)直接警示**：当年投资者无法获取 ⇒ 等价"过度稀疏" ⇒ 伪 OOS 为正也勿直接归因 alpha，须核查历史可得性与前视、优先用反向 OOS 复核。
- **结构变化/因子衰减**：可预测性被发现即被套利衰减(IS→OOS 大幅缩水是高维学习指纹)，回测须滚动窗/指数加权/递归更新，超参也会漂移；A股摩擦(T+1/涨跌停/融券稀缺/停牌)可作差异化收缩抓手，散户/噪声交易者多正契合"老练者用 ML 剥离错误定价"场景。
