## 本块讲什么

第2章上半部，搭建监督学习的统一框架并铺开三大回归族的**目标函数与机理**：

- **§2.1 监督学习 ＝ 函数逼近**：定义 `y=f(x)+ε`、分类 vs 回归、高维(K 接近甚至超过 N)是 ML 方法的共同战场；引出 **no-free-lunch 定理** → 全书主旨"必须注入领域先验"。
- **§2.2.1 线性正则化回归**：OLS → **Ridge → Lasso → Elastic Net**，逐一给目标函数、解、以及正交特征下的"收缩几何"，并解释 lasso 为何产生稀疏、为何怕相关变量、elastic net 为何能补救；最后强调"解不随尺度不变 → 需标准化"。
- **§2.2.2 回归树与随机森林**：阶梯函数逼近、递归二叉切分、贪心算法、剪枝惩罚、bagging+特征随机 → 降方差；与核回归的类比。
- **§2.2.3 神经网络(开篇)**：单/多隐层表达式、ReLU、为何"非线性激活"是命门、深度网络的参数量爆炸、交互效应的捕捉。

贯穿主线＝**偏差-方差权衡(bias–variance tradeoff)**：高维 OLS 过拟合(拟合噪声 ε 而非信号 f)＝高方差；所有正则化方法本质都是"主动引入一点偏差，换取方差大幅下降"。书中这几页用的措辞是 overfitting / regularization / shrinkage，未直呼其名，但机理即此。

## 方法论 / 机理要点（重点·展开）

### 1. 监督学习 ＝ 函数逼近（§2.1）

- 任务：用 K×1 预测向量 `x`(如公司特征集)预测结果 `y`(如股价/收益)。给训练数据 `{y_i, x_i}, i=1..N`，在函数族 `F` 中找未知映射 `f`，满足 `y_i = f(x_i) + ε_i`，其中 `ε` 是**均值为零、x 不可预测的噪声**。
- 关键假设：样本外数据由**同一统计模型**生成，故训练得到的 `f̂` 才能做好样本外预测。这是所有"训练→预测"逻辑的前提，也是资产定价里最脆弱的一环(下一章会讲低信噪比、分布漂移)。
- 两类问题：**Classification**(y 离散，如违约/不违约) vs **Regression**(y 连续)。资产定价以回归为主，本书聚焦回归；但分类对二元事件(corporate default)也有用。
- 方法按函数族 `F` 分两支：① 线性族(计量里无处不在的线性回归变体)；② 高度非线性族(与非参方法如**核回归 kernel regression** 同源)。
- **所有 ML 监督方法的共同点**：为高维设计——`x` 含极多特征，**特征数可超过观测数**。这是 ML 与传统计量最根本的分野。

### 2. No-free-lunch 定理：为什么"全自动通用学习器"不存在（§2.1 核心思想）

- ML 的理想：不对 `f` 做强假设(不预设线性、不预设低维/稀疏)，让数据自己说话；能否造出**完全自动、放之四海皆准**的通用学习器，任何问题(图像/生物/资产定价)都能从数据中复原 `f` 并推广到测试集？
- **Wolpert (1996) 证明：不存在这样的通用算法 = no-free-lunch theorem。** 推论：
  - 没有先验知识，就没有理由偏好任何一个算法胜过另一个；
  - 若多个算法在训练集上拟合得一样好，**仅凭训练集无法判断**谁在未见测试集上预测更好。
- 因此"找到一个好算法"**必然**需要关于该问题的**领域先验**：`f` 属于哪个函数族、`f` 从训练到测试的**稳定性**、`ε` 和 `x` 的统计分布。成功预测要求我们把算法选择**有意识地偏置(bias)**向"契合问题性质"的方向(Wilson & Martinez 1997)。
- → **全书底层主旨**：把资产价格数据的特性当作先验，去筛选/约束海量可选算法与设定。货架上的算法无穷多，但**并非都适配资产定价**。(机理上这正是后文 Bayesian 先验、经济结构约束的出发点。)

### 3. 线性正则化回归：OLS → Ridge → Lasso → Elastic Net（§2.2.1，本块最重）

线性模型 `y_i = x_i'g + ε_i`，`g` 为系数向量。注意：**"线性"只指对 g 线性**，`x_i` 本身可装入非线性变换——尤其是**交互项**：原始特征 `h=(h1,h2)'` 可扩成 `x=(h1, h2, h1·h2)'`，把两特征的交互效应纳入回归。这是本书反复使用的"用形式线性的模型表达非线性"的手法。

**(a) OLS 与高维失效。** 最小化残差平方和得闭式解 `ĝ=(X'X)⁻¹X'y`。问题在高维：
- K 相对 N 不小、甚至 K>N 时，OLS 预测**不可靠**：样本内 R² 极高，样本外 R² 极低甚至为负。
- 机理：协变量太多 → OLS 通过微调 `ĝ` 去**拟合噪声 ε 而非真信号 f**，这些"看似可预测"的虚假规律样本外不重现。
- 极端 K>N：解**不唯一**，有无穷多组 `ĝ` 完美拟合训练集，但完美拟合大半是在虚假地拟合 ε。
- → 这是过拟合(高方差)的根，下面三种惩罚都是解药。

**(b) Ridge（L² 惩罚，Hoerl & Kennard 1970）。** 目标 = 同样的平方损失 + **L²-范数惩罚** `γ·g'g`：
- 两部分：第一项是 loss(OLS 只管它)；第二项是惩罚，超参数 `γ` 控制惩罚力度。
- 解：`ĝ=(X'X + γI_K)⁻¹X'y`——在 `X'X` 对角线上加一条"岭(ridge)"`γI_K`，逆矩阵把系数**整体向 0 收缩**；`γ=0` 退化为 OLS。
- **收缩几何(正交特征 X'X=I_K 时最直观)**：每个系数按**同一比例**缩小，`ĝ_l = ĝ_{l,OLS}/(1+γ)`——**比例收缩，永不恰为 0**(除非 OLS 本身为 0)。这就是 ridge 的几何本质：均匀压缩、不做变量选择。
- 定性：L²-收缩是**正则化(regularization)**防过拟合的范例。

**(c) Lasso（L¹ 惩罚，Tibshirani 1996）。** 把惩罚换成 **L¹-范数** `||g||₁ = Σ|g_j|`：
- 解**对 y 非线性、无闭式**，须数值求解(如 **LARS** 最小角回归算法)。
- 同样向 0 收缩，但**收缩性质不同**：lasso 能给出**稀疏(sparse)解**——只有少数 `ĝ` 非零，其余恰为 0(自动变量选择)。
- **收缩几何(正交特征下)＝软阈值(soft-thresholding)**：`ĝ_l = sgn(ĝ_{l,OLS})·(|ĝ_{l,OLS}| − γ)₊`——把 OLS 估计**向 0 平移固定量 γ**；若平移会让它**变号**，则直接**置 0**。这一"截断到 0"正是稀疏性的来源(对比 ridge 的比例收缩永不为 0)。
  - [标准几何补充·非书中明文] 用约束域看：lasso 约束域是 L¹ 的"菱形"(带尖角)，ridge 是 L² 的"圆球"(光滑)；等高线先碰到菱形的**尖角**(某些坐标=0)，故 lasso 出稀疏；圆球无尖角，故 ridge 不出 0。本书是用上面的软阈值公式 + 下面的相关变量论证来解释稀疏，未画此图。

**(d) Lasso 的软肋：相关变量。** 设两个**强相关**协变量，真实对应同一个系数值：
- 只留其一(系数翻倍)、或两个都留(各取约真值)，对 lasso 目标(2.8)的 loss 和 penalty **几乎都无影响** → lasso 对"留谁/留几个"**近乎无所谓**，最终选择可能取决于数据噪声里**无关的细节**；数据稍动,lasso 就可能从选 A 翻成选 B(不稳定)。
- 但为**最大化预测力**,更好的做法常是把两个相关变量**取平均**纳入,以**平均掉两者的噪声**——而这恰恰是 **ridge 会做的事**。

**(e) Elastic Net（Zou & Hastie 2005）＝ Ridge + Lasso 双惩罚。** 目标 = 平方损失 + `γ₁·Σ|g_j|` + `γ₂·g'g`：
- 兼得两者：像 lasso 一样能把部分系数**恰好置 0**(仍做变量选择),但对变量选择的**依赖更轻**,同时施加 ridge 式收缩 → **平均掉相关变量的噪声**。这是第4章资产定价估计量的雏形(那里再加入预测误差协方差)。

**(f) 尺度敏感性(重要工程坑)。** Ridge/Lasso/Elastic Net 的解**不随协变量尺度不变(not scale-invariant)**：
- 以 ridge、`X'X` 对角为例,加 `γI_K` 对系数的收缩程度,取决于 `X'X` 对角元(即各协变量方差)的大小——**小方差协变量**的对角元小,加同样的 γ 对它的**相对冲击更大**,被收缩得更狠。
- → 实务常**先把输入标准化到单位标准差**再做正则化回归。但"一律标准化"未必最优:**先验**有时告诉我们应当对某些协变量收缩得比另一些**更多/更少**(对接 Bayesian 先验 → 各系数差异化先验方差)。

### 4. 回归树与随机森林（§2.2.2）

- **回归树**用**多维阶梯函数(step function)**逼近非线性 `f`(Breiman et al. 1984)。把特征空间切成若干区域/叶子(leaves),每个叶子是某点 `x_i` 周围的邻域;叶内用 `y_i` 的**等权平均 ȳ_b** 作为 `f` 的估计。
- 一般最优划分难找,故用**递归二叉切分(recursive binary partition)** → 可表示成**决策树**(故名)。
- **全局最优不可行**:即便限定递归二叉,可能的树数目也极大,枚举求全局最小 RSS 计算上不可行。树方法的成功靠**找到"够好"的划分**的算法,而非全局最优。
- **贪心算法(greedy)**:逐步**局部**择优——从全样本起,对每个特征找"使两分区残差平方和下降最多"的**阈值**,选下降最多的特征切一刀;再在两个子区域内**递归重复**;直到每叶观测数达到预设**终止门槛**。
- **与核回归的类比**:都靠"邻域内观测取平均"做预测,差别在**"邻域"的定义与权重方式**不同(树是硬划分等权,核是软加权)。
- **树有多大？过拟合权衡**:大树能更精细逼近高度非线性(低偏差),但易**过拟合训练噪声**(高方差,样本外差)。
- **剪枝(pruning)控复杂度**:先长一棵必然过拟合的大树,再**塌叶剪枝**。准则＝在 RSS 上加**叶数惩罚**:目标 `Σ_h Σ_{x_i∈R_b}(y_i−ȳ_b)² + γ·H`,`H` 为叶数,`γ` 控剪枝力度(`γ=0` 不剪,越大剪越狠)。
- **随机森林(Breiman 2001)＝另一条路:不剪枝 + bagging**:
  1. 把树长到底(只设很小的最小叶尺寸),**不剪枝**;
  2. **bootstrap aggregation(bagging)** 降方差:从训练集有放回抽一个**同样大小**的 bootstrap 样本,在其中**随机选 m<J 个特征**长一棵树;
  3. 重复抽样建多棵树;`x_i` 处的预测＝**各树预测的平均**。`m` 是调参(类比惩罚参数)。
- **为何 bagging 降方差(机理)**:每个 bootstrap 样本噪声不同 → 各棵满树过拟合噪声的方式不同;再加"每样本随机选特征子集"**放大**树间差异 → **跨树取均值把噪声诱发的波动平均掉**。
- **再次类比核回归(Athey, Tibshirani, Wager 2019)**:随机森林对 `x_i` 处的预测,效果类似"离 `x_i` 越近的点被纳入越频繁",与核回归"近邻给更大权重"异曲同工。

### 5. 神经网络（§2.2.3，开篇）

- 本质＝估计**高度非线性回归函数**。结构:J 个协变量 `x_i` 作**输入** → 一个 **H 节点隐层**(视作潜变量) → 一个输出 `y_i`。
- **单隐层表达式**:`f(x_i)=a_2 + w_2'·g(a_1 + W_1 x_i)`。其中 `a_1+W_1 x_i` 是潜变量向量;**激活函数 g 非线性、逐元素作用**于每个潜变量;输出层把激活结果用 `w_2` 加权、`a_2` 平移得最终输出。
- **ReLU**:`g(z)=max(0,z)`,分段线性,正则原样输出、负则归零(当下流行选择)。
- **非线性是命门**:若 `g` 线性(`g(z)=Az`),整个网络会**塌缩成普通线性回归** `y_i=a+g'x_i+ε_i`(其中 `g'=w_2'AW_1`,`a=a_2+w_2'Aa_1`)。所以**激活的非线性才是网络能逼近非线性的唯一来源**。
- **隐层节点数 = 灵活度旋钮**:节点足够多,单隐层即可把任意非线性函数**逼近到任意精度**(Cybenko 1989;Hornik-Stinchcombe-White 1989)。图像/NLP 网络动辄数十万节点。
- **加深(多隐层)**:两隐层 `f=a_3+w_3'·g(a_2+W_2·g(a_1+W_1 x_i))`;**深度网络**常叠 10–20 层＝"线性映射 + 交织的非线性变换"的序列。
- **参数量爆炸**:单隐层(H 节点、1 输出、J 输入)参数数 `=H·J + H + 1`;取 `J=1000, H=10000` → **逾千万参数**;全连接下每加一层同尺寸隐层再添 `H·H + H` 个参数。
- **深 vs 浅**:理论上单隐层已能逼近任意非线性;但实务中加层常更好。是否"越深越好"仍是开放问题(Ba & Caruana 2013);Lin-Tegmark-Rolnick (2017) 认为取决于数据/目标函数性质——**深网尤其适合有层级结构(hierarchical)的函数**。
- **交互效应(资产定价高度相关)**:即多个输入的**联合效应非可加**。书用一个双节点 ReLU 例子(式 2.13)展示网络如何"分段激活"出交互:`f=max(0,−3/2+x_{i,1}+x_{i,2}) + max(0,−3/2−x_{i,1}−x_{i,2})`——第一项仅当两特征之和**足够大**才激活,第二项仅当其和**足够小**才激活;多加隐节点即可拼出更多分段线性片段。(对应图 2.2,图本身在下一块。)

## 关键公式 · 图表 · 定义（图表要转述其内容）

**函数逼近**
- (2.1) `y_i = f(x_i) + ε_i` ——监督学习的总设定,`ε` 均值零且 x 不可预测。

**OLS**
- (2.2) `y_i = x_i'g + ε_i`(对 g 线性,x 可含非线性变换/交互项)
- (2.3) `min_g (y − Xg)'(y − Xg)`(平方损失;`y` 为 N×1,`X` 为 N×K)
- (2.4) `ĝ = (X'X)⁻¹X'y`(OLS 解)
- (2.5) `ŷ = X ĝ`(样本内拟合值)

**Ridge（L²）**
- (2.6) `min_g [ (1/N)(y − Xg)'(y − Xg) + γ·g'g ]`
- (2.7) `ĝ = (X'X + γI_K)⁻¹X'y`,`I_K` 为 K×K 单位阵;`γ=0` 即 OLS
- 正交特征收缩:`ĝ_l = ĝ_{l,OLS}/(1+γ)`(**比例收缩,永不为 0**)

**Lasso（L¹）**
- L¹ 惩罚定义:`||g||₁ = Σ_{j=1}^K |g_j|`
- (2.8) `min_g [ (1/N)(y − Xg)'(y − Xg) + γ·Σ_{j=1}^K |g_j| ]`(对 y 非线性、无闭式,用 LARS 等数值解)
- 正交特征软阈值:`ĝ_l = sgn(ĝ_{l,OLS})·(|ĝ_{l,OLS}| − γ)₊`(**平移+截断到 0,出稀疏**)

**Elastic Net**
- (2.9) `min_g [ (1/N)(y − Xg)'(y − Xg) + γ₁·Σ_{j=1}^K |g_j| + γ₂·g'g ]`(双惩罚:既稀疏又平均相关变量噪声)

**回归树**
- (2.10) 剪枝准则 `Σ_{h=1}^H Σ_{x_i∈R_b}(y_i − ȳ_b)² + γ·H`(RSS + 叶数惩罚;`ȳ_b`＝叶 b 内 y_i 等权均值)

**神经网络**
- (2.11) 单隐层 `f(x_i) = a_2 + w_2'·g(a_1 + W_1 x_i)`
- (2.12) 双隐层 `f(x_i) = a_3 + w_3'·g(a_2 + W_2·g(a_1 + W_1 x_i))`
- (2.13) 交互例 `f(x_i) = max(0, −3/2 + x_{i,1} + x_{i,2}) + max(0, −3/2 − x_{i,1} − x_{i,2})`
- 线性塌缩:`g(z)=Az` ⇒ `y_i=a+g'x_i+ε_i`,`g'=w_2'AW_1`,`a=a_2+w_2'Aa_1`
- 参数量:单隐层 `=H·J+H+1`;每加一同尺寸全连接层 `+H·H+H`

## 与因子研究 & 回测的关联（构造/检验/成本/A股 视情况）

- **过拟合即"拟合 ε 非 f"**:因子研究里 K(候选特征数,含原始特征×非线性×两两交互)极易接近/超过 N(股票数),OLS 样本内 R² 漂亮但样本外崩——所有单因子/多因子回归**必须以样本外为准绳**,警惕 in-sample 显著、out-of-sample 失效。
- **联合建模优于逐个看显著性**:面对海量公司特征因子,用 **elastic net** 类联合估计(既选变量又压缩),比"一个个看单因子 t 值"更稳——尤其因子高度相关时,lasso 会随机二选一且不稳定,elastic net/ridge 取平均更可投资。这直接对接本地 `factor_lib`(已积累大量首席/特征因子)的建模姿态。
- **务必先标准化**:正则化解不随尺度不变,小方差因子会被收缩过度 → 入库前对因子做**横截面标准化**(z-score)再喂正则化模型;但若有经济先验认为某类因子(如低频基本面 vs 高频微结构)该差异化收缩,可设差异化先验方差而非一刀切。
- **交互项是非线性的低成本入口**:形式线性模型只需把 `h1·h2` 当新列即可表达交互;构造因子时把"特征两两乘积/比值"纳入候选,是用线性框架抓非线性的实用手法(第4章会系统用)。
- **树/随机森林＝分块常数的稳健逼近**:适合抓阈值型、非单调的因子效应(如规模分段、流动性分层);随机森林的 bagging+特征随机是天然降方差,适配低信噪比的 A 股截面;但其"邻域平均"本质对**极端样本/停牌缺口**敏感,回测需配合 A 股涨跌停/停牌/T+1 的样本清洗。
- **no-free-lunch → 别迷信"全自动 ML"**:资产定价数据低信噪比、分布漂移,纯数据驱动"好得不真实";必须注入无套利/风险-收益等经济先验来约束算法选择(全书主线,第4–5章落地)。
