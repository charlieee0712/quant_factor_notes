## 本块讲什么

这是定量附录的后半段，集中讲三组"把微观结构现象量化"的统计工具，全部以微结构里的真实问题（市场份额集中度、价差—波动率关系、成交量日内季节性）作为载体：

1. **§A.9 比例/分布的描述与检验**：Gini 系数（衡量集中度/不平等）+ Kolmogorov–Smirnov 检验（非参数地检验"某样本是否来自某分布"或"两样本是否同分布"）。落地案例：2008–2012 年 SBF120 在 Euronext Paris 与 Chi-X 两个交易场所的市场份额分布如何演化、是否趋同。
2. **§A.10 简单线性回归**：用最小二乘把两个金融指标（如 价差 ψ 与 波动率 σ）建立线性依赖；推导 OLS 估计量、其无偏性/方差/分布、方差分解与 R²；再用 Rio Tinto 真实数据演示"先看分布→发现对数正态→做对数回归→回变换时必须做偏差修正"这一完整方法论。
3. **§A.11 时间序列与季节性**：序列三成分分解（趋势 + 季节 + 噪声）、加法/乘法模型、移动平均估趋势、去季节性（deseasonality）的完整流程，最后给出一个**成交量的乘法季节模型**——这正是执行算法（VWAP/日内成交量曲线预测）的理论原型。

主线方法论：**先刻画分布、再做假设检验、再建回归、再分解时间结构**，并贯穿"看数据形状决定模型形式"的实证纪律。

## 方法论 / 机理要点（重点·展开）

### 1. Gini 系数：集中度的一把标尺（§A.9.1 收尾）

- **构造逻辑（Lorenz 曲线法）**：把个体（人/股票/成交量份额）从小到大累积排序，横轴是"累积个体占比"，纵轴是"累积量占比"。若完全平均，曲线就是 45°对角线（完全平等线）；越不平等，曲线越往右下方弯。Gini = 对角线与 Lorenz 曲线之间的面积 A 占对角线以下总三角形 (A+B) 的比例：`G = A/(A+B)`。由于坐标都归一到 [0,1]，A+B=0.5，故 **`G = 2A`**。G∈[0,1]，0=完全平均，1=完全集中。
- **等价定义（相对平均差）**：Gini 等于"相对平均差(relative mean difference)的一半"。相对平均差 = 从总体中随机抽两个个体、其差值绝对值的期望，再除以均值做尺度归一。这个定义把 Gini 解释成"任意两个体的平均相对差距"，比面积法更利于直接用数据估计。
- **机理直觉/用途**：在微结构里 Gini 用来度量**集中度/碎片化程度**——例如某只股票的成交量在多个交易场所之间是集中在一家还是分散；或一篮子股票里成交是否集中在少数几只。**B 越小（Lorenz 越弯）⇒ 越集中 ⇒ Gini 越大**。

### 2. Kolmogorov–Smirnov 检验：非参数地比分布（§A.9.2）

- **核心对象——经验分布函数 (ECDF)**：对 n 个 i.i.d. 观测，`F_n(x) = (1/n)Σ_k 1{X_k ≤ x}`，就是"≤x 的样本占比"构成的阶梯函数。Glivenko–Cantelli 定理保证：若样本真来自 F，则 F_n 几乎必然一致收敛到 F。
- **检验统计量**：`D_n = sup_x |F_n(x) − F(x)|`，即经验 CDF 与参照 CDF 之间**最大的竖直距离**。直觉上 D_n 抓的是"两条累积曲线偏离最厉害的那一点"。
- **为什么强大——分布无关 (distribution-free)**：在 H₀（样本来自连续分布 F）下，`√n·D_n` 依分布收敛到一个**与 F 无关**的 Kolmogorov 分布 K。正因为极限分布不依赖被检分布的具体形状，同一套临界值 `k_{1−α}` 可用于检验任意连续分布——这是 KS 相对卡方拟合优度检验的最大优点。
- **决策规则**：拒绝 H₀ 当 `√n·D_n > k_{1−α}`（k_{1−α} 是 Kolmogorov 分布的 (1−α) 分位）；或等价地看 p 值是否 < α。
- **两样本版本**：不指定共同分布，直接检验两组样本 (X₁..Xₙ) 与 (Y₁..Yₘ) 是否同分布，统计量 `D_{n,m} = sup_x|F_n^X(x) − F_m^Y(x)|`，拒绝当 `√(nm/(n+m))·D_{n,m} > k_{1−α}`。这一版在量化里最实用（见下）。

### 3. 把 Gini + KS 用到市场份额演化（§A.9.3 实操案例）

- **问题**：欧洲股票从 2007 MiFID 后出现交易场所碎片化。作者比较 SBF120 成分股在 **Euronext Paris (ENPA)** 与新兴 **Chi-X (CHIX)** 上的市场份额分布，看 2008→2012 如何演化、两场所是否趋同。
- **方法组合（值得学的范式）**：
  - 用 **Gini** 量化"份额在成分股之间的集中度"——每年每个场所一条 Lorenz 曲线、一个 Gini。
  - 用 **两样本 KS** 检验"两个场所的份额分布在该年是否相同"。
- **结论与机理**：Chi-X 份额逐年逼近 Euronext；KS 检验显示 2008、2009 拒绝 H₀（两分布显著不同），但 **2010、2012 接受 H₀（两分布已无法区分）**。即新进入者用约 4 年把自己的份额分布"磨"到与在位龙头一致——碎片化从早期不对称走向成熟均衡。**方法论要点：一个集中度指标(Gini) + 一个分布检验(KS) 就能刻画一个市场结构变量的纵向演化。**

### 4. 简单线性回归：连接两个微结构指标（§A.10）

- **典型用途**：在微结构里把两个指标做线性挂钩——价差↔波动率、市场份额↔价差、日换手↔价差、电子化程度↔成交单笔规模等。
- **"线性"的精确含义**：模型对**参数**线性。一般默认建模的是 y 给定 x 的**条件期望**为 x 的仿射函数；但也可建模条件**中位数**或任意**分位数**为仿射——这点为分位数回归留了口子。
- **Gauss–Markov 假设（OLS 成立的前提，按重要性记）**：① x 无观测误差；② 均值正确设定 `E[ε_i]=0`；③ 同方差 `E[ε_i²]=σ_ε²`；④ 误差无自相关 `E[ε_iε_j]=0 (i≠j)`；⑤ x 与 ε 不相关 `E[x_iε_i]=0`。
- **OLS 求解逻辑**：最小化残差平方和 `S=Σ(y_i−a_0−a_1x_i)²`，对 a₀、a₁ 求偏导置零，得闭式解：斜率 = 样本协方差/样本方差 `â₁ = Σ(x_i−x̄)(y_i−ȳ) / Σ(x_i−x̄)²`，截距 `â₀ = ȳ − â₁x̄`（回归线必过样本均值点 (x̄,ȳ)）。
- **估计量性质**：OLS 估计量**无偏**（E[âₖ]=aₖ）；其方差随 x 的离散度 `Σ(x_i−x̄)²` 增大而减小（x 铺得越开、斜率估得越准）。σ_ε² 未知，需用**残差**估计，且分母用 **n−2**（扣掉两个被估参数的自由度）才无偏：`σ̂_ε² = (1/(n−2))Σe_i²`。
- **推断与分布**：若误差服从高斯 `ε_i ~ N(0,σ_ε²)`，则 OLS 估计与极大似然一致；标准化后 `(âₖ−aₖ)/σ_{âₖ} ~ N(0,1)`，用估计方差替代后服从**自由度 n−2 的 Student-t**——这就是回归系数 t 检验的来源。
- **拟合优度——方差分解**：`Σ(y_i−ȳ)² = Σ(ŷ_i−ȳ̂)² + Σe_i²`，即 总变异 TV = 可解释变异 EV + 残差变异 RV。`R² = EV/TV = 1 − RV/TV ≤ 1`；简单回归下 **R² = r²**（r 为 x,y 的线性相关系数）。
- **概念澄清（作者特别强调）**："线性模型"是一类模型，"最小二乘"只是一种估计方法（还可用极大似然、贝叶斯）；二者常一起出现但不是一回事。

### 5. 实证范式：价差 vs 波动率，先看分布再选模型（§A.10.2，Rio Tinto）

这一节是全附录方法论密度最高处，给出一条**可复制的实证流程**：

1. **先做朴素线性回归** ψ=a₀+a₁σ+ε（15 分钟区间数据，2012 起）。结果 â₀=2.49、â₁=0.0485、**R²=0.24**——拟合很差，正相关但解释力低。
2. **诊断：查自变量/因变量的分布**。直方图显示 σ 与 ψ **都不是高斯、而像对数正态**（右偏、长右尾）。这一步是关键纪律：**回归效果差时，先怀疑变量分布而非急着加变量**。
3. **对数变换**后再看直方图，log σ 与 log ψ **近似高斯**——满足回归对误差/变量的隐含假设。
4. **做对数—对数回归** logψ = a₀+a₁logσ+ε（â₀=0.5609、â₁=0.2252、R²=0.1553）。回到原变量即得**乘法模型** `ψ = A₀·σ^{a₁}·e^ε`，其中 A₀=e^{a₀}，误差也变成乘性。
5. **回变换偏差修正（最重要的坑）**：直接对 logψ 的预测取指数会**系统性低估** ψ，因为 `E[e^ε]=e^{σ_ε²/2}≠1`。要得到无偏预测须乘上对数正态均值修正项：
   `ψ̂ = Â₀·σ^{â₁}·e^{σ̂_ε²/2}`（σ̂_ε=0.2509）。对应的"原空间"决定系数 R'²=0.1097。
   - **机理**：log 空间的最小二乘只保证 log 预测无偏，不保证原空间无偏；漏掉 e^{σ²/2} 会让所有成本/价差预测偏低——这是成本模型里极常见的实务错误。
6. **R² 的可比性警示**：原空间 R²=0.24、log 空间 R²=0.1553、乘法模型 R'²=0.1097，数值变小**不代表模型变差**——不同变换空间/不同模型的 R² 不可直接横比；要看模型形状是否抓住了数据的形状（乘法模型的凹形幂律曲线明显比直线更贴合点云的弯曲）。

### 6. 时间序列分解与去季节性（§A.11.1）

- **三成分分解**：任一序列 (X_t) 拆成 长期趋势 C_t + 周期性季节 S_t + 不可解释噪声 ε_t。两个经典模型：**加法** `X_t=C_t+S_t+ε_t`（季节振幅与水平无关）、**乘法** `X_t=C_t·S_t+ε_t`（季节振幅随水平按比例放大——金融量价多属此类）。
- **移动平均 (MA) 估趋势**：
  - 奇数长 2k+1 的**中心化 MA** = 前后各 k 个点的简单平均。
  - 偶数长 2k 的中心化 MA = 首尾两端各加权 0.5、中间权 1，再除以 2k（这样前后取点数对称，故称"中心化"）。
  - **两条性质**：① MA **平滑噪声同时保留线性趋势**（若 C_t=bt+a，MA 仍近似该直线，MA 越长越贴合）；② **边缘效应**——序列前 k 个与后 k 个点算不出 MA（缺 X_{t−k} 或 X_{t+k}）。
- **去季节性的核心技巧**：**MA 窗口长度必须等于季节周期 p（或其整数倍）**，这样一个完整周期内的季节波动在平均中相互抵消，MA 就只剩趋势——于是 MA 成了趋势 C 的估计。
- **季节因子估计流程（记号：t=(i−1)p+j，i 数周期、j 数周期内位置）**：
  - **加法模型**：`X_{ij}−MA_{ij} ≈ S_j`（差值约等于季节项）。对每个 j 在各周期上取均值（或中位数）得初估 `Ŝ_j=(1/n)Σ_i(X_{ij}−MA_{ij})`；再**中心化**使季节因子之和为 0：`S_j=Ŝ_j−(1/p)Σ_k Ŝ_k`。
  - **乘法模型**：改用**比值** `X_{ij}/MA_{ij} ≈ S_j`；初估 `Ŝ_j=(1/n)Σ_i X_{ij}/MA_{ij}`；再**归一**使季节因子平均为 1（总和为 p）：`S_j=Ŝ_j/[(1/p)Σ_k Ŝ_k]`。
  - **识别约束的意义**：加法因子和=0、乘法因子均值=1，是把"趋势 vs 季节"的尺度不确定性钉死的**可识别性约束**——否则可把常数在 C 和 S 之间任意搬移。

### 7. 成交量的乘法季节模型（§A.11.2）——执行算法的原型

- **建模对象**：年 y、月 m、日 d、日内时段 n 的成交量，写成**多因子乘法分解**：
  `V(y,m,d,n,ω) = (f(n) + g(d,n))·F(m)·G(y)·v(n,ω)`
  - `f(n)`：时段均值效应（即著名的日内 **U 型成交量曲线**）；
  - `g(d,n)`：特定"日 × 时段"效应，专门捕捉**特定日子的异常**（宏观数据公布日、期权/期货到期日等）；
  - `F(m)`：月度均值效应；`G(y)`：年度均值效应（长期趋势/电子化扩张）；
  - `v(n,ω)`：刻画"当天整体活跃度高低"的随机水平变量。
- **聚合到日总量**：对时段 n 求和，月/年因子可提到外面：
  `V^D = F(m)G(y)·(Σ_n f(n)v(n,ω) + Σ_n g(d,n)v(n,ω)) = F(m)G(y)·(f̃(ω)+g̃(d,ω))`。
  即"日内形状项"与"特定日项"被分别聚合成两个随机量 f̃、g̃，乘上月/年因子。
- **机理价值**：这套**乘法可分离**结构把"日内形状、特殊日、月季节、年趋势、当日水平"五种异质效应解耦，每一块都能单独估计/预测——正是 VWAP 等执行算法预测日内成交量曲线所用的模型骨架。

## 关键公式 · 图表 · 定义（图表逐张转述）

**公式速查**
- Gini：`G=A/(A+B)=2A`；= 相对平均差 / 2。
- ECDF：`F_n(x)=(1/n)Σ1{X_k≤x}`；KS 统计量：`D_n=sup|F_n−F|`，两样本 `D_{n,m}=sup|F_n^X−F_m^Y|`；极限 `√n·D_n →(law) K`（K 不依赖 F）。
- OLS：`â₁=Σ(x−x̄)(y−ȳ)/Σ(x−x̄)²`，`â₀=ȳ−â₁x̄`；`σ̂_ε²=(1/(n−2))Σe_i²`；`(âₖ−aₖ)/σ̂_{âₖ}~T_{n−2}`。
- 方差分解：`TV=EV+RV`，`R²=EV/TV=1−RV/TV=r²`（简单回归）。
- 对数正态回归无偏预测：`ψ̂=Â₀σ^{â₁}e^{σ̂_ε²/2}`。
- 季节去除：加法 `S_j=Ŝ_j−mean(Ŝ)`（和=0）；乘法 `S_j=Ŝ_j/mean(Ŝ)`（均值=1，和=p）。

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **Gini → 集中度/拥挤度类因子**：可直接构造"成交量集中度""持仓/资金集中度""跨场所(或跨券商)份额集中度"等因子；A股可做"行业内成交集中度""龙头股成交占比"。Gini 也适合衡量**因子暴露的拥挤程度**（暴露是否集中在少数票上）。
- **KS 检验 → 因子/分布稳健性工具（量化里用途极广）**：
  - **样本内/样本外漂移检测**：两样本 KS 检验 IS vs OOS 的因子值分布是否一致，及早发现因子失真。
  - **截面分布稳定性**：逐期对因子做单样本 KS 拟合优度（如检验是否近正态），决定是否需要标准化/分位变换。
  - **状态/regime 识别**：用两样本 KS 比较不同市场状态下收益或因子的分布差异。
  - 非参数、不依赖分布形状，适合金融数据的厚尾/偏态场景。
- **线性回归 → 因子检验与成本建模的主力**：截面因子收益回归、Fama-MacBeth、价差/冲击成本对波动率/换手的回归都用它；注意 R²=r² 仅限单变量，残差方差用 n−2。
- **对数变换 + 回变换偏差修正（成本模型必读）**：价差、买卖冲击、成交额等微结构量普遍**对数正态**；建模时应在 log 空间回归，但**回到原空间预测必须乘 e^{σ²/2}**，否则系统性低估成本——这是搭建交易成本模型(TCA)/冲击模型时最易踩的坑。
- **R² 跨空间不可比**：评估对数模型 vs 线性模型时不要直接比 R²，要看原空间预测的无偏性与形状贴合度。
- **时间序列季节性 → 日内 U 型与去季节化**：A股价量有强日内季节性（开盘/收盘集合竞价、午休断点、尾盘放量），在算日内因子(如日内动量、流动性)前应**先去季节性**（MA 窗=日内 bar 数；乘法模型用比值法、令季节因子均值=1），否则季节形状会污染因子信号。
- **成交量乘法模型 → 执行/VWAP 直接可用**：§A.11.2 的 `V=(f(n)+g(d,n))F(m)G(y)v(n,ω)` 就是 VWAP 算法预测日内成交量曲线的标准骨架；g(d,n) 项专门留给**到期日/数据公布日**等特殊日的成交异常——在A股可映射到调仓日、指数成分调整日、月末/季末等。L2 逐笔/分钟数据正适合估这套时段因子 f(n) 与特殊日因子 g(d,n)。

## 术语与存疑

- **术语对照**：Gini coefficient 基尼系数；Lorenz curve 洛伦兹曲线；relative mean difference 相对平均差；empirical distribution function (ECDF) 经验分布函数；Kolmogorov–Smirnov test KS 检验；Glivenko–Cantelli theorem 格利文科–坎泰利定理；goodness-of-fit 拟合优度；homoscedastic 同方差；ordinary least squares (OLS) 普通最小二乘；coefficient of determination R² 决定系数；total/explained/residual variability (TV/EV/RV) 总/可解释/残差变异；centered moving average 中心化移动平均；deseasonality 去季节性；additive/multiplicative model 加法/乘法模型；time slot 时段。
- **人名/参考**：Kolmogorov (1933) 原始论文；线性回归参考 Tassi (2004)、Bourbonnais (2011)；时间序列参考 Brockwell & Davis (1991)、Hamilton (1994)。
- **案例市场**：ENPA = Euronext Paris；CHIX = Chi-X；标的指数 SBF120（后文表注用 SBF 120）。
- **存疑**：① 书内 p.300 一句关于序列末端 MA 可算范围的表述（"one can calculate MA_T,…,MA_{T−k+1} since we must know X_{t+k}"）字面读来与"边缘 k 个点算不出 MA"的本意相反，疑为原文笔误，应理解为**末端 k 个点同样无法计算**——已按此本意转述 [原文措辞存疑]。② 图 A.13/A.18 中部分点云密度为视觉估读，具体坐标以原图为准 [图中数值为估读]。
