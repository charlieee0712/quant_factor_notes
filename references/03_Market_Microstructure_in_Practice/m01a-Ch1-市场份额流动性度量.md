## 本块讲什么

MiFID 之后欧洲股票从单一主板裂变为多场所（multi-venue）交易，本节先问一个看似简单的问题：**怎么度量一只股票/一个市场的"碎片化"程度？** 答案是"没有干净答案"——市场份额（market share）是最常用却**远非完美**的代理：它的数值随你怎么定义（按金额/按股数/按笔）、统计什么时间窗、是否含集合竞价、是否含 OTC、跨股票如何加权而剧烈变化。§1.1.1 把这套"度量的自由度"逐条拆开，给出三个核心公式，并立下贯穿全书的方法论铁律：**度量必须服从它要回答的问题**。§1.1.2 转入史实，用逐国异步开放（表 1.1）和主板份额的持续下滑（图 1.3）刻画碎片化"第一阶段"的机理，并解剖 MiFID 用极少数简单机制（最佳执行、场所三身份、盘前/盘后透明度）如何催生出极高的市场复杂度，最后以 Turquoise 的失败收尾。

## 方法论 / 机理要点（重点·展开）

### 1. 碎片化是"多尺度"现象 → 注定没有单一度量
作者开宗明义：碎片化同时发生在多个层级，每个层级各有其"合适的度量"（"one adequate metric for each scale"），因此**任何单一指标都只能照亮一个尺度**。五个层级（机理）：
- **市场运营商（market operators）**：竞争初期冒出大量新运营商（美国 Archipelago、Direct Edge、BATS；欧洲 Turquoise、Chi-X），随后为分摊运营固定成本而**合并**。
- **交易场所（trading venues）**：同一运营商可经营多个场所（如 NYSE Euronext 同时运营 Lit 订单簿 + Smartpool + NYSE Arca）。竞争开放后还出现 BCN（Broker Crossing Networks，经纪商撮合网络）这类场所。
- **订单簿（orderbooks）**：同一场所可提供多个订单簿（Chi-X 同时有 Chi-X Lit 与 Chi-Delta 暗池）。
- **订单本身（orders）**：碎片化加深后，一张大单不仅要**沿时间**拆分（降低市场冲击，见第3章§3.4），还要**沿空间**拆分，把单子铺到各个合规可达的订单簿上"追逐流动性"。
- **技术（technology）**：竞争使对接协议数量上升——行情可用公共协议（欧洲 ITCH），但各运营商争相提供最快的私有下单协议（OUCH、UTP、Millennium）。

> 机理意义：市场份额"最常用"，**仅因为它直接挂钩场所收入**（成交额经手续费转化为营收，见下"度量该回答什么问题"），并不是因为它最准。

### 2. 市场份额的"定义自由度"——每一个选择都改变数字
这是"为何不完美"的核心论证。**同一片市场，换一种口径，份额就大不一样**：

**(a) 按金额 vs 按股数 vs 按笔。** 三种定义（公式 1.1、1.2，见下）。按金额份额与按股数份额"在价格大致恒定时近似相等"，但**在价格大幅波动的长周期上可能差很远**——因为份额是对 P·V 的加权，价格漂移会重标历史成交的权重。

**(b) 时间窗。** 份额依赖统计区间：**开盘第一小时与收盘最后一小时的份额通常不同**（日内节奏，见第2章§2.1.2）。

**(c) 集合竞价 vs 连续竞价（关键机理）。** 集合竞价（fixing auction）**通常被某一个场所垄断**（一般是历史主板）。因此"含集合竞价"会系统性抬高主板份额。书给出**剔除集合竞价的修正公式**（见下"关键公式"）：若场所在集合竞价中的平均份额为 m(n)、集合竞价占全天成交比重为 p，则可由"全口径份额"反推"连续竞价份额"。图 1.1(a)→(b)、图 1.2(a)→(b) 都是这条机理的实证。

**(d) 统计范围：可见/Lit vs 全口径（含 OTC/暗池）。** 把 OTC（如 Markit BOAT 报告的场外成交）纳入，会**大幅稀释主板的表观份额**（图 1.1：Deutsche Boerse 从 (b) 的 67% 掉到 (c) 全口径的 31%）。

**(e) 跨股票加权方案。** 多股汇总时三选一：①按成交额加权（最贴合上面金额份额的定义）；②按指数权重 w_k 加权（但指数权重常正比于自由流通/市值，**不等于**日成交额）；③等权。三者结果不同。

> 隐含机理：**份额不是一个客观事实，而是一组建模选择的函数**。报告同一市场时，口径不透明就能"做"出想要的份额。

### 3. 铁律：度量必须匹配"它要回答的问题"
全书反复出现的方法论核心。书把市场份额的"用途"对应到三个不同问题、三种不同的正确口径：
- **想看某场所的定位与营收** → 用"全口径（含集合竞价）金额份额"；但要记住集合竞价的手续费率与连续竞价不同。
- **想估"当一笔成交发生在股票 k 上时，它落在场所 𝒯(n) 的概率"** → 不要把集合竞价和连续竞价相加，而要**分别**跟踪两个指标；此时自然的度量是**按笔份额** 𝔐_T^k(n)（公式 1.2）。
- **想估"当一单位货币（欧元/美元）在 k 上成交时，它落在 𝒯(n) 的概率"** → 用经典的**按金额份额** 𝔐^k(n)（公式 1.1），并把集合竞价与日内分开各测一次（集合竞价时某些场所可能 100% 垄断）。

### 4. 份额本质是"概率估计"→ 须用统计估计的严谨性
上面三种用途里有两种是在"estimating the probability that…"。书因此明确：份额是对"市场参与者平均在哪交易"的**粗略估计**（"a rough estimate of where the market participants are trading, on average"），既然是估计，就该借用文献里**比例（proportion）的点估计与置信区间**方法（书设专栏指向 Appendix A.8）。机理提醒：小样本/短窗的份额数字有估计噪声，不能当精确值用。

### 5. 碎片化随流动性上升（异质性）
经验规律：**越流动的股票越碎片化**（"fragmentation increases with liquidity"）。推论：碎片化在指数成分间**不均匀**，所以需要一个专门刻画"碎片化在指数内是否齐整"的指标——coverage（第2章§2.4）。图 1.2 也佐证：不同测法给出不同结果，说明碎片化因股而异。

### 6. Fragmentation Metrics 专栏（方法论收口）
书用方框总结："碎片化难以度量、可被歧义地定义；**碎片化指标应由其运营用途来引导**"。举例用途：能指示场所营收；能估某笔成交落在某场所的概率；能监测市场份额的**（熵式）离散度**（entropic dispersion，即把份额向量当分布、用熵衡量其分散程度——本节只点题，公式在 §1.1.3）。

### 7. 第一阶段碎片化的机理（§1.1.2）
- **触发与观测设计**：MiFID 放开新运营商后，聚焦 Chi-X、Turquoise、BATS Europe、NASDAQ OMX Europe 四家即可看清主要机制。关键的**研究设计红利**：各场所**逐国异步开放**（表 1.1），历史交易所成员被迫按渐进日历开通——这种"异步碎片化"天然构成**自然实验/事件研究**素材（如第2章§2.5.1 对英国与西班牙市场的比较）。
- **现象**：主板份额自竞争开始**持续单调下滑**（图 1.3）。
- **MiFID 的目标（事后归纳）**：①引入竞争以打破历史交易所的垄断；②促成泛欧微观结构。四年后大体兑现：Chi-X 与 BATS Europe 于 2012 合并成大型泛欧平台；LSE Group 纳入 LSE+Borsa Italia+Turquoise；加上 Euronext，三强主导。后续并购：2013 年 ICE 收购 NYSE Euronext（拿下 Liffe），Euronext 重新独立；2017 年 2 月 CBOE 收购 BATS Global Markets，形成跨洲场所。
- **复杂度不降反升的机理**：三强反而把流动性**切成众多利基池（niche pools）**，每个池提供不同"性质"的流动性，混合两类：**自然流动性**（来自中长期持仓投资者）与**机会型流动性**（来自高频交易者与做市商，见 [Menkveld, 2010]）。借 Akerlof(1970) 的"柠檬"比喻，运营商早就发现客户接受**分仓供给**——一个池盛"柠檬流动性（lemon）"、一个盛"樱桃流动性（cheery）"。LSE 集团让已在 LSE 上市的股票同时在 Turquoise 交易，正是暗示"LSE 多樱桃、Turquoise 多柠檬"。这催生了中点暗池、仅买方池、按规模优先（size-priority）订单簿、随机集合竞价等。**自由度↑→资源配置可更优，但配置决策的复杂度也↑**：聚合视图与统一接入的成本随之上升，故 2018 年的智能订单路由/最佳执行举证比 2006 年复杂得多。
- **MiFID 用"极少数简单机制"撬动巨大复杂度**，三大支柱：
  - **最佳执行（best execution）**：欧洲**不**提供美式"合并行情带（consolidated tape）"，也不强制场所把可在别处获更优价的订单系统性路由出去（原因：欧洲清算结算渠道与费率多样，含全成本后"最优价"可能反转）。于是用"最佳执行政策"取代美国的 trade-through 规则——中介须披露其路由/拆单方式，形成在场所竞争之上**叠加一层**的"元竞争（meta competition）"。两层竞争虽理论上推动更高效，但**收敛更慢**，使整个组织的效率难以判断。（脚注：合并行情带汇总各所最优限价，构成 NBBO 全国最优买卖价。）
  - **场所三/四身份**：交易须发生在 **RM**（受监管市场，托管撮合引擎+公共订单簿，负责价格形成过程 PFP 完整性，含熔断）、**MTF**（多边交易设施，撮合引擎+公共订单簿，常显式/隐式锚定 RM 盘口）、**SI**（系统性内部撮合商，双边协议、一方为做市商），或 **OTC**（场外）。无论哪种都须上报披露以维护 PFP 一致性。该划分把重心压在 RM 成交上。**两件意外**重塑了格局：① OTC 规模远超预期（连重复计数都扣除后仍大）；② 极少参与者申请 SI 身份（其盘前/盘后透明度约束让做市商不愿暴露库存，尤其后危机时代）。结果碎片化坍缩成 **RM 与 MTF 的二元结构**。（脚注：MiFID 2 新增 OTF，介于 MTF 与 OTC 之间；意大利 2013 年中对 OTC 征双倍金融交易税，OTC 笔数骤降。）
  - **盘前透明度（pre-trade transparency）**：原理是"披露给决策者的信息越多，行为越理性，PFP 越高效（成交价越接近'公允价'）"。四类豁免：**大额豁免（large-in-scale）**（足够大的单可全隐藏挂在簿中）、**订单管理设施豁免**（保护 facilitation 交易免于信息泄露）、**协商成交豁免**（须落在参考市场成交量加权价差内）、**参考价豁免（reference price waiver）**（任意规模隐藏单，只要不通过买卖对撞**形成价格**，可按外部固定价成交，透明度委托给参考场所——**暗池据此运作**）。
  - **盘后透明度（post-trade transparency）**：要求成交"尽快"公开披露。**核心张力**：投资者一方面想要尽量多信息以理性决策，另一方面想隐藏自己的成交，怕短线者套利。机理细节：理想化的市场冲击（[Gabaix et al., 2006]）下大资管把大单卖给做市商不会自伤（做市商次日转手对冲）；但实务中大单被切成**时间片+名单**，对手方观察到上半天的买压会**提前抢跑**，反噬原资管；做市商也不愿暴露刚吃下的大单（人人皆知他要对冲）。又因为成交所在场所**总是可见**，**市场越碎片，利基池里的对手方越易被识别**（专做市商的池 → 池内多为对冲单）。这反而让参与者**偏好把 Lit/暗池/全隐藏单混在同一场所**（如 Chi-X），使其汇入同一盘后通道、降低单笔信息含量——这也解释了为何 SI 申请少、替代场所增长有限。
- **失败案例（机理教训）——Turquoise**：由九家大投行（BNP Paribas、Citi、Credit Suisse、Deutsche Bank、Goldman Sachs、Merrill Lynch、Morgan Stanley、Société Générale、UBS）创立，**创始股东自任做市商**。但投行不是日内交易专家，开业即未能提供预期流动性，其**日内成交曲线更随日内波动率走**（而非像其他场所那样的稳定日内形态），说明**该池做市商比别处更风险厌恶**。一年后（2009 年 3 月）专职做市商退出，份额立刻受冲击；2009 年 12 月被 LSE 集团收购。教训：**流动性供给的可持续性取决于供给者是否专业的日内做市方**，而非品牌或资本。

## 关键公式 · 图表 · 定义（图表逐一转述）

### 公式
记号：t1→t2 期间在 N 个场所共 M 笔成交；第 ℓ 笔价 P_ℓ、量（股数）V_ℓ、时刻 τ_ℓ；δ_ℓ=n 表示第 ℓ 笔发生在场所 n；指示函数 **1_n(δ_ℓ)**=（该笔在场所 n 则 1，否则 0）。上标 k 表股票。

- **(1.1) 按金额（成交额）的市场份额**：
  𝔐^k(n) = [ Σ_{t1≤τ_ℓ^k≤t2} P_ℓ^k · V_ℓ^k · 1_n(δ_ℓ^k) ] / [ Σ_{t1≤τ_ℓ^k≤t2} P_ℓ^k · V_ℓ^k ]
  （分子=场所 n 的成交额，分母=全市场成交额。）
- **价格近似恒定时退化为按股数份额**：当 ∀ℓ: P_ℓ^k ≈ P，
  𝔐^k(n) ≈ [ Σ V_ℓ^k · 1_n(δ_ℓ^k) ] / [ Σ V_ℓ^k ].
- **(1.2) 按笔（per trade）的市场份额**：
  𝔐_T^k(n) = [ Σ_{1≤ℓ≤T} 1_n(δ_ℓ^k) ] / T，  T=期间总成交笔数。
- **剔除集合竞价的修正**：设场所 n 在集合竞价中的平均份额 m(n)，集合竞价占全天比重 p，T=该股全市场日均成交额，则
  𝔐(n; ex-fixing) ≈ [ (1 − m(n)·p)·𝔐(n)·T ] / [ (1 − p)·T ] = [ (1 − m(n)·p) / (1 − p) ] · 𝔐(n; all included).
  （含义：从"全口径份额"剥离被某场所垄断的集合竞价部分，还原连续竞价份额。）
- **跨股票加权三式**：
  ① 按成交额：𝔐^{1..K}(n) = [ Σ_{1≤k≤K} T^k · 𝔐^k(n) ] / [ Σ_{1≤k≤K} T^k ]；
  ② 按指数权重：𝔐^𝔍(n) = Σ_{k∈𝔍} w_k · 𝔐^k(n)；
  ③ 等权：𝔐_u^{1..K}(n) = (1/K) Σ_{1≤k≤K} 𝔐^k(n).
- **脚注 1（重要口径提示）**：用"成交"度量份额是**有意选择**，好处是成交额经手续费直接对应场所营收；但也可改用"已发送订单（sent orders）"或"盘口一档挂单量（quantity at first limit）"来度量份额——口径不同，含义不同。

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **A股不存在欧洲式多场所碎片化**（沪/深是分板上市而非竞争撮合，无 MTF/暗池竞争），所以"市场份额=流动性"不能直接搬。但本节的**度量方法论可直接迁移**到因子构造：
  - **"度量须匹配问题"是因子设计的总纪律**：先想清因子要回答"概率/营收/金额去向"哪类问题，再选口径，避免拿一个口径回答所有问题。
  - **按金额 / 按股数 / 按笔 三分（公式 1.1/1.2）= 成交/换手类因子的口径分叉**：量价因子用成交额、成交股数还是成交笔数（订单数），在价格波动期会显著分歧；构造换手率、量比、VWAP 偏离等务必固定并标注口径。
  - **脚注 1 → L2 因子的口径选择**：份额可基于"成交 / 已发送订单 / 盘口一档量"——对应 L2 里用**成交流、委托流、还是盘口一档/多档量**构造因子，三者信息与噪声不同（成交=已实现，委托=意图含撤单噪声，盘口量=可被刷量虚增）。
  - **集合竞价 vs 连续竞价必须分开统计**：A股有开盘/收盘集合竞价（收盘集合竞价 2018 年于沪深引入），其量价行为与连续竞价不同；做日内/分钟因子时，集合竞价时段应**单独处理或剔除**，否则会像图 1.1(a)→(b) 那样被系统性扭曲。书给的 ex-fixing 修正式提供了"从全口径剥离集合竞价"的思路。
  - **跨股票加权三式 = 截面因子聚合的三种选择**：等权 / 按成交额（流动性）权 / 按指数权重（市值/自由流通），会给出不同的组合层面读数；构造行业或全市场聚合指标时需明确并做稳健性对比。
  - **熵式离散度（entropic dispersion）可移植为"成交分散度"特征**：即便没有多场所，也可把份额概念换成"跨时间桶 / 跨价位 / 跨成交规模"的分布，用熵度量其分散程度，作为流动性/微观结构因子的候选。
  - **比例估计 + 置信区间（Appendix A.8）= 对短窗份额/占比类因子做统计严谨化**：小样本占比噪声大，因子检验时应给区间而非点值。
  - **"盘口深度 ≠ 真实可成交流动性 / 重复流动性"警示**：盘前透明度可被多场所重复挂单与高隐藏单失真；A股虽无跨场所，但盘口量同样可被刷量/瞬时撤单虚增——L2 流动性因子应结合撤单率、成交回报校正，盘后成交流信息更实。
  - **"碎片化随流动性上升"的异质性**：提示流动性类因子在不同流动性分层的股票上行为不同，截面上需分层或中性化。
- **成本/执行**：市场冲击 vs 市场风险的权衡、子单时间×空间拆分、以执行结果（非延迟）评价 SOR——可对接 factor_library 回测库的交易成本/执行模块（细节在第3章 §3.4，本块仅作前瞻）。
