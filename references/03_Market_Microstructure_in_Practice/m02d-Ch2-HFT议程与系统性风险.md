## 本块讲什么

承接 §2.4 对 HFT 的讨论，本块分两大主题：

1. **§2.4.3 HFT 对投资者的影响并不均匀（HFT do not impact all investors equally）**：先给出 SEC 对 HFT 的官方定义，再用一个"两玩家 + HFT 插入"的微观博弈，论证一个反直觉的核心结论——**HFT 确实压窄了有效价差，但并没有降低（甚至略微抬高了）真实投资者的成交成本**。随后把"成本"建模成一个随机变量，用概率树给出其**期望 E(C) 与方差 V(C)**，并借"成本—不确定性"前沿（类比有效前沿）与中心极限定理（CLT），说明 HFT 对**零售/养老金/对冲基金**三类投资者的冲击在"期望成本"上几乎一致、但在"成本不确定性"上严重不均。

2. **§2.5 碎片化与系统性风险（起）**：界定系统性风险，提出核心研究问题——**MiFID 后价格形成过程（PFP）的改变，是否从日内尺度传导到了跨日尺度**（日内波动 vs 跨日波动、日内相关 vs 隔夜相关）。本块覆盖到：
   - **§2.5.1 西班牙自然实验**：因清算结算立法滞后，西班牙在 2007–2011 几乎未被碎片化波及，成为研究碎片化效应的"对照组"；用 Garman–Klass 日内波动做 FTSE 100 vs IBEX 35 的差分比较。
   - **隔夜 vs 日内相关性**：相关性飙升主要由隔夜侧驱动，日内相关充当基线；用"夜间=集合竞价定盘 / 日间=连续双向竞价"两种微结构解释。
   - **§2.5.2 Flash Crash（2010-05-06，起）**：引入闪崩作为系统性风险的疑点，本块到此开篇。

## 方法论 / 机理要点（重点·展开）

### 一、HFT 的定义（SEC 五条标准）

HFT 是"为多池竞争付出的代价（the price to pay）"。它们专门套利同一标的在多个交易池之间的流动性失衡，靠 MTF 的 maker/taker 费率结构获利，借助技术用**极小订单、极高频率**交易来压低库存风险。其在成交中的对手方占比：**美国 >80%、欧洲 40%、日本 35%**。

SEC 的五条认定标准（机理上即"低延迟 + 高撤单 + 不留隔夜头寸"）：
1. 使用极高速、精密的计算机程序生成、路由、执行订单；
2. 使用交易所/MTF 的**co-location（主机托管）与专属数据源**以最小化网络及各类延迟；
3. 建仓与平仓的时间窗极短；
4. 提交大量订单、又在提交后很快撤单；
5. 收盘时尽量保持**平头寸**（不持有显著的、未对冲的隔夜头寸）。

### 二、核心机理：HFT 如何"压窄价差却不降低成本"

这是本块最重要、最反直觉的论证。设市场只有两个玩家 A、B，都挂在盘口第一档等待成交，有效买卖价差记为 ψ。

- **无 HFT（图2.21a、b）**：谁先"不耐烦（impatient）"，谁就主动越过价差去吃对手价，成本 = 有效价差 ψ。下一次轮到对方不耐烦，对方同样付 ψ。长期看 A、B **平均各付 ψ/2**。
- **有 HFT 插入（图2.21c）**：HFT 抢在 B 内侧挂单，成为 A 的第一档对手。A 不耐烦时吃掉 HFT 的价，成本仅 **ψ/2 + ε**（略高于原价差的一半）。HFT 随即极快地在 B 前方重新挂单；B 的单未被撮合，只能去吃 HFT 的新价，成本同为 **ψ/2 + ε**。
- **结论（关键）**：
  - 最终**有效价差确实被压窄**（约为原价差的一半多一点）；
  - 但 A、B 的**平均成交成本仍是 ψ/2 + ε**，即比原来的 ψ/2 略高——HFT 抽走了那一份 ε；
  - **HFT 自己的成交成本为零（nil）**（它总是被动方/总能被撮合）。

也就是说：**"挂牌/有效价差变窄" ≠ "投资者成本下降"**。盯着报价价差去判断交易成本会系统性低估真实成本。

**额外的不对称成本**：HFT 偏好小单 → 逼迫其他参与者把单子拆细 → 拆细后成交更多、向客户/监管上报更频繁 → **上报成本（reporting costs）上升**，而这类成本**只有非自营机构（non-proprietary firms）才承担**，叠加技术成本，构成一组**分布不均的固定损失**，最终使终端投资者的交易条件恶化。

### 三、把"成本"建成随机变量：期望与不确定性

定义 ψ_B = HFT 出现前（"Before"）的价差；ψ_N = 新（"New"）价差。若 HFT 参与了 q% 的成交，则：

> **ψ_N = q·ψ_HFT + (1−q)·ψ_B**

其中投资者吃到 HFT 价时多付的部分 ε = **ψ_HFT/2**（即 HFT 把价差进一步压窄后的"半价差"）。

**实证标定（CAC 40，仅取主市场 Euronext Paris 数据）**：以 2007-04~11 为"before"、2011-04~11 为"new"，q=40%，得
> ψ_B = **5.41 bp**，ψ_N = **4.73 bp**。

即有效价差确实下降了。但单个投资者下一单的真实成本 C 取决于其遭遇（概率树，图2.22）：
- 以概率 r **遇到 HFT** → 被迫主动成交，付 C = **ψ_HFT/2 + ψ_B/2**；
- 不遇 HFT 但因其他原因主动成交 → 付 C = **ψ_B**；
- 不遇 HFT 且被动成交 → 付 C = **0**。

由此（在"不遇 HFT 时主动/被动各半"的假设下）：
> **E(C) = (ψ_B + r·ψ_HFT) / 2**
> **V(C) = r·((ψ_HFT + ψ_B)/2)² + (1−r)·ψ_B²/2 − E(C)²**

（图中末项写作 "E(r)²"，按方差定义应为 E(C)²，疑为排版/OCR 笔误 — [符号存疑]。）

### 四、"成本—不确定性"前沿（图2.23）与有效前沿类比

把每个投资者画在 (不确定性 √V(C), 期望成本 E(C)) 平面上，得到一条凹形曲线，**与组合管理的有效前沿同构**：
- **成本越低，对成本的不确定性越高**——无法在不增加风险的前提下降低期望收益/成本；
- 沿曲线，**提高遇到 HFT 的概率 r**：期望成本上升 ε = ψ_HFT/2，但**成本的不确定性下降**（HFT 让结果更"确定地略贵"）。

### 五、交易次数 N 与中心极限定理：不均匀的真正来源（图2.24）

单笔交易讲完后，考虑做大量交易的投资者。设 N 笔交易成本 C₁,…,C_N。由 CLT：
> E[(1/N)·Σ Cᵢ] = E(C)，Var[(1/N)·Σ Cᵢ] = **Var(C) / N**

故平均成本的标准差按 **1/√N** 衰减。**交易笔数越多的投资者，其成本不确定性天然越低**（其他条件相同）。在图2.23 上表现为：N 增大时曲线按**位似（homothety）整体左移**——同样的遇 HFT 概率下，期望成本不变，但不确定性已经更低。

三类投资者：**Small=零售、Medium=养老金、Large=对冲基金**（N 依次增大，曲线依次更靠左）。把 HFT 占比从 30% 提到 50%（图中 ★ Q=30% → ● Q=50%）：
- **期望成本对所有人都约升 0.5 bp**（几乎一致）；
- 不确定性都下降，但**极不均匀**：零售投资者的成本不确定性**大幅下降**；而养老金（大 N）的不确定性**早已被大量交易压到很低，几乎无可再降**。

**总结（本节落点）**：买卖价差既是成本来源，其**期望值**与**不确定性**都是投资者的关键指标。随交易笔数 N 与遇 HFT 概率 r 变化，**期望成本几乎不变，但不确定性变化显著**——这正是"HFT 对投资者影响不均"的量化体现。

### 六、§2.5 系统性风险与"日内→跨日"传导问题

- **系统性风险定义**：用尽所有可能的金融工具对冲掉一切可对冲风险后，组合中**残余的那部分风险**。
- **争论**：HFT 支持方称高频活动增加流动性、**降低波动**；反对方称 MiFID + HFT 使**系统性风险上升**，常援引 2010-05-06 美股 Flash Crash。
- **核心研究问题**：区分**日内波动（intraday）**与**跨日波动（extraday）**。投资者受跨日约束，直接受跨日波动影响、并不直接受日内波动影响。问题是：**日内尺度在多大程度上传导到跨日尺度？MiFID 引起的 PFP（价格形成过程）改变，是否在跨日尺度产生了可见效应？**
- 论证路线：① 从西班牙市场视角看 MiFID 对日内波动的影响；② 指数成分股之间的相关性与指数波动；③ Flash Crash 与欧洲市场宕机。

### 七、§2.5.1 西班牙自然实验（对照组设计）

**设计思想**：MiFID 在西班牙与其他欧洲国家同日生效（2007-11-01），但因西班牙立法复杂，**MTF 直到 2010 年市场份额仍极低**（Chi-X 在 IBEX 35 上的份额 2009-08 才首次破 0.5%，2010 年初峰值仅 1.43%；Turquoise、BATS 更弱）。于是西班牙在 2007-11~2010-01 几乎**未被碎片化波及**，成为研究"正确实施 MiFID 的其他欧洲市场"的**对照参照系（control frame）**。

**机理（为何 MTF 进不来）**：症结在清算与结算立法。Titulo V（2011-06）之前，唯一获授权的清算机构是 **Iberclear**，其持有者**独家**为西班牙主市场 **BME**。登记代码（registration code）的签发是 Iberclear 的特权且收费高昂 → MTF 无法在西班牙落地。Titulo V 后，MTF 自有清算所获准签发该代码，清算结算费用才放松。
- **临时绕道（2011-06 前）**：MTF 在日终把全部成交量与 BME 经纪商对敲（crossed），但仍要向 BME 额外付费，竞争力受损。
- **结论**：西班牙在 2007–2011 维持着近乎 MiFID 前的运行模式——**理想的"未处理组"**。

**图2.25（每日成交笔数）佐证**：成交笔数是衡量高频活动强度的良好代理。CAC 40、FTSE 100（尤其 FTSE）笔数在整个区间持续上升，反映高频活动加剧；**IBEX 35 则一直低位平稳**，印证西班牙的"未碎片化"。

### 八、波动率分析：微结构、Garman–Klass 与差分比较

- **波动率 = PFP 的随机（未预期）成分**，源于上市公司经营不确定性，经资产结构传导为市场不确定性；日波动 = 投资者承担的风险，波动越高应提供越高回报。
- **微结构通过 PFP 把市场价格与基本面相连**：微结构越有效，日内波动越低。日内波动是日内市场风险的代理，也是交易隐性成本（市场冲击）的一部分；又因波动与买卖价差有**经济学上成立的关系**，故"价差下降"理应"波动下降"。
- **但现实更复杂**：MiFID 后价差确有收窄（**主因是 tick size 变化**），而波动下降并不明显——**部分被 2008–2010 金融危机掩盖**。
- 故只能在**情境相似**的市场间比较：取 **UK（被碎片化）vs 西班牙（维持 MiFID 前模式）**。
- **Garman–Klass 15 分钟波动率**（用 OHLC 的区间估计量）：
  > Vol_GK = √( ½·(high−low)² − (2·log2 − 1)·(close−open)² ) / 区间平均价
  其中 open/high/low/close 在 15 分钟区间内计算。

### 九、§2.5 阶段性结论：碎片化对日内波动无显著可见效应

- **图2.26(a)** 与 **图2.26(b)** 都**未显示**英国市场碎片化（2007-11 后）对日内波动有显著效应；也**无法断言**英国市场碎片化后日内波动是升还是降。
- 方法学上值得学的：用**比值（日内/日波动）的差分**来抵消"两国危机暴露程度不同"这一混淆因子（详见图表节）。

### 十、隔夜 vs 日内相关性：传染链中的中介标志

- 考察指数成分股之间**平均日内相关 / 隔夜相关**的趋势。**相关性飙升往往由隔夜侧解释**：2010 夏季高相关事件中，日内相关也同步飙升（**有记录以来最高**）；2011-08 又一次高相关。
- **日内相关反映日内尺度的协同运动**，是传染过程（contagion）中把"微观成因"放大为"宏观效应"的**中介标志（intermediary marker）**。
- 计算（六个月滑动窗口）：
  > 日内相关 = Corr(日内收益ᴬ, 日内收益ᴮ)  …(2.2)
  > 隔夜相关 = Corr(隔夜收益ᴬ, 隔夜收益ᴮ)  …(2.3)
  > 日内收益ₜ = log(closeₜ) − log(openₜ)
  > 隔夜收益ₜ = log(openₜ) − log(close₍ₜ₋₁₎)
- **发现（图2.27/2.28）**：在 CLES 60、DAX、CAC 40、IBEX 35 上，相关性的上升**主要由隔夜侧驱动**；**日内相关充当基线（baseline/floor），隔夜相关不断在其上方跳动**。对 CAC 40、CLES 60、IBEX 35 三个指数，**日内相关在 2010-10 达到 1991 年以来最高**；而隔夜相关 2008、2009 的尖峰其实也曾同样高——故"日内相关创新高"是新现象，可与"日内相关增长 + HFT 影响"相联系。
- **两种微结构的解释（关键机理）**：差异源于**夜间与日间是两种不同的微结构**。**夜间 = 集合竞价定盘（auction fixing）**，**日间 = 连续双向竞价（continuous double auction）**。**开盘定盘较少被嘈杂的微结构噪声污染**，比收盘定盘更能反映投资者对资产价格的共识；因此**隔夜相关更接近跨日相关**。据此，2010 年 3–10 月跨日相关的尖峰，部分源于**日内相关上升**，而后者又是 **PFP 噪声降低**的结果——**HFT 活动增长可能是其中一个解释**。

### 十一、§2.5.2 Flash Crash（2010-05-06，起）

- **问题**：Flash Crash 是否说明市场碎片化后系统性风险上升？作者立场谨慎：闪崩虽是坏消息，但**并非系统性风险上升的明显证据**。
- **令人不安的反事实**：若闪崩**恰好发生在临近收盘**，后果会严重得多——会把一些基金的估值"扫"进去（按受污染价格估值），并使市场在更长时间内被扰乱。
- 书中标注将给出**闪崩的小型动力学模型**（见 Appendix A.5）。
- 史实开篇：2010-05-06 美股突遭急跌后又快速反弹；S&P 500 当日开盘起本已处于下行趋势……（叙述延续到本块之后）。

## 关键公式 · 图表 · 定义（图表逐一转述）

**关键公式汇总**
- 新价差：ψ_N = q·ψ_HFT + (1−q)·ψ_B；额外抽成 ε = ψ_HFT/2。
- 成本期望：E(C) = (ψ_B + r·ψ_HFT)/2。
- 成本方差：V(C) = r·((ψ_HFT+ψ_B)/2)² + (1−r)·ψ_B²/2 − E(C)²（末项符号见上 [符号存疑]）。
- 组合化：E[均值成本] = E(C)，Var[均值成本] = Var(C)/N，标准差 ∝ 1/√N。
- Garman–Klass：Vol_GK = √( ½(high−low)² − (2log2−1)(close−open)² ) / 区间均价。
- 日内收益 = log(close) − log(open)；隔夜收益 = log(openₜ) − log(close₍ₜ₋₁₎)。
- 标定值：ψ_B = 5.41 bp，ψ_N = 4.73 bp（CAC 40，2007 vs 2011，q=40%，仅主市场数据）。

**图2.21（书p.164）三种盘口情形——HFT 如何插入价差**
横轴价格、纵轴挂单量的盘口示意，三联图：
- (a) 无 HFT、第一玩家主动：A、B 各挂一档，间距为有效价差 ψ（标注 ψ/2 两段）；箭头表示 A 越过价差去吃 B 的价。
- (b) 无 HFT、第二玩家主动：轮到 B 越过价差吃 A 的价，箭头反向。两情形长期平均各付 ψ/2。
- (c) 有 HFT：HFT 在 B（及 A）内侧各插一档，把不耐烦玩家与对手之间的有效距离压到约 ψ/2；箭头显示不耐烦玩家先吃 HFT 的内侧价。直观传达：**HFT 把自己塞进原价差内部，压窄了有效价差，但被动玩家被迫去吃 HFT 的价**。

**关键定义**
- **PFP（Price Formation Process，价格形成过程）**：把基本面/经营不确定性经微结构映射为市场价格的过程；其随机成分即波动。
- **系统性风险**：穷尽对冲后组合中的残余风险。
- **auction fixing（集合竞价定盘）vs continuous double auction（连续双向竞价）**：夜间靠开/收盘定盘，日间靠连续撮合；开盘定盘噪声更小、更反映共识价。

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **成本建模（最重要）**：本块给出"有效价差 ≠ 真实成交成本"的硬结论。做 TCA / 回测时，**不能用挂牌价差或有效价差直接当成本**——HFT 会抽走 ε≈ψ_HFT/2，且会逼迫拆单从而把你推成主动方。可把单笔成本建成概率树式随机变量：以"遇对手类型"的概率 r 混合 {主动付价差、被动付 0} 等情形，分别估 **E(C)** 与 **V(C)**。
- **成本的方差 ∝ 1/N（容量/换手视角）**：组合层面平均成本的不确定性按 1/√N 衰减。高换手策略（大 N）期望成本不一定更高，但**成本的可预测性更强**；低频/低换手组合（小 N）则成本尾部风险更大。做容量分析、滑点预算时应同时报告成本的**期望与方差**，而非只报均值。
- **Garman–Klass 区间波动**：一个只需 OHLC 的高效日内波动估计量，A 股日频/分钟 OHLC 齐备，可直接做**range-based 波动因子 / 风险度量**，比收盘价方差更省样本、更抗噪。注意公式里 (2log2−1)≈0.386 对 (close−open)² 的修正项。
- **隔夜 / 日内收益分解（A 股直接可用）**：日内收益=log(close)−log(open)、隔夜收益=log(open_t)−log(close_{t−1})。这正是 A 股**隔夜跳空因子、日内动量/反转因子**的标准构造；本块还提示**隔夜相关 > 日内相关、且更接近跨日相关**，因为**开盘集合竞价噪声更小**——A 股有 9:15–9:25 开盘集合竞价与收盘集合竞价（2018 起 14:57–15:00），该"定盘更干净"的机理可迁移，开盘竞价价可能比连续段更能代表共识价。
- **相关性 / 拥挤 / 系统性风险因子**：日内相关被定位为"传染链中介标志"，其抬升与 PFP 噪声下降、HFT 活动相关。可据此构造**成分股平均两两相关（隔夜/日内分开）**作为系统性风险/拥挤度的监测指标；相关性飙升常由隔夜侧先行，对**风险预警**有用。
- **方法学模板：自然实验 + 差分**：西班牙因清算立法滞后而成为"未处理组"，是干净的**对照组/事件研究设计**范例；图2.26(b) 用"日内/日波动比值的差分"抵消两国危机暴露差异，是一个可直接套用的**双重差分 + 比值归一去混淆**模板——做 A 股政策/制度变更（如涨跌停改革、注册制、做市制度）对波动/价差影响的研究时尤其适用。
- **闪崩 / 尾部场景**：作者强调"若闪崩临近收盘则会污染基金估值"。回测与风控应专门压力测试**收盘前流动性骤降**情形，并警惕用受污染收盘价做估值/再平衡。
