## 本块讲什么

这是全书量化附录里**高频（tick 级）随机建模**的核心三节，是 L2/逐笔因子的理论根基：

1. **§A.12 流动性聚簇（Clusters of Liquidity）** —— 用**点过程（point process）**为成交/订单到达建模；从 Poisson 过程（无记忆）讲到 **Hawkes 自激励过程**（能产生"聚簇/爆发"），再到 **propagator（传播子）模型**这一逐笔价格冲击的替代框架，以及多类别 agent 的推广。
2. **§A.13 Signature Plot 与 Epps 效应** —— 高频下两个著名的**偏差现象**：已实现波动率（realized variance）在采样步长 Δ→0 时**发散/爆炸**（signature plot），已实现相关系数在 Δ→0 时**塌缩归零**（Epps effect）。根源都是微观结构噪声 + 异步成交。
3. **§A.14 平均效应（Averaging Effect）** —— 警示"求均值"的两个陷阱：日内均值剖面（mean profile）是不存在于任何真实单日的人造形状；在聚合量（指数）上做回归 ≠ 对成分回归系数求加权平均（聚合偏差）。

贯穿主线：**真实市场不是 Poisson**——事件成簇、价格有微观结构噪声、跨资产异步——所以朴素的"squared-returns 求和""高频相关""均值剖面"都会系统性失真，必须用自激励过程/噪声稳健估计来纠偏。

> §A.11 尾（书页 303–304，Fig A.19/A.20）只是把成交量季节性分解 `V = G(年)·F(月)·H(日内)` 收尾：Fig A.19 为 Total SA 周一至周五的**日内成交量季节性**（每天重复的开盘高→盘中低→收盘抬升形态，周五正午因衍生品到期 derivative expiry 有小凸起，故模型额外加 `g(d,n)` 项）；Fig A.20 为**年度季节性**（2007–2011，2008 危机年有明显尖峰）。本块重点不在此，下面集中 §A.12–§A.14。

## 方法论 / 机理要点（重点·展开）

### 1. 点过程基础：为什么需要它

把"成交时刻""限价单到达时刻"看作一列随机时点 `(τ_i)`，这就是**简单点过程（simple point process）**。围绕它定义四件事：

- **计数过程（counting process）** `N_t = Σ_i 1{τ_i ≤ t}`：到 t 为止发生了几次（阶梯函数，每事件 +1，见 Fig A.22）。
- **持续期过程（duration）** `Δτ_i = τ_i − τ_{i−1}`：相邻事件间隔。逐笔研究里这就是 inter-trade duration（交易间隔，ACD 类因子的对象）。
- **强度过程（intensity）** `λ(t) = lim_{h↓0} (1/h)·E[N_{t+h}−N_t | F_t]`：单位时间事件发生的瞬时期望率，即"此刻市场活跃程度"的连续刻画。默认用自然 filtration `F_t^N`（只用历史自身信息）。
- **积分强度（integrated intensity）** `Λ(τ_{i−1},τ_i)=∫λ(s)ds`：把时间按强度"拉伸"。

**齐次 Poisson 过程（homogeneous Poisson）**是基准：强度 λ 恒定、不依赖历史，事件在 (t,t+h) 出现概率 `λh+o(h)`，间隔 `Δτ_i` 是 iid 指数分布。它是 **Markov、无记忆**的——这正是它**不适合真实市场**的原因：现实中成交会成簇爆发。

**时间变换定理（Time Change Theorem，Thm 1，机理关键）**：任何点过程经过"按其积分强度重置时钟"后，都退化为强度=1 的标准 Poisson 过程。这是鞅论 Dubins–Schwarz 定理（连续局部鞅 = 布朗运动的时间变换 `M_t=B_{⟨M⟩_t}`）在点过程上的对应物。
- **用途**：①模型拟合优度检验——把估出的 λ(t) 积分，得到的残差间隔若不是 iid 指数(1)，说明模型设定有误（Hawkes 校准的标准诊断）；②理论上支持"成交时钟/成交量时钟（trade time / volume clock）"——在变换后的时间里建模更干净。

### 2. Hawkes 过程：自激励 = 聚簇的生成机制（本块第一根基）

当数据**非 Markov、有聚簇**时，用 **Hawkes 线性自激励过程（linear self-exciting process）**：

```
λ(t) = λ_0(t) + ∫_{-∞}^t ν(t−s) dN_s = λ_0(t) + Σ_{τ_i<t} ν(t−τ_i)
```

- `λ_0` 是确定性的**基准强度（base/baseline intensity）**＝外生（exogenous）到达；
- `ν(·)≥0` 是**激励核（kernel）**：每发生一次事件，就给未来强度抬升一块 ν，体现"过去事件对当前强度的正向影响"。

Hawkes(1971) 取**指数核** `ν(t)=Σ_j α_j e^{−β_j t}`：α 是激励幅度（一次事件抬升多少强度），β 是衰减速度（影响多快消退）。最简单 P=1、λ_0 常数：

```
λ(t) = λ_0 + Σ_{τ_i<t} α·e^{−β(t−τ_i)}
```

**机理直觉**：一次成交→强度跳升 α→更可能引发下一次成交→再跳升……形成**自我强化的爆发（cluster）**，事后指数衰减回基准。这就是"流动性聚簇"的生成模型。

**平稳性 / 分支比（branching ratio，最重要的一个数）**：
- 平稳时 `E[λ]=μ=λ_0 / (1 − ∫ν(u)du)`（式 A.30）；P=1 时 `E[λ]=λ_0/(1−α/β)`。
- 平稳/非爆炸条件：`Σ_j α_j/β_j < 1`（P=1 即 **α/β<1**）。
- **`n=α/β` 即分支比**：可解读为"平均每个外生事件最终诱发多少后代事件"，度量市场活动的**内生性（endogeneity）**。n→1 表示成交几乎全是市场对自身的连锁反应（自反性 reflexivity 极高）、外生信息占比极低——这种状态脆弱、易触发闪崩（flash crash）。这是把 Hawkes 用作"市场稳定性/拥挤度"监测指标的理论出发点。

**Hawkes 的核心价值（书中明确）**：它能在**精细尺度复现微观结构噪声、signature plot、Epps 效应**，同时在**大尺度收敛到标准布朗扩散**——即用一个 tick 级模型统一解释了 §A.13 的两个偏差现象，又不破坏宏观有效性。这是它成为高频建模主力的原因（Bacry et al., 2012）。

**多维/互激励（mutually exciting）**：把买单流、卖单流（乃至不同 agent 类别）写成多维 Hawkes，核 φ 描述"某类事件激励另一类事件"。式 (A.39) 给出按 agent 类别 a 的卖出强度：
```
λ_t^-(a) = μ_a^- + Σ_{a'} ∫_{s<t} φ_{a-}^{a'+}(t−s) dN_s^+(a') + ∫_{s<t} φ_{a-}^{a'-}(t−s) dN_s^-(a')
```
非爆炸条件推广为 `‖φ‖_1<1`。这把"买单自相关、买卖交叉影响、不同参与者互相反应"统一进一个物理时间（physical time）驱动的框架。

### 3. Propagator（传播子）模型：逐笔价格冲击的另一条路（本块第二根基）

这是 Hawkes 之外、且**早于** Hawkes 用于高频金融的框架（综述见 Taranto et al., 2016）。它在**成交时间（trade time）**里直接写中间价：

```
m_k = Σ_{k'≤k} ( G(k−k')·ε_{k'} + η_{k'} ) + m_{-∞}      (A.31)
```

- `ε_k ∈ {−1,+1}`：第 k 笔成交的**符号**（买方/卖方发起，trade sign）；**关键：ε 不是 iid**，有长记忆自相关 `C(ℓ)=E[ε_{i+ℓ}ε_i]`。
- `G(·)`：**传播子核（propagator kernel）= 价格冲击的衰减形状**，通常取指数 `exp(−λδt)` 或幂律 `(1+δt)^{−γ}`。
- `η`：与 ε 独立的 iid 公允价格随机扰动（无方向的"新闻"）。

**机理直觉（本节灵魂）**：每笔成交都把价格往其方向推 `G(0)`，此后这个冲击随后续成交**逐步衰减**（由 `Δ_1G(δk)=G(δk+1)−G(δk)` 刻画）。于是单步价格变动（式 A.32）：
```
r_k(1) = m_{k+1}−m_k = G(0)ε_{k+1} + η_{k+1} + Σ_{δk≥0} Δ_1G(δk)·ε_{k−δk}
```

**长记忆—有效性悖论的解（核心 insight）**：成交符号有强正自相关（下面解释来源），按理价格应可预测；但价格近似鞅（收益无自相关）。化解之道——**传播子 G 的衰减形状被"精调"得正好抵消可预测的订单流**，使价差不可预测。数学上体现为响应函数与符号自相关的卷积关系：
```
S(ℓ) := E[r_k(1)·ε_{k−ℓ+1}] = G(0)C(ℓ) + Σ_{δk≥0} Δ_1G(δk)·C(ℓ+δk−1)      (A.33)
```

**估计法（去卷积 / deconvolution）**：把 (A.33) 写成矩阵 `S̃(L)=C̄(N,L)·Ḡ(N)`，则核增量 `Ĝ(N)=Ĉ(N,L)^{-1}·Ŝ(L)`（式 A.34），再累加还原 `G(k)=G(0)+Σ Δ_1G(ℓ)`。即：**只要测出符号自相关 C 和响应 S，就能反解出价格冲击核 G。** 实务上非参数 G(逐区间 Δ_1G)不光滑、不现实，建议用少数光滑函数（指数/幂律）分段拼接 `G(ℓ)=Σ_u g_u(ℓ)·1{τ_{u-1}≤ℓ<τ_u}`（式 A.35，参 Jaisson PhD）。

**价格响应曲线（price response curve）** `R(h)=E[r_k(h)|ε_k]`（式 A.37）：给定某笔成交方向，h 笔之后价格的期望累计移动——这就是**经验冲击曲线**（凹形、随 h 缓慢增长）。

**波动率标度（volatility scaling）/ Hurst 指数**：噪声水平 `∝ σ_η·√h`；不同 horizon 的方差比 `ρ²(h_1,h_2)=E r_k(h_2)/E r_k(h_1)`（式 A.38）作为 `h_2/h_1` 的函数，与 **Hurst 指数**密切相关，用来刻画微观价格动力学是否偏离 √时间 标度。

**符号自相关从哪来（stylized fact，务必记住）**：实证强支持"成交符号正自相关"——买发起成交后更可能再来买发起成交。两大解释：
- **拆单（splitting）**：大额母单（metaorder）被资管拆成长串同向子单；
- **羊群（herding）**：相似机构几乎同时做相似决策。
书中脚注提醒：**每拿到新数据集都该重新检验这条事实**。

**用 C 检验模型 + agent 异质性**：因为 G 完全由 C 决定（式 A.34），可做稳健性检验——在数据集 D_1 上估 G、拿到 C' 不同的 D_2 上用，若 `‖C_1^{-1}S_1 − C_2^{-1}S_2‖²<ε²` 则 G 稳健（说明冲击力学不依赖"谁在交易"，投资者结构变化会被价格变化自动吸收）。若 G 变了，标准解释是**参与者结构（mix of investors）变了**。由此引出按类别（投行/经纪/HFT…）建模 ε 的扩展：可用 `2A×2A` 常数强度矩阵 Λ，或 `2A` 维 Hawkes（式 A.39），让不同类别的买卖到达互相激励。物理时间 t 在这里是必需的——因为"上一笔由谁、哪个方向"决定下一笔强度（式见书页 316）。

### 4. Signature Plot：高频波动率的发散偏差（本块第三根基·上）

设对数价 `X=log S`，步长 Δ 的对数收益 `r_Δ(j)=X_{Δj}−X_{Δ(j−1)}`。理论上积分方差 `∫σ²ds = lim_{Δ→0} Σ r_Δ(j)²`，对应**已实现方差估计** `V̂_R(Δ)=Σ_{j} r_Δ(j)²`。

**问题（核心）**：实际把 Δ 取得越来越小时，`V̂_R(Δ)` **不收敛反而发散/爆炸**。把 `V̂_R(Δ)` 画成 Δ 的函数就是 **signature plot**（特征图）。

**根源——微观结构噪声**：观测价 = 有效价 + 噪声，`X=M+ε`（M 是半鞅，ε 是 bid-ask bounce/离散 tick 等噪声）。于是
```
V̂_R(Δ) = Σ r_Δ^M(j)²  +  Σ η_Δ(j)²  +  2Σ r_Δ^M(j)η_Δ(j)
```
第一项随 Δ→0 收敛到真积分方差；但**第二项（噪声增量平方和）≈ 2·Var(ε)·(T/Δ)，随采样点数 T/Δ 线性放大→∞**。即采样越密，每格收益越被买卖价跳动主导，噪声累积压倒信号。

**实务用途**：①诊断噪声强度；②选**最优采样频率**（signature plot 由陡降转平的拐点附近）；③改用**噪声稳健估计**——高低价法（Garman–Klass 1980）、多尺度/两尺度（Zhang et al. 2005；Aït-Sahalia & Jacod 2007）、bid-ask 建模（Robert & Rosenbaum 2012）、realized kernel、pre-averaging 等。

### 5. Epps 效应：高频相关系数的塌缩偏差（本块第三根基·下）

两资产协方差 `∫ρσ¹σ²ds = lim_{Δ→0} Σ r_Δ¹(j)r_Δ²(j)`，估计量 `Ĉ_R(Δ)=Σ r_Δ¹(j)r_Δ²(j)`。**Epps(1979)**：**测得的相关系数随测量区间缩短而下降**（Δ→0 时趋零）。

**根源**：①**异步成交（asynchronicity）**——两资产不在同一时刻成交，高频格子里大量"一动一停"，错配的零/陈旧收益稀释了协方差（最主要）；②同板块的 **lead–lag**（领先-滞后）；③ tick 效应等微观结构。

**实务用途**：高频估相关/β/协方差矩阵会**系统性偏低**——做高频配对、统计套利、风险平价、协方差矩阵时必须警惕。补救：用 **Hayashi–Yoshida 估计**（重叠区间、无需同步化）、或干脆放粗采样区间到 Epps 曲线的平台段（参 Barndorff-Nielsen & Shephard 2004；Hayashi & Yoshida 2005；Zhang 2010）。

### 6. 平均效应：两个"求均值"陷阱（§A.14）

**(a) 均值剖面 ≠ 真实路径（mean vs path）**：把某资产 3 个月的日内波动率**逐时刻平均**得到一条光滑 U 形剖面，但**任何单一交易日的真实路径都不长这样**（剧烈锯齿、尖峰）。教训：日内"平均成交量/平均波动率/平均价差"剖面是统计人造物，**不能当作典型单日来用**——对 VWAP/执行排程、日内季节性因子尤其危险。

**(b) 聚合回归偏差（aggregation bias）**：指数 `I=Σ w_k S^k`。价差线性聚合：`ψ^I = Σ w_k ψ^k`；**但波动率不线性聚合**（因分散化/相关性，指数波动率 < 成分波动率加权平均）。于是在指数量上回归"价差~波动率"得到的系数，**高于**在各成分上回归再加权平均的系数。即"对聚合量回归" ≠ "回归后再聚合"，是一种生态学谬误式（ecological/Simpson 型）偏差。教训：成本/冲击模型、风险模型在指数层 vs 成分层标定结果会系统性不同，不能互相替代。

## 关键公式 · 图表 · 定义（图表转述其内容）

**核心公式速查**
- 强度：`λ(t)=lim_{h↓0}(1/h)E[N_{t+h}−N_t|F_t]`
- Hawkes（指数核，P=1）：`λ(t)=λ_0+Σ_{τ_i<t} α e^{−β(t−τ_i)}`；平稳 `α/β<1`；`E[λ]=λ_0/(1−α/β)`
- Propagator：`m_k=Σ_{k'≤k}(G(k−k')ε_{k'}+η_{k'})+m_{-∞}`；去卷积 `Ĝ=Ĉ^{-1}Ŝ`（A.34）
- 已实现方差：`V̂_R(Δ)=Σ r_Δ(j)²`；噪声项 `∝(T/Δ)·Var(ε)` 致 Δ→0 发散
- 已实现协方差：`Ĉ_R(Δ)=Σ r_Δ¹(j)r_Δ²(j)`；Δ→0 时相关塌缩（Epps）

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **L2/逐笔因子的"为什么"**：成交、撤改单、价格跳变都不是独立到达——它们**自激励、成簇**。任何基于逐笔/快照的活跃度、订单流、波动率因子，背后的统计基准应是 Hawkes 而非 Poisson。把买/卖单流拟成多维 Hawkes，可直接构造：①**分支比 α/β（内生性/拥挤度因子）**——逼近 1 预警脆弱、易闪崩；②**激励强度/衰减 (α,β)** 作为微观活跃度状态；③用**时间变换定理**做"成交时钟/成交量时钟"重采样，再在变换时间里算其它因子（去季节性、去聚簇噪声）。
- **价格冲击与交易成本**：propagator 模型给出**冲击的衰减形状 G 与价格响应曲线 R(h)**——这正是执行算法（VWAP/IS）和成本（market impact）模型要标定的对象。A 股逐笔可估 G/C：先量成交符号自相关 C（拆单+羊群导致的长记忆），再去卷积出冲击核，校准临时/永久冲击的衰减。"长记忆订单流 + 价格近鞅"靠 G 精调抵消——这解释了为何冲击会衰减而非永久。
- **高频波动率/相关因子的去偏（必做）**：①算 RV/已实现波动率时务必看 **signature plot**，别在 Δ→0 端取值，或改用两尺度/realized kernel/pre-averaging；②算高频相关/β/协方差矩阵时警惕 **Epps 塌缩**，用 Hayashi–Yoshida 或放粗到平台段——否则配对/统计套利/风险模型的相关性被系统性低估。A 股的撮合节奏（集合竞价、3 秒快照、涨跌停停摆）会加剧异步与 tick 效应，Epps 偏差可能更重。
- **日内剖面与回测陷阱**：用"平均日内波动率/成交量剖面"排程或当特征时，记住 **mean ≠ path**（Fig A.27）——单日会大幅偏离均值，回测里用平滑剖面会低估执行风险与因子方差。
- **聚合层一致性**：在指数/组合层标定的回归（如 波动率~价差、成本~规模）**不能**等同于成分层加权平均（Fig A.28 聚合偏差）——构造组合级因子或成本曲线时，应明确在哪一层标定、避免跨层套用。
- **数据集稳健性检验**：propagator 给的"G 应跨数据集稳定，否则是参与者结构变了"提供了一个**因子/模型漂移诊断**思路——把同一冲击核拿到不同时段/不同票上验证，漂移即提示 mix-of-investors 变化（可作市场状态/风格切换信号）。
