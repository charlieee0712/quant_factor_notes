# 微观结构 → 回测成本模型 & L2 因子理论根基

> 综合 Market Microstructure in Practice（导论+§3.2/§3.3+附录A）+ Factor Zoo 第6章 + 《因子投资》§7.3 · 生成 2026-06-30

把整本微观结构的机理收敛到两个落点：**回测该用什么成本模型**，以及**微观结构噪声为何是 L2 因子的理论根基**。一句话纲领——流动性不是一个标量（固定 bid-ask spread / 固定 bps 滑点），而是一条**随委托量上升的 round-trip 成本曲线 c(Q)**（MMiP 导论）：买入沿 ask 逐档累加、卖出沿 bid 累加，单位成本 `c(Q)=P_buy(Q)−P_sell(Q)` 关于 Q 单调递增且凹，Q→0 时退化为盘口价差。回测按固定费率扣滑点会系统性低估大单成本、高估策略容量。

## 一、回测该用什么成本模型

把交易成本拆成**显性 + 隐性 + 时机**三类分开建模（《因子投资》§7.3.5 + MMiP 切片理论），缺一不可。

**显性成本（按成交额固定比例）**
- 佣金 + 规费（券商实际费率）+ 印花税（A股卖出单边千一）；
- **半价差**（bid-ask/2）：中点到成交价的确定性损耗。有 L2 报价直接取 (Ask−Bid)/2；无 L2 时用 Roll 价差 `2·√(−cov(ΔPₜ,ΔPₜ₋₁))`（式6.6，源于买卖价差弹跳致负自协方差）或 Corwin-Schultz 高低价估计（式6.5，仅需日频高/低价）反推。有效价差 ES（式6.1/6.3）比报价价差更真实（大单逐层吃单、有隐藏单）；且 **ES ≈ 已实现价差 RS + 价格冲击**，可据此拆出逆向选择分量。

**隐性成本 = 市场冲击**，遵循**平方根律**（MMiP §3.2，式3.1 标定 γ≈0.5）：单位冲击 **Δ ≈ Y·σ·√(Q/V)**，对参与率与久期均**凹**（边际冲击递减）；Y 按个股标定、σ 为波动率、Q 为待交易量、V 为 ADV；冲击还随执行久期幂律上升（拖得越久越被识别跟随）。务必区分**临时冲击**（约 1 日内随弹性 resilience 衰减回补）与**永久冲击**（留存、进持仓成本）。三篇元订单实证（Bacry / Brokmann / Waelbroeck-Gomes 2015）收敛结论：**剔除 alpha 与订单流自相关后，元订单自身几乎不留永久冲击，残余主要由系统性 beta 解释**——故有真 alpha 的长持有期策略别用单边冲击高估成本，否则把 alpha 误记为成本。去偏两法可搬进 TCA：①逐日扣"执行后每天自身参与率按平方根律产生的临时冲击"（订单流有强自相关）；②用 CAPM 式系统/特质分解（式3.2）先净化大盘 beta、再看特质残差。

**时机成本（timing / opportunity）**：大单切片须在"快冲→高冲击"与"慢拖→高市场风险漂移"间权衡（MMiP 最优交易理论）。拖延期间价格随机漂移的暴露即时机成本，与冲击此消彼长，是执行节奏（IS/VWAP/PoV）优化的另一半。

**进优化器的成本函数模板**（Δωᵢ:=ωᵢ−ωᵢ⁰，相对调仓前权重）：
```
TC(ω) = Σ cᵢ|Δωᵢ|  +  Σ dᵢ(Δωᵢ)²        (线性式7.79 + 二次式7.80)
目标:  max f(μ,Σ,θ,ω) − γ_TC·TC(ω)        (式7.81)
```
- **cᵢ（线性项）** ≈ 佣金率 + 印花税率 + 半价差率；
- **dᵢ（二次项）** ≈ 市场冲击，是平方根律 √(Q/V) 在优化器里的**凸近似**（√ 不可直接微分，用二次凸项代理），须随个股流动性（ADV / 换手分位）**分档标定**，大单小单不同价；
- **γ_TC** 做敏感性扫描、与净夏普曲线对照；
- **关键：成本进目标函数同步权衡，而非"先满仓再事后扣"**，并与换手率约束 `Σ|Δωᵢ|≤Φ`（式7.73）双管。

**A股特殊项（不可成交即剔除）**：涨跌停 / 一字板 / 停牌当日不可成交；T+1 使"计划换手"与"可成交换手"产生缺口；tick（最小变动价位 0.01 元）须按真实档位撮合，低价股 tick 占比大、价差类指标需价位归一；融券难 → 卖空约束 ω≥0 近乎常态。

## 二、微观结构噪声为何是 L2 因子的理论根基

**噪声从哪来**：观测价 = 真实有效价 + 噪声（X = M + η，MMiP A.13）。三源——**买卖价差弹跳**（价格在 bid/ask 间来回跳，Roll 测度负自协方差之源）、**离散 tick**、**异步交易**。

**噪声怎样污染低频估计（去偏对象）**：
- **signature plot**：已实现方差 = 真实增量² + 噪声² + 交叉项，噪声项 ≈ 2·Var(η)·(T/Δ)，采样步长 Δ→0 时随采样点数线性放大而**发散**——已实现波动率随采样频率漂移，故存在噪声与离散化折中的**最优采样频率**（signature plot 由陡降转平的拐点附近）；补救用两尺度 / realized kernel / pre-averaging。
- **Epps 效应**：已实现协方差需同步数据，而成交异步，使高频相关随采样间隔缩短而**塌缩归零**（Epps 1979，主因异步，含 lead-lag / tick）；高频估相关 / β / 协方差矩阵系统性偏低，补救用 Hayashi-Yoshida 或放粗到平台段。A股集合竞价、3 秒快照、涨跌停停摆会加剧异步、Epps 偏差更重。

**但正因有噪声，L2 盘口/逐笔才含可提取 alpha**——噪声编码了做市商定价与订单流信息：
- **Kyle's λ**：净订单流 → 价格冲击的斜率，信息不对称代理，深度 ≈ 1/λ（Kyle 1985）；
- **PIN / VPIN**：知情交易概率；VPIN 用成交量时钟等量分桶 + BVC 软分类，由订单失衡 `Σ|V^B−V^S|/(nV)` 预警订单流毒性（式6.24–6.27），比 Lee-Ready tick rule 在 A股 L2 更稳；
- **有效 / 已实现价差**（式6.1/6.4）：成交价偏中点的真实成本 / 做市商真实赚取；
- **订单流不平衡 OFI / 盘口失衡**：一档失衡对短期价格方向有预测力，但幅度常 **< 1 tick** → 定位执行择时 / 降本信号、非独立套利 alpha（MMiP §3.3 队列反应模型）；
- **三类事件强度**：insert / cancel / trade 建成泊松强度 λ⁺/λᶜ/λˣ，但**齐次 Poisson 复现不出真实队列规模分布**，必须按本档及邻档规模**条件化**（Queue-Reactive Model I/II/III，Huang 2015）；**A/R 比** `λ⁺/(λᶜ+λˣ)` 作逐档净流入状态量（一档天然<1、二档天然>1，靠"二档晋升一档"机制达遍历稳态），裸订单簿信号天然均值回复、需 reset / Hawkes 记忆修正；
- **订单流聚簇强度**：成交 / 挂撤单建成 **Hawkes 自激点过程** `λ(t)=λ₀+Σα·e^{−β(t−τᵢ)}`，**分支比 n=α/β** 度量内生性 / 拥挤度，n→1 预警自反性过高、脆弱易闪崩；Hawkes 能在精细尺度复现噪声 / signature plot / Epps、大尺度收敛布朗运动；
- **传播子 propagator**：成交符号 ε 有长记忆自相关（拆单 + 羊群），冲击核 G 被精调到正好抵消可预测订单流、使价格近鞅——解释了冲击为何衰减而非永久；可去卷积 `Ĝ=Ĉ⁻¹Ŝ` 反解冲击核，校准临时/永久衰减；
- **弹性 RES**（Hua 2020）：开盘 30 分钟 vs 当日其余收益的协方差，区分永久 vs 暂时价格变动；
- **Amihud ILLIQ**（式6.7）：每单位成交额的价格冲击，纯日频代理，A股横截面稳健。

这些因子能解释横截面收益，根因在**流动性溢价**：低流动性资产要求更高预期收益（Amihud-Mendelson 1986；Pástor-Stambaugh 2003 证被定价），但市场压力期因流动性枯竭剧烈回撤（liquidity spirals，Factor Zoo §6.2）——L2 因子既是 alpha 来源、也是需在回测中压力测试的风险敞口。

→ **噪声既是"敌"又是"友"**：做高频波动率 / 相关性时它是估计偏差（signature plot 发散、Epps 塌缩），**回测里要去偏**；做 L2 因子时它是信息源——价差弹跳里有价差、订单失衡里有方向、到达强度里有知情，**做因子时要利用**。这正是 L2 因子区别于低频量价因子的理论分水岭：低频世界要消灭的噪声，在盘口逐笔层面恰是 alpha 的载体。

## 三、对回测框架的直接落点（checklist）

- [ ] **成本四件套**：显性（佣金+印花税）线性 + 半价差线性 + 冲击二次 + 时机成本；最小版先上线性(7.79)，进阶上二次(7.80)。
- [ ] **冲击系数 dᵢ 用平方根律 Y·σ·√(Q/V) 标定**，按 ADV / 换手分位分档，禁止全市场一刀切；用 round-trip 曲线 c(Q) 校验大单超线性总成本。
- [ ] **成本进目标函数 −γ_TC·TC 同步权衡**，γ_TC 敏感性扫描；换手约束(7.73) + 成本惩罚双管。
- [ ] **区分临时 / 永久冲击**：长持有期用永久（去偏后≈系统性 beta），别用单边冲击高估、误把 alpha 计成本。
- [ ] **A股不可成交过滤**：涨跌停 / 一字板 / 停牌剔除，T+1 占用，tick 档位撮合，卖空 ω≥0。
- [ ] **L2 因子采样频率按 signature plot 拐点设**；异步协方差用 Hayashi-Yoshida（或拉长窗口）去 Epps，VPIN 用成交量时钟等量分桶；强度类因子固定并记录采样起点口径。
- [ ] **队列因子按 tick 档位分别检验**（结论建立在大 tick 股），按队列规模分桶估 λ、AES 归一；imbalance 当执行信号而非套利 alpha。
- [ ] **因子侧从噪声提取 alpha**：Kyle's λ、PIN/VPIN、有效/已实现价差、OFI/盘口失衡、Hawkes 分支比、弹性 RES——数据落 L2（通联 / 同花顺两套，ClickHouse.56，见 DATA_SOURCES.md），回测落 47 factor_library。
