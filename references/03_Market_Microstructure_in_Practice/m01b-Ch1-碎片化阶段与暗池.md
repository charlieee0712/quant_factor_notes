## 本块讲什么

承接 §1.1 对欧洲 MiFID 后市场碎片化（fragmentation）的历史叙事，本块把碎片化进程归纳为**三个阶段**并逐一展开机理，核心方法论是：

1. **Phase 1（挑战者登场）收尾**——BATS Europe、NASDAQ OMX Europe、Chi-X 三家新平台的成败对比（图 1.4–1.6）。
2. **Phase 2（收敛 / Convergence towards a European offer）**——引入**熵（entropy）**这一统计热力学工具来**量化**碎片化程度，进而定义可跨股票/指数/时间比较的标量指标 **FEI（Fragmentation Efficiency Index，碎片化效率指数）**（图 1.7–1.8、表 1.2）；并指出"碎片化"有两层含义：成交额在场所间的分布 + 接入流动性的技术手段（通信协议）也是碎片化的。
3. **Phase 3（BCN 与暗池出现）**——HFT 兴起 + 金融危机双重压力下，券商内部撮合网络 **BCN（Broker Crossing Network）** 与**暗池（dark pool）**为何产生、如何演化为受监管的 MTF 暗池，以及 MiFID 2 对暗池的"双重上限（double cap）"。
4. **大/中/小盘碎片化差异**——用对数-对数散点（图 1.9–1.10）揭示碎片化与市值/成交额的乘法关系，配 **log-linear 回归**给出可检验的幂律，并从**借券成本/自由流通(free float)**与**相关品种对冲**两条经济机理解释为何大盘更碎片化。
5. **从竞争到合并（From competition to merger）**——平台并购潮（LSE 收 Turquoise、BATS 并 Chi-X、ICE 收 NYSE Euronext 等）的全球时间线。

## 方法论 / 机理要点（重点·展开）

### 1. 碎片化三阶段总览（叙事框架）
作者用"演化阶段"而非"事件流水账"来组织：
- **Phase 1**：MiFID（2007/11 生效）放开后，新交易场所（MTF）涌入与历史交易所竞争，市场份额被切碎。
- **Phase 2 收敛**：经历爆发式扩张后，少数赢家胜出，市场进入"趋同"——盘中量价节律、技术接口逐步收敛；此阶段催生了**量化碎片化的需求**（→ 熵/FEI）。
- **Phase 3**：HFT 改变了各资金池（pool）的"流动性成色"，倒逼券商开 BCN、各方上暗池，交易开始大量发生在**不可见的池子**里。
- 之后（超出"阶段"框架）是**合并潮**：场所从"多而小"走向"少而大"。

### 2. Phase 1 收尾：挑战者的成败（technology is not enough）
- **NASDAQ OMX Europe（2008/09–2010/05）**：把美国经验照搬过来，主打 MTF 侧的 **SOR（Smart Order Router，智能订单路由）**，接近 MiFID 的"trade-through rule view（穿价规则视角）"。**但失败**——欧洲的路由由"**best execution policies（最优执行政策）**"驱动，而非美式"blink routing（按最优价瞬时跳转）"；该 MTF 全周界市占**几乎从未越过 1%**（图 1.6b）。教训标题即"**当技术不够用时**"：把美国微结构假设直接移植到欧洲制度环境会失败。
- **Chi-X**：最成功的挑战者（图 1.6a，FTSE 60 段市占冲到 ~30%+）。
- **BATS Europe**：次于 Chi-X 的有力挑战者（图 1.5a）。
- 2010 年末格局定型，2012 年主要玩家：**历史大所**＝LSE Group、Euronext、Deutsche Börse；**挑战者**＝BATS Europe、Chi-X。

### 3. Phase 2 核心方法论：用熵度量碎片化 + FEI
**思想**：把"合并后的整体市场（consolidated market）"看作一个**系统**，碎片化＝系统的"无序度（disorder）"。全部成交集中在单一场所＝完全有序（熵=0）；成交越分散，熵越大。这是从**统计热力学**借来的（作者点名：布朗运动、Black–Scholes 也源出于此）。

**两个极端**：若某股有 5 个可选场所，
- **最大碎片化**＝各占 1/5（均分 → 熵最大）；
- **最小碎片化**＝全部在 1 个场所（熵最小）。

**Kolmogorov–Shannon 熵**（原始、未归一）：
$$\text{Entropy} = -\sum_{n=1}^{N} \mathfrak{M}_n \log \mathfrak{M}_n$$
其中 $\mathfrak{M}_n$＝第 n 个场所的市占。书中例子：分布 70/20/10 → 熵 ≈ **0.34**（按 log₁₀ 计算可复现：−(0.7·log0.7+0.2·log0.2+0.1·log0.1) ≈ 0.348）。

**问题**：原始熵的取值随场所数 N 变化，不可横向比较 → **归一化**。用该 N 下的**最大可能熵 log(N)** 作分母：
$$H = -\frac{1}{\log N}\sum_{n=1}^{N}\mathfrak{M}_n \log \mathfrak{M}_n$$
这个归一化熵作者称为 **FEI（Fragmentation Efficiency Index）**。它度量的是"**相对于在现有场所数下本可以达到的碎片化，实际碎片化到了几成**"——即"碎片化效率"。70/20/10 三场所 → ≈ **72–73%**（书中 p.52 正文写 72%，表 1.2 写 73%，系四舍五入差异 [数值存疑·属舍入]）。

**关键设计选择 / 坑**：
- **可比性**：归一化后 FEI 是 0–100% 的标量，可跨股票、跨指数、跨时间比较（图 1.7 把单只股票 CAGR.PA 与其指数 CAC40 放一起，曲线几乎重合 → 证明该指标在个股与指数层面一致捕捉碎片化）。
- **"分母里的 N"很敏感**：同样的实际成交分布 70/20/10，若把"可用场所数"算作 3，FEI=73%；若算作 4（即 70/20/10/0），FEI 骤降到 **58%**。所以**纳入哪些场所进 N，结论会变**。
- **纳入阈值**：把一个新 MTF 计入 FEI 的门槛设为**市占 ≥ 1%**（低于此忽略，避免长尾噪声把 N 撑大）。
- **物理对应保真**：因为 FEI 复刻了熵的物理灵感，**重大碎片化事件会在 FEI 曲线上留下印记**（图 1.8 中 Turquoise 流动性协议于 2008/03 终止，可见对其他指数的冲击）——即该指标对结构性事件有可读的响应，可用作事件研究的载体。

### 4. "碎片化的是什么？"——两层含义（易被忽略的第二层）
- **第一层（上面 FEI 度量的）**：成交的"欧元"如何在各场所间切分（traded euros split between venues）。其中一种市占定义＝在欧洲市场成交的"1 欧元"平均有多大比例落在某个特定场所。
- **第二层（同样重要）**：**接入流动性的手段也碎片化**。对参与者而言，与每个交易平台建立连接都有成本，因此**通信协议的收敛**至关重要。MiFID 初期欧洲在三类协议上高度碎片化：
  - **应用层协议**：XDP、ITCH、EBS、Pitch、EMDI、RWF；
  - **网络层协议**：组播 UDP vs TCP（且各家组播 UDP 的初始化步骤、丢包/乱序处理实现各异）；
  - **压缩协议**：FAST vs ZLIB vs 不压缩。
  - 后果：很难做一个**通用 feed handler（行情解析器）**同时对接大多数场所（引 eTrading Software 2012 白皮书）。
  - 启示：碎片化的成本不只在"流动性被切碎"，也在"**为接入每个场所付出的固定工程/连接成本**"——这是后面 Phase 3 券商技术成本飙升、被迫开 BCN 的伏笔。

### 5. Phase 3 核心机理：HFT → BCN → 暗池（本块重点）
**5.1 技术集中（像移动运营商网络）**
碎片化的演化形似移动运营商：**少数公司拥有"硬件"（技术），网络上跑着大量（含细分）的"业务"**。LSE 集团是典型——一套技术（Millennium）支撑多个 orderbook：常规 LSE、Turquoise 的 **Lit 与 Dark** 两套（覆盖含米兰所在内的全英股）。BATS Europe 与 Chi-X Europe 共用同一技术跑四个 orderbook（两个场所各自的 Lit + Dark）。NYSE Euronext 用同一技术跑 NYSE、Euronext 巴黎/里斯本/布鲁塞尔/阿姆斯特丹及 NYSE Arca。**要点：同一技术栈下"明池(Lit)+暗池(Dark)"成对出现**。

**5.2 信任与盈利双重压力**
- **HFT 兴起**（部分是自营 prop、部分是高科技做市商）+ 欧洲流动性因经济原因整体下降 → **传统投资者对价格形成过程(price formation process)的信任被削弱**。Flash Crash 强化了"某些 HFT 赚取了买卖价差(bid–ask spread)的一部分，这部分必然来自其他参与者"的模糊认知（引 Menkveld 2010）。
- **金融危机**让传统券商盈利承压：为接入所有新 orderbook、为把客户单**高效路由**到合适场所组合、并能**事后举证最优执行**，**技术成本大涨**。

**5.3 BCN 为何诞生——Akerlof"柠檬市场"机理（关键论证）**
- Phase 2 造出了多个**成分不同的流动性池**，每个池子托管的是"特定投资者组合"下的单。
- 套用 Akerlof（1970）"**market for lemons（柠檬市场，信息不对称下劣币逐良）**"：MTF 提供的更像"**二手流动性(second-hand liquidity)**"，而历史大所池子里"**好货(cherries)/一手流动性(first-hand liquidity)**"更多。
- 执行券商（既因碎片化导致**固定成本上升**，又因危机导致**管理资产规模(AUM)下降**）被诱导去开自家池子，主打"**100% 一手流动性**"。但只有少数券商付得起配套开销（技术、监管所需人手、营销）→ 于是诞生了首批 **BCN（Broker Crossing Network，券商撮合网络）**。
- **BCN 的隐患**：它不仅托管"natural liquidity（自然/客户流动性）"，**有时也掺入拥有该运营券商的投行的自营(prop)流**——这就埋下利益冲突（自营台能否窥见客户流）。

**5.4 从 BCN 到受监管的 MTF 暗池**
- 历史所与 MTF 怕丢份额 → MTF（及 Euronext 等部分历史所）**主动上暗池**。
- 部分券商把 BCN **转制为 MTF 暗池**，以向客户保证"**没有自营台能太清楚地看到来单流**"（用监管身份背书来对冲 5.3 的利益冲突质疑）。
- **MiFID one 最后阶段三特征**：①技术手段集中到少数人手里；②强势 BCN 崛起；③部分 BCN 转为受监管(MTF)暗池。

**5.5 MiFID 2 的暗池"双重上限(double cap)"**
为压制暗池对价格发现的侵蚀，MiFID 2 对股票暗池市占设双帽：
1. **单池单股帽**：任一暗池在某只股票上的成交额占比不得超过某阈值（"似乎是 4%"）；
2. **全暗池单股帽**：所有欧洲暗池在该股上的合计成交额不得超过另一阈值（"应为 8%"）。
- **执行机制**：很可能按**年度**计算；任何暗池突破帽线，**未来六个月**须在相关股票上停止交易。
- 启示：监管把"暗池占比"作为可量化、可处罚的硬约束 → 暗池占比本身成为一个需要被持续监测的指标。

### 6. 大/中/小盘碎片化差异 + 可检验幂律
**6.1 现象（图 1.9、1.10）**：碎片化路径在所有市值段上一致，但**大盘最先、最深地被碎片化**，分三步：①2009 年中前，大盘显著比中小盘更碎片化；②2009 年中后，小盘主市场份额的侵蚀放缓；③2012 年中，中盘与大盘差异变得很小（趋同）。

**6.2 两条经济机理（为何大盘更易碎片化 / 更易被 HFT 提供流动性）**：
1. **借券/自由流通(free float) 的不对称**：要在**卖方(ask)** 提供流动性而又不正式做市，常需**借券**。借券成本随**自由流通股数**增大而下降（大盘借券便宜）。于是流动性供给存在**不对称**：在**买方(bid)** 挂单是"免费"的，但在**卖方**可能要借券——这与高频做市商的效用函数不符。2011/08 中欧洲卖空禁令期间，部分 HFT 申请接近 MTF 做市商的身份以便在卖方提供"**naked（裸）流动性**"；MiFID 1 缺欧洲统一做市商身份，留下监管漏洞。
2. **相关品种对冲库存**：HFT/做市商可买卖**相关金融工具**来对冲不平衡库存。典型：用**卖出股指期货**对冲指数现货成分股的多头库存。**期货买卖价差通常小于现货**，对冲方便。**因此只有"属于指数成分 / 市值大到成为期货标的"的股票，才容易承受不平衡库存** → 这类（大盘）股票更容易获得 HFT 的流动性供给，也就更碎片化。

**6.3 log-linear 回归（可检验的乘法关系）**：把 2011 年每一天配成对：(1) 某市值段指数上 MTF 的市占，(2) 当天该指数的成交额(turnover)。图 1.10 用**对数-对数坐标**散点，揭示乘法（幂律）关系：
$$\text{Market Share of MTFs} = \frac{b_i}{\text{Turnover}^{\gamma}} + \epsilon$$
- $b_i$＝每个指数（理论上可细到每只股票）特有的"截距"；
- $\gamma$＝**universal（与指数/个股无关的）通用斜率**；
- $\epsilon$＝噪声项（捕捉个股偏离幂律的外生因素）。

**6.4 碎片化对流动性的"双重"依赖（重要结论）**：
- **截面看（比两只不同股票）**：市值越大 → MTF 市占越大 → 碎片化随**市值**上升；
- **时序看（同一只股票比不同交易日）**：当日成交额越大 → MTF 市占越小 → 碎片化在**高成交日下降**。
- **高成交日机理**：高量日历史大所反而抢到更多成交，因为 ①"**race to liquidity（奔向流动性）**"——流动性紧张时投资者回流到历史大所以确保成交；②高量日里在场的投资者交易流程更不成熟（更倾向用主市场）。

### 7. 从竞争到合并（merger wave）
快速扩张与收敛后，趋势转向**场所整合**——"少而大"以在残酷竞争中存活并摊薄技术投入。时间线（含技术迁移代价）：
- **LSE 收购 Turquoise（2011/02）**；同月 LSE 主英股 orderbook 迁到 **Millennium IT** 平台（Turquoise 早 5 个月迁移）。新技术：订单确认延迟降至 **126 微秒**、处理容量潜在提升约 3 倍；但 Turquoise 曾于 2010/10/04（周一）宕机。
- **BATS 并购 Chi-X Europe**：2010/12 进入排他谈判，2011/06 英国 OFT 提交竞争委员会，2011/11 BATS 完成收购。技术迁移同样踩坑——选用 BATS 技术后，Chi-X 在多年无大故障后接连宕机；两所 2011/12 与 2012/01 均现技术问题。
- **Deutsche Börse 与 NYSE Euronext**：2011/02 进入深入谈判；NASDAQ OMX Group + ICE 2011/04 出竞争性报价被拒；2011/07 中德所股东批准；但**欧盟委员会以衍生品业务潜在垄断为由否决**。
- **ICE 收购 NYSE Euronext（2013/01）**：主要冲着衍生品业务；合并后 **Euronext 的股票现货业务独立**出来。
- **全球并购潮**：LSE 与加拿大 TMX 拟 2011/06 末合并未果（未达三分之二多数）；Maple Group（本土银行财团）两度被拒后于 10 月底获 TMX 董事会支持。南美（智利/哥伦比亚/秘鲁）2011/05/30 设单一接入点（首例合作）。日本：Chi-X 市占从 2011/01 的 0.8% 升至 2011/06 的 1.7%，东证(91.2%)与大阪所(5.3%)2011/11 达成合并。新加坡所收购澳交所被澳财政部 2011/04 否决。**CBOE 于 2017/02 收购 BATS(Chi-X) Global Markets**，形成大型跨洲平台。

## 关键公式 · 图表 · 定义（图表均转述其传达内容）

**公式**
- 原始熵：$\text{Entropy} = -\sum_n \mathfrak{M}_n \log \mathfrak{M}_n$（$\mathfrak{M}_n$＝第 n 场所市占；70/20/10 → 0.34，按 log₁₀）。
- 归一化熵 = FEI：$H = -\frac{1}{\log N}\sum_n \mathfrak{M}_n \log \mathfrak{M}_n$（除以最大熵 log N；纳入门槛市占≥1%）。
- MTF 市占幂律：$\text{Market Share} = b_i/\text{Turnover}^{\gamma} + \epsilon$（$b_i$ 指数特有截距，$\gamma$ 通用斜率）。

**表 1.2（FEI 典型取值）**——核心读法：均分=100%（最碎片化）；同一实际分布下"可用场所数 N 越大、FEI 越低"。

| 市占分布(%) | FEI | 市占分布(%) | FEI |
|---|---|---|---|
| 25/25/25/25 | 100% | 70/20/5/5 | 63% |
| 50/50 | 100% | 64/24/5/5/1 | 60% |
| 25/25/25/25/0 | 86% | 70/20/10/0 | 58% |
| 70/20/10 | 73% | 80/10/5/5 | 51% |

（对照：70/20/10 当 N=3 → 73%，同一分布写成 70/20/10/0 即 N=4 → 58%，凸显"分母 N"的影响。）

**图 1.4（Turquoise 盘中量曲线 vs 其他欧洲场所；a=2009 初、b=2012）**：三条曲线 Euronext / Turquoise / Chix，纵轴＝该时点占全日量比例(0.4%–2.4%)，横轴＝09:30–17:30。**(a) 2009**：三场所盘中节律**明显分化**（形状各异，Chix 上午偏低，15:30 美股开盘附近有跳升），说明碎片化早期各场所量价行为不同步。**(b) 2012**：三条曲线**高度重合**，共同呈经典 **U 形日内季节性**（开盘高、约 13:00 午盘低谷、尾盘抬升至 17:30 收盘峰）。**传达**：2009→2012 各场所盘中量节律**收敛**——是 Phase 2"趋同"的微观证据。

**图 1.5（BATS Europe (a) / Turquoise (b) 在不同流动性段的市占）**：分段 CAC NEXT 20 / FTSE 60 / CAC40，时间 2007/11–2010/10。两图均显示市占从 ~0 爬升：**英股 FTSE 60 段碎片化最快最高**（BATS 到 ~16–18%，Turquoise 早期到 ~13%），法股 CAC40/CAC NEXT 20 较低（~5–8%）。**传达**：挑战者在不同市场进度不同，UK 比法国更早被碎片化。

**图 1.6（Chi-X (a) / NASDAQ OMX MTF (b) 的市占）**：**Chi-X**——FTSE 60 段冲到 ~30–35%，CAC 段 ~15–25%，是最成功挑战者。**NASDAQ OMX MTF**——仅在 2009 短暂达 ~3–6% 即崩回近 0，全周界**从未越 1%**。**传达**：成败对照，呼应"技术不够用"。

**图 1.7（CAC40 与 Crédit Agricole 股 的归一化熵/FEI）**：两条曲线 Sliding CAC40 与 Sliding CAGR.PA，从 2008/01 的 ~10% 升到 2011–2012 的 ~70%，**几乎重合**。**传达**：单只个股的碎片化紧贴其所属指数 → 碎片化是全市场现象，FEI 在个股/指数两层一致有效。

**图 1.10（MTF 市占 vs 成交额，对数-对数散点，2011 每日一点）**：三市值段形成**三个分离的点簇**——SMALL 在低成交额(~10⁹)、MID 居中、LARGE 在高成交额(~10¹⁰)，且 LARGE 簇整体 MTF 市占更高。每簇内有一条**向下倾斜**的拟合线。**传达**：①截面——市值越大 MTF 市占越高；②簇内（时序）——当日成交额越大 MTF 市占越低；log-log 直线关系＝**乘法/幂律**，正是 §6.3 回归式的依据。

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **碎片化作为可比标量（FEI/熵）**：归一化熵的思路是把"任意离散占比向量"压成 0–1 的有序度标量，并用 log N 归一以便横向比较。这套方法可直接迁移到因子研究里任何"分散度/集中度"度量——例如**成交在交易所/券商/订单类型上的分布**、**持仓集中度**、**资金流在板块间的分散度**，都能用 $-\frac{1}{\log N}\sum p\log p$ 做成可跨期比较的因子，并继承"对结构性事件敏感、留痕可读"的优点。注意两坑：**纳入阈值**（低占比长尾要截断，否则 N 被噪声撑大）与**分母 N 的口径敏感**。
- **盘中 U 形与 VWAP/执行排程**：图 1.4 的"各场所日内量曲线 2009 分化→2012 收敛到 U 形"，提示**日内成交季节性**是执行算法（VWAP/TWAP 的量曲线、参与率约束）的基础输入；做成本模型或冲击因子时，需用**该标的当前的日内量分布**而非历史平均，且要意识到不同场所/时期的曲线可能不同步。
- **可检验幂律（power law）模板**：$\text{MTF 市占}=b_i/\text{Turnover}^\gamma$ 给了一个"**截距随个体变、斜率通用**"的回归范式——在因子检验里对应"**固定效应(个体截距) + 共同弹性(通用 β)**"的面板设定；用 log-log 散点先目检乘法关系、再做 log-linear 回归估通用弹性，是稳健的建模顺序。其结论"高成交日碎片化下降 / 大市值碎片化上升"也提醒：**流动性的截面效应与时序效应可能方向相反**，建因子时必须区分 cross-section 与 time-series。
- **暗池 / 场所选择与执行成本（L2 相关）**：Phase 3 的 BCN/暗池机理（一手 vs 二手流动性、自营流是否掺入、MiFID 2 的 4%/8% 双帽）直接关系**执行场所质量**与**逆向选择成本**——在有 L2/逐笔的市场做执行分析时，"成交落在明池还是暗池""暗池占比是否触帽"可作为**交易成本/对手方毒性(toxicity)** 的特征。
- **A 股适配**：A 股是**单一交易所集中竞价**（沪/深各自挂牌、无跨所碎片化、无 MTF/暗池），本块的"跨场所碎片化"叙事在 A 股**不直接成立**——这恰是重要对照：A 股不需要 SOR/跨场所最优执行，价格发现集中、无明暗池分流问题。但**方法**可迁移：熵/FEI 可用于度量 A 股**成交在席位/营业部、或资金在板块的分散度**；幂律回归框架可用于 A 股的"流动性 vs 成交额"关系；HFT 提供流动性的**借券不对称（free float / 卖空约束）** 机理在 A 股（融券稀缺、T+1）下更极端，是解释 A 股多空流动性不对称、卖空因子的有用直觉。

## 术语与存疑

- **MTF**（Multilateral Trading Facility，多边交易设施）：MiFID 下与历史交易所竞争的新型受监管交易场所。
- **SOR**（Smart Order Router，智能订单路由）；**trade-through rule（穿价规则）** vs **best execution policy（最优执行政策）**：美式按最优价瞬时跳转 vs 欧式按执行政策综合判定。
- **Entropy / FEI**（熵 / 碎片化效率指数）：见上式；FEI＝归一化熵。
- **Lit pool / Dark pool（明池/暗池）**：盘前可见报价 vs 不可见；**BCN**（Broker Crossing Network，券商内部撮合网络）。
- **first-hand / second-hand liquidity（一手/二手流动性）**、**cherries / lemons**：Akerlof 信息不对称隐喻——好货 vs 次货。
- **free float（自由流通）**、**naked liquidity（裸流动性）**、**race to liquidity（奔向流动性）**。
- **double cap（双重上限）**：MiFID 2 暗池单池 4% / 全暗池 8%（书中措辞为"似乎/应为"，[数值存疑·属监管草案口径]）。
- **存疑**：70/20/10 三场所 FEI 正文写 72%、表 1.2 写 73%，系四舍五入差异 [数值存疑·属舍入]，机理无歧义。
