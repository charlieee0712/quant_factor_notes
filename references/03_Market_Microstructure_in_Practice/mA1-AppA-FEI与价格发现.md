## 本块讲什么

附录A 是全书的"玩具模型作坊"：每个微观结构现象都配一个**最小可解析模型**，目的不是预测而是**把机理的因果链条讲透**。本块覆盖前四节：

- **§A.1 从熵到 FEI**——把"流动性在 N 个交易场所(venue)有多分散"用信息熵压成一个 [0,1] 标量。
- **§A.2 信息搜寻与价格发现(price discovery)**——Foucault et al. (2009) 模型，解释高频做市商(HFMM)、技术投入与挂单/吃单费率(maker/taker fees)如何共同决定价格形成过程(PFP)的效率，并给出 maker-taker 返佣结构的理论解。
- **§A.3 SOR 驱动的自然碎片化**——智能订单路由(SOR)如何把垄断"元市场"推向碎片化；强化学习式拆单 + 市占反馈耦合成随机动力系统。
- **§A.4 Kyle 做市模型**——一期版本，知情交易者 + 噪声交易者 + 做市商共存，导出经典 **Kyle's lambda λ = σ_p/(2σ_n)**。

贯穿主线：**碎片化 / 流动性分散是费率、技术与路由算法的内生均衡结果**，而价格冲击(market impact)是做市商对"逆向选择"的线性自保反应。

## 方法论 / 机理要点（重点·展开）

### 1. FEI：用熵把碎片化"无量纲化"（§A.1）

**思想迁移**：物理里熵度量物质的无序度（热力学第二定律：孤立系统熵不减；气体在房间里会自发均匀弥散，弛豫过程中熵上升）。信息论里熵度量任意映射/消息的有序度（Shannon 1948）；统计里用熵发现变量关系——系统 (Y,X) 比单独 Y 更"有序"就说明 X 能解释 Y（回归的熵视角）。

**类比落点**：若微观结构服从"物质定律"，流动性会像气体一样自发、等成本地铺满所有可用场所，均匀填充。于是**把当前流动性分布的熵和它的最大可能熵相比，就能量出微观结构离"100% 弛豫态"还有多远**。建模假设：N 个场所，在弛豫态下流动性能以相同成本流向任一场所，"理想"分布是每个场所各 1/N。

**为什么用归一化的熵而不是直接用方差/HHI**：熵对"完全集中"和"完全均匀"两个极端有干净的端点值（0 与 log N），除以 log N 后得到与场所数 N 无关、可跨时间/跨标的比较的 [0,1] 标尺。这正是把"分散程度"做成可监控时间序列 F(t) 的关键——只跟一条标量曲线，而不是 N 条市占曲线。

**坑与正确做法（新场所的机械效应）**：N 增加（新场所开市）会机械地抬高分母 log N，即便新场所当天零成交，FEI 也会从 H/log N 掉到 H/log(N+1)。书给出修正系数 𝔉(t+1)=[log N/log(N+1)]·𝔉(t)，并建议**经验规则：只把市占≥1% 的场所计入 N**（否则空壳场所会污染指标）。量级感：N=3 时 log N/log(N+1)≈80%，N=6 时≈90%——即新增场所对既有 FEI 的"稀释"在场所数小时相当可观。

**与最优交易的连接**：FEI 越低（越集中）→ 微观结构"低效"，每个池子提供的流动性不等价，下单前必须精挑场所、谨慎做智能路由；FEI 越高（越均匀）→ 各流动性来源在参与者眼中"等价"，碎片化是"高效"的，此时**交易员无需在 smart routing 上花太多心思**，因为各来源已被市场隐含定价为等值。

### 2. 价格发现：费率与技术的内生均衡（§A.2，Foucault et al. 2009）

**问题动机**：MiFID 后欧洲冒出三类争论——HFMM 的相对角色、其活动的负外部性（比别人更依赖技术）、以及费率表(fee schedule)对 PFP 效率的影响。该模型用一个可解析框架同时回答这三件事。

**双方设定**（只写买方投资者，卖方对称）：市场分成 M 个 HFMM 与 N 个长期投资者(market taker)。投资者在现价 v_0 买入、内心估值 v_0+Γ；HFMM 愿在 v_0+s·Δ 卖出（s 是其索取的**流动性溢价**，以 tick 大小 Δ 计）。双方向场所付费：吃单方(taker, 移除流动性)付 c_t，做市方(maker, 增加流动性)付 c_m。一笔成交里三方收益：HFMM 得 π_M=s·Δ−c_M，投资者得 π_T=Γ−s·Δ−c_T，场所得 c̄=c_T+c_M。

**核心创新——把"到达订单簿的时间"建进模型**：这是理解"技术投入如何改变微观结构动态"的钥匙。第 m 个做市商平均到达时间 1/μ_m，第 n 个投资者（用券商交易算法）平均到达时间 1/τ_n，按 Poisson 过程建模以得闭式解。聚合速率 μ̄=Σμ_m、τ̄=Στ_n。竞争结构：**做市商之间竞速**（最快者抢到与投资者成交），**投资者之间竞速**（最快者抢到做市商提供的流动性），但做市商与投资者之间无直接竞争。两笔成交平均间隔 (μ̄+τ̄)/(μ̄·τ̄)，单位时间成交数 R=μ̄·τ̄/(μ̄+τ̄)。

**低延迟的成本不对称**：抢速度要花钱，且 HFMM 与投资者成本不同（投资者受监管约束更重）。达到时间参数 1/t 的成本是平方型 C(1/t)=½·(系数)·T/t²（T=一个交易日典型时长），HFMM 系数 β、投资者系数 γ——**β、γ 就是两类参与者获取低延迟的"成本差异因子"**。每方平均收益 = 自身参与份额 × 单笔收益 × 成交数 − 技术成本（见公式区 Π_M、Π_T）。

**关键结论——maker-taker 返佣的理论解 (A.4)**：当各方都优化完技术投入后，均衡的吃单费 c_T* 与做市费 c_M* 的失衡有闭式刻画。解读三点：(i) Γ−c̄（投资者净收益）必然大于 s·Δ（做市商收益）；(ii) **做市商少(M 小)时，高效做市费可为负**——即给 HFMM 返佣；(iii) **做市商多(M 相对 N 大)、或投资者技术成本高于 HFMM 时，高效吃单费可为负**——即给投资者返佣。这正是现实中交易所 maker-taker 返佣结构的微观经济学解释：返佣不是促销，而是把稀缺一侧（做市能力或吃单流）补贴进来的均衡价格。

### 3. SOR 如何制造碎片化：强化学习 × 市占反馈（§A.3）

**问题与定位**：模型不追求最终均衡的精细刻画，而是抓"从垄断元市场走向碎片化"的**驱动成分**。垄断被 MiFID 打破后竞争者迅速涌现（Chi-X、Turquoise、NASDAQ OMX），初始碎片化 p(0)=(p_1,…,p_K)'（如 (80%,15%,2%,3%)）刻画各场所最初的"吸单能力"——来自费率表、做市协议、参与者连接等。

**A.3.1 SOR 内部的拆单学习**：某场所被询价时可得流动性 D_k(ω)（ω 是流动性的随机性，逐次请求都不同）。SOR 收到规模 V(ω) 的单，按拆分键 r(t)=(r_1,…,r_K)' 分配，目标是**最大化捕获的流动性** C=Σ min(r_k·V, D_k)（注意 min：分给某场所多于其可得量是浪费）。因为 D 随机，SOR 能做的最好策略是"边交易边学"(learn by trading)的**强化学习更新** (A.7)：满足率（实际被全额成交的比例）低于平均的场所被惩罚（下次少分），高于平均的被奖励（下次多分）。其不动点 (A.6) 是"各场所满足率相等"——只要时间平均≈系综平均（(V,D) 的随机律 i.i.d. 或遍历）。

**A.3.2 SOR 活动反过来改变市占**：若无人用 SOR（所有吃单都发去同一个最老场所），新场所的吸单能力就毫无用处；反之若所有 SOR 不看成交率盲目给某场所 5%，该场所很快就拿到约 5% 市占。设 a=装备 SOR 的参与者比例，市占演化 p_k(t+1)=(1−a)p_k(t)+a·r_k(t)：**市占是"被 SOR 的拆单键拖动"的**。

**A.3.3 耦合系统**：把 (A.7) 拆单学习与 (A.8) 市占演化连续化，得一组**随机微分方程** (A.9)：r 受满足率失衡驱动，p 以速率 a 向 r 收敛。约束 (A.10) 要求各场所期望"相对可得流动性"等于其市占；并设流动性服从对数正态(log-normal)，**关键附加假设：某场所提供流动性的标准差与其数量成正比**（多一个参数 ν）——这把"流动性质量/稳定性"显式建了进来。

**A.3.5 定性结论（最有洞见的部分）**：短期 SOR 想抓可得流动性，但被随机性、流动性的瞬时消失、重复挂单、隐藏单等模糊掉；最终 SOR 只能利用**超过信噪阈值**的场所间失衡——一个场所若不够"规整"（标准差大）相对其提供量，SOR 要花更久才能利用它的流动性。均衡时 p(∞)=r(∞)：收敛越快，p(∞) 越接近 p(0)；r 动得慢，则 p(∞) 可能远离 p(0)。两条规律：(i) D 方差小 且 SOR 初始权重 r(0) 更靠近 p(0) → p(∞)≈p(0)；(ii) D 方差大 且 r(0) 更靠近均匀分布 r^u(=1/K) → p(∞) 趋向 r^u。**现实推论：流动性"质量"（方差）老牌大所比新进者更难维持，于是碎片化会发生、把龙头市占压到 60% 以下；但小所若能把自己提供的流动性"清晰贴标签"（让 SOR 的初始配置更贴近 p(0) 以补偿其流量方差），仍可作为利基(niche)池存活**。

### 4. Kyle 模型与 λ：价格冲击是逆向选择的线性自保（§A.4，Kyle 1985）

**三方共存的最简框架**（一期）：一个大型**知情交易者(informed trader)**知道收盘价 p_ω~N(P*, σ_p²)；**噪声交易者(noise trader)**因别的原因交易，净方向 n_ω~N(0, σ_n²)；**做市商(MM)**只看到合计净流 Q+n，按线性规则定公共价 f_P=P̃+λ(Q+n)，且要在自己的信息集(filtration)下给出 p_ω 给定 Q+n 的**最佳线性估计**。

**两步求解的逻辑**：
- **Step 1（知情者最优）**：知情者按 p_ω 选参与量 Q 最大化期望利润 argmax_Q E[(p_ω−f_P(Q+n))·Q | p_ω]，解得线性策略 **Q(p_ω)=(p_ω−P̃)/(2λ)**——估值越偏离做市商基准价、交易越多，但 λ 越大（冲击越贵）交易越收敛。
- **Step 2（做市商响应）**：要求定价等于条件期望 → 退化为 p 对 Q+n 的**线性回归**，斜率 λ=Cov(p, Q+n)/Var(Q+n)。代入 Step 1 的 Q 得自洽方程，解出 P̃=P*、**λ=σ_p/(2σ_n)**。

**Proposition 1 的经济含义（重点）**：为抵御**逆向选择(adverse selection)**，做市商把价格**正比于观测净流线性移动**，比例系数就是 Kyle's lambda。两条直觉：
- σ_p 大（潜在信息驱动的价格波动大）→ λ 大 → 同样净流推动价格更多（信息越值钱，冲击越大）；
- σ_n 大（噪声/非信息流多）→ λ 小 → 信息被噪声淹没、做市商越难从净流里识别信息，于是越不敢/不需要大幅调价。

**λ 的标尺意义**：它一身二用——既是**价格对净订单流的敏感度**（流动性/深度的倒数刻画，λ 大=浅、冲击成本高），又是**知情交易强度**的代理（信息越多 λ 越高）。这是把"流动性"与"信息不对称"统一进一个可估系数的经典做法。

## 关键公式 · 图表 · 定义（图表要转述其内容）

### 公式（视觉读取，符号尽量还原）

**§A.1 FEI**
- 熵（市占 q_n，Σq_n=1，约定 0·log0=0）：`H(q_1,…,q_N) = − Σ_{1≤n≤N} q_n log q_n`  (A.1)
- 均匀分布的熵 = 最大熵：`H(1/N,…,1/N) = −N·(1/N)·log(1/N) = log N`  (A.2)
- 最大熵程序：`max −Σ q_n log q_n s.t. Σ q_n=1`，Lagrange 乘子 λ 给 `log q_n − 1 = λ` → 全部 q_n=1/N → 最大可能熵 = log N
- **定义1 FEI**：`𝔉(q_1,…,q_N) = H(q)/log N = (−Σ q_n log q_n)/log N`  (A.3)，取值 [0,1]：=0 高度集中、=1 均匀分散
- 新场所修正：`𝔉(t+1) = [log N / log(N+1)] · 𝔉(t)`（仅当新场所空仓时）

**§A.2 价格发现**
- 单笔收益：`π_M = s·Δ − c_M`，`π_T = Γ − s·Δ − c_T`，`c̄ = c_T + c_M`
- 聚合速率：`μ̄ = Σ_{m=1}^M μ_m`，`τ̄ = Σ τ_n`[符号存疑：原书第二式上标写作 m=1..M，按语义应为对投资者 n 求和]
- 成交速率：`R = μ̄·τ̄/(μ̄+τ̄)`（两笔间隔 (μ̄+τ̄)/(μ̄·τ̄)）
- 低延迟成本：`C_M(1/t)=½β·T/t²`，`C_T(1/t)=½γ·T/t²`
- 平均收益：`Π_M(m)=(μ_m/μ̄)·π_M·R − ½β μ_m²`，`Π_T(n)=(τ_n/τ̄)·π_T·R − ½γ τ_n²`，`Π_E=c̄·R=(c_M+c_T)·R`
- **均衡费率 (A.4)**：`c_M* = s·Δ − (Γ − c̄)/[1 + (M/N · γ/β)^{1/3}]`，其中 `c̄ = c_T* + c_M*`

**§A.3 SOR 碎片化**
- 捕获流动性：`C(r,ω) = Σ_{k=1}^K min(r_k·V(ω), D_k(ω))`
- 优化 (A.5)：`opt_r E_ω[Σ min(r_k V, D_k)] s.t. Σ r_k=1`[公式存疑：原书印作 "Minimize"，但准则是"最大化捕获流动性"，疑为 Maximize 笔误]
- 满足率相等 (A.6)：`E[(1/K)Σ_ℓ δ_{r_ℓ V_ℓ<D_ℓ} − δ_{r_k V_k<D_k}] = 0`（δ 为指示函数=全额成交事件）
- 强化学习更新 (A.7)：`r_k(ω+1) = r_k(ω) + γ_ω·[(1/K)Σ_ℓ δ_{r_ℓ V_ℓ<D_ℓ} − δ_{r_k V_k<D_k}]`（ω 作迭代器）
- 市占演化 (A.8)：`p_k(t+1) = (1−a)·p_k(t) + a·r_k(t)`（a=装 SOR 比例）
- 耦合 ODE (A.9)：`dr_k/dt = γ[(1/K)Σδ − δ_k]`，`dp_k/dt = a·(r_k − p_k)`
- 约束 (A.10)：`E[D_k·(Σ_ℓ D_ℓ)^{−1}] = p_k`；附 `E(V) ≃ Σ E(D_k)`，D 取对数正态、std∝数量（参数 ν）

**§A.4 Kyle**
- 分布：`p_ω ~ N(P*, σ_p²)`，`n_ω ~ N(0, σ_n²)`
- 公共价：`f_P(Q+n) = P̃ + λ·(Q+n)`
- Step1：`argmax_Q E[(p_ω − f_P(Q+n))·Q | p_ω]` → `Q(p_ω) = (p_ω − P̃)/(2λ)`
- Step2：定价=条件期望 → 线性回归 `λ = Cov(p, Q+n)/Var(Q+n)`；代入得 `P̃=P*`、自洽式 `λ = [σ_p²/(2λ)] / [σ_p²/(2λ)² + σ_n²]`（分母首项即 Var(Q)=σ_p²/(4λ²)）
- **Proposition 1**：`λ = σ_p/(2σ_n)`（自洽求解：σ_p²+4λ²σ_n²=2σ_p² → 4λ²σ_n²=σ_p² → λ=σ_p/(2σ_n)，已校验）

## 与因子研究 & 回测的关联（构造 / 检验 / 成本 / L2 / A股）

- **Kyle's lambda 作流动性 / 知情交易因子**：逐笔（或分钟）回归"价格变动 ~ 净订单流(净主买减主卖量)"，斜率即 λ 的样本估计，作横截面**价格冲击 / 知情交易强度**因子——λ 高者更脆弱、冲击成本高、可能被知情资金主导。本项目有通联 Level2 与同花顺 Level2 两套逐笔/盘口数据（见 DATA_SOURCES.md §2/§3/§5），具备估 λ 的原料；估计时注意按 σ_p（短窗收益波动）/σ_n（噪声流方差）分解做稳健化。
- **FEI 作市场状态 / 择场(venue selection)因子**：A 股单一主交易所场景下场所碎片化弱，但 FEI 思想可迁移到"成交量在不同价位 / 不同会员席位 / 主板-科创-北交所之间的分散度"，作流动性结构因子；低 FEI 状态指示执行需重路由、高 FEI 可简化路由。监控时务必套用书里"≥1% 才计入 N"的修正，避免新场所/新板块机械污染。
- **maker-taker 费率 (A.4) → 成本模型**：A 股无 maker-taker 返佣，但 (A.4) 的"稀缺侧被补贴"逻辑可用于理解港股 / 美股跨市场执行的费率优化，以及衍生品做市返佣对挂单激进度的影响；建因子时可把"挂单 vs 吃单的净费率"作执行成本模块的输入。
- **SOR 耦合模型 → 执行算法母本**：A.3 的"满足率反馈调权"就是多场所/多席位拆单算法的理论原型；其"质量(方差)越稳的场所长期吸单越多"对应执行里"优先打稳定深度的池子"。在 47 回测库(factor_library, `/mnt/sda2/lichenchen/factor_library/`)可把 λ、FEI、满足率做成横截面流动性因子并回测，A.2/A.3 框架作执行成本/冲击模型模块的理论基线。
- **检验设计提示**：λ 与 FEI 都对**采样频率与场所/席位定义**敏感（A.1 的 N 选择、Kyle 的单期假设），回测前先做频率稳健性与定义敏感性测试，再上横截面。

## 术语与存疑

**术语**：FEI = Fragmentation Efficiency Index(碎片化效率指数)；entropy(熵)；venue/liquidity pool(交易场所/流动性池)；market share(市占)；price discovery(价格发现)；PFP = Price Formation Process(价格形成过程)；HFMM = High Frequency Market-Maker(高频做市商)；market taker(吃单/流动性移除方)；maker/taker fees(挂单/吃单费率)、rebate(返佣)；liquidity premium(流动性溢价 s)；tick size(最小报价单位 Δ)；time to market / latency(到达订单簿时间/延迟)；Poisson process；SOR = Smart Order Router(智能订单路由)；repartition key(拆分键 r)；reinforcement learning / learn by trading(强化学习/边交易边学)；fill/satisfaction rate(满足率)；log-normal(对数正态)；adverse selection(逆向选择)；informed/noise trader(知情/噪声交易者)；Kyle's lambda(价格冲击系数 λ)；market impact / depth(市场冲击/深度)。

**存疑**：
- (A.4) maker-taker 均衡费率式的 (M/N·γ/β)^{1/3} 指数与括号结构按视觉读取还原，符号细节以原书 p.253 为准。[公式存疑]
- 聚合速率第二式 τ̄ 原书上标印作 Σ_{m=1}^M，按语义应为对 N 个投资者求和(Σ_{n=1}^N τ_n)，疑为排版笔误。[符号存疑]
- (A.5) 优化方向原书印作 "Minimize"，但 §A.3.1 全文准则是"最大化捕获流动性 C"，疑为 Maximize 笔误（含义按"最大化"理解）。[公式存疑]
- Kyle 自洽式分母首项 σ_p²/(2λ)² 应理解为 Var(Q)=σ_p²/(4λ²)；据此求解 λ=σ_p/(2σ_n) 已独立校验无误，最终结论可靠。
- 本块为数学密集附录，已尽量按"看图"还原公式；§A.4 完整解(Proposition 1)落在原书 p.261(PDF p.288)，§A.5 闪崩模型自 p.261 起为下一块。
