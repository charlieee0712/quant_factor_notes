## 全书定位（碎片化 / HFT / 流动性 / 最优执行，偏欧洲股票与执行）

本书由两位实务派（CA Cheuvreux 量化执行团队）写就，立足 **MiFID（2007）/ Reg NMS 之后的欧洲与美国股票市场**，回答三个递进问题：**如何定量描述、如何理解、如何在一个碎片化（fragmented）市场里最优地交易**。重心是"机理与为什么"，而非制度清单或选股 alpha——它是一本**卖方/交易台视角的微观结构与执行**专著，核心命题贯穿全书：流动性不是一个标量（bid-ask spread），而是一条**带数量、时间、信息博弈维度的 round-trip 成本曲线**。结构=导论（搭概念骨架）+ 三章（描述碎片化 / 理解其利害 / 组织最优执行）+ 一个把每个现象都配上"最小可解析玩具模型"的定量附录 A + 术语表 B。

## 对本项目最有价值的点

- **执行成本建模**：核心方法论——流动性=随委托量上升的 round-trip 成本曲线 c(Q)，否定"固定 bps 滑点"；冲击须拆**临时(随弹性 resilience 衰减、约 1 日内消散)+永久(随公允价移动)**；平均成交价可做 `即时价+计划价+意外漂移+交易效率+市场冲击` 五项分解；**关键反直觉结论：永久冲击在剔除 alpha 与后续自相关执行后趋近 0**——成交后同向走多半是信息兑现，不应全记为成本。
- **流动性因子**：价差–波动–活跃度近似普适关系 `ψ≈a+bσ/√N`，其残差=流动性错配/拥挤信号；Kyle's λ 作价格冲击/知情交易强度因子；覆盖度/Gini 作集中度；book size 不是可成交上界(订单簿会回补)。
- **订单簿动态 & Hawkes**：三类事件强度 λ⁺/λᶜ/λˣ 必须**按队列规模条件化**(队列反应模型)而非全局平均；一档失衡有方向预测力但幅度常＜1 tick(定位为执行择时而非套利)；**Hawkes 分支比 α/β→1 = 内生性/拥挤度高、易闪崩**，是市场脆弱性监测利器；propagator 给逐笔冲击核 G。
- **signature plot & Epps 高频波动率**：已实现波动率在 Δ→0 时被微观结构噪声**放大发散**(signature plot)、高频相关被异步成交**塌缩归零**(Epps)——算 RV 要用 Garman–Klass/两尺度，算相关要用 Hayashi–Yoshida 或放粗采样。
- **成交量季节性**：日内量=乘法可分离结构 `V=(f(n)+g(d,n))·F(m)·G(y)·v`，f(n)=U/J 形、g(d,n)=到期/数据日异常——VWAP 量曲线预测的标准骨架；做日内因子前须**去季节化**，做执行量曲线须**剔除到期/witching/宏观日**。
- **A 股映射差异**：A 股单一交易所、无 MTF/暗池/SOR 跨场竞争，碎片化/路由叙事基本不适用(可作对照系)；但**时间拆分、tick 地板(Harris 净化低价股虚高价差)、集合竞价(四步撮合/凹凸撮合曲线)、L2 三事件/失衡/队列因子框架完全适用**，且融券稀缺+T+1 使多空流动性不对称更极端。

## 可借鉴工具 · 模型清单

- **Round-trip 成本曲线 c(Q)** — 用 L2 逐档重建，把流动性当曲线而非标量。
- **市场冲击平方根律 MI=κσ(v/V)^γ (γ≈0.5)** — 回测的凹形冲击惩罚项，κ/γ 逐股标定。
- **Almgren–Chriss 均值-方差最优调度** — VWAP/TWAP/IS 执行曲线母本；λ=激进度旋钮。
- **Cartea–Jaimungal 随机控制(HJB)** — 把订单流/alpha 预测原生纳入执行节奏。
- **Kyle 模型 λ=σ_p/(2σ_n)** — 价格对净订单流的回归斜率=深度倒数+知情强度因子。
- **Hawkes 自激点过程** — 成交聚簇生成器；分支比作内生性/拥挤度因子；时间变换→成交量时钟。
- **Propagator 传播子模型** — 逐笔冲击核 G 与价格响应曲线 R(h)，解释"长记忆订单流+价格近鞅"。
- **Queue-Reactive 队列反应模型** — 盘口失衡/撤单率/成交率的条件化强度估计。
- **FEI(归一化熵) / Gini / KS 检验** — 集中度-碎片化标量、分布漂移与稳健性检验。
- **价差模型 ψ=a+bσ/√N (大 tick 加 η 弹跳项)** — 流动性残差因子；η=N^c/(2N^a)。
- **Harris 离散价差模型** — 剥离 tick 地板，还原"连续价差"，A 股低价股价差净化预处理。
- **Garman–Klass / 两尺度 RV / Hayashi–Yoshida** — 高频波动率与相关的去偏估计。
- **成交量乘法季节模型** — VWAP 日内量曲线预测；g(d,n) 捕捉特殊日。
- **比例估计+CLT 置信区间(np>5)** — 给占比类因子(主买占比/份额/HFT 占比)配误差棒。

## 一句话总评

一本把"流动性"从一个数重写成"带数量/时间/信息维度的成本曲线"的实务派经典——其执行成本、市场冲击与订单簿玩具模型可直接喂进本项目的成本建模与 L2 因子研究；碎片化/SOR/暗池那条主线对单一交易所的 A 股主要是**对照系**，真正可搬的是冲击-调度-订单簿-高频去偏这套定量工具箱。
