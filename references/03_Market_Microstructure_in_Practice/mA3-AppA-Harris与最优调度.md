## 本块讲什么

两件事，互不相关但都是微观结构落地的"算盘"：

1. **A.7 最优交易调度 (optimal trade scheduling)**——把一笔大单 v* 在日内拆成 N 段怎么拆最好。核心是"**市场冲击要你慢、市场风险要你快**"这对矛盾，用不同目标函数解出不同调度：
   - **A.7.1–A.7.2 Almgren–Chriss (AC)**：离散时间、确定性解。纯期望最小化→正比于 V_n/σ_n^{1/γ}（恒定市况退化为均匀拆单）；纯方差最小化→立刻全成交；均值-方差→凸的前重执行曲线（递推方程数值解）。
   - **A.7.3 Cartea–Jaimungal (CJ)**：连续时间、随机控制解。改 AC 的效用函数，把确定解变随机解，得后向 HJB 偏微分方程；妙处是能把"对未来订单流 μ_t 的观点（alpha/信号）"原生塞进调度。
   - **Optimal Trading Rate**：AC vs CJ 何时用哪个。

2. **A.8 比例及其置信区间 (estimation of proportion)**——市占率、HFT 占比、买/卖侧成交概率这类"比例型"量怎么估、误差多大。二项分布 + CLT 给点估计与置信区间，再推广到 venue 市占、指数市占的加权聚合，并比较两种估计量的方差。

（末页 A.9 Gini 仅起头，归下一块。）

## 方法论 / 机理要点（重点·展开）

### 一、AC 交易模型的三个建模决策 (A.7.1)

把 Markowitz 均值-方差思想从"组合的各资产成分"搬到"日内的各时间切片"——这是全章的题眼：**调度问题 = 在时间维度上做组合优化**。

模型只靠三条假设站起来：

- **公平价是鞅**：`S_{n+1} = S_n + σ_{n+1} ξ_{n+1}` (A.16)，离散化的算术布朗运动，ξ 标准正态。**关键含义**：不对价格趋势下任何判断（脚注明示 "No view is taken on the trend"）。因为是鞅，`E[S_n] = S_0`，期望成本里价格项就塌成常数 `S_0·v*`，**唯一要"防"的只剩冲击**。这条假设直接决定了后面期望最小化解的形态。
- **冲击是幂律、且随波动率放大**：第 n 段买 v 股的成交价 `S̃_n(v) = S_n + κ σ_n (v/V_n)^γ` (A.17)。三个要素：
  - **交易率 v/V_n 是冲击的核心自变量**（你的量相对市场量越大，冲击越大）——这正是估计冲击时要盯的参数；
  - σ_n 进冲击项：**高不确定期冲击更大**（同样的单子在高波动时更"推得动"价格）；
  - γ、κ 是个股相关常数，γ 刻画冲击的非线性曲率（经验上 γ<1，凹冲击）。
- **全额拆分约束**：`Σ_{n=1}^N v_n = v*` (A.18)。N 段必须把 v* 分完。

总成本 `W = Σ v_n·(S_n + κσ_n (v_n/V_n)^γ)` (A.19)。

### 二、三种目标 → 三种调度（AC 的核心结论，A.7.2）

**(1) 纯期望最小化** → Proposition 2/3
- 期望成本 `E[W] = S_0·v* + κ Σ σ_n v_n^{γ+1}/V_n^γ` (A.20)（鞅性吃掉了价格随机项）。
- 在 (A.18) 下用 Lagrange 乘子 λ 求解（Bellman 1956），得每段 `v_n ∝ (λ/(σ_n(γ+1)))^{1/γ}·V_n`，定出乘子 (A.21)。
- **结论 (A.22)**：最优速率**线性正比于 V_n/σ_n^{1/γ}**：

  `v_n^exp = [V_n/σ_n^{1/γ}] / [Σ_ℓ V_ℓ/σ_ℓ^{1/γ}] · v*`

  **直觉**：市场量 V_n 大的时段多交易（冲击被稀释）、波动 σ_n 大的时段少交易（单位冲击更贵）。这本质上是一条**跟随日内成交量曲线的 VWAP 式调度**（再按波动率微调）。指数 1/γ 来自冲击模型的非线性。
- **恒定市况退化 (A.23)**：若 σ_n≡σ、V_n≡V，则 `v_n = v*/N`——**均匀拆单**。书中点名这是自营盘常用的速率（用来消 tracking error，因为是"他们自己的风险"、一个月里能平均掉，对风险不敏感）。

**(2) 纯方差最小化** → Proposition 4
- 换元：记 `x_n = Σ_{ℓ≥n} v_ℓ`（第 n 段起的**剩余待成交量**）。分部求和后 `Σ v_n S_n = v*·S_0 + Σ x_n σ_n ξ_n`，于是成本方差极简：

  `Var(W) = Σ_{n=1}^N σ_n^2 x_n^2` (A.24)

  约束变成两个端点条件 `x_0 = v*`（开头啥都没做）、`x_N = 0`（结尾必须做完）(见 A.27)。
- **结论 (A.25)**：`v_1 = v*`，其余全 0——**一上来全部成交**。**直觉**：冲击根本不进方差表达式，你拖得越久、剩余 x_n 暴露在价格波动里的时间越长，方差越大；既然冲击不罚方差，最优就是瞬间清仓。

**(3) 均值-方差**（合并 1 与 2）→ Proposition 5
- 用风险厌恶 λ（与前面 Lagrange 乘子同质）线性组合：

  `C = E[W] + λ·Var(W) = v*·S_0 (立即买) + κ Σ σ_n v_n^{γ+1}/V_n^γ (市场冲击) + λ Σ x_n^2 σ_n^2 (市场风险)` (A.26)

- 两股力清晰对立：**冲击项要你慢**（最优是线性/VWAP 式），**风险项要你快**（最优是立即清）。平衡后必得**次线性（凸）的剩余量曲线 x_n**——即"前重"执行（早期多做、尾部收敛）。
- 线性冲击特例（γ=1）可写成 x 的函数并对每个 x_n 求导置零，得**递推方程 (A.28)**：

  `x_{n+1} = (1 + (σ_{n-1}/σ_n)(V_n/V_{n-1}) + (λ/κ)V_nσ_n)·x_n − (σ_{n-1}/σ_n)(V_n/V_{n-1})·x_{n-1}`

  配端点 (A.27)。**数值解法**：(a) 把 x_N^2 当 x_1 的函数做梯度下降；(b) 打靶法（shooting）——视 (A.28) 为某 PDE 的离散化，用 x_1 当参数迭代。**最优性原理**：一旦定下 x_1（首段交易量），递推唯一确定 x_N；x_N>0 说明 x_1 选大了（太慢、N 段做不完），x_N<0 说明选小了（太快）。调参 x_1 直到 x_N=0 即得对应任意 λ 的最优调度。
- Prop 2 与 Prop 4 都是 Prop 5 的特例（λ→小 / λ→大两端）。

### 四、Cartea–Jaimungal 随机控制框架 (A.7.3)

**动机**：AC 是确定性解（时间一离散就把轨迹钉死了）。CJ 对 AC 的效用函数做"一个小改动"，就把**确定解变成随机解**（能随市况/信号在线调整）。

- **目标泛函**（写的是卖出方向）：

  `H_t(ν) = E_t[ X_T^ν + Q_T^ν(S_T − A·Q_T^ν) − φ ∫_t^T (Q_s^ν)^2 ds ]`

  - Q_t 剩余待成交量、X_t 现金账户、ν 待优化的**交易速率**、A 是大额罚项（罚结尾没清完的残量 `−A·Q_T^2`）。
  - **风险厌恶 φ 罚的是"沿整条轨迹的剩余持仓平方积分"**：φ>0 时交易者有动力**快交易**以尽早把剩余量压小（这就是 CJ 把风险厌恶"连续化"的方式，对应 AC 的 λ）。
- **价格动态含外生订单流**：`dS_t = b·(μ_t − ν_t)dt + σ dW_t`。μ_t 是**所有其他交易者的外生买卖流**，b 是**永久冲击**系数（线性于净流 μ−ν）。再叠加买卖价差 ψ 与**临时冲击** κ（瞬时、100% 即时衰减），卖方瞬时成交价 `Ŝ_t = S_t − (ψ/2 + κν_t)`。
- 交易者视角的状态方程：`dQ^ν = −ν dt`，`dX^ν = Ŝ_t ν_t dt`。
- **Proposition 6（CJ 主方程）**：由动态规划原理(DPP)，价值函数 H* 满足后向 HJB-PDE（含 `∂_t + ½σ²∂²_S + L^μ` 算子、`−φq²` 风险罚、以及对 ν 取 sup 的控制项），终端条件 `H*(T,·) = x + q(S − ψ/2) − Aq²`。
- **求解套路（很典范，值得记）**：
  1. **降维 ansatz**：令 `H* = x + q(S − ψ/2) + h(t,μ,q)`，代入后控制项里对 ν 的最优解析显式可得：`ν* = −(bq + ∂_q h)/(2κ)`（一个二次型取极值），回代得 h 的 PDE。
  2. **对 q 做二次展开**：`h = h_0(t,μ) + q·h_1(t,μ) + q²·h_2(t,μ)`，按 q 幂次拆成三条方程：
     - **h_2：Riccati 方程**，与订单流 μ **无关**——它装的是纯粹的"风险 vs 冲击"权衡核。解 `h_2(t) = −ξ(t) − b/2`，`ξ(t)=√(κφ)·(1+ζe^{2γ(T−t)})/(1−ζe^{2γ(T−t)})`，其中 `γ=√(φ/κ)`、ζ 由端点定。
     - **h_1：Feynman–Kac**（b=0 时 `h_1(t,μ)=∫_t^T e^{(1/κ)∫ h_2} E_{t,μ}[μ_s] ds`）——**这条携带"对未来订单流的预测 E[μ_s]"**，即信号/alpha 进入调度的通道。
     - **h_0：直接积分** `h_0=(1/4κ)∫_t^T E_{t,μ}[h_1²] ds`。
  3. 合起来即给出"**当交易者对未来交易流 μ_t 有观点时**"的最优执行解。

  **要点**：CJ 的价值不在公式本身，而在它把"订单流预测"做成框架的一等公民——h_1 那一支就是预测信号的载体。

### 五、AC vs CJ 怎么选 (Optimal Trading Rate)

- **AC**：时间离散（实务 5–30 分钟尺度），适合**价格形成在介观尺度(mesoscopic, 5–30min)已被摸清**的情形；要引入套利/alpha 需交易者另外知道这些机会的统计量 [Lehalle 2009]。
- **CJ**：连续时间，适合**有一套聚焦于订单流的连续时间价格形成动力学**；套利机会只要能用交易流表达就能**原生纳入**。
- 还能更进一步：**双尺度同时建模**——介观(5–30min) + 订单簿驱动 [Bouchard et al. 2011]，或直接在挂单价格上建模 [Guéant et al. 2012b]。

### 六、比例估计与置信区间 (A.8)

**为什么微观结构里到处要估比例**：市占率、市场中 HFT 占比、"成交落在买价 vs 卖价侧"的概率……都是比例/概率。

- **模型**：二项分布 / 伯努利试验。n 个独立 {0,1} 变量 X_1..X_n，成功概率 p 的估计就是经验均值 `p̂ = (1/n)Σ X_k`。
- **三性质**：无偏 `E[p̂]=p`；方差 `Var(p̂)=p(1−p)/n`；相合（SLLN：`p̂ →a.s. p`）；渐近正态（CLT, A.29）：`√n·(p̂−p)/√(p(1−p)/n) → N(0,1)`。
- **置信区间**（p 未知，用 p̂ 估方差）：

  `p̂ ± z_{1−α/2}·√(p̂(1−p̂)/n)`

  95% 置信 → α=5% → z_{0.975}=1.96。
- **坑与正确做法**（书里专门 Remark）：
  - CLT 对二项即便 n<30 也常够用，**前提是 p 不太靠近 0 或 1**；p 极端时即便 n≥30 也可能不够。
  - p̂ 恰为 0 或 1 时正态近似**彻底失效**。
  - **经验法则：要求 np>5 且 n(1−p)>5**；小样本/极端比例应改用更稳的公式 [Agresti & Caffo 2000]（书中因不涉及故略）。
- **A.8.1 venue 在某资产上的市占**：`p̂_k^i = (资产 k 在场所 i 的成交额) / (k 在所有场所成交额之和)`——用成交额比值当比例估计量。
- **A.8.2 指数层市占聚合**：指数由 K 个权重 w_k 的成分构成，则场所 i 的指数市占是各成分市占按权重加权：`p̂^i = Σ_{k} w_k·p̂_k^i`。
- **A.8.3 两个估计量比较**：比较"单一成分估计量 p̂_k^i"与"聚合估计量 p̂^i"。在成分独立假设下推得差值方差：

  `Var(p̂_k^i − p̂^i) = (1 − 2w_k)·p_k^i(1−p_k^i)/n_k^i + p^i(1−p^i)/n^i`

  **解读**：阈值在 **w_k=0.5**——w_k<0.5 时两估计量之差的方差**大于**指数市占估计本身的方差；w_k>0.5 时**更小**。即某成分权重过半时，用它单独去近似整指数才相对稳。

## 与因子研究 & 回测的关联

**A. 执行 / 冲击成本（A.7）**
- **冲击成本模型直接可用**：(A.17) `κσ(v/V)^γ` 是回测里"市场冲击成本"的标准函数形；交易率 v/V、波动率 σ 是两个必备输入。给因子回测加成本约束时，这是比"固定 bps"更贴近 L2 现实的成本曲线。
- **执行基准的母本**：(A.22)→VWAP 式调度、(A.23)→TWAP/均匀拆单、(A.26)/(A.28)→IS（implementation shortfall）类前重曲线。λ（或 CJ 的 φ）就是"激进度"旋钮，回测换手成本估计可按目标算法选对应曲线。
- **CJ 的 h_1 通道 = alpha 进执行**：若有日内订单流/方向预测信号 E[μ_s]，CJ 框架给出"信号如何改写最优速率"的解析路径——把**预测因子**与**执行**统一在一个最优控制里，而非两段式（先选股再无脑拆）。

**B. 比例估计（A.8）= 给"占比类因子/统计量"配误差棒**
- 凡是用 L2/逐笔数据算的**比例型指标**——买侧成交占比 P(trade@bid)、主动买卖比、HFT/做市占比、某 venue 市占——都该带 (A.29) 的置信区间，而不是只报点值。np>5 且 n(1−p)>5 是判断"这个比例因子在当前样本量下可不可信"的硬门槛（薄盘/低频股容易踩线）。
- **市占/集中度因子**：(A.8.1) 成交额比值、(A.8.2) 加权聚合，可做横截面"流动性集中度/碎片化"因子；(A.8.3) 的 w_k=0.5 阈值提醒：用单一权重成分近似整体时方差何时可接受。

**C. 与本项目（L2 高频 / 47 回测库）**
- 通联 Level2 + 同花顺 Level2 两套逐笔/盘口数据（见仓库 `DATA_SOURCES.md` §2/§3/§5）正是估 (A.17) 冲击参数 κ/γ、交易率 v/V、以及各类买卖侧占比的原料。
- 在 47 `factor_library`（`/mnt/sda2/lichenchen/factor_library/`）落地时：把 (A.17) 作为成本模块的冲击曲线默认形、(A.26)/(A.28) 作为执行成本的理论基线；占比类因子统一套 (A.29) 置信区间做显著性闸门。
- **A 股注意**：A.7 假设连续可拆、可双向；A 股 T+1、涨跌停、单边集合竞价会破坏部分前提，纯方差解"立即全清"在涨跌停/流动性枯竭时不可行——冲击参数与可行调度需按 A 股微观结构重估。
